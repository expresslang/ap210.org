Description:

Contains 10 2D HO-8 components rotated 0, 90, 180, 270 and 360 degrees and mounted on both sides of the board. The components rotated the same degree are mounted at the same location but on different sides of the board. The locations are asymmetrical to help detect errors, in placement.
A detail description of the matrix transformation required to go from 2D to 3D can be found in the Matrix Tutorial.
