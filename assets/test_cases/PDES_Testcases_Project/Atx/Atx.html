Description:

The ATX specification defines the physical and function requirements for a ATX PC. Detailed information on the ATX specification can be found at http://developer.intel.com/design/motherbd/atx.htm and http://www.formfactors.org This testcase currently contains only the geometric physical requirements for the PCB. Additional information describing the requirements in the ATX specification would need to be added manually or via a prototype tool.
