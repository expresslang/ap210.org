4.2.76 Bound_volume_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Bound_volume_shape is a type of Shape_definition. A Bound_volume_shape is one of an Advanced_b_rep, a Csg_solid, an Edge_based_wireframe, a Geometrically_bounded_surface, a Shell_based_wireframe, or a Solid_of_linear_extrusion. A Bound_volume_shape is a three-dimensional geometric representation of a volume.
