4.2.654 Physical_connectivity_element

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_connectivity_element is a type of Managed_design_object. A Physical_connectivity_element is the object that describes a link between two nodes for the graph representation.

The data associated with this application object are the following:

  • connectivity_context
  • end_terminus
  • name
  • start_terminus

4.2.654.1 connectivity_context

The connectivity_context specifies the role of the Physical_connectivity_structure_definition for the Physical_connectivity_element. The connectivity_context contains all the link elements of the graph, of which a Physical_connectivity_element is one.

4.2.654.2 end_terminus

The end_terminus specifies either a Component_terminal or a Topological_junction as the end of the Physical_connectivity_element.

4.2.654.3 name

The name identifies the member of Physical_connectivity_element within the context of the assembly.

4.2.654.4 start_terminus

The start_terminus specifies either a Component_terminal or a Topological_junction as the beginning of the Physical_connectivity_element.

Formal constraints:

UR1: The combination of the name and the connectivity_context shall be unique within a population of Physical_connectivity_element.

WR1: The start_terminus shall not be the end_terminus;

WR2: Each object referenced by the Physical_connectivity_element shall be in the design view specified by connectivity_context.associated_definition.

ENTITY physical_connectivity_element
  SUBTYPE OF (managed_design_object);
    start_terminus       : termination_or_junction;
    name                 : ee_name;
    end_terminus         : termination_or_junction;
    connectivity_context : physical_connectivity_structure_definition;
  UNIQUE
    ur1 : name, connectivity_context;
END_ENTITY; -- physical_connectivity_element
