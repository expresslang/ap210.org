4.2.665 Physical_unit_3d_keepout_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_unit_3d_keepout_shape is a type of Bound_volume_shape and a type of Non_feature_shape_definition. A Physical_unit_3d_keepout_shape is the shape in three orthogonal dimensions of a keepout constraint for a Physical_unit.

NOTE: Keepout regions typically include some minimal spacing around the extreme shape.

The data associated with this application object are the following:

  • application_technology_constraint
  • centroid_location
  • constrained_design_object_category
  • shape_characterized_physical_unit
  • shape_environment
  • shape_material_condition
  • shape_purpose
  • shape_representation

4.2.665.1 application_technology_constraint

The application_technology_constraint specifies the role of Ee_requirement_occurrence for the Physical_unit_3d_keepout_shape. The application_technology_constraint is the relationship between keepout shapes amd process constraints under which those keepout shapes are valid.

EXAMPLE: An example of an application_technology_constraint is the relationship of the shape of a clearance area to an assembly or testing process which requires the clearance.

4.2.665.2 centroid_location

The centroid_location is a Cartesian_point which is the center of a Physical_unit_3d_keepout_shape. A centroid_location need not be specified for a particular Physical_unit_3d_keepout_shape.

4.2.665.3 constrained_design_object_category

The constrained_design_object_category specifies the category for which the Physical_unit_3d_keepout_shape is a constraint.

4.2.665.4 shape_characterized_physical_unit

The shape_characterized_physical_unit specifies the role of the Physical_unit for the Physical_unit_3d_keepout_shape.

4.2.665.5 shape_environment

The shape_environment specifies either manufacturing or an end_user_application as the environment for the Physical_unit_3d_keepout_shape.

4.2.665.6 shape_material_condition

The shape_material_condition specifies either the maximum_material_condition, the minimum_material_condition, or the nominal_material_condition as the material condition for the Physical_unit_3d_keepout_shape.

4.2.665.7 shape_purpose

The shape_purpose specifies either electro_magnetic_compatibility, shock, thermal, vibration, or an external_definition as the purpose for the Physical_unit_3d_keepout_shape.

4.2.665.8 shape_representation

The shape_representation specifies the role of the Bound_volume_shape for the Physical_unit_3d_keepout_shape.
