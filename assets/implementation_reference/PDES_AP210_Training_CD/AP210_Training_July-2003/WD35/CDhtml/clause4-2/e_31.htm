4.2.31 Assembled_with_fasteners

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Assembled_with_fasteners is a type of Assembly_joint that establishes the assembly by using fasteners to mechanically position a component so that its component mounting features are connected to component mounting features of another component in the assembly. The fasteners are associated with the base component using Product_association.

EXAMPLE: The usual case for this is the mounting hardware associated with a power device mounted to the Interconnect_module_component. Figure awf illustrates the concept of the joint category described herein and the roles of the related Application Objects.

The data associated with this application object are the following:

  • associated_assembly

4.2.31.1 associated_assembly

The associated_assembly specifies the Assembly_group_component that identifies the components required to be considered together in the assembly process for the Assembled_with_fasteners. The associated_assembly does not describe a formal product structure hierarchy.
