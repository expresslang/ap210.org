4.2.301 Dimensional_size

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimensional_size is a type of Dimensional_characteristic that defines the size of a Shape_element. A Dimensional_size is either a Dimensional_size_based_on_opposing_boundaries, or a Single_boundary_dimension. Additionally, a Dimensional_size may be an Angular_size_dimension.

The data associated with this application object are the following:

  • envelope
  • size_of

4.2.301.1 envelope

The envelope specifies the boolean value that indicates whether or not the Shape_element for which the size is characterized by the Dimensional_size has the additional requirement that it shall not violate the envelope of perfect form at the maximum material size of the Shape_element. A value of TRUE indicates that the requirement is in effect. A value of FALSE indicates that the requirement is not in effect.

NOTE: The envelope requirement is described in 6.1 of ISO 8015.

4.2.301.2 size_of

The size_of specifies the Shape_element for which the size is characterized by the Dimensional_size.
