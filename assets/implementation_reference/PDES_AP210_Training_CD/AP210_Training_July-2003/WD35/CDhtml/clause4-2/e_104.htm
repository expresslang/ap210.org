4.2.104 Closed_curve

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Closed_curve is a type of Curve that is traced by a moving point that returns to an arbitrary starting Cartesian_point.

Informal constraints:

IP1: A Closed_curve shall not be self intersecting.

NOTE: 
*)
ENTITY closed_curve
  SUBTYPE OF (curve);
END_ENTITY; -- closed_curve
