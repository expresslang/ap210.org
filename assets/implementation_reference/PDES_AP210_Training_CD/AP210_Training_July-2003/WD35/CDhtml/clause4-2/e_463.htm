4.2.469 Interface_component_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interface_component_allocation is a type of Managed_design_object. An Interface_component_allocation is the assignment relationship between an Interface_component on an assembly and the Mating_connector_component to which the Interface_component must mechanically and electrically mate in the next assembly level.

The data associated with this application object are the following:

  • mating_requirement
  • mating_solution

4.2.469.1 mating_requirement

The mating_requirement specifies the role of the Mating_connector_component for the Interface_component_allocation.

4.2.469.2 mating_solution

The mating_solution specifies the role of the Interface_component for the Interface_component_allocation.
