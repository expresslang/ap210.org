4.2.859 Stratum_feature_shape_stratum_surface_shape_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_feature_shape_stratum_surface_shape_relationship is a type of Managed_design_object. A Stratum_feature_shape_stratum_surface_shape_relationship is the association between a Stratum_feature_non_planar_2d_shape and a Stratum_surface_shape for the purpose of specifying that the points on the Stratum_feature_non_planar_2d_shape shall be in the domain of the points on the Stratum_surface_shape.

The data associated with this application object are the following:

  • of_stratum_feature_shape
  • of_stratum_surface_shape

4.2.859.1 of_stratum_feature_shape

The of_stratum_feature_shape specifies the dependent Stratum_feature_non_planar_2d_shape for the Stratum_feature_shape_stratum_surface_shape_relationship.

4.2.859.2 of_stratum_surface_shape

The of_stratum_surface_shape specifies the independent Stratum_surface_shape for the Stratum_feature_shape_stratum_surface_shape_relationship.
