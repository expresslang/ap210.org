4.2.939 Unsupported_passage

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Unsupported_passage is a type of Inter_stratum_feature that has no metallization in the realization of its walls.

The data associated with this application object are the following:

  • usage_definition

4.2.939.1 usage_definition

The usage_definition specifies the role of the Unsupported_passage_template for the Unsupported_passage.

Formal constraints:

WR1: An Unsupported_passage shall be an inner feature.

ENTITY unsupported_passage
  SUBTYPE OF (inter_stratum_feature);
    SELF\assembly_component.usage_definition : unsupported_passage_template;
END_ENTITY; -- unsupported_passage
