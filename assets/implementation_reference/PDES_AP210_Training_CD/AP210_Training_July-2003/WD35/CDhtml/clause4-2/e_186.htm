4.2.186 Connection_zone_based_assembly_joint

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Connection_zone_based_assembly_joint is a type of Assembly_joint. A Connection_zone_based_assembly_joint consists of a zone_1 and a zone_2.

The data associated with this application object are the following:

  • zone_1
  • zone_2

4.2.186.1 zone_1

A zone_1 specifies one role of a Connection_zone for a Connection_zone_based_assembly_joint.

4.2.186.2 zone_2

A zone_2 specifies one role of a Connection_zone for a Connection_zone_based_assembly_joint.

NOTE: There are no semantics associated with the sequence of zones in a Connection_zone_based_assembly_joint.

Formal constraints:

WR1: The zone_1 shall not be zone_2.

NOTE: 
*)
ENTITY connection_zone_based_assembly_joint
  SUBTYPE OF (assembly_joint);
    zone_1 : connection_zone;
    zone_2 : connection_zone;
WHERE
  WR1: zone_1 :<>: zone_2;
(* The zone_1 shall be referenced by assembled_features_1 inherited from

the assembly_joint supertype either directly or indirectly.
The zone_2 shall be referenced by assembled_features_2 inherited from

The zone_2 shall not be referenced by assembled_features_1 inherited from
