4.2.655 Physical_connectivity_interrupting_cutout

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_connectivity_interrupting_cutout is a type of Cutout that is included in the design of an Interconnect_module for the purpose of interrupting one or more connectivity_elements in an Interconnect_module.

NOTE: A Physical_connectivity_interrupting_cutout is used where a relayout of the design that would remove the connectivity elements from the design is not provided, but where traceability to a Physical_connectivity_definition is required.

The data associated with this application object are the following:

  • interrupted_connectivity_element

4.2.655.1 interrupted_connectivity_element

The interrupted_connectivity_element specifies the Conductive_interconnect_element that is interrupted with the Physical_connectivity_interrupting_cutout. There shall be one or more Conductive_interconnect_element for a Physical_connectivity_interrupting_cutout.
