4.2.681 Planned_effectivity

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Planned_effectivity is a type of Managed_design_object. A Planned_effectivity may be a Planned_date_effectivity, Planned_lot_effectivity, or Planned_serial_number_effectivity. A Planned_effectivity is the specification by a design organization of the intended use of an Ee_product or of an Ee_product within a particular Ee_product_configuration. The Planned_effectivity details the circumstances under which the usage will take place.

The data associated with this application object are the following:

  • component_quantity
  • configuration
  • effective_assembly
  • effectivity_approval

4.2.681.1 component_quantity

The component_quantity specifies the number of parts as a Data_element that are included in the affected Ee_product_configuration. The component_quantity need not be specified for a particular Planned_effectivity.

4.2.681.2 configuration

The configuration specifies the Ee_product_configuration that is effected by the Planned_effectivity.

4.2.681.3 effective_assembly

The effective_assembly specifies the role of the Assembly_composition_relationship for the Planned_effectivity.

4.2.681.4 effectivity_approval

The effectivity_approval specifies the Ee_approval that is associated with the Ee_product changes having the Planned_effectivity.

Formal constraints:

UR1: The combination of the configuration and the effective_assembly shall be unique within a population of Planned_effectivities.

WR1: The component_quantity, if it exists, shall be an integer greater than zero.

NOTE:

ENTITY planned_effectivity
  SUPERTYPE OF (ONEOF (planned_date_effectivity,
      planned_serial_number_effectivity,planned_lot_effectivity))
  SUBTYPE OF (managed_design_object);
    effective_assembly   : assembly_composition_relationship;
    effectivity_approval : ee_approval;
    configuration        : ee_product_configuration;
    component_quantity   : OPTIONAL data_element;
  UNIQUE
    ur1 : effective_assembly, configuration;
END_ENTITY; -- planned_effectivity
