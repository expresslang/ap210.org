4.2.435 Inter_stratum_feature_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Inter_stratum_feature_template is a type of Part_template. An Inter_stratum_feature_template may be one of a Component_termination_passage_template, an Unsupported_passage_template, or a Via_template. An Inter_stratum_feature_template is a definitional construct that defines the cross-section shape of layered constructs.

The data associated with this application object are the following:

  • of_passage_technology

4.2.435.1 of_passage_technology

The of_passage_technology specifies the role of the Passage_technology for the Inter_stratum_feature_template. The op_passage_technology specifies the material and layered properties for the Inter_stratum_feature_template.
