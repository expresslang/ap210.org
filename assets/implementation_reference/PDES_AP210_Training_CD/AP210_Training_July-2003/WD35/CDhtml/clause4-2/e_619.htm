4.2.615 Part_linear_array_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_linear_array_feature is a type of Part_array_feature. The location_uncertainty refers to originating CAD system geometric uncertainty, not material processing tolerance.

The data associated with this application object are the following:

  • element
  • initial_node_positive_x_displacement
  • terminus_node_positive_x_displacement

4.2.615.1 element

The element specifies the role of a Part_feature_placement_link for the Part_linear_array_feature. There shall be one or more Part_feature_placement_link for a Part_linear_array_feature.

4.2.615.2 initial_node_positive_x_displacement

The initial_node_positive_x_displacement specifies a Length_data_element for the Part_linear_array_feature.

4.2.615.3 terminus_node_positive_x_displacement

The terminus_node_positive_x_displacement specifies a Length_data_element for the Part_linear_array_feature.
