4.2.73 Bond_assembly_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Bond_assembly_position is a type of Managed_design_object. A Bond_assembly_position is the placement of an Assembled_with_bonding within the Physical_unit_3d_shape of an assembly. The Bond_assembly_position is an approximation to the actual position since it uses a default shape for the bond. There may be considerable variation between the default shape and actual shapes in teh "as delivered" product.

The data associated with this application object are the following:

  • assembly_model
  • bond
  • bond_definition_placement
  • bond_model

4.2.73.1 assembly_model

The assembly_model specifies the Physical_unit_3d_shape of the assembly for the Bond_assembly_position.

4.2.73.2 bond

The bond specifies the role of the Assembled_with_bonding for the Bond_assembly_position.

4.2.73.3 bond_definition_placement

The bond_definition_placement specifies the role of the Axis_placement_3d for the Bond_assembly_position. The placement specifies the transformation applied to the bond model to bring it into the assembly model.

4.2.73.4 bond_model

The bond_model specifies the Default_assembly_bond_shape of the assembly bond for the Bond_assembly_position.
