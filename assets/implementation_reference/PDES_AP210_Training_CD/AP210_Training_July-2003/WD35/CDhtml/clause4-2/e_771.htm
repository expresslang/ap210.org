4.2.771 Requirement_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Requirement_definition is a type of Ee_product_definition.

The data associated with this application object are the following:

  • associated_requirement_occurrence

4.2.771.1 associated_requirement_occurrence

The associated_requirement_occurrence is an inverse relationship that specifies that the existence of the Requirement_definition is dependent on the existence of a Ee_requirement_occurrence to reference the Requirement_definition.
