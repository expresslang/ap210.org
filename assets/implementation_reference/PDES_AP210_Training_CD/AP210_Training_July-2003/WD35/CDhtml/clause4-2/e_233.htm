4.2.233 Datum_point

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_point is a type of Datum that is a theoretically exact point to which toleranced Shape_element are related.

Formal constraints:

WR1: If the Datum_point is not specified as a reference_origin of a Datum_reference_frame then the Datum_point shall be either a Common_datum, or a Single_datum.

NOTE: 
*)
ENTITY datum_point
  SUPERTYPE OF (edge_segment_vertex)
  SUBTYPE OF (datum);
  location_in_2d : OPTIONAL cartesian_point;
  location_in_3d : OPTIONAL cartesian_point;
  associated_volume_shape_intersection : OPTIONAL volume_shape_intersection;
WHERE
 wr1: NOT(EXISTS(location_in_2d) AND EXISTS(location_in_3d)) OR
      EXISTS(associated_volume_shape_intersection);
 wr2: NOT(EXISTS(location_in_2d)) OR (location_in_2d.geometric_context.dimensionality 
                                                       = two_dimensional);
 wr3: NOT(EXISTS(location_in_3d)) OR (location_in_3d.geometric_context.dimensionality 
                                                       = three_dimensional);
END_ENTITY; -- datum_point
