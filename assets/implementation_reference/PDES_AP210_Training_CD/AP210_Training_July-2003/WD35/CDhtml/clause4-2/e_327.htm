4.2.327 Ee_product_specific_parameter_value_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_product_specific_parameter_value_assignment is a type of Managed_design_object. An Ee_product_specific_parameter_value_assignment is the assignment of a specific value to a specific characteristic of an Ee_product_category for a specific product.

The data associated with this application object are the following:

  • assigned_parameter
  • associated_product_category
  • of_product

4.2.327.1 assigned_parameter

The assigned_parameter specifies the Parameter_assignment that is assigning the specific value for this Ee_product_specific_parameter_value_assignment.

4.2.327.2 associated_product_category

The associated_product_category specifies the Ee_product_related_product_category that includes the characteristic being assigned a value with this Ee_product_specific_parameter_value_assignment.

4.2.327.3 of_product

The of_product specifies the Ee_product assigned a characteristic value by this Ee_product_specific_parameter_value_assignment.

Formal constraints:

WR1: Each Model_parameter in the role of assigned_parameter.assigned_parameter shall be in the role of category_parameter for the Ee_product_related_product_category specified by associated_product_category.

WR2: Each Ee_product in the role of of_product shall be in the role of of_product for the Ee_product_related_product_category specified by associated_product_category.
