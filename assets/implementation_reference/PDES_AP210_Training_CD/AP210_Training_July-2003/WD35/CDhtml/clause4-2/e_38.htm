4.2.38 Assembly_component_placement_link

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Assembly_component_placement_link is a type of Managed_design_object.

The data associated with this application object are the following:

  • precedent_element
  • referenced_by
  • subsequent_element

4.2.38.1 precedent_element

The precedent_element specifies an Assembly_component for the Assembly_component_placement_link.

4.2.38.2 referenced_by

The referenced_by is an inverse relationship that specifies that the existence of the Assembly_component_placement_link is dependent on the existence of the Linear_array_placement_group_component that specifies the Assembly_component_placement_link as its element.

4.2.38.3 subsequent_element

The subsequent_element specifies an Assembly_component for the Assembly_component_placement_link.
