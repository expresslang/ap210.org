4.2.569 Organization

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Organization is a type of Managed_design_object. An Organization is a collection of persons grouped for a specific purpose.

The data associated with this application object are the following:

  • address
  • identifier

4.2.569.1 address

The address specifies the string that describes the location where the Organization may be reached by postal mail or in person. The address need not be specified for a particular Organization.

4.2.569.2 identifier

The identifier specifies the name of the Organization.

EXAMPLE: The identifier may be a concatenation of the following: "Country code" = country code (as established by UN); "Organization code name" = identifying name the country gives the code; "Organization code" = organization code (as established by country assigning the organization code); "Sub Organization code" = sub organization code (as established by organization). This illustrates the usage: "Country code" = "USA", "Organization code name" = "CAGE", "Organization code" = "12345", "Sub Organization code" = "Telecommunications".

NOTE: Reference ISO 6523 amendment 1:1994.

Formal constraints:

UR1: The identifier shall be unique within a population of Organizations.
