4.2.281 Design_view_connection_zone_position_relationship

A Design_view_connection_zone_position_relationship is a type of Managed_design_object. A Design_view_connection_zone_position_relationship is the association between a Physical_unit_design_view, as a target and a Connection_zone as the source. This relationship positions and orients the geometric model of the source into the model of the target using a transformation.

NOTE: There can be only one terminal that uses the Connection_zone related so there is no need to reference the terminal in this relationship.

The data associated with this application object are the following:

  • associating_design_view_shape
  • associated_usage
  • associated_connection_zone_shape_definition
  • associated_usage_placement
  • associating_design_view

4.2.281.1 associating_design_view_shape

The associating_design_view_shape specifies the Feature_shape_definition, that is the shape that contains the target for the Design_view_connection_zone_position_relationship.

4.2.281.2 associated_usage

The associated_usage specifies the Connection_zone, that is the source item for the Design_view_connection_zone_position_relationship.

4.2.281.3 associated_connection_zone_shape_definition

The associated_connection_zone_shape_definition specifies the Connection_zone_shape, that is the source shape for the Design_view_connection_zone_position_relationship.

4.2.281.4 associated_usage_placement

The associated_usage_placement specifies the Axis_placement that is the target item in the coordinate system of the terminal shape for the Design_view_connection_zone_position_relationship, including any transformation required.

4.2.281.5 associating_design_view

The associating_design_view specifies the Physical_unit_design_view for which the shape is composed for the Design_view_connection_zone_position_relationship.

Informal constraints:

IP1: The feature_shape_definition of the associating_design_view shall be the associating_design_view_shape.

IP2: The associated_usage connection_zone shall be a connection_zone in the scope of the associating_design_view.

 END_ENTITY; -- design_view_connection_zone_position_relationship
