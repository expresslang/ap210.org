4.2.652 Physical_connectivity_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_connectivity_definition is a type of Managed_design_object. A Physical_connectivity_definition may be a Physical_connectivity_structure_definition. A Physical_connectivity_definition is the root node in the classification hierarchy of connectivity in the physical domain. All types of connectivity in this domain contain additional information to that of the base type. A Physical_connectivity_definition is a definition of one node, in an electrical network, consisting of physical terminals.

The data associated with this application object are the following:

  • associated_terminals
  • name

4.2.652.1 associated_terminals

The associated_terminals specify the Component_terminal of the Physical_connectivity_definition. There shall be two or more Component_terminal for a Physical_connectivity_definition.

4.2.652.2 name

The name identifies the Physical_connectivity_definition.

Formal constraints:

UR1: The name shall be unique within the context of the associated_definition.

WR1: Each design view that references a member of associated_terminals shall reference all members of associated_terminals.

WR2: The associated_definition shall reference all members of associated_terminals.
