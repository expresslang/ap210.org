4.2.249 Default_component_termination_passage_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Default_component_termination_passage_definition is a type of Passage_technology that defines a default passage based on characteristics of the associated Part_terminal.

The data associated with this application object are the following:

  • deposition_material
  • maximum_allowed_component_terminal_extent
  • minimum_allowed_component_terminal_extent
  • minimum_as_finished_deposition_thickness
  • minimum_as_finished_passage_extent

4.2.249.1 deposition_material

The deposition_material specifies the role of the Ee_material for the Default_component_termination_passage_definition.

4.2.249.2 maximum_allowed_component_terminal_extent

The maximum_allowed_component_terminal_extent specifies one role of the Length_data_element for the Default_component_termination_passage_definition.

4.2.249.3 minimum_allowed_component_terminal_extent

The minimum_allowed_component_terminal_extent specifies one role of the Length_data_element for the Default_component_termination_passage_definition.

4.2.249.4 minimum_as_finished_deposition_thickness

The minimum_as_finished_deposition_thickness specifies one role of the Length_data_element for the Default_component_termination_passage_definition.

4.2.249.5 minimum_as_finished_passage_extent

The minimum_as_finished_passage_extent specifies one role of the Length_data_element for the Default_component_termination_passage_definition.

Formal constraints:

WR1: The passage deposition material shall be conductive.
