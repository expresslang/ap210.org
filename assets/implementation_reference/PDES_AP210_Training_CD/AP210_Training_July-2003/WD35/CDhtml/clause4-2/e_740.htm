4.2.740 Process_specification

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Process_specification is a type of Ee_specification that identifies and describes an activity to be performed on an Ee_product or Ee_material. A Process_specification may be an Assembly_technology_specification, a Fabrication_technology_specification or a Lead_form_specification.

EXAMPLE: Figure ps illustrates a detail from a Process_specification for an etching process.
