4.2.793 Rule_set_group

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_set_group is a type of Rule_set. A Rule_set_group shall have at least two Rule_sets.

The data associated with this application object are the following:

  • elements

4.2.793.1 elements

The elements specify the Rule_set for the Rule_set_group. There may be more than two Rule_set for the Rule_set_group.
