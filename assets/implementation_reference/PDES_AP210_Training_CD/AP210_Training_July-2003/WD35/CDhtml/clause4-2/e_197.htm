4.2.197 Coordinated_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Coordinated_characteristic is a type of Managed_design_object that may include more than one discrete Characteristic and that is dependent upon a specified environment for its validity. A Coordinated_characteristic may be a Tolerance_characteristic or may be a Range_characteristic. The members of component_characteristic shall be coordinated in accordance with the definition associated with the type_name in the reference_document.

NOTE: This part of ISO 10303 predefines several categories of Tolerance_characteristic.

EXAMPLE: An example of coordinated_characteristic is Poisson's ratio. This measurement, for some materials, is dependent upon the orientation of the measurement relative to the specimen being measured. Poisson's ratio, as a characteristic, is specified by three values of stress strain ratio along three orthogonal axes where one axis is along a characteristic direction of the material being described.

The data associated with this application object are the following:

  • component_characteristic
  • dependent_environment
  • reference_document
  • type_name

4.2.197.1 component_characteristic

The component_characteristic specifies the Characteristic for the Coordinated_characteristic. There shall be one or more Characteristic for a Coordinated_characteristic.

4.2.197.2 dependent_environment

The dependent_environment specifies the text that describes the environment under which the Coordinated_characteristic is valid. The dependent_environment need not be specified for a particular Coordinated_characteristic.

4.2.197.3 reference_document

The reference_document specifies the Ee_document or an identifier of the standard document for the type of Coordinated_characteristic.

4.2.197.4 type_name

The type_name specifies the string, defined in the reference_document, for the type of Coordinated_characteristic.

ENTITY coordinated_characteristic
  SUPERTYPE OF (ONEOF(range_characteristic,tolerance_characteristic))
  SUBTYPE OF (managed_design_object);
    type_name                : STRING;
    component_characteristic : SET [1:?] OF characteristic;
    dependent_environment    : OPTIONAL STRING;
    reference_document       : ee_document;
END_ENTITY; -- coordinated_characteristic
