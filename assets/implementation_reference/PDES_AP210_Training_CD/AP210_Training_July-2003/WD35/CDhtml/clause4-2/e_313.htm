4.2.313 Edge_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Edge_shape is a type of Planar_shape and a type of Feature_shape_definition. An Edge_shape may be one or more of a Bevel_edge_feature_shape, a Radius_edge_feature_shape, or a Step_edge_feature_shape. Edge_shape is provied to describe parametric data necessary to fabricate commonly used shapes on segments of interconnect substrates. Edge_shape applies to the entire arc length of the feature that is referencing it.
