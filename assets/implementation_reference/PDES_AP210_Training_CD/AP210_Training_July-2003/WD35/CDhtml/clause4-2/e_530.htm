4.2.530 Material_composition_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Material_composition_relationship is a type of Managed_design_object. A Material_composition_relationship is the association between one Ee_material and another when describing a composite material.

The data associated with this application object are the following:

  • composition_basis
  • composition_class
  • constituent_amount
  • determination_method
  • related_product
  • relating_product

4.2.530.1 composition_basis

The composition_basis specifies the string that represents the basis on which a material product is decomposed into its constituents.

4.2.530.2 composition_class

The composition_class specifies the string that represents the category of composition for the Material_composition_relationship.

4.2.530.3 constituent_amount

The constituent_amount specifies the Data_element that represents the amount of material.

4.2.530.4 determination_method

The determination_method specifies the string that represents the procedure by which constituent_amount is determined.

4.2.530.5 related_product

The related_product is Ee_material that is the constituent for the Material_composition_relationship. There shall be one or more Ee_material for the Material_composition_relationship.

4.2.530.6 relating_product

The relating_product is Ee_material that is the composition for which this Material_composition_relationship is specifying a constituent.

NOTE: This relationship is intended for compounds and simple mixtures only, not for assemblies.

ENTITY material_composition_relationship
  SUBTYPE OF (managed_design_object);
    composition_basis    : STRING;
    determination_method : ee_text;
    composition_class    : STRING;
    constituent_amount   : data_element;
    related_product      : ee_material;
    relating_product     : ee_material;
END_ENTITY; -- material_composition_relationship
