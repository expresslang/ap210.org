4.2.481 Join_two_physical_connectivity_definition_supporting_printed_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Join_two_physical_connectivity_definition_supporting_printed_component is a type of Printed_component. The Join_two_physical_connectivity_definition_supporting_printed_component is included specifically for cases in which analog and digital grounds are kept topologically and physically separate completely through the design process to minimize noise but need to be physically connected for reference voltage purposes.

NOTE: This connections is usually required to be at one and only one location.

The data associated with this application object are the following:

  • joined_definitions

4.2.481.1 joined_definitions

The joined_definitions specify the roles of exactly two Physical_connectivity_definition for the Join_two_physical_connectivity_definition_supporting_printed_component.
