4.2.293 Dimension_location_with_specified_datum_system

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimension_location_with_specified_datum_system is a type of Dimensional_location that is related to a Datum_reference_frame.

The data associated with this application object are the following:

  • referenced_datum_system_defining_conditions

4.2.293.1 referenced_datum_system_defining_conditions

The referenced_datum_system_defining_conditions specifies the Datum_system_defining_conditions_without_material_conditions that defines the order of precedence for the datums of a Datum_system which in turn establishes the Datum_reference_frame to which the Dimension_location_with_specified_datum_system is related.

NOTE: The specification of the relationship between linear and angular dimensions to a datum reference frame is discussed in 4.4 of ASME Y14.5M.
