4.2.371 Feature_group_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Feature_group_relationship is a type of Managed_design_object. A Feature_group_relationship is the association of one or more Part_feature and a Part_group_feature that are used in the composition of that Part_group_feature.

The data associated with this application object are the following:

  • related_feature
  • relating_feature

4.2.371.1 related_feature

The related_feature specifies the Part_feature used in the Feature_group_relationship.

4.2.371.2 relating_feature

The relating_feature specifies the Part_group_feature in the Feature_group_relationship.

ENTITY feature_group_relationship
