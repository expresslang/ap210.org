4.2.88 Cartesian_transformation_operator_2d

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Cartesian_transformation_operator_2d is a type of Managed_design_object. A Cartesian_transformation_operator_2d defines a geometric transformation in two-dimensional space composed of translation, rotation, mirroring and uniform scaling. A precise mathematical definition for Cartesian_transformation_operator_2d is provided in ISO 10303-42.

The data associated with this application object are the following:

  • axis_1
  • axis_2
  • mirror
  • scale
  • translation

4.2.88.1 axis_1

The axis_1 specifies one role of a Direction for the Cartesian_transformation_operator_2d.

4.2.88.2 axis_2

The axis_2 specifies one role of a Direction for the Cartesian_transformation_operator_2d.

4.2.88.3 mirror

The mirror is a derived attribute that specifies the value of the determinant of the transformation matrix for the Cartesian_transformation_operator_2d.

4.2.88.4 scale

The scale specifies the role of the Positive_ratio_measure for the Cartesian_transformation_operator_2d.

4.2.88.5 translation

The translation specifies the role of the Cartesian_point for the Cartesian_transformation_operator_2d.

Informal constraints:

IP1: Mirroring shall be only about the local Y axis, which "local Y axis" is defined as a): being parallel to the Y axis of the current coordinate system and b): intersecting the Cartesian_point which is in the role of translation for this Cartesian_transformation_operator_2d.

EXAMPLE: A point at (X = 3, Y = 2) would be at (X = -3, Y = 2) after being transformed using the Cartesian_transformation_operator_2d if no rotation were applied and no translation were applied, but mirroring were applied.

NOTE: Typically the transformation matrix is the result of applying the ordered sequence of "rotate, mirror, translate" operations to an identity matrix. Placing translate last in the sequence satisfies the requirement that mirroring shall be about a local axis.

IP2: The determinant of the transformation matrix shall have a value of -1.0 or +1.0.

NOTE: In a received file the value of the determinant may not be exactly 1.0 or -1.0 due to roundoff error. Since these are the only two allowed values, standard techniques to deal with the error are easily applied.

IP3: The scale value for the transformation shall be one.

ENTITY cartesian_transformation_operator_2d
  SUBTYPE OF (managed_design_object);
    axis_1             : direction;
    axis_2             : direction;
    translation        : cartesian_point;
    scale              : positive_ratio_measure;
DERIVE
   mirror : BOOLEAN := derive_mirror(SELF);
END_ENTITY; -- cartesian_transformation_operator_2d
