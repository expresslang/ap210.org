4.2.405 Functional_unit_network_terminal_definition_node_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_unit_network_terminal_definition_node_assignment is a type of Managed_design_object. A Functional_unit_network_terminal_definition_node_assignment is the assignment relationship between a node definition of the functional unit network definition and a terminal definition of that network definition. This assignment provides visibility into the node definition from higher levels in the functional decomposition.

The data associated with this application object are the following:

  • composed_node
  • connected_terminal

4.2.405.1 composed_node

The composed_node specifies the role of the Functional_unit_network_node_definition for the Functional_unit_network_terminal_definition_node_assignment.

4.2.405.2 connected_terminal

The connected_terminal specifies the role of the Functional_unit_usage_view_terminal_definition for the Functional_unit_network_terminal_definition_node_assignment.
