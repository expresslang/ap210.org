4.2.931 Transmission_line_functional_unit_terminal_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Transmission_line_functional_unit_terminal_allocation is a type of Managed_design_object. A Transmission_line_functional_unit_terminal_allocation is the relationship between the occurrence of a Functional_unit_terminal of a transmission line and the physical implementation of that occurrence as a Printed_component_join_terminal.

The data associated with this application object are the following:

  • functional_unit_design_path
  • implementation
  • transmission_line_functional_unit_terminal

4.2.931.1 functional_unit_design_path

The functional_unit_design_path specifies the Transmission_line_functional_unit_to_routed_printed_component_allocation path from the Functional_unit_terminal to the Printed_component_join_terminal for the Transmission_line_functional_unit_terminal_allocation.

4.2.931.2 implementation

The implementation specifies the role of the Printed_component_join_terminal for the Transmission_line_functional_unit_terminal_allocation.

4.2.931.3 transmission_line_functional_unit_terminal

The transmission_line_functional_unit_terminal specifies the role of the Functional_unit_terminal for this Transmission_line_functional_unit_terminal_allocation.

Formal constraints:

UR1: The combination of the implementation and transmission_line_functional_unit_terminal shall be unique within a population of Transmission_line_functional_unit_terminal_allocation.
