4.2.899 Theoretically_exact_dimension_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Theoretically_exact_dimension_representation is a type of Singular_dimension_representation that describes the theoretically exact size, form, orientation, or location of one or more Shape_element.

NOTES:

  1. Theoretically exact dimensions are described in clause 10 of ISO 1101.
  2. The term in ASME Y14.5M corresponding to Theoretically_exact_dimension_representation is basic dimension. Basic dimensions are defined in 1.3.9 of ASME Y14.5M.

NOTE: 
*)
ENTITY theoretically_exact_dimension_representation
  SUBTYPE OF (singular_dimension_representation);
END_ENTITY; -- theoretically_exact_dimension_representation
