4.2.424 Geometrically_bounded_2d_wireframe

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Geometrically_bounded_2d_wireframe is a type of Feature_shape_definition. A Geometrically_bounded_2d_wireframe is a geometrically bounded wireframe with a dimensionality equal to 2.

NOTE: The Geometrically_bounded_2d_wireframe, being a wireframe-based feature, does not support the representation of areas.
