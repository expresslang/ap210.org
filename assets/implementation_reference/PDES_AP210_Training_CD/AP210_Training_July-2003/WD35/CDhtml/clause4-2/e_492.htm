4.2.492 Land_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Land_join_terminal is a type of Component_terminal included in the design to mate directly with a terminal of another fabricated feature by use of a Fabrication_joint internally in a layout.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.492.1 associated_component

The associated_component specifies the role of the Land for the Land_join_terminal.

4.2.492.2 definition

The definition specifies the role of the Land_template_join_terminal for the Land_join_terminal.

Formal constraints:

WR1: A Land_join_terminal shall participate in a relationship with a Fabrication_joint as a fabrication_features.
