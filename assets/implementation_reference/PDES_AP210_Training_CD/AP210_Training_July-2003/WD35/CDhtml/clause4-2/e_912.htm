4.2.912 Tolerance_specific_restraint_condition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
The Tolerance_specific_restraint_condition is a type of Restraint_condition that is only applied to specified instances of Geometric_tolerance and Dimensional_characteristic.

The data associated with this application object are the following:

  • affected_tolerances

4.2.912.1 affected_tolerances

The affected_tolerances specifies the Geometric_tolerance and Dimensional_characteristic to which the restraint condition applies. There shall be one or more Geometric_tolerance or Dimensional_characteristic specified as the affected_tolerances by a Tolerance_specific_restraint_condition.
