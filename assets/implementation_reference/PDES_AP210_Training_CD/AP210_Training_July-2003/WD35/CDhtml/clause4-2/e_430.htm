4.2.430 Inspection_requirement_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Inspection_requirement_occurrence is a type of Ee_requirement_occurrence that specifies a production inspection related requirement. An Inspection_requirement_occurrence may be a Layout_line_width_tolerance_requirement_occurrence or a Layout_land_width_tolerance_requirement_occurrence or a Fabrication_thickness_requirement_occurrence.
