4.2.558 Non_conductive_cross_section_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Non_conductive_cross_section_template is a type of a Part_template.

The data associated with this application object are the following:

  • template_technology
  • nominal_width

4.2.558.1 template_technology

The template_technology specifies a Stratum_technology for the Non_conductive_cross_section_template.

4.2.558.2 nominal_width

The nominal_width specifies a Length_data_element for the Non_conductive_cross_section_template.
