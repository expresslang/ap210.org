4.2.805 Separate_geometric_tolerance_requirement

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Separate_geometric_tolerance_requirement is a type of Managed_design_object. A Separate_geometric_tolerance_requirement is a specification that indicates that a set of one or more Geometric_tolerance_with_referenced_datum_system is to be treated as a separate requirement. That is, when geometric tolerances (excluding the composite profile refinement tolerances and composite feature relating position tolerances) reference the same datum system, all the physical features to which the geometric tolerances are applied are treated as a single pattern with regard to the geometric tolerances. If this type of interrelationship is not desired, the Separate_geometric_tolerance_requirement may be used to specify a set of one or more Geometric_tolerance that are to be treated as a separate requirement.

NOTE: Separate requirements are discussed in 4.5.12 of ASME Y14.5M.

The data associated with this application object are the following:

  • separated_geometric_tolerances

4.2.805.1 separated_geometric_tolerances

The separated_geometric_tolerances specifies the Geometric_tolerance_with_referenced_datum_system that are to be treated as a separate requirement. There shall be one or more Geometric_tolerances_with_referenced_datum_system for a Separate_geometric_tolerance_requirement.

NOTE: If the goal is for an individual Geometric_tolerance to be a separate requirement, then it should be specified as the separated_geometric_tolerances by a Separate_geometric_tolerance_requirement that only specifies that Geometric_tolerance.

Formal constraints:

WR1: The Geometric_tolerance_with_referenced_datum_system that are specified as the separated_geometric_tolerances shall all specify the same instance of Datum_system_defining_conditions as their datum_system_definition.

WR2: The Geometric_tolerance_with_referenced_datum_system that are specified as the separated_geometric_tolerances shall not either be Composite_linear_profile_refinement_tolerance, Composite_surface_profile_refinement_tolerance, or Composite_feature_relating_position_tolerance.

WR3: The Geometric_tolerance_with_referenced_datum_system that are specified as the separated_geometric_tolerances shall either be Profile_of_any_line_tolerance, Profile_of_any_surface_tolerance, or Position_tolerance.
