4.2.202 Csg_solid

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Csg_solid is a type of Bound_volume_shape. A Csg_solid is a solid represented as a CSG model that is defined by a collection of primitive solids, combined using regularized boolean operations. The allowed operations are intersection, union and difference. Csg_solids may be used in conjunction with other solid models and boolean operations to construct larger solid models.
