4.2.6 Advanced_b_rep

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Advanced_b_rep is a type of Bound_volume_shape that represents the shape of a part or an aspect of the shape of a part by an advanced boundary representation solid model. This representation allows for the curves and surfaces and the topology that binds them. Boundaries are explicitly defined only by topology. All of the geometry that defines the shape of a part or an aspect of the shape of a part must be associated with topology.
