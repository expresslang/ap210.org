4.2.957 Viewing_plane_based_orientation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Viewing_plane_based_orientation is a type of Managed_design_object. A Viewing_plane_based_orientation is a definition that characterizes the orientation of a Tolerance_zone with respect to a Viewing_plane.

The data associated with this application object are the following:

  • orienting_relationship
  • viewing_plane_specification

4.2.957.1 orienting_relationship

The orienting_relationship specifies the Tolerance_zone_and_viewing_plane_orienting_relationship that associates the Tolerance_zone with the Viewing_plane with which it is oriented.

4.2.957.2 viewing_plane_specification

The viewing_plane_specification specifies a Viewing_plane_definition that characterizes the Viewing_plane with which the Tolerance_zone is oriented.

Formal constraints:

WR1: The Viewing_plane_based_orientation shall be specified as the zone_orientation by exactly one Tolerance_zone_definition_with_specified_orientation.

WR2: The Viewing_plane specified as the defined_viewing_plane by the Viewing_plane_definition that is specified as the viewing_plane_specification shall be the same instance of Viewing_plane as specified as the orienting_viewing_plane by the Tolerance_zone_and_viewing_plane_orienting_relationship that is specified as the orienting_relationship.
