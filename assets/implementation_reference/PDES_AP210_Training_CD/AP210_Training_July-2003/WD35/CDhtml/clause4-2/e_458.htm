4.2.465 Interconnect_module_usage_view

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_usage_view is a type of Physical_unit_usage_view. An Interconnect_module_usage_view may be a Pcb_usage_view. Either the maximum_thickness_over_dielectric_requirement or the maximum_thickness_over_metal_requirement shall be provided. If both the maximum_thickness_over_dielectric_requirement and the maximum_thickness_over_metal_requirement are provided, the maximum_thickness_over_metal_requirement shall be greatere than the maximum_thickness_over_dielectric_requirement.

The data associated with this application object are the following:

  • implemented_function
  • located_thickness_requirement
  • maximum_thickness_over_dielectric_requirement
  • maximum_thickness_over_metal_requirement
  • measurement_condition
  • minimum_thickness_over_dielectic_requirement
  • minimum_thickness_over_dielectic_requirement

4.2.465.1 implemented_function

The implemented_function specifies the role of the Functional_unit_usage_view for the Interconnect_module_usage_view. The implemented_function need not be specified for a particular Interconnect_module_usage_view.

4.2.465.2 located_thickness_requirement

The located_thickness_requirement specifies the role of the Located_interconnect_module_thickness_requirement for the Interconnect_module_usage_view. A Located_interconnect_module_thickness_requirement need not be specified for an Interconnect_module_usage_view.

4.2.465.3 maximum_thickness_over_dielectric_requirement

The maximum_thickness_over_dielectric_requirement specifies the role of the Length_data_element for the Interconnect_module_usage_view. The maximum_thickness_over_dielectric_requirement need not be specified for a particular Interconnect_module_usage_view. The maximum_thickness_over_dielectric_requirement references a base dielectric and not final finish dielectrics.

4.2.465.4 maximum_thickness_over_metal_requirement

The maximum_thickness_over_metal_requirement specifies the role of the Length_data_element for the Interconnect_module_usage_view. The maximum_thickness_over_metal_requirement need not be specified for a particular Interconnect_module_usage_view.

4.2.465.5 measurement_condition

The measurement_condition specifies the role of the Restraint_condition for the Interconnect_module_usage_view. The measurement_condition need not be specified for a particular Interconnect_module_usage_view.

4.2.465.6 minimum_thickness_over_dielectric_requirement

The minimum_thickness_over_dielectric_requirement specifies the role of the Length_data_element for the Interconnect_module_usage_view. The minimum_thickness_over_dielectric_requirement need not be specified for a particular Interconnect_module_usage_view. The minimum_thickness_over_dielectric_requirement references a base dielectric and not final finish dielectrics.

4.2.465.7 minimum_thickness_over_metal_requirement

The minimum_thickness_over_metal_requirement specifies the role of the Length_data_element for the Interconnect_module_usage_view. The minimum_thickness_over_metal_requirement need not be specified for a particular Interconnect_module_usage_view.

Formal constraints:

WR1: At least one thickness property shall be provided.

ENTITY interconnect_module_usage_view
  SUPERTYPE OF (pcb_usage_view)
  SUBTYPE OF (physical_unit_usage_view);
    minimum_thickness_over_metal_requirement      : OPTIONAL length_data_element;
    maximum_thickness_over_metal_requirement      : OPTIONAL length_data_element;
    minimum_thickness_over_dielectric_requirement : OPTIONAL length_data_element;
    maximum_thickness_over_dielectric_requirement : OPTIONAL length_data_element;
    measurement_condition                         : OPTIONAL restraint_condition;
    located_thickness_requirement                 : 
	OPTIONAL SET [1:?] OF located_interconnect_module_thickness_requirement;
    implemented_function                          : OPTIONAL functional_unit_usage_view;
WHERE
 WR1: SIZEOF(QUERY ( pf <* USEDIN (SELF,
   'AP210_ARM.INTERCONNECT_MODULE_SURFACE_FEATURE.ASSOCIATED_DEFINITION') |
    ('AP210_ARM.' +
     'INTERCONNECT_MODULE_PRIMARY_SURFACE' IN TYPEOF (pf))))
     <= 1;
 WR2: SIZEOF(QUERY ( pf <* USEDIN (SELF,
   'AP210_ARM.INTERCONNECT_MODULE_SURFACE_FEATURE.ASSOCIATED_DEFINITION') |
    ('AP210_ARM.' +
     'INTERCONNECT_MODULE_SECONDARY_SURFACE' IN TYPEOF (pf)))) 
     <= 1;
 WR3: SIZEOF(QUERY ( pf <* USEDIN (SELF,
   'AP210_ARM.INTERCONNECT_MODULE_SURFACE_FEATURE.ASSOCIATED_DEFINITION') |
    ('AP210_ARM.' +
     'INTERCONNECT_MODULE_EDGE_SURFACE' IN TYPEOF (pf))))
     <= 1;
  WR4: EXISTS(maximum_thickness_over_metal_requirement) OR
       EXISTS(maximum_thickness_over_dielectric_requirement);
  WR5: NOT(EXISTS(minimum_thickness_over_metal_requirement)) OR
       (minimum_thickness_over_metal_requirement\data_element.measure_value > 0.0);
  WR6: NOT(EXISTS(maximum_thickness_over_metal_requirement)) OR
       (maximum_thickness_over_metal_requirement\data_element.measure_value > 0.0);
  WR7: NOT(EXISTS(minimum_thickness_over_dielectric_requirement)) OR
       (minimum_thickness_over_dielectric_requirement\data_element.measure_value > 0.0);
  WR8: NOT(EXISTS(maximum_thickness_over_dielectric_requirement)) OR
       (maximum_thickness_over_dielectric_requirement\data_element.measure_value > 0.0);
END_ENTITY; -- interconnect_module_usage_view
