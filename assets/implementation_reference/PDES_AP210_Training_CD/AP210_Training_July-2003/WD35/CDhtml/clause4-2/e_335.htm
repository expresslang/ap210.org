4.2.335 Ee_rule_premise_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
The Ee_rule_premise_definition is a type of Managed_design_object. An Ee_rule_premise_definition is the premise for a rule construct. The Ee_rule_premise_definition uses an 'OR' construct so that if any of the parts of the premise are true, the rule will fire.

The data associated with this application object are the following:

  • rule_definition
  • identifier
  • disjunctive_content

4.2.335.1 rule_definition

The rule_definition specifies the Ee_rule_definition for the Ee_rule_premise_definition. The rule_definition is the Ee_rule_definition this Ee_rule_premise_definition is helping to compose.

4.2.335.2 identifier

The identifier specifies the name for the Ee_rule_premise_definition.

4.2.335.3 disjunctive_content

The disjunctive_content specifies an Ee_rule_complex_clause. There shall be one or more disjunctive_content specified for an Ee_rule_premise_definition.

Formal constraints:

UR1: The combination of rule_definition and identifier shall be unique within a population of Ee_rule_conclusion_definition.
