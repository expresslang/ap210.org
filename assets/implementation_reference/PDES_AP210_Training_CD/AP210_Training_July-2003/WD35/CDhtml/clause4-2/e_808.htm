4.2.808 Shape_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shape_definition is a type of Managed_design_object A Shape_definition is one of a Bound_volume_shape, a Curve, an Edge_based_2d_wireframe, a Geometrically_bounded_2d_wireframe, an Open_shell_based_surface, a Planar_shape,or a Shell_based_2d_wireframe. A Shape_definition is also one of a Feature_shape_definition or of a Non_feature_shape_definition. A Shape_definition is the aggregation of geometry elements that represents the shape of an object or that represents additional properties of an object or that represents annotation information. The dimensionality of the geometric elements is determined by the dimensionality of the object being represented.

The data associated with this application object are the following:

  • geometric_context

4.2.808.1 geometric_context

The geometric_context specifies the role of the Cartesian_coordinate_system for the Shape_definition.

ENTITY shape_definition
  ABSTRACT SUPERTYPE OF ((ONEOF (
      bound_volume_shape,
      open_shell_based_surface,
      manifold_subsurface_shape,
      planar_shape)) AND
      (ONEOF(feature_shape_definition,
       non_feature_shape_definition)))
  SUBTYPE OF (managed_design_object);
    geometric_context : cartesian_coordinate_system;
    element : OPTIONAL SET [1:?] OF shape_item;
END_ENTITY; -- shape_definition
