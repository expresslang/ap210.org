4.2.528 Material_addition_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Material_addition_feature is a type of Design_intent_modification that describes the material that is to be added in the realization of the feature specified by the design_intent attribute inherited from Design_intent_modification. This requirement shall be interpreted as being mathematically equivalent to the application of the regularized Boolean geometric OR operation where the design_intent is OR'd with the shape of the Material_addition_feature.
