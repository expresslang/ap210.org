4.2.673 Pitch_class

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Pitch_class is a type of Managed_design_object. A Pitch_class is a category in which a particular group of Part_feature may fall, depending on the value of their paired spacing. The Pitch_class controls the width of the mating feature which is measured perpendicular to the primary axis of a terminal in the class. The only Data_element type permitted in the range is a Length_data_element.

EXAMPLE: The Pitch_class is included so that tolerances may be applied to all members of the same class, and specified only once. E.g., all pitchs with a design spacing greater than 50 mils and less than 75 mils are in the 'big' class.

NOTE: Other Application Objects will be populated in order to apply the class property to tolerances of the product being designed .

The data associated with this application object are the following:

  • class_member
  • class_name

4.2.673.1 class_member

The class_member specifies the role of the Range_characteristic for the Pitch_class. The range specifies the spacings which are included as a member of the Pitch_class. The Range_characteristic shall use Length_data_element for the Numeric_parameter.

4.2.673.2 class_name

The class_name specifies the name for the Pitch_class.
