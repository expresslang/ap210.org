4.2.632 Part_text_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_text_template is a type of Part_template that is the information required to support the creation of text messages in the stratum material on the Interconnect_module or in a marking application.

The data associated with this application object are the following:

  • font
  • maximum_font_horizontal_extent
  • maximum_font_vertical_extent
  • slant_angle
  • stroke_width

4.2.632.1 font

The font specifies either the Font_definition or the Registered_font for the Part_text_template.

4.2.632.2 maximum_font_horizontal_extent

The maximum_font_horizontal_extent specifies the role of the Length_data_element for the width of the Part_text_template.

4.2.632.3 maximum_font_vertical_extent

The maximum_font_vertical_extent specifies the role of the Length_data_element for the height of the Part_text_template.

4.2.632.4 slant_angle

The slant_angle specifies the Angle_data_element that is the deviation from vertical for the Part_text_template.

4.2.632.5 stroke_width

The stroke_width specifies the Length_data_element that is the distance across a single line that draws the glyph for the Part_text_template.

NOTE: A per character shape is required because the message is not available in the template definition. This limits the capability to support proportional font by causing the bounding box to be the maximum size required rather than the minimum that would occur if the bounding box were calculated strictly on the basis of the font proportions in the actual design.

Formal constraints:

WR1: font glyph shall not extend beyond the shape established by the template_shape.

ENTITY part_text_template
  SUBTYPE OF (part_template);
    font                           : font_select;
    maximum_font_horizontal_extent : length_data_element;
    maximum_font_vertical_extent   : length_data_element;
    slant_angle                    : angle_data_element;
    stroke_width                   : length_data_element;
END_ENTITY; -- part_text_template
