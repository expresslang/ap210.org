4.2.529 Material_addition_feature_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Material_addition_feature_template is a type of Part_template that describes the shape of material to be added to a design feature.

The data associated with this application object are the following:

  • of_stratum_technology

4.2.529.1 of_stratum_technology

The of_stratum_technology specifies the role of the Stratum_technology for the Material_addition_feature_template.
