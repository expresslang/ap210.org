4.2.215 Datum_axis

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_axis is a type of Datum that is a theoretically exact axis to which toleranced Shape_element are related.

Formal constraints:

WR1: If the Datum_axis is not specified as a reference_axes by a Datum_reference_frame then the Datum_axis shall be either a Common_datum, or a Single_datum.

NOTE: 
*)
ENTITY datum_axis
  SUBTYPE OF (datum);
END_ENTITY; -- datum_axis
