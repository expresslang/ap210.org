4.2.841 Statistical_geometric_tolerance

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Statistical_geometric_tolerance is a type of Geometric_tolerance that specifies design requirements that are based on sound statistics. Associated with the Statistical_geometric_tolerance is the requirement that a specified level of process control be maintained during the manufacture of the Physical_unit. A Statistical_geometric_tolerance may specify another Geometric_tolerance as an alternate to the Statistical_geometric_tolerance in the event that the specified level of process control cannot be met.

NOTE: Statistical tolerancing is discussed in 2.16 of ASME Y14.5M.

The data associated with this application object are the following:

  • alternate_tolerance
  • statistical_control_requirement

4.2.841.1 alternate_tolerance

The alternate_tolerance specifies the Geometric_tolerance that shall be used if the specified level of process control cannot be met. The alternate_tolerance need not be specified for a Statistical_geometric_tolerance.

4.2.841.2 statistical_control_requirement

The statistical_control_requirement specifies the Ee_requirement_occurrence that states the statistical process control requirements that the manufacturing facility shall meet for the Statistical_ geometric_tolerance.

Formal constraints:

WR1: If a Geometric_tolerance is specified as an alternate_tolerance, the specified Geometric_tolerance shall not be a Statistical_geometric_tolerance.

WR2: If a Geometric_tolerance is specified as an alternate_tolerance, the Shape_element specified as the toleranced_shape_element by the Statistical_geometric_tolerance shall be the same instance of Shape_element as specified as the toleranced_shape_element by the Geometric_tolerance specified as the alternate_tolerance.

NOTE: The attribute toleranced_shape_element is declared in the Geometric_tolerance entity of which this entity is a subtype.

ENTITY statistical_geometric_tolerance
  SUBTYPE OF (geometric_tolerance);
