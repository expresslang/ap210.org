4.2.153 Component_termination_passage_template_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_termination_passage_template_terminal is a type of Shape_element. A Component_termination_passage_template_terminal is one of a Component_termination_passage_template_interface_terminal or a Component_termination_passage_template_join_terminal.

The data associated with this application object are the following:

  • associated_definition
  • name
  • terminal_connection_zone

4.2.153.1 associated_definition

The associated_definition specifies the role of the Component_termination_passage_template for the Component_termination_passage_template_terminal.

4.2.153.2 name

The name specifies the string identifying the Component_termination_passage_template_terminal.

4.2.153.3 terminal_connection_zone

The terminal_connection_zone specifies the role of the Connection_zone for the Component_termination_passage_template_terminal. The terminal_connection_zone is optional and need not be supplied if the data is unavailable in the source system. If the data is available in the source system, it shall be supplied.

Formal constraints:

UR1: The combination of the associated_definition and the name shall be unique within a population of Component_termination_passage_template_terminals.

WR1: The points comprising the terminal_connection_zones shall be defined in the coordinate system of the device being accessed via the terminal_connection_zones.
