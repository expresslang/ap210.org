4.2.503 Layout_line_width_tolerance_requirement_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Layout_line_width_tolerance_requirement_occurrence is a type of Inspection_requirement_occurrence that characterizes the line width range that is expected by the design organization to be manufactured for a particular Stratum_technology.

The data associated with this application object are the following:

  • characterized_stratum_technology
  • maximum_negative_deviation
  • maximum_positive_deviation
  • tolerance_line_width_class

4.2.503.1 characterized_stratum_technology

The characterized_stratum_technology specifies the role of the Stratum_technology for the Layout_line_width_tolerance_requirement_occurrence.

4.2.503.2 maximum_negative_deviation

The maximum_negative_deviation specifies the largest negative Length_data_element for the Layout_line_width_tolerance_requirement_occurrence.

4.2.503.3 maximum_positive_deviation

The maximum_positive_deviation specifies the largest positive Length_data_element for the Layout_line_width_tolerance_requirement_occurrence.

4.2.503.4 tolerance_line_width_class

The tolerance_line_width_class specifies the role of the Line_width_class for the Layout_line_width_tolerance_requirement_occurrence.
