4.2.376 Fill_area

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Fill_area is a type of Managed_design_object. A Fill_area is the area bounded by a Curve that is defined to be filled with material using a pattern. Each Curve shall be an outer or inner boundary. Each boundary shall be closed. There shall be no more than one outer boundary. No boundary shall self-intersect or intersect with any other boundary.

NOTE: This Application Object is describing geometric patterns. The association of material with the patterns occurs elsewhere, for instance in the Design_intent_modification_styled_area_representation.

The data associated with this application object are the following:

  • boundary

4.2.376.1 boundary

The boundary specifies the Closed_curve that describes the outline of the Fill_area. There shall be one or more Curve for a Fill_area. The curves shall be non-self intersecting and non-overlapping.
