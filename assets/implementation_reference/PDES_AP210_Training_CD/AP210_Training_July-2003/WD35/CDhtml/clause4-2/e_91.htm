4.2.91 Centre_of_symmetry

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Centre_of_symmetry is a type of Derived_shape_element that is derived by determining the geometric centre of one or more Shape_element. A Centre_of_symmetry is one of a Centre_axis, a Centre_plane, or a Centre_point.
