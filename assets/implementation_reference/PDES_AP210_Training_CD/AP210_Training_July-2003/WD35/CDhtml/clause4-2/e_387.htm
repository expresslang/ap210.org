4.2.387 Font_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Font_definition is a type of Managed_design_object. A Font_definition is the definition of a font. Font support shall include outlines using glyphs, as well as font and font family.

NOTES:

  1. This description is intended to be a complete reference to a font definition. Items to be included are usually font size, aspect ratio, type, etc.
  2. Font_description supports Part_text_template.

NOTE: 
*)
ENTITY font_definition
  SUBTYPE OF (managed_design_object);
END_ENTITY; -- font_definition
