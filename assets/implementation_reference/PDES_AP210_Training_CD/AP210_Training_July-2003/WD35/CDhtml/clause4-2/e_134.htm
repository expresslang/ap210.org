4.2.134 Component_part_3d_geometric_representation_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_part_3d_geometric_representation_relationship is a type of Geometric_representation_relationship. A Component_part_3d_geometric_representation_relationship is the association between a Part_template_3d_shape or a Physical_unit_3d_shape and the Assembly_component_3d_shape that is the occurrence of the three-dimensional shape. If the type of the shape representation of the assembly is different from the type of the shape representation of the part, the Component_part_3d_geometric_representation_relationship allows a processor to create a shape rep of the assembly type and establish an equivalency relationship between that shape representation and the part shape representation. If the shape representation of the assembly is of the same type as the part shape representation, the Component_part_3d_geometric_representation_relationship shall reference the same shape representation as both the part_shape and the component_shape. That instance shall be a complex instance consisting of a combination of a part shape and of a component shape.

The data associated with this application object are the following:

  • component_shape
  • part_shape

4.2.134.1 component_shape

The component_shape specifies the role of the Assembly_component_3d_shape for the Component_part_3d_geometric_representation_relationship.

4.2.134.2 part_shape

The part_shape specifies the Part_template_3d_shape, or the Part_template_non_planar_2d_shape, or the Physical_unit_3d_shape for the Component_part_3d_geometric_representation_relationship.

ENTITY component_part_3d_geometric_representation_relationship
  SUBTYPE OF (geometric_representation_relationship);
    DERIVE
      part_shape      : part_template_or_physical_unit_3d_shape_select := 
               SELF\geometric_representation_relationship.shape_1;
      component_shape : assembly_component_3d_shape := 
               SELF\geometric_representation_relationship.shape_2;
