4.2.714 Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship is a type of Volume_shape_intersection. A Pre_defined_perpendicular_datum_plane_symbol_3d_2d_relationship establishes the geometric reference relationship between a Datum_plane representation in a 3 dimensional coordinate system and the related representation in a 2 dimensional coordinate system where the Datum_plane is perpendicular to the plane that represents the 2 dimensional coordinate system in the 3 dimensional coordinate system. The controlled information is the line representing the normal of the plane.

NOTE: Due to differences in CAD systems resolutions the lines used as base and arrowhead may sightly deviate from intersecting at the start and end of the line.
