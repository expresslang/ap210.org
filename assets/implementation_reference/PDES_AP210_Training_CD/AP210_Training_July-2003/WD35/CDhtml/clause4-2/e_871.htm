4.2.871 Stratum_technology

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_technology is a type of Managed_design_object. A Stratum_technology may be either a Design_layer_technology or a Documentation_layer_technology. A Stratum_technology provides material properties, parametric data associated with the technology, and material identification.

The data associated with this application object are the following:

  • laminate_stiffness_class
  • layer_position
  • maximum_feature_size_requirement
  • maximum_stratum_thickness
  • minimum_finished_feature_size
  • minimum_finished_feature_spacing
  • minimum_stratum_thickness
  • name
  • specification
  • stratum_material

4.2.871.1 laminate_stiffness_class

The laminate_stiffness_class specifies either a conformal_coat, a fluid_like, or a stiff_laminate stiffness class of which the Stratum_technology is a member. The laminate_stiffness_class need not be specified for a particular Stratum_technology.

4.2.871.2 layer_position

The layer_position specifies one of an internal, external, all, primary, or secondary position of the Stratum for the Stratum_technology. Primary and secondary shall only be used if the member of Stratum_technology is a Design_layer_technology.

4.2.871.3 maximum_feature_size_requirement

The maximum_feature_size_requirement specifies the maximum Length_data_element of any feature of the Stratum. The maximum_feature_size_requirement need not be specified for a particular Stratum_technology.

4.2.871.4 maximum_stratum_thickness

The maximum_stratum_thickness specifies the maximum Length_data_element for the thickness of the Stratum_technology.

4.2.871.5 minimum_finished_feature_size

The minimum_finished_feature_size specifies one role of the Length_data_element of any feature of the Stratum. The minimum_finished_feature_size need not be specified for a particular Stratum_technology.

4.2.871.6 minimum_finished_feature_spacing

The minimum_finished_feature_spacing specifies one role of the Length_data_element of any feature of the Stratum. The minimum_finished_feature_spacing need not be specified for a particular Stratum_technology.

4.2.871.7 minimum_stratum_thickness

The minimum_stratum_thickness specifies the minimum Length_data_element for the thickness of the Stratum_technology.

4.2.871.8 name

The name specifies the alphanumeric identifier of a Stratum_technology.

4.2.871.9 specification

The specification specifies the role of the Process_specification for the Stratum_technology. The specification need not be specified for a particular Stratum_technology.

4.2.871.10 stratum_material

The stratum_material specifies the role of the Ee_material for the Stratum_technology.

Formal constraints:

UR1: The name shall be unique within a population of Stratum_technologies.

ENTITY stratum_technology
  SUPERTYPE OF (ONEOF (design_layer_technology,
      documentation_layer_technology))
  SUBTYPE OF (managed_design_object);
    layer_position                   : layer_position_type;
    stratum_material                 : ee_material;
    maximum_stratum_thickness        : length_data_element;
    minimum_stratum_thickness        : length_data_element;
    name                             : ee_name;
    minimum_finished_feature_size    : OPTIONAL length_data_element;
    laminate_stiffness_class         : OPTIONAL stiffness_class;
    minimum_finished_feature_spacing : OPTIONAL length_data_element;
    maximum_feature_size_requirement : OPTIONAL length_data_element;
    specification                    : OPTIONAL process_specification;
  UNIQUE
    ur1 : name;
WHERE
  wr1: NOT(layer_position = layer_position_type.primary) OR ('AP210_ARM.DESIGN_LAYER_TECHNOLOGY' IN
       TYPEOF(SELF));
  wr2: NOT(layer_position = layer_position_type.secondary) OR ('AP210_ARM.DESIGN_LAYER_TECHNOLOGY' IN
       TYPEOF(SELF));
END_ENTITY; -- stratum_technology
