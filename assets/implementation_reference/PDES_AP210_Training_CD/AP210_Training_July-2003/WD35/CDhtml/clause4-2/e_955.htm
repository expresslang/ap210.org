4.2.956 Viewing_plane

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Viewing_plane is a type of Non_feature_shape_element that is a plane to which a Tolerance_zone_boundary is oriented.

The data associated with this application object are the following:

  • viewing_plane_specification

4.2.956.1 viewing_plane_specification

A viewing_plane_specification is an inverse relationship that specifies that the existence of the Viewing_plane is dependent on the existence of the Viewing_plane_definition that specifies the Viewing_plane as its defined_viewing_plane.
