4.2.863 Stratum_interconnect_module_3d_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_interconnect_module_3d_position is a type of Managed_design_object. A Stratum_interconnect_module_3d_position is the position of the Stratum model in the Interconnect_module model in three dimensions.

The data associated with this application object are the following:

  • interconnect_module_shape
  • stratum_component
  • stratum_shape
  • transformation

4.2.863.1 interconnect_module_shape

The interconnect_module_shape specifies the Physical_unit_3d_shape that is the Interconnect_module model for the Stratum_interconnect_module_3d_position.

4.2.863.2 stratum_component

The stratum_component specifies the role of the Interconnect_module_stratum_assembly_relationship for the Stratum_interconnect_module_3d_position.

4.2.863.3 stratum_shape

The stratum_shape specifies the Stratum_3d_shape for the Stratum model for the Stratum_interconnect_module_3d_position.

4.2.863.4 transformation

The transformation specifies the role of the Axis_placement_3d for the Stratum_interconnect_module_3d_position.
