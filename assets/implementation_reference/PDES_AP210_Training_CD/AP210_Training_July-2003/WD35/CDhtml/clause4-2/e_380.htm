4.2.378 Fill_area_style

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Fill_area_style is a type of Managed_design_object. A Fill_area_style is the selected style used in a Fill_area.

The data associated with this application object are the following:

  • fill_style
  • name

4.2.378.1 fill_style

The fill_style specifies either an Externally_defined_hatch_style, an Externally_defined_tile_style, a Fill_area_style_hatching, a Fill_area_style_tile, or a Solid_fill_style for the Fill_area_style.

4.2.378.2 name

The name specifies the string for the identifier for the Fill_area_style.

Formal constraints:

UR1: The name shall be unique within a population of Fill_area_style.
