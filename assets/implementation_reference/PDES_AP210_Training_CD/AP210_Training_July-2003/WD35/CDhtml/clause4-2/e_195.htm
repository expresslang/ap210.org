4.2.195 Contact_size_dependent_non_functional_land

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Contact_size_dependent_non_functional_land is a type of Non_functional_land whose shape is dependent on the size of the contact area of the Component_terminal that will be joined to the Contact_size_dependent_land in the assembly.

The data associated with this application object are the following:

  • usage_definition

4.2.195.1 usage_definition

The usage_definition specifies the role of the Default_attachment_size_based_land_physical_template for the Contact_size_dependent_non_functional_land.
