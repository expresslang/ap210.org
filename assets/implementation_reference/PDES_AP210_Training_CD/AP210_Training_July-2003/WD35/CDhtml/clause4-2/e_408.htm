4.2.408 Functional_unit_terminal_bus_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_unit_terminal_bus_assignment is a type of Managed_design_object. A Functional_unit_terminal_bus_assignment is the relationship between the structural definition of a bus and a terminal that is assigned to the bus. The members of Functional_unit_terminal that are assigned to the same bus by this application object are defined to be connected to each other.

NOTE: There are cardinality constraints levied to ensure that the cardinality as resolved through all the references is identical.

The data associated with this application object are the following:

  • connected_bus
  • connected_terminal

4.2.408.1 connected_bus

The connected_bus specifies the role of the Bus_structural_definition for the Functional_unit_terminal_bus_assignment.

4.2.408.2 connected_terminal

The connected_terminal specifies the role of the Functional_unit_terminal for the Functional_unit_terminal_bus_assignment.
