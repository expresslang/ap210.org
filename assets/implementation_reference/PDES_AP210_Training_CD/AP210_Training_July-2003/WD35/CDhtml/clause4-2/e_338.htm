4.2.338 Ee_specification

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_specification is a type of Ee_document. An Ee_specification may be either a Design_specification, a Language_reference_manual, a Material_specification, a Process_specification, or a Test_specification.

The data associated with this application object are the following:

  • source

4.2.338.1 source

The source specifies the Organization or the Person_and_organization responsible for the information of an Ee_specification.
