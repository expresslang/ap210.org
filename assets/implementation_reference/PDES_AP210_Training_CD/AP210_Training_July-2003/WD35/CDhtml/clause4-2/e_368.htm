4.2.368 Fabrication_joint

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Fabrication_joint is a type of Managed_design_object. A Fabrication_joint is a joint that is created in the fabrication of a Interconnect_module.

EXAMPLES:

  1. Figure fj is a plan view of the construction of a Fabrication_joint between a Land and a Conductive_interconnect_element that results in a Conductor segment. This figure illustrates the geometric relationship of the element terminals in the construction of the segment.
  2. Figure fj2 is a cross-sectional view of the construction of a Fabrication_joint between a Land and a Component_termination_passage.

The data associated with this application object are the following:

  • auxiliary_joint_material
  • fabrication_features

4.2.368.1 auxiliary_joint_material

The auxiliary_joint_material specifies the Stratum_feature_template_component for the Fabrication_joint. The auxiliary_joint_material need not be specified for a particular Fabrication_joint.

4.2.368.2 fabrication_features

The fabrication_features specify exactly two from among the following: a Component_termination_passage_join_terminal, a Conductive_interconnect_element_terminal, a Land_join_terminal, a Non_functional_land_join_terminal, a Printed_component_join_terminal, or a Via_terminal for the Fabrication_joint.

Formal constraints:

WR1: A member of Conductive_interconnect_element_terminal and a member of Non_functional_land_join_terminal shall not be in the same set of fabrication_features.

ENTITY fabrication_joint
  SUBTYPE OF (managed_design_object);
    fabrication_features_1   : fabricated_feature_select;
    fabrication_features_2   : fabricated_feature_select;
    auxiliary_joint_material : OPTIONAL stratum_feature_template_component;
    associated_interconnect_module : interconnect_module;
END_ENTITY; -- fabrication_joint
