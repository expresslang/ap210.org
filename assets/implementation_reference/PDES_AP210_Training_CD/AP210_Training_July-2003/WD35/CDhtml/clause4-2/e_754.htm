4.2.754 Projected_zone_height_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Projected_zone_height_characteristic is a type of Managed_design_object. A Projected_zone_height_characteristic is a definition that characterizes the height that a Tolerance_zone is projected from a Physical_feature_or_part_template.

The data associated with this application object are the following:

  • defined_height_of
  • projected_height_value
  • zone_definition

4.2.754.1 defined_height_of

The defined_height_of specifies the Projected_zone_and_base_relationship that associates a Tolerance_zone with the Physical_feature_or_part_template from which the Tolerance_zone is projected. The specified Projected_zone_and_base_relationship is characterized by the Projected_zone_height_characteristic.

4.2.754.2 projected_height_value

The projected_height_value specifies the Length_data_element that represents the height that the associated Tolerance_zone is projected relative to the associated Physical_feature_or_part_template for the Projected_zone_height_characteristic.

4.2.754.3 zone_definition

The zone_definition is an inverse relationship that specifies that the existence of the Projected_zone_height_characteristic is dependent on the existence of the Projected_tolerance_zone_definition that specifies the Projected_zone_height_characteristic as its projected_zone_height.

ENTITY projected_zone_height_characteristic
  SUBTYPE OF (managed_design_object);
    defined_height_of      : projected_zone_and_base_relationship;
    projected_height_value : length_data_element;
  INVERSE
    zone_definition : projected_tolerance_zone_definition FOR 
                      projected_zone_height;
END_ENTITY; -- projected_zone_height_characteristic
