4.2.377 Fill_area_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Fill_area_occurrence is a type of Managed_design_object. A Fill_area_occurrence is an instantiation of a Fill_area. The boundary_style shall be applied to each boundary of the instance fo the Fill_area. A zero width boundary_style shall be interpreted as using the boundaries of the Fill_area as clipping regions. A non-zero width boundary_style shall be interpreted as the regularlized Boolean "OR" operation on the instance of the Fill_area, applied at each boundary after the clip operation required by the zero width boundary style. The orientation of the Fill_area instances shall be parallel with the orientation of the Fill_area.

The data associated with this application object are the following:

  • boundary_style
  • filled_area
  • fill_style_target
  • style

4.2.377.1 boundary_style

The boundary_style specifies the role of the Curve_style for the Fill_area_occurrence. The boundary_style is applied to each instance of the Fill_area boundaries individually. The boundary_style end parameters shall be ignored. The boundary_style curve_font shall not be specified.

4.2.377.2 filled_area

The filled_area specifies the role of the Fill_area for the Fill_area_occurrence. The Fill_area provides shape and orientation information for the Fill_area_occurrence.

4.2.377.3 fill_style_target

The fill_style_target specifies the Cartesian_point that defines the location of the Fill_area_style for the Fill_area_occurrence.

4.2.377.4 style

The style specifies the role of the Fill_area_style for the Fill_area_occurrence. There shall be one or more Fill_area_style for a Fill_area_occurrence.
