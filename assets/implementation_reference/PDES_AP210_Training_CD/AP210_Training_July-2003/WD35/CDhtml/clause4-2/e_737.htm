4.2.737 Printed_part_template_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_part_template_terminal is a type of Managed_design_object. A Printed_part_template_terminal is a Printed_part_cross_section_template_terminal or additionally one of a Printed_part_template_interface_terminal or a Printed_part_template_join_terminal. A Printed_part_template_terminal is a terminal of a Printed_part_template.

The data associated with this application object are the following:

  • associated_definition
  • connection_requirement
  • connection_zone_category
  • feature_shape
  • functional_usage_view_terminal
  • name
  • terminal_connection_zone

4.2.737.1 associated_definition

The associated_definition specifies the role of the Printed_part_template for the Printed_part_template_terminal.

4.2.737.2 connection_requirement

The connection_requirement is an inverse relationship that specifies that the existence of the Printed_part_template_terminal is dependent on the existence of the Printed_part_template_connected_terminals_definition that specifies the Printed_part_template_terminal as its connected_terminals. There shall be no more than one Printed_part_template_connected_terminals_definition for a Printed_part_template_terminal.

4.2.737.3 connection_zone_category

The connection_zone_category specifies either an area_edge_segment, a curve_edge_segment, a surface_area, or a surface_point for the Printed_part_template_terminal.

4.2.737.4 feature_shape

The feature_shape specifies the role of the Feature_shape_occurrence for the Printed_part_template_terminal. There shall be one or more Feature_shape_occurrence for the Printed_part_template_terminal.

4.2.737.5 functional_usage_view_terminal

The functional_usage_view_terminal is an inverse relationship that specifies the Functional_usage_view_to_printed_part_template_terminal_assignment that specifies the Printed_part_template_terminal as its physical_usage_view_terminal. There shall be no more than one Functional_usage_view_to_printed_part_template_terminal_assignment for a Printed_part_template_terminal.

4.2.737.6 name

The name identifies the descriptive label for the Printed_part_template_terminal.

4.2.737.7 terminal_connection_zone

The terminal_connection_zone specifies the role of the Connection_zone for the Printed_part_template_terminal. There may be one or more Connection_zone for a Printed_part_template_terminal. The terminal_connection_zone is optional and need not be supplied if the source system does not contain the data. If the source system contains the data, then the terminal_connection_zone shall be provided. The Feature_shape_occurrence specified by the terminal or mounting feature may be the same Feature_shape_occurrence that is specified by the associated Connection_zone. In this case the whole terminal is specified as the Connection_zone.

Formal constraints:

UR1: The combination of the name and the associated_definition shall be unique within a population of Printed_part_template_terminals.

WR1: The points comprising the attachment surface must reside in the shape of the electrical device being accessed.

WR2: The points comprising the terminal_connection_zone shall be defined in the coordinate system of the device being accessed via the terminal_connection_zone.

ENTITY printed_part_template_terminal
  ABSTRACT SUPERTYPE OF (
      printed_part_cross_section_template_terminal
      ANDOR (ONEOF (printed_part_template_join_terminal,
      printed_part_template_interface_terminal)))
  SUBTYPE OF (managed_design_object);
    associated_definition    : printed_part_template;
    connection_zone_category : printed_part_template_terminal_class;
    terminal_connection_zone : OPTIONAL SET [1:?] OF connection_zone;
    name                     : ee_name;
  INVERSE
    connection_requirement         : 
	SET [0:1] OF printed_part_template_connected_terminals_definition FOR 
	connected_terminals;
    functional_usage_view_terminal : 
	SET [0:1] OF functional_usage_view_to_printed_part_template_terminal_assignment FOR 
	physical_usage_view_terminal;
    feature_shape         : SET [1:?] OF usage_concept_usage_relationship FOR
    associated_usage;
  UNIQUE
    ur1 : name, associated_definition;
END_ENTITY; -- printed_part_template_terminal
