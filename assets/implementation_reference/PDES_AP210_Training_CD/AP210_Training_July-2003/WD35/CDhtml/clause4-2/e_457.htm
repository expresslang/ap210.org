4.2.464 Interconnect_module_terminal_surface_constituent_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_terminal_surface_constituent_relationship is a type of Shape_element_constituent_relationship.

The data associated with this application object are the following:

  • relating_shape_element
  • related_shape_element

4.2.464.1 relating_shape_element

The relating_shape_element specifies the role of the Interconnect_module_surface_feature for the Interconnect_module_terminal_surface_constituent_relationship.

4.2.464.2 related_shape_element

The related_shape_element specifies the role of the Interconnect_module_terminal for the Interconnect_module_terminal_surface_constituent_relationship.
