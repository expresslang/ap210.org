4.2.324 Ee_product_definition_based_constraint_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_product_definition_based_constraint_occurrence is a type of Constraint_occurrence that has a basis in the characteristics of an Ee_product_definition and that identifies the basis definition. AnEe_product_definition_based_constraint_occurrence may be a Shape_and_ee_product_definition_based_constraint_occurrence.

The data associated with this application object are the following:

  • constraining_definition

4.2.324.1 constraining_definition

The constraining_definition specifies the role of the Ee_product_definition for the Ee_product_definition_based_constraint_occurrence.
