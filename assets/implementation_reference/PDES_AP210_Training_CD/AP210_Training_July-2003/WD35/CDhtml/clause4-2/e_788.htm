4.2.788 Rule_modification

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_modification is a type of Rule_action that states a version of an Ee_rule is a result of a request to change an existing version of an Ee_rule.

The data associated with this application object are the following:

  • modification_rationale

4.2.788.1 modification_rationale

The modification_rationale specifies the role of the Rule_change_request for the Rule_modification.
