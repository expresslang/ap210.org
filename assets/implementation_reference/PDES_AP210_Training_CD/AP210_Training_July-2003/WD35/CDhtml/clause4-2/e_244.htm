4.2.244 Default_assembly_bond_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Default_assembly_bond_shape is a type of Bound_volume_shape and a type of Feature_shape_definition. A Default_assembly_bond_shape is the shape of an Assembly_bond_definition.

The data associated with this application object are the following:

  • characterized_bond_definition

4.2.244.1 characterized_bond_definition

The characterized_bond_definition specifies the role of the Assembly_bond_definition for the Default_assembly_bond_shape.
