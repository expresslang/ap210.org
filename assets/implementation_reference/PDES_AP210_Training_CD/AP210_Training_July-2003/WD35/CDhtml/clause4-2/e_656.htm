4.2.656 Physical_connectivity_make_from_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_connectivity_make_from_relationship is a type of Managed_design_object. A Physical_connectivity_make_from_relationship is an association between one Physical_connectivity_definition that is used as a basis for modification into another Physical_connectivity_definition and the modified Physical_connectivity_definition that results from the modification.

The data associated with this application object are the following:

  • resultant_definition
  • reusable_definition

4.2.656.1 resultant_definition

The resultant_definition specifies the Physical_connectivity_definition that results from modification of the reusable_definition of the Physical_connectivity_make_from_relationship.

4.2.656.2 reusable_definition

The reusable_definition specifies the Physical_connectivity_definition for the basis for modification into the resultant_definition in the Physical_connectivity_make_from_relationship.
