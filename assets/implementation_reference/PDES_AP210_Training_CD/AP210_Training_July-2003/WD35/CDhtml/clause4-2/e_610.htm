4.2.606 Part_device

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_device is a type of Managed_design_object. A Part_device is a specific occurrence of a Bare_die in a Packaged_part.

The data associated with this application object are the following:

  • bonding_agent
  • defined_packaged_part
  • included_device
  • device_2d_position
  • device_3d_position
  • item_identification

4.2.606.1 bonding_agent

The bonding_agent specifies the Ee_material used to attach the Bare_die within the Packaged_part. The bonding_agent need not be specified for a particular Part_device.

4.2.606.2 defined_packaged_part

The defined_packaged_part specifies the Packaged_part that contains this Part_device.

4.2.606.3 device_2d_position

The device_2d_position is an inverse relationship that specifies the existence of the Device_2d_position that specifies the Part_device as its placed_device. There shall be no more than one Device_2d_position for a Part_device.

4.2.606.4 device_3d_position

The device_3d_position is an inverse relationship that specifies the existence of the Device_3d_position that specifies the Part_device as its placed_device. There shall be no more than one Device_3d_position for a Part_device.

4.2.606.5 included_device

The included_device specifies the Bare_die used to provide the function of the Part_device.

4.2.606.6 item_identification

The item_identification specifies the name for the reference designation of the Bare_die instance within the Packaged_part.

Formal constraints:

UR1: The combination of the defined_package_part and the item_identification shall be unique within a population of Part_devices.

ENTITY part_device
  SUBTYPE OF (managed_design_object);
    defined_packaged_part : packaged_part;
    included_device       : bare_die;
    item_identification   : ee_name;
    bonding_agent         : OPTIONAL ee_material;
  INVERSE
    device_3d_position : SET [0:1] OF device_3d_position FOR placed_device;
    device_2d_position : SET [0:1] OF device_2d_position FOR placed_device;
  UNIQUE
    ur1 : defined_packaged_part, item_identification;
END_ENTITY; -- part_device
