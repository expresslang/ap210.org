4.2.735 Printed_part_template_material

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_part_template_material is a type of Managed_design_object. A Printed_part_template_material is the Ee_material characterized in Printed_part_cross_section_template, for the realization of the Stratum containing part of the Printed_component during fabrication of the Interconnect_module.

NOTE: A Printed_component may span more than one Stratum, as defined by the Printed_part_template.

The data associated with this application object are the following:

  • maximum_thickness
  • minimum_thickness
  • required_material

4.2.735.1 maximum_thickness

The maximum_thickness specifies the maximum Length_data_element for the thickness of the Printed_part_template_material.

4.2.735.2 minimum_thickness

The minimum_thickness specifies the minimum Length_data_element for the thickness of the Printed_part_template_material.

4.2.735.3 required_material

The required_material specifies the role of the Ee_material for the Printed_part_template_material.
