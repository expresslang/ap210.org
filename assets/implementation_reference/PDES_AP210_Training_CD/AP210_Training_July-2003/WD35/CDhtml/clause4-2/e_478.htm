4.2.478 Item_restricted_requirement_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Item_restricted_requirement_occurrence is a type of Ee_requirement_occurrence. The Item_restricted_requirement_occurrence provides the ability for a design organization to track the dependency of a requirement on a specific design object.

EXAMPLE: Thermal requirements may, for example, be dependent on the processor module or the drive motor.

The data associated with this application object are the following:

  • basis

4.2.478.1 basis

The basis specifies the role of one of an Assembly_component, a Component_external_reference, a Component_feature; or a Component_feature_external_reference; or one of an Ee_requirement_occurrence, a Stratum; or one of an Inter_stratum_feature, a Laminate_component, a Stratum_feature, or a Stratum_surface for the Item_restricted_requirement_occurrence. The basis is the reference design object for the requirement.
