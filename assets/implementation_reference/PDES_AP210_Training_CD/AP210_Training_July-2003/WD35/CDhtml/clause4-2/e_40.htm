4.2.40 Assembly_group_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Assembly_group_component is a type of Assembly_component. An Assembly_group_component may be either one or both of an Array_placement_group or an Interfaced_group_component. Each component that is part of a group is in the same design view as that group.

EXAMPLE: This application object is provided to support, for instance, the hardware items required for power device mounting.

NOTES:

  1. This application object is not included to support process plans.
  2. The intent is for all members of Assembly_component that are also in the role of related_components for Sub_assembly_relationship to be placed as a group in a placement activity using a CAD tool. However, each member of the group is placed individually in this standard. A receiving system should treat the relative placement of members of the group as derived data.

EXAMPLE: This application object supports paired-line layout by making each individual conductive_interconnect_element associated by the sub_assembly_relationship entity to this application object one of the lines to be paired. Extensions to higher number of lines is obvious.

The data associated with this application object are the following:

  • components

4.2.40.1 components

The components is an inverse relationship that specifies that the existence of the Assembly_group_component is dependent on the existence of the Sub_assembly_relationship that specifies the Assembly_group_component as its relating_component. There shall be one or more Sub_assembly_relationship for an Assembly_group_component.

NOTE: The usual case is for one member of Sub_assembly_relationship to associate all relating components, but more than one is permitted. There is no semantic difference in this part of ISO 10303 between all the components being related with one member of Sub_assembly_relationship or each of the component being related having its own Sub_assembly_relationship.

ENTITY assembly_group_component
  SUPERTYPE OF (interfaced_group_component ANDOR array_placement_group ANDOR
   (ONEOF(footprint_occurrence, padstack_occurrence)))
  SUBTYPE OF (assembly_component);
  INVERSE
    components : SET [1:?] OF sub_assembly_relationship FOR relating_component;
END_ENTITY; -- assembly_group_component
