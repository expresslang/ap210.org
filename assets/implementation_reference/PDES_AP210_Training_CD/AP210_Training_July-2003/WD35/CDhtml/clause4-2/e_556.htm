4.2.555 Next_higher_assembly_interface_requirement

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Next_higher_assembly_interface_requirement is a type of Managed_design_object. A Next_higher_assembly_interface_requirement is the association between the version of the next higher assembly and a shape that simulates the fit characteristics of the next higher assembly. The coordinate system of the shape is congruent with the coordinate system of the current design. The shape simulates a maximum material condition state of the next higher assembly.

The data associated with this application object are the following:

  • shape
  • version_of_next_higher_assembly

4.2.555.1 shape

The shape specifies the Bound_volume_shape for the model for the Next_higher_assembly_interface_requirement.

4.2.555.2 version_of_next_higher_assembly

The version_of_next_higher_assembly specifies the Ee_product_version of the Next_higher_assembly_interface_requirement.
