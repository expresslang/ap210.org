4.2.321 Ee_product_category_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_product_category_relationship is a type of Managed_design_object. An Ee_product_category_relationship establishes an inheritance relationship that is a parent-child relationship between two classes in a class hierarchy of Ee_product_category.

The data associated with this application object are the following:

  • category
  • sub_category

4.2.321.1 category

The category specifies the Ee_product_category that is in the role of higher level category in the classification structure.

4.2.321.2 sub_category

The sub_category specifies the Ee_product_category that inherits properties from the higher level category in the classification structure.

Formal constraints:

WR1: The Ee_product_category_relationship shall not establish a cyclic path in the classification structure it is a member of.
