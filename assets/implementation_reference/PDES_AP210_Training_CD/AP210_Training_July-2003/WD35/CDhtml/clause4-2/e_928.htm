4.2.928 Topological_junction

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Topological_junction is a type of Managed_design_object. A Topological_junction is used when at least three Component_terminal are connected and the explicit topology must be stated.

The data associated with this application object are the following:

  • name
  • scope

4.2.928.1 name

The name identifies the Topological_junction.

4.2.928.2 scope

The scope specifies the role of the Physical_connectivity_structure_definition for the Topological_junction.

Formal constraints:

UR1: The combination of the name and the scope shall be unique within a population of Topological_junction.
