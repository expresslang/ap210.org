4.2.389 Footprint_definition_planar_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Footprint_definition_planar_shape is a type of Managed_design_object. A Footprint_definition_planar_shape is the planar shape of a Footprint_definition.

The data associated with this application object are the following:

  • shape_characterized_footprint_definition
  • shape_environment
  • shape_material_condition
  • shape_representation

4.2.389.1 shape_characterized_footprint_definition

The shape_characterized_footprint_definition specifies the role of the Footprint_definition for the Footprint_definition_planar_shape.

4.2.389.2 shape_environment

The shape_environment specifies either manufacturing or an end_user_application as the valid environment for the Footprint_definition_planar_shape. The shape_environment need not be specified for a particular Footprint_definition_planar_shape.

4.2.389.3 shape_material_condition

The shape_material_condition specifies either the maximum_material_condition, the minimum_material_condition, or the nominal_material_condition as the material condition for the Footprint_definition_planar_shape.

4.2.389.4 shape_representation

The shape_representation specifies the Planar_projected_shape that is the model for the Footprint_definition_planar_shape.
