4.2.565 Numeric_parameter

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Numeric_parameter is a type of Characteristic. A Numeric_parameter is a quantitative value in an identified unit of measure. Uncertainty may be applied to the value by referencing the Qualified_data_element type of Data_element.

NOTE: Actual values of occurrence data will be dependent on many variables.

The data associated with this application object are the following:

  • property_value

4.2.565.1 property_value

The property_value specifies the Data_element for the quantitative value and unit of measure for the Numeric_parameter.

NOTE: 
*)
ENTITY numeric_parameter
  SUBTYPE OF (characteristic);
    SELF\characteristic.property_value : data_element;
END_ENTITY; -- numeric_parameter
