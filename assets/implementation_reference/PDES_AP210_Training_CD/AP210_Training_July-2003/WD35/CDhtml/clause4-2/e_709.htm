4.2.707 Pre_defined_dimension_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Pre_defined_dimension_qualifier is a type of Dimension_qualifier the intent of which is unambiguous and is commonly used in geometric dimensioning.

The data associated with this application object are the following:

  • description

4.2.707.1 description

The description specifies the label that describes the manner in which the Dimensional_characteristic that are specified as the qualified_dimensions are qualified by the Pre_defined_dimension_qualifier. The description shall be either "average", or "free_state".

NOTE: The attribute qualified_dimensions is declared in the Dimension_qualifier entity of which this entity is a subtype.
