4.2.8 Altered_package

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Altered_package is a type of Package. An Altered_package describes a Package that has been modified in some manner from a reference Package. The material or shape properties of the Altered_packaged are different from the reference Package.

EXAMPLES:

  1. Examples of alteration include trimming of the terminals, prebending of the terminals, tinning of the terminals for solderability, or more than one of these.
  2. Figure prepac illustrates some of the different forms a package may take when it is installed into an assembly. The original form on the left is in the role of to_be_altered for the forms on the right. The altered forms are identified and tracked by an enterprise.

NOTE: Most Altered_packages are a result of terminal alteration.

The data associated with this application object are the following:

  • package_to_be_altered

4.2.8.1 package_to_be_altered

The package_to_be_altered specifies the Package that is a reference for the Altered_package.

Formal constraints:

WR1: A Package shall not be in the role of package_to_be_altered for itself.
