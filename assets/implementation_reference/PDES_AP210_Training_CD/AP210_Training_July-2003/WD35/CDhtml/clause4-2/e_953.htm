4.2.953 Via_template_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Via_template_terminal is a type of Shape_element. A Via_template_terminal is a terminal in a Via_template. A Via_template_terminal is a terminal that is congruent with the exterior surface of the cylinder of the Via, including material covered by joined members of Land. The fabrication join capability is optionally constrained by a set of Stratum.

The data associated with this application object are the following:

  • associated_definition
  • disallowed_inter_stratum_extent
  • name
  • terminal_connection_zone

EXAMPLE: Figure vtt illustrates an abstraction of the cylinder associated with a Via and the associated terminal. Because a Via is always an internal design object, there are no interface or join subtypes. A usage view is included that defines the terminal to be just the external wall of the Via.

4.2.953.1 associated_definition

The associated_definition specifies the role of the Via_template for the Via_template_terminal.

4.2.953.2 disallowed_inter_stratum_extent

The disallowed_inter_stratum_extent specifies the Inter_stratum_extent for which a connection to the Via_template_terminal is disallowed. The disallowed_inter_stratum_extent need not be supplied for a Via_template_terminal. There may be more than one Inter_stratum_extent for a Via_template_terminal.

4.2.953.3 name

The name identifies the Via_template_terminal.

4.2.953.4 terminal_connection_zone

The terminal_connection_zone specifies the role of the Connection_zone for the Via_template_terminal. There may be one or more Connection_zone for the Via_template_terminal. The terminal_connection_zone is optional and need not be supplied if the data is unavailable in the source system. If the data is available in the source system, the data shall be provided.

Formal constraints:

UR1: The combination of the name and associated_definition shall be unique within a population of Via_template_terminal.

  SUBTYPE OF (shape_element);
    associated_definition           : via_template;
    disallowed_inter_stratum_extent : OPTIONAL inter_stratum_extent;
    SELF\shape_element.name         : ee_name;
    terminal_connection_zone        : OPTIONAL SET [1:?] OF connection_zone;
INVERSE
    terminal_shape         : SET [0:1] OF usage_concept_usage_relationship FOR
    associated_usage;
  UNIQUE
    ur1 : name, associated_definition;
END_ENTITY; -- via_template_terminal
