4.2.370 Fabrication_thickness_requirement_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Fabrication_thickness_requirement_occurrence is a type of Inspection_requirement_occurrence that characterizes the interconnect thickness range over metallic material or dielectric material and assigns a tolerance to that range for fabrication purposes.

The data associated with this application object are the following:

  • characterized_class
  • characterized_class_range
  • tolerance

4.2.370.1 characterized_class

The characterized_class specifies either "over_finished_conductor" or "over_finished_dielectric" for the Fabrication_thickness_requirement_occurrence. This is the class to which the range and tolerance is applied.

4.2.370.2 characterized_class_range

The characterized_class_range specifies the role of the Range_characteristic for the Fabrication_thickness_requirement_occurrence. The characterized_class_range specifies the set of measurements to which the tolerance applies.

4.2.370.3 tolerance

The tolerance specifies the role of the Tolerance_characteristic for the Fabrication_thickness_requirement_occurrence. The tolerance is applicable over the range specified by the characterized_class_range.
