4.2.297 Dimension_set

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimension_set is a type of Managed_design_object. A Dimension_set is a set of Dimensional_characteristic that specify the theoretically exact shape of a Physical_feature_or_part_template.

The data associated with this application object are the following:

  • associated_dimensions

4.2.297.1 associated_dimensions

The associated_dimensions specifies the Dimensional_characteristic that are associated with the Dimension_set. There shall be one or more Dimensional_characteristic specified as the associated_dimensions for the Dimension_set.

Formal constraints:

WR1: The Dimension_set shall be specified as the profile_representation by at least one True_profile.

NOTE: 
*)
ENTITY dimension_set
  SUBTYPE OF (managed_design_object);
    associated_dimensions : SET [1:?] OF dimensional_characteristic;
END_ENTITY; -- dimension_set
