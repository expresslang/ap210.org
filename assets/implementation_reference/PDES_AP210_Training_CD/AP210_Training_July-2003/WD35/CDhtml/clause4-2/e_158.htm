4.2.159 Composite_group_shape_element

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Composite_group_shape_element is a type of Composite_shape_element in which the composite may be thought of as a group of individual Shape_element in which any property that is applied to the Composite_group_shape_element is applied to each of the constituent Shape_elements on an individual basis.

EXAMPLE: A group of cylindrical holes in a printed circuit board may be grouped with a Composite_group_shape_element for the purpose of applying the same Cylindricity_tolerance to each of the constituent holes in the group.

Formal constraints:

WR1: If all the Shape_elements that compose the Composite_group_shape_element are of the same type then the Composite_shape_element shall also be of that type.

EXAMPLE: If all the Shape_elements that compose a Composite_group_shape_element are of type Centre_axis then, the Composite_group_shape_element will also be of type Centre_axis.
