4.2.667 Physical_unit_design_view

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_unit_design_view is a type of Physical_unit. A Physical_unit_design_view may be one or both of an Assembly_module or an Interconnect_module. A Physical_unit_design_view shall fully describe a specifically designed item in sufficient detail to permit production of that item in a manufacturing process and analysis of that item's response to a simulated environment in a computer based experimental environment.

EXAMPLE: Figure pudv illustrates an assembly design view.

The data associated with this application object are the following:

  • usage_view

4.2.667.1 usage_view

The usage_view specifies the role of the Physical_unit_usage_view for the Physical_unit_design_view. The usage_view provides sufficient detail to allow the Physical_unit to be utilized in its next assembly, but does not provide sufficient detail to allow reproduction of the Physical_unit. The usage_view specifies the features visible in the next assembly and the terminals of the Physical_unit, if there are any. The usage_view shall not be supplied for the Layout_macro_definition but shall be supplied for all other types of Physical_unit_design_view.
