NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Analytical_model_make_from_relationship is a type of Managed_design_object.

The data associated with this application object are the following:

  • reusable_model
  • resultant_model

4.2.16.1 reusable_model

The reusable_model specifies an Analytical_model for the Analytical_model_make_from_relationship.

4.2.16.2 resultant_model

The resultant_model specifies an Analytical_model for the Analytical_model_make_from_relationship.

Formal constraints:

UR1: The combination of reusable_model and resultant_model shall be unique within a population of Analytical_model_make_from_relationship.

WR1: The reusable_model shall not be the resultant_model within a population of Analytical_model_make_from_relationship.
