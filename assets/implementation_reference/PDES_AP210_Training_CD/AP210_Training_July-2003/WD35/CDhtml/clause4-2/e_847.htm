4.2.847 Stratum_average_surface

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_average_surface is a type of Managed_design_object. A Stratum_average_surface specifies the constructed geometry that is the average of the primary and secondary surface for a Stratum. A Stratum_average_surface is provided as a construction aid in defining a Stratum.

The data associated with this application object are the following:

  • of_shape
  • of_stratum

4.2.847.1 of_shape

An of_shape is an inverse relationship that specifies the role of a Stratum_average_surface_shape for a Stratum_average_surface. An of_shape need not be specified for a Stratum_average_surface.

4.2.847.2 of_stratum

An of_stratum specifies the role of a Stratum for a Stratum_average_surface.
