4.2.112 Component_assembly_2d_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_assembly_2d_position is a type of Managed_design_object. A Component_assembly_2d_position is the placement and orientation for the positioning of the Assembly_component_2d_shape in the Physical_unit_planar_shape specified by the Next_higher_assembly_relationship.

The data associated with this application object are the following:

  • assembly_model
  • component
  • component_model
  • transformation

4.2.112.1 assembly_model

The assembly_model specifies the role of the Physical_unit_planar_shape for the Component_assembly_2d_position.

4.2.112.2 component

The component specifies the role of the Next_higher_assembly_relationship for the Component_assembly_2d_position. The Next_higher_assembly_relationship specifies the actual component being placed.

4.2.112.3 component_model

The component_model specifies the role of the Assembly_component_2d_shape for the Component_assembly_2d_position.

4.2.112.4 transformation

The transformation specifies the role of the Component_2d_location for the Component_assembly_2d_position.

NOTE: This application object is included to support exchange where a CAD geometric model is constrained to be two-dimensional models, and there is additional non-geometric characteristic information available.
