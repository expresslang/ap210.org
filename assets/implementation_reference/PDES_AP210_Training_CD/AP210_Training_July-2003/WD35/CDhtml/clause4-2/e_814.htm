4.2.814 Shape_element_deriving_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shape_element_deriving_relationship is a type of Shape_element_relationship that associates a Derived_shape_element with a Shape_element from which the Derived_shape_element is derived.

The data associated with this application object are the following:

  • relating_shape_element

4.2.814.1 relating_shape_element

The relating_shape_element specifies the Derived_shape_element that is the Shape_element that is derived from the Shape_element specified as the related_shape_element for the Shape_element_deriving_relationship.

NOTE: The attribute related_shape_element is declared in the Shape_element_relationship entity of which this entity is a subtype.

EXAMPLE: Consider a centre axis that is derived from a cylindrical hole. The nature of the relationship between the centre axis and the cylindrical hole is indicated by a Shape_element_deriving_relationship in which the cylindrical hole is specified as the related_shape_element and the centre axis is specified as the relating_shape_element.

Formal constraints:

UR1: The combination of the relating_shape_element inherited from Shape_element_relationship and related_shape_element inherited from Shape_element_relationship shall be unique within a population of Shape_element_deriving_relationship.
