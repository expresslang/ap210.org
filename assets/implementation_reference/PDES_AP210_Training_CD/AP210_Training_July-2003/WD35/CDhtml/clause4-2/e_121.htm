4.2.121 Component_feature_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_feature_relationship is a type of Managed_design_object. A Component_feature_relationship is the association between two Component_feature.

The data associated with this application object are the following:

  • related_component_feature
  • relating_component_feature

4.2.121.1 related_component_feature

The related_component_feature specifies the dependent Component_feature for the Component_feature_relationship.

4.2.121.2 relating_component_feature

The relating_component_feature specifies the independent Component_feature for the Component_feature_relationship.

Formal constraints:

WR1: The related_component_feature shall not be the relating_component_feature either directly or indirectly.

NOTE: 
*)
ENTITY component_feature_relationship
  SUBTYPE OF (managed_design_object);
    related_component_feature  : component_feature;
    relating_component_feature : component_feature;
END_ENTITY; -- component_feature_relationship
