4.2.732 Printed_part_template_interface_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_part_template_interface_terminal is a type of Printed_part_template_terminal, an occurrence of which is an external access to the printed circuit board of which it is a component.

The data associated with this application object are the following:

  • external_connector_function

4.2.732.1 external_connector_function

The external_connector_function is an inverse relationship that specifies the Printed_connector_template_terminal_relationship that specifies the Printed_part_template_interface_terminal as its interface_terminal. There shall be one or more Printed_connector_template_terminal_relationship for a Printed_part_template_interface_terminal.
