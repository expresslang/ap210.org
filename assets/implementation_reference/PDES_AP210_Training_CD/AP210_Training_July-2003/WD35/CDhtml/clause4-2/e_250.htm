4.2.250 Default_land_physical_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Default_land_physical_template is a type of Land_physical_template. A Default_land_physical_template is either a Default_passage_based_land_physical_template or a Default_attachment_size_based_land_physical_template.

The data associated with this application object are the following:

  • of_stratum_technology

4.2.250.1 of_stratum_technology

The of_stratum_technology specifies the role of the Design_technology for the Default_land_physical_template.
