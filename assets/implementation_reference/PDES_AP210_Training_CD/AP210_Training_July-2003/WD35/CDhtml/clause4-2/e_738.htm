4.2.738 Printed_part_template_terminal_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_part_template_terminal_relationship is a type of Managed_design_object. A Printed_part_template_terminal_relationship is an association between two Printed_part_template_terminal in a Printed_part_template.

The data associated with this application object are the following:

  • related_printed_part_terminal
  • relating_printed_part_terminal

4.2.738.1 related_printed_part_terminal

The related_printed_part_terminal specifies the dependent Printed_part_template_terminal in the Printed_part_template_terminal_relationship.

4.2.738.2 relating_printed_part_terminal

The relating_printed_part_terminal specifies the independent Printed_part_template_terminal in the Printed_part_template_terminal_relationship.
