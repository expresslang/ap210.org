4.2.409 Functional_unit_terminal_node_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_unit_terminal_node_assignment is a type of Managed_design_object. A Functional_unit_terminal_node_assignment is the assignment relationship between a Functional_unit_network_node_definition and a Functional_unit_terminal.

The data associated with this application object are the following:

  • composed_node
  • connected_terminal

4.2.409.1 composed_node

The composed_node specifies the role of the Functional_unit_network_node_definition for the Functional_unit_terminal_node_assignment.

4.2.409.2 connected_terminal

The connected_terminal specifies the role of the Functional_unit_terminal for the Functional_unit_terminal_node_assignment.
