4.2.692 Plus_minus_tolerance_range_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Plus_minus_tolerance_range_representation is a type of Shape_dimension_representation that represents a Dimensional_characteristic with three Data_element from which the limits of the Dimensional_characteristic can be calculated.

The data associated with this application object are the following:

  • basic_dimension_value
  • lower_deviation
  • upper_deviation

4.2.692.1 basic_dimension_value

The basic_dimension_value specifies the Data_element from which the lower and upper limits of the associated Dimensional_characteristic may be calculated.

NOTES:

  1. The basic_dimension_value corresponds to the basic size component of toleranced dimensions and is described in ISO 286-1 and ISO 406.
  2. The basic_dimension_value should not be confused with basic dimensions defined in 1.3.9 of ASME Y14.5M.

4.2.692.2 lower_deviation

The lower_deviation specifies the Data_element that is added to the Data_element specified by the basic_dimension_value to determine the lower limit of the associated Dimensional_characteristic.

4.2.692.3 upper_deviation

The upper_deviation specifies the Data_element that is added to the Data_element specified by the basic_dimension_value to determine the upper limit of the associated Dimensional_characteristic.

Formal constraints:

WR1: The units of the Data_elements specified by the basic_dimension_value, lower_deviation, and upper_deviation shall be the same.

WR2: The value of the Data_element specified by the lower_deviation shall be less than the value of the Data_element specified by the upper_deviation.
