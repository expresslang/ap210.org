4.2.892 Termination_constraint

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Termination_constraint is a type of Managed_design_object. A Termination_constraint is a constraint that requires a set of two or more Mating_connector_termination to satisfy an Ee_requirement_occurrence.

The data associated with this application object are the following:

  • constrained_termination
  • termination_usage_constraint

4.2.892.1 constrained_termination

The constrained_termination specifies the role of the Mating_connector_termination for the Termination_constraint. There shall be two or more Mating_connector_termination for the Termination_constraint.

4.2.892.2 termination_usage_constraint

The termination_usage_constraint specifies the role of the Ee_requirement_occurrence for the Termination_constraint.
