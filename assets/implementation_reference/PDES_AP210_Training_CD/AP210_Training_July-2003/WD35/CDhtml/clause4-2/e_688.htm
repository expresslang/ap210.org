4.2.689 Plated_interconnect_module_edge_segment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Plated_interconnect_module_edge_segment is a type of Plated_inter_stratum_feature that forms a section of the edge of an interconnect substrate.

EXAMPLE: Figure pimes illustrates an edge segment that is plated. Note that the opposite side of the cross-section is unplated. There is not enough information in the figure to define the extent of the plated segment.

The data associated with this application object are the following:

  • composed_edge

4.2.689.1 composed_edge

The composed_edge specifies the Partially_plated_interconnect_module_edge that contains the Plated_interconnect_module_edge_segment.

4.2.689.2 end_vertex

The end_vertex specifies one role of an Edge_segment_vertex for the Plated_interconnect_module_edge_segment.

4.2.689.3 start_vertex

The start_vertex specifies one role of an Edge_segment_vertex for the Plated_interconnect_module_edge_segment.

Formal constraints:

WR1: The start_vertex shall not be the end_vertex.
