4.2.75 Boolean_csg_result

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Boolean_csg_result is a type of Shape_item. A Boolean_csg_result is a Bound_volume_shape, or a Csg_primitive, which is the result of using a boolean operator. The Boolean_csg_result allows for the composition or decomposition of a Bound_volume_shape, or a Csg_primitive.

The data associated with this application object are the following:

  • csg_operator
  • operand_1
  • operand_2

4.2.75.1 csg_operator

The csg_operator specifies the difference, the intersection, or the union that determines the Boolean_csg_result.

4.2.75.2 operand_1

The operand_1 specifies the Bound_volume_shape, or a Csg_primitive which is the first operand for the Boolean_csg_result.

4.2.75.3 operand_2

The operand_2 specifies the Bound_volume_shape, or a Csg_primitive which is the second operand for the Boolean_csg_result.
