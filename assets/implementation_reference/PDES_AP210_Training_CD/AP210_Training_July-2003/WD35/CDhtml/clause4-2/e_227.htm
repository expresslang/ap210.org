4.2.227 Datum_feature_material_condition_property

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_feature_material_condition_property is a type of Managed_design_object. A Datum_feature_material_condition_property is a specification of a material condition property that defines the manner in which Geometric_tolerance_with_referenced_datum_system requirements are dependent on the produced size of a Datum_feature of the Datum_system.

The data associated with this application object are the following:

  • applied_to
  • datum_system_conditions_definition
  • name

4.2.227.1 applied_to

The applied_to specifies the Datum_feature_usage_in_datum_system_relationship that identifies the usage of a Datum_feature in a Datum_system and to which the material condition property is applied.

4.2.227.2 datum_system_conditions_definition

The datum_system_conditions_definition is an inverse relationship that specifies that the existence of the Datum_feature_material_condition_property depends on the existence of the Datum_system_defining_conditions_with_material_conditions that specifies the Datum_feature_material_condition_property as its applied_material_condition_properties.

4.2.227.3 name

The name specifies the type of the applied material condition property, either "least_material_requirement" or "maximum_material_principle" for the Datum_feature_material_condition_property.

NOTES:

  1. The application of the maximum material principle to a Datum feature is described in 4.3 ISO 2692.
  2. The application of the least material requirement to a Datum feature is described in clause 3 of ISO 2692-Amendment 1.

ENTITY datum_feature_material_condition_property
  SUBTYPE OF (managed_design_object);
    applied_to : datum_feature_usage_in_datum_system_relationship;
    name       : material_condition_property_names;
  INVERSE
    datum_system_conditions_definition : 
	datum_system_defining_conditions_with_material_conditions FOR 
	applied_material_condition_properties;
END_ENTITY; -- datum_feature_material_condition_property
