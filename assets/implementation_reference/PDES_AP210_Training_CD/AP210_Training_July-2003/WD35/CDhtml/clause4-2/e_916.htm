4.2.916 Tolerance_zone_boundary_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_zone_boundary_definition is a type of Managed_design_object that characterizes a Tolerance_zone_boundary. A Tolerance_zone_boundary_definition is either a Boundary_zone_definition_with_specified_size or a Conical_tolerance_zone_boundary_definition.

The data associated with this application object are the following:

  • defined_zone_boundary
  • zone_definition

4.2.916.1 defined_zone_boundary

The defined_zone_boundary specifies the Tolerance_zone_boundary that is characterized by the Tolerance_zone_boundary_definition.

4.2.916.2 zone_definition

The zone_definition is an inverse relationship that specifies that the existence of the Tolerance_zone_boundary_definition is dependent on the existence of the Tolerance_zone_definition that specifies the Tolerance_zone_boundary_definition as its boundary_definition.

ENTITY tolerance_zone_boundary_definition
  ABSTRACT SUPERTYPE OF (ONEOF (
      boundary_zone_definition_with_specified_size,
      conical_tolerance_zone_boundary_definition))
  SUBTYPE OF (managed_design_object);
    defined_zone_boundary : tolerance_zone_boundary;
  INVERSE
    zone_definition : tolerance_zone_definition FOR boundary_definition;
END_ENTITY; -- tolerance_zone_boundary_definition
