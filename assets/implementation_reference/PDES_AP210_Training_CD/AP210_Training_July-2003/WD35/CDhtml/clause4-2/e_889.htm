4.2.889 Template_stratum_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Template_stratum_assignment is a type of Managed_design_object. A Template_stratum_assignment is the relationship between a Stratum and any one of an Stratum_feature_template, a Material_addition_feature_template, or a Material_removal_feature_template for the purpose of binding a template to a specific Stratum for a Padstack_definition.

The data associated with this application object are the following:

  • bound_template
  • bound_stratum

4.2.889.1 bound_template

The bound_template specifies any one of the Stratum_feature_template, Material_addition_feature_template, Material_removal_feature_template as the assigned object for the Template_stratum_assignment.

4.2.889.2 bound_stratum

The bound_stratum specifies the role of the Stratum for the Template_stratum_assignment.
