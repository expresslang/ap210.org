4.2.840 Statistical_dimensional_tolerance

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Statistical_dimensional_tolerance is a type of Dimensional_characteristic that specifies design requirements that are based on sound statistics. Associated with the Statistical_dimensional_tolerance is the requirement that a specified level of process control be maintained during the manufacture of the Physical_unit. A Statistical_dimensional_tolerance may specify another Dimensional_characteristic as an alternate to the Statistical_dimensional_tolerance in the event that the specified level of process control cannot be met.

NOTE: Statistical tolerancing is discussed in 2.16 of ASME Y14.5M.

The data associated with this application object are the following:

  • alternate_dimension
  • statistical_control_requirement

4.2.840.1 alternate_dimension

The alternate_dimension specifies the Dimensional_characteristic that shall be used if the specified level of process control cannot be met. The alternate_dimension need not be specified for a Statistical_dimensional_tolerance.

4.2.840.2 statistical_control_requirement

The statistical_control_requirement specifies the Ee_requirement_occurrence that states the statistical process control requirements that the manufacturing facility shall meet for the Statistical_dimensional_tolerance.

Formal constraints:

WR1: If a Dimensional_characteristic is specified as an alternate_dimension, the specified Dimensional_characteristic shall not be a Statistical_dimensional_tolerance.

WR2: If a Dimensional_characteristic is specified as an alternate_dimension, the Dimensional_characteristic shall characterize the same Shape_element or Shape_element_locating_relationship as characterized by the Statistical_dimensional_tolerance.

  SUBTYPE OF (dimensional_characteristic);
