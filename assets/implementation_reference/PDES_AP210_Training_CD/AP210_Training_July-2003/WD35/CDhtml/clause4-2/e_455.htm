4.2.462 Interconnect_module_surface_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_surface_feature is a type of Part_feature. An Interconnect_module_surface_feature is one of an Interconnect_module_edge_segment_surface, an Interconnect_module_primary_surface, or an Interconnect_module_secondary_surface. An Interconnect_module_surface_feature identifies a surface of the Interconnect_module for fabrication and assembly purposes, particular to support Component_2d_location.
