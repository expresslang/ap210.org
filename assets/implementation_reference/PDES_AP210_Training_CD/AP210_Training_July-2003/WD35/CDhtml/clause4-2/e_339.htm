4.2.339 Ee_text

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_text is a type of Managed_design_object. An Ee_text is a string used to convey information.

The data associated with this application object are the following:

  • content

4.2.339.1 content

The content specifies the organized list of one or more words that comprise a message.
