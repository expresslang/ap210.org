4.2.540 Mating_connector_termination

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Mating_connector_termination is a type of Managed_design_object. A Mating_connector_termination is a single terminal of a Mating_connector_component.

The data associated with this application object are the following:

  • accessed_connector
  • definition
  • name
  • signal_definition

4.2.540.1 accessed_connector

The accessed_connector specifies the role of the Mating_connector_component for the Mating_connector_termination.

4.2.540.2 definition

The definition specifies either a Packaged_part_terminal or a Printed_part_template_terminal for the Mating_connector_termination.

4.2.540.3 name

The name specifies the identifier for the Mating_connector_termination.

4.2.540.4 signal_definition

The signal_definition specifies the role of the Ee_requirement_occurrence for the Mating_connector_termination. The signal_definition need not be specified for a particular Mating_connector_termination.

Formal constraints:

UR1: The combination of the accessed_connector and the definition shall be unique within a population of Mating_connector_termination.

ENTITY mating_connector_termination
  SUBTYPE OF (managed_design_object);
    definition         : packaged_or_printed_terminal;
    accessed_connector : mating_connector_component;
    name               : ee_name;
    signal_definition  : OPTIONAL ee_requirement_occurrence;
  UNIQUE
    ur1 : definition, accessed_connector;
END_ENTITY; -- mating_connector_termination
