4.2.385 Filled_area_material_removal_structure

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Filled_area_material_removal_structure is a type of Material_removal_feature. A Filled_area_material_removal_structure is either an Electrical_isolation_removal_structure or a Thermal_isolation_removal_structure.

The data associated with this application object are the following:

  • design_intent

4.2.385.1 design_intent

The design_intent specifies the Conductive_filled_area for the requirement for the Filled_area_material_removal_structure.
