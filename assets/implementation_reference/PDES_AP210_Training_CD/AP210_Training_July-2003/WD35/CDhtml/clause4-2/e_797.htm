4.2.797 Runout_tolerance_zone_orienting_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Runout_tolerance_zone_orienting_relationship is a type of Managed_design_object. A Runout_tolerance_zone_orienting_relationship is a relationship that associates a Tolerance_zone with the Directed_axis with which it is oriented.

The data associated with this application object are the following:

  • angular_size_characteristic
  • oriented_zone
  • orienting_axis

4.2.797.1 angular_size_characteristic

The angular_size_characteristic is an inverse relationship that specifies that the existence of the Runout_tolerance_zone_orienting_relationship is dependent on the existence of the Runout_zone_orientation that specifies the Runout_tolerance_zone_orienting_relationship as its oriented_relationship.

4.2.797.2 oriented_zone

The oriented_zone specifies the Tolerance_zone that is oriented with respect to the specified Directed_axis by the Runout_tolerance_zone_orienting_relationship.

4.2.797.3 orienting_axis

The orienting_axis specifies the Directed_axis to which the specified Tolerance_zone is oriented by the Runout_tolerance_zone_orienting_relationship.

Informal constraints:

IP1: The Centre_axis upon which the Directed_axis is based shall correspond with the axis upon which the unit would be rotated if the corresponding run-out tolerance (i.e., either a Circular_runout_tolerance or Total_runout_tolerance) were to be verified.

ENTITY runout_tolerance_zone_orienting_relationship
  SUBTYPE OF (managed_design_object);
    oriented_zone  : tolerance_zone;
    orienting_axis : directed_axis;
  INVERSE
    angular_size_characteristic : runout_zone_orientation FOR oriented_relationship;
END_ENTITY; -- runout_tolerance_zone_orienting_relationship
