4.2.491 Land_interface_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Land_interface_terminal is a type of Component_feature included to identify the interface available to be fastened to in the next level of assembly.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.491.1 associated_component

The associated_component specifies the role of the Land for the Land_interface_terminal. Only the following types of Land shall be referenced: Contact_size_dependent_land; Component_termination_passage_and_contact_size_dependent_land; Via_and_contact_size_dependent_land.

4.2.491.2 definition

The definition specifies the role of the Land_template_interface_terminal for the Land_interface_terminal.

NOTE: Use of this capability occurs when a member of Component_feature_to_physical_usage_view_assignment references a member of Land_interface_terminal to define a traceability relationship to an Interconnect_module_terminal.

ENTITY land_interface_terminal
  SUBTYPE OF (laminate_component_interface_terminal);
    SELF\component_feature.definition           : land_template_interface_terminal;
    SELF\component_feature.associated_component : land;
WHERE
  WR1: SIZEOF(['AP210_ARM.CONTACT_SIZE_DEPENDENT_LAND',
               'AP210_ARM.COMPONENT_TERMINATION_PASSAGE_AND_CONTACT_SIZE_DEPENDENT_LAND',
               'AP210_ARM.VIA_AND_CONTACT_SIZE_DEPENDENT_LAND'] *
       TYPEOF(SELF\component_feature.associated_component))
         = 1;
END_ENTITY; -- land_interface_terminal
