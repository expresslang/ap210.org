4.2.742 Profile_boundary_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Profile_boundary_definition is a type of Boundary_zone_definition_with_specified_size that characterizes a Tolerance_zone_boundary of a Tolerance_zone that is defined by either a Profile_of_any_line_tolerance or a Profile_of_any_surface_tolerance. A Profile_boundary_definition may be a Profile_boundary_definition_with_offsets.

The data associated with this application object are the following:

  • basic_profile
  • defined_zone_boundary

4.2.742.1 basic_profile

The basic_profile specifies the True_profile that characterizes the theoretically exact profile of the toleranced Physical_feature_or_part_template along which the Tolerance_zone_opposing_boundary_set is disposed.

4.2.742.2 defined_zone_boundary

The defined_zone_boundary specifies the Tolerance_zone_opposing_boundary_set that is characterized by the Profile_boundary_definition. If a Tolerance_zone_implicit_boundary_set is specified as the defined_zone_boundary, the Tolerance_zone_boundary is equally disposed about the True_profile.

Formal constraints:

WR1: The Tolerance_zone_definition specified as the zone_definition shall be specified as the tolerance_zone_or_boundary_definition by either a Profile_of_any_line_tolerance or a Profile_of_any_surface_tolerance.
