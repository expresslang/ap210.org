4.2.96 Change_order

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Change_order is a type of Work_order that authorizes the development of a modified design of an Ee_product, resulting in a new Ee_product_version.

The data associated with this application object are the following:

  • adopted_solution

4.2.96.1 adopted_solution

The adopted_solution specifies the Design_object_management_relationship, or Ee_text which is to be implemented by the Change_order.

NOTE: 
*)
ENTITY change_order
  SUBTYPE OF (work_order);
    adopted_solution : design_object_management_relationship_or_description;
END_ENTITY; -- change_order
