4.2.152 Component_termination_passage_template_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_termination_passage_template_join_terminal is a type of Component_termination_passage_template_terminal, an occurrence of which provides a fabrication join capability, within an optionally constrained set of Stratum.

The data associated with this application object are the following:

  • disallowed_inter_stratum_extent

4.2.152.1 disallowed_inter_stratum_extent

The disallowed_inter_stratum_extent specifies the Inter_stratum_extent for which a connection to the Component_termination_passage_template_join_terminal is disallowed. The disallowed_inter_stratum_extent need not be supplied for a Component_termination_passage_template_join_terminal. There may be more than one Inter_stratum_extent for a Component_termination_passage_template_join_terminal.
