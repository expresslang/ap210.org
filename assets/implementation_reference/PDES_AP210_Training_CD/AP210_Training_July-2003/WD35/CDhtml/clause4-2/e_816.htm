4.2.816 Shape_element_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shape_element_relationship is a type of Managed_design_object. A Shape_element_relationship is one of a Parallel_shape_element_relationship, a Perpendicular_shape_element_relationship, a Shape_element_boundary_relationship, a Shape_element_constituent_relationship, a Non_feature_shape_element_relationship, or a Shape_element_deriving_relationship. Additionally, a Shape_element_relationship may be a Datum_feature_usage_in_datum_relationship. A Shape_element_relationship is an association between two Shape_element.

The data associated with this application object are the following:

  • description
  • name
  • related_shape_element
  • relating_shape_element

4.2.816.1 description

The description specifies the descriptive label that relates the nature of the Shape_element_relationship. The description need not be specified for a Shape_element_relationship.

4.2.816.2 name

The name specifies the identifying label for the Shape_element_relationship. The name need not be specified for a Shape_element_relationship.

4.2.816.3 related_shape_element

The related_shape_element specifies the Shape_element to which the relating_shape_element refers for the Shape_element_relationship. If one Shape_element of the relationship is dependent upon the other, this attribute shall be the dependent one.

4.2.816.4 relating_shape_element

The relating_shape_element specifies the Shape_element that relates to another, for the Shape_element_relationship.

Formal constraints:

WR1: If the Shape_element_relationship is a Datum_feature_usage_in_datum_relationship it shall not also be a Parallel_shape_element_relationship, a Perpendicular_shape_element_relationship, a Shape_element_boundary_relationship, or a Shape_element_constituent_relationship.
