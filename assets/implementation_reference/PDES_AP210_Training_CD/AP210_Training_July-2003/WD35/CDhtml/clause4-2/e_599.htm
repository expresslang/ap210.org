4.2.599 Padstack_occurrence

A Padstack_occurrence is a type of Assembly_group_component. A Padstack_occurrence organizes the various Land and Inter_stratum_features so that one placement may suffice for all Application Objects that are derived from the placement of one padstack.

The data associated with this application object are the following:

  • usage_definition

4.2.599.1 usage_definition

The usage_definition inherited from the Assembly_component specifies the role of the Padstack_definition for the Padstack_occurrence.
