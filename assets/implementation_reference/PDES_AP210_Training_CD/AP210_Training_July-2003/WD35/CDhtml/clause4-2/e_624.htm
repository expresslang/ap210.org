4.2.620 Part_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_template is a type of Physical_feature_or_part_template. A Part_template may be one of an Inter_stratum_feature_template, a Material_addition_feature_template, a Material_removal_feature_template, a Printed_part_template, a Special_symbol_part_template, a Stratum_feature_template, or a Trace_template, or additionally may be a Part_text_template. One reason for including information provided by members of Part_template is that they are collectors for shape data templates for layout items.

The data associated with this application object are the following:

  • analytical_part_template_representation
  • part_template_analytical_model
  • physical_characteristic

4.2.620.1 analytical_part_template_representation

The analytical_part_template_representation specifies the role of the Analytical_representation for the Part_template. There may be more than one Analytical_representation for the Part_template. The analytical_part_template_representation need not be specified for a particular Part_template.

4.2.620.2 part_template_analytical_model

The part_template_analytical_model specifies the role of the Analytical_model for the Part_template. The part_template_analytical_model need not be specified for a particular Part_template.

4.2.620.3 physical_characteristic

The physical_characteristic specifies either the Characteristic or Coordinated_characteristic for the Part_template. There may be more than one Characteristic for the Part_template. The physical_characteristic need not be specified for a particular Part_template.

ENTITY part_template
  SUPERTYPE OF (ONEOF (
      inter_stratum_feature_template,
      material_addition_feature_template,
      material_removal_feature_template,
      non_conductive_cross_section_template,
      printed_part_cross_section_template,
      printed_part_template,
      special_symbol_part_template,
      stratum_feature_template,
      trace_template)
      ANDOR part_text_template)
  SUBTYPE OF (physical_feature_or_part_template);
    part_template_analytical_model          : OPTIONAL analytical_model;
    physical_characteristic                 : OPTIONAL SET [1:?] OF 
                                              correlated_or_independent;
    analytical_part_template_representation : OPTIONAL SET [1:?] OF 
                                              analytical_representation;
    associated_definition                   : template_definition;
WHERE
 WR1: SIZEOF (QUERY( prpc <* 
  USEDIN(associated_definition\ee_product_definition.version.versioned_item,
       'AP210_ARM.EE_PRODUCT_RELATED_PRODUCT_CATEGORY.OF_PRODUCT') | 
              (prpc.name = 'template model')
             )) >= 0;
 WR2: SELF\shape_element.name = 'NULL';
END_ENTITY; -- part_template
