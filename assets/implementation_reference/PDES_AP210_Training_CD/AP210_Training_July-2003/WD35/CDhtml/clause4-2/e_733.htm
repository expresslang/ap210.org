4.2.733 Printed_part_template_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_part_template_join_terminal is a type of Printed_part_template_terminal an occurrence of which provides access to the functionality of the Printed_part_template from within a printed circuit board.

NOTE: Join is a concept that explicitly states the purpose is for completing the network connections internal to a physical design.

The data associated with this application object are the following:

  • internal_connector_function

4.2.733.1 internal_connector_function

The internal_connector_function is an inverse relationship that specifies the Printed_connector_template_terminal_relationship that specifies the Printer_part_template_join_terminal as its join_terminal. There shall be no more than one Printed_connector_template_terminal_relationship for a Printed_part_template_join_terminal.
