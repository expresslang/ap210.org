4.2.749 Profile_of_any_surface_tolerance_without_referenced_datum_system

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Profile_of_any_surface_tolerance_without_referenced_datum_system is a type of Geometric_tolerance_without_referenced_datum_system and a type of Profile_of_any_surface_tolerance. That is, it is a Profile_of_any_surface_tolerance in which the Physical_feature_or_part_template to which the Profile_of_any_surface_tolerance_without_referenced_datum is applied is not oriented or located in relation to a Datum_system.
