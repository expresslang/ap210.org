4.2.288 Dielectric_crossover_area

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dielectric_crossover_area is a type of Stratum_concept_relationship that relates one Stratum_feature with another that results in the creation of a dielectric crossover feature in a dielectric Stratum.

The data associated with this application object are the following:

  • related_concept
  • relating_concept
  • resulting_dielectric_crossover

4.2.288.1 related_concept

The related_concept specifies the source Stratum_feature for the Dielectric_crossover_area.

4.2.288.2 relating_concept

The relating_concept specifies the target Stratum_feature for the Dielectric_crossover_area.

4.2.288.3 resulting_dielectric_crossover

The resulting_dielectric_crossover specifies the Stratum_feature in a stratum of dielectric material for the Dielectric_crossover_area.

Formal constraints:

WR1: The Stratum on which the Stratum_features reside related to this Dielectric_crossover_area shall be arranged such that the resulting_dielectric_crossover is between the relating_concept and the related_concept.

NOTE: There is no directionality implied by the use of "related", "relating" in the attribute names.

WR2: The relating_concept shall not be the related_concept.
