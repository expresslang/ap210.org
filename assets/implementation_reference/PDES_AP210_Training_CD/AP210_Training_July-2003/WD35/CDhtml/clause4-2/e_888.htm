4.2.888 Template_material_cross_section_boundary

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Template_material_cross_section_boundary is a type of Non_feature_shape_element.

The data associated with this application object are the following:

  • associated_definition

4.2.888.1 associated_definition

The associated_definition specifies a Printed_part_template_material for the Template_material_cross_section_boundary.

Formal constraints:

UR1: The combination of associated_definition and the name attribute inherited from Shape_element shall be unique within a population of Template_material_cross_section_boundary.

WR1: The name attribute inherited from Shape_element shall be one of "top", "bottom", "left", or "right" within a population of Template_material_cross_section_boundary.

NOTE: 
*)
ENTITY template_material_cross_section_boundary 
  SUBTYPE OF (non_feature_shape_element);
    associated_definition : printed_part_template_material;
UNIQUE
  ur1: associated_definition,SELF\shape_element.name;
WHERE
  wr1: SELF\shape_element.name IN ['top','bottom','left','right'];
END_ENTITY; -- template_material_cross_section_boundary
