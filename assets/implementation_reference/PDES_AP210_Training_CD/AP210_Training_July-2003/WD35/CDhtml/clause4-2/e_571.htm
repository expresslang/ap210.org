4.2.570 Owned_design_object

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Owned_design_object is a type of Managed_design_object. An Owned_design_object identifies the Managed_design_object which participates in the Owned_assignment.

NOTE: The purpose of Owned_design_object is identify the single authority ownership of design aspects. The typical use is to negotiate between the owners of the different design aspects. If there are multiple joint owners, then the Frozen_design_object is more appropriate since that uses Ee_approval which supports multiple signatures.

NOTE: 
*)
ENTITY owned_design_object
  SUBTYPE OF (managed_design_object);
INVERSE
  owner : owner_assignment for owned_item;
END_ENTITY; -- owned_design_object
