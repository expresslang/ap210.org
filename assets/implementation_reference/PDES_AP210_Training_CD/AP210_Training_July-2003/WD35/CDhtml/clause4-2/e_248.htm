4.2.248 Default_component_termination_passage_based_land_physical_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Default_component_termination_passage_based_land_physical_template is a type of Default_passage_based_land_physical_template. A Default_component_termination_passage_based_land_physical_template may be a Default_attachment_size_and_component_termination_passage_based_land_physical_template. The shape of the land template is dependent upon the requirements from Passage_technology.

The data associated with this application object are the following:

  • of_passage_technology

4.2.248.1 of_passage_technology

The of_passage_technology specifies the role of the Default_component_termination_passage_definition for the Default_component_termination_passage_based_land_physical_template.
