4.2.172 Conductive_interconnect_element_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Conductive_interconnect_element_terminal is a type of Component_terminal. The Conductive_interconnect_element_terminal is a terminal for the trace defined by Conductive_interconnect_element.

NOTE: This is used for detailed design and analysis, and also as a means to check the quality of geometry representation and transfer.

The data associated with this application object are the following:

  • associated_component
  • connection
  • location
  • name

4.2.172.1 associated_component

The associated_component specifies the role of the Conductive_interconnect_element for the Conductive_interconnect_element_terminal.

4.2.172.2 connection

The connection specifies the role of the Connection_zone for the Conductive_interconnect_element_terminal. The connection is optional and need not supplied if the data is not available in the source system. If the data is available in the source system, it shall be supplied.

4.2.172.3 location

The location is an inverse attribute that specifies the Design_view_terminal_component_shape_relationship that the Conductive_interconnect_element_terminal is existence dependent on. There may be one or two Design_view_terminal_component_shape_relationship depending on whether there is a single shape representation for the design or whether there is a two dimensional shape and a three dimensionsal shape.

4.2.172.4 name

The name specifies the string for the human-interpretable identifier for the Conductive_interconnect_element_terminal.

Formal constraints:

UR1: The combination of name and associated_component inherited from Component_feature shall be unique within a population of Conductive_interconnect_element_terminal.

WR1: The same Stratum shall be referenced by the path starting at location and the path starting at associated_component inherited from the Component_feature.

ENTITY conductive_interconnect_element_terminal
  SUBTYPE OF (laminate_component_feature);
    SELF\component_feature.associated_component : conductive_interconnect_element;
    connection                                  : OPTIONAL connection_zone;
    name                                        : STRING;
    location 					: layer_connection_point;
UNIQUE
  ur1: name, SELF\component_feature.associated_component;
WHERE
  wr1: NOT(EXISTS(SELF\component_feature.
    associated_component\conductive_interconnect_element.
    composed_conductor)) OR
    (location.resident_design_layer_stratum = SELF\component_feature.
    associated_component\conductive_interconnect_element.
    composed_conductor.resident_stratum);
END_ENTITY; -- conductive_interconnect_element_terminal
