4.2.583 Package_terminal_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Package_terminal_occurrence is a type of Managed_design_object. A Package_terminal_occurrence is an occurrence of a Package_terminal in a Packaged_component for the case where the equivalent Packaged_part_terminal is not populated. This application object is provided to specify the cases where no Packaged_part_terminal is defined for a Package_terminal because there is no access to functionality specified through that Package_terminal. There is a requirement to support the fact that the Assembly_joint connects the realized terminal of the component to the realized terminal of an Interconnect_module_component or to other members of Assembly_component.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.583.1 associated_component

The associated_component specifies the role of the Packaged_component for the Package_terminal_occurrence.

4.2.583.2 definition

The definition specifies the role of the Package_terminal that has no defined electrical functionality for the Package_terminal_occurrence.
