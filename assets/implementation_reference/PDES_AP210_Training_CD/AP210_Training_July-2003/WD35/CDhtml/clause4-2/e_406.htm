4.2.406 Functional_unit_requirement_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_unit_requirement_allocation is a type of Managed_design_object. A Functional_unit_requirement_allocation is the relationship between the Ee_requirement_occurrence and the Functional_unit that satisfies the Ee_requirement_occurrence.

The data associated with this application object are the following:

  • functional_requirement
  • functional_unit_assignment

4.2.406.1 functional_requirement

The functional_requirement specifies the role of the Ee_requirement_occurrence for the Functional_unit_requirement_allocation.

4.2.406.2 functional_unit_assignment

The functional_unit_assignment specifies the Functional_unit that satisfies the Ee_requirement_occurrence for the Functional_unit_requirement_allocation.
