4.2.613 Part_group_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_group_feature is a type of Part_feature that is an aggregation of Part_features. A Part_group_feature may be a Part_array_feature.

NOTE: 
*)
ENTITY part_group_feature
  SUPERTYPE OF (part_array_feature)
  SUBTYPE OF (part_feature);
WHERE
 WR1:SIZEOF(QUERY( fgr <* USEDIN(SELF,
       'AP210_ARM.FEATURE_GROUP_RELATIONSHIP.RELATING_FEATURE') |
       fgr.related_feature\part_feature.associated_definition <>
       SELF\part_feature.associated_definition)) = 0;
END_ENTITY; -- part_group_feature
