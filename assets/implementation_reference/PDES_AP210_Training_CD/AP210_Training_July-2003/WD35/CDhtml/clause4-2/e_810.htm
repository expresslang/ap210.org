4.2.810 Shape_element

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shape_element is a type of Managed_design_object. A Shape_element is one of a Non_feature_shape_element or a Physical_feature_or_part_template or a Land_template_terminal or a Component_termination_passage_template_terminal or a Via_template_terminal. A Shape_element may additionally be a Composite_shape_element. A Shape_element is an identifiable element of the shape of a Physical_unit.

The data associated with this application object are the following:

  • description
  • name

4.2.810.1 description

The description specifies the label that conveys the nature of the Shape_element. The description need not be specified for a Shape_element.

4.2.810.2 name

The name specifies the identifier of the Shape_element.

ENTITY shape_element
  ABSTRACT SUPERTYPE OF (ONEOF (physical_feature_or_part_template,
      component_termination_passage_template_terminal,
      land_template_terminal,
      via_template_terminal,
      non_feature_shape_element) ANDOR composite_shape_element)
  SUBTYPE OF (managed_design_object);
    name        : label;
    description : OPTIONAL label;
WHERE
  WR1: NOT('AP210_ARM.COMPOSITE_SHAPE_ELEMENT' IN TYPEOF(SELF)) OR
       (SIZEOF(TYPEOF(SELF)) > 2);
  WR2: NOT(('AP210_ARM.COMPOSITE_SHAPE_ELEMENT' IN TYPEOF(SELF)) AND
       ('AP210_ARM.COMPONENT_TERMINATION_PASSAGE_TEMPLATE_TERMINAL' IN 
       TYPEOF(SELF)));
  WR3: NOT(('AP210_ARM.COMPOSITE_SHAPE_ELEMENT' IN TYPEOF(SELF)) AND
       ('AP210_ARM.LAND_TEMPLATE_TERMINAL' IN 
       TYPEOF(SELF)));
  WR4: NOT(('AP210_ARM.COMPOSITE_SHAPE_ELEMENT' IN TYPEOF(SELF)) AND
       ('AP210_ARM.VIA_TEMPLATE_TERMINAL' IN 
       TYPEOF(SELF)));
END_ENTITY; -- shape_element
