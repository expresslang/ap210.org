4.2.927 Tolerance_zone_per_unit_size_specification

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_zone_per_unit_size_specification is a type of Managed_design_object. A Tolerance_zone_per_unit_size_specification is a specification of the Tolerance_zone_boundary that defines the extent of the Tolerance_zone_boundary.

The data associated with this application object are the following:

  • per_unit_size
  • specified_tolerance_zone
  • zone_definition

4.2.927.1 per_unit_size

The per_unit_size specifies the role of the Length_data_element that represents the extent of the Tolerance_zone for the Tolerance_zone_per_unit_size_specification. There may be one or two Length_data_elements specified as the per_unit_size for the Tolerance_zone_per_unit_size_specification. If one Length_data_element is specified, it represents the linear extent of the Tolerance_zone. If two Length_data_elements are specified, they represent the rectangular extent of the Tolerance_zone.

4.2.927.2 specified_tolerance_zone

The specified_tolerance_zone specifies the Tolerance_zone the extent of which is defined by the Tolerance_zone_per_unit_size_specification.

4.2.927.3 zone_definition

The zone_definition is an inverse relationship that specifies that the existence of the Tolerance_zone_per_unit_size_specification is dependent on the existence of the Tolerance_zone_definition_with_per_unit_size_specification that specifies the Tolerance_zone_per_unit_size_specification as its per_unit_size_specification.

ENTITY tolerance_zone_per_unit_size_specification
  SUBTYPE OF (managed_design_object);
    specified_tolerance_zone : tolerance_zone;
    per_unit_size            : SET [1:2] OF length_data_element;
  INVERSE
    zone_definition : tolerance_zone_definition_with_per_unit_size_specification FOR 
                      per_unit_size_specification;
END_ENTITY; -- tolerance_zone_per_unit_size_specification
