4.2.553 Multi_layer_stratum_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Multi_layer_stratum_feature is a type of Stratum_feature. A Multi_layer_stratum_feature causes shapes to be realized in the material of multiple sequential Stratum which are all bases on the geometry for the feature. Each shape is effectively a Stratum_feature in that Stratum, but due to the identical nature of the geometry, there is no need to separately model the geometry. Each shape in the sequential Stratum is isolated from other shapes in that Stratum.

EXAMPLE: Gold plating on Conductors that is based directly on the underlying Copper shape is an example of this Application Object.

The data associated with this application object are the following:

  • derived_layer_geometry

4.2.553.1 derived_layer_geometry

The derived_layer_geometry specifies the role of the Stratum for the Multi_layer_stratum_feature.
