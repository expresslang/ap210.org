4.2.683 Planned_serial_number_effectivity

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Planned_serial_number_effectivity is a type of Planned_effectivity. A Planned_serial_number_effectivity is the specification by the design organization of the intended use of an Ee_product identified by a range of serial numbers or of an Ee_product within Ee_product_configuration identified by a range of serial numbers.

The data associated with this application object are the following:

  • from_effectivity_id
  • through_effectivity_id

4.2.683.1 from_effectivity_id

The from_effectivity_id specifies the string for the initial serial number of the range of serial numbers for the Planned_serial_number_effectivity.

4.2.683.2 through_effectivity_id

The through_effectivity_id specifies the string for the final serial number of the range of serial numbers for the Planned_serial_number_effectivity. The through_effectivity_id need not be specified for a particular Planned_serial_number_effectivity.
