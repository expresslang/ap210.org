4.2.213 Date_and_time

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Date_and_time is a type of Date that includes the clock time together with the calendar Date. Resolution to the second is provided by this object.

The data associated with this application object are the following:

  • hour
  • minute
  • second
  • time_zone

4.2.213.1 hour

The hour specifies the integer for the period of clock time consisting of sixty minutes, for the Date_and_time.

4.2.213.2 minute

The minute specifies the integer for the period of clock time consisting of sixty seconds, for the Date_and_time.

4.2.213.3 second

The second specifies the integer for the second portion of the clock time for the Date_and_time.

4.2.213.4 time_zone

The time_zone specifies the integer number of hours that the local time is offset from coordinated universal time, and the direction of the offset, for the Date_and_time.
