4.2.120 Component_feature_make_from_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_feature_make_from_relationship is a type of Managed_design_object. A Component_feature_make_from_relationship is an association between two Component_feature, where one Component_feature is used as a basis for the development of another Component_feature.

The data associated with this application object are the following:

  • resultant_feature
  • reusable_feature

4.2.120.1 resultant_feature

The resultant_feature specifies the Component_feature that results from modifying or reusing the reusable_feature in the Component_feature_make_from_relationship.

4.2.120.2 reusable_feature

The reusable_feature specifies the Component_feature that is modified or reused to become the resultant_feature in the Component_feature_make_from_relationship.

Formal constraints:

WR1: The resultant_feature shall not be the reusable_feature either directly or indirectly.

WR2: The type of the resultant_feature shall be the same as the type of the reusable_feature.

ENTITY component_feature_make_from_relationship
