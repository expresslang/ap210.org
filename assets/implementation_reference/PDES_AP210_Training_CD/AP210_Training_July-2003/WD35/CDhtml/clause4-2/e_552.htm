4.2.572 Movable_packaged_component_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Movable_packaged_component_join_terminal is a type of Packaged_component_join_terminal. A Movable_packaged_component_join_terminal is one termination for a Movable_packaged_component. The terminal provides access to the electrical functionality of the movable_packaged part being terminated.

The data associated with this application object are the following:

  • maximum_wire_terminal_length
  • minimum_wire_terminal_length

4.2.572.1 maximum_wire_terminal_length

The maximum_wire_terminal_length specifies the maximum dimension of the Length_data_element of the Movable_packaged_component_join_terminal.

4.2.572.2 minimum_wire_terminal_length

The minimum_wire_terminal_length specifies the minimum dimension of the Length_data_element of the Movable_packaged_component_join_terminal.
