4.2.855 Stratum_feature_in_stratum_planar_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_feature_in_stratum_planar_position is a type of Managed_design_object. A Stratum_feature_in_stratum_planar_position is the two-dimensional position and orientation of a two-dimensional model of a Stratum_feature in the two-dimensional model of the Stratum.

The data associated with this application object are the following:

  • feature_shape
  • stratum_shape
  • transformation

4.2.855.1 feature_shape

The feature_shape specifies either a Stratum_feature_planar_shape or a Stratum_feature_styled_area_representation for the Stratum_feature_in_stratum_planar_position.

4.2.855.2 stratum_shape

The stratum_shape specifies the role of the Stratum_planar_shape for the Stratum_feature_in_stratum_planar_position.

4.2.855.3 transformation

The transformation specifies the role of the Axis_placement_2d for the Stratum_feature_in_stratum_planar_position.
