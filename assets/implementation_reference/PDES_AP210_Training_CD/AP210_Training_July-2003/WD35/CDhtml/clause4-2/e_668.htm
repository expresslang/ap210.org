4.2.668 Physical_unit_network_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_unit_network_definition is a type of Ee_product_definition. A Physical_unit_network_definition specifies a network of physical nodes, each of which references several physical terminals. Each node in the network is isolated from the other nodes in the network and is separately identifiable.

NOTE: This Application object corresponds to the Functional_unit_network_definition but the nodes and terminals in that AO are abstract.

EXAMPLE: A Physical_unit_network_definition is referenced by members of Physical_connectivity_definition that correspond to the connectivity input to a design process for interconnect design.
