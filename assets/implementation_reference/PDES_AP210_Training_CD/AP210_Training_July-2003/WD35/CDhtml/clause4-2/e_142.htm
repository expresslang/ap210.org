4.2.142 Component_terminal_to_pca_terminal_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_terminal_to_pca_terminal_assignment is a type of Managed_design_object. A Component_terminal_to_pca_terminal_assignment is the relationship between a Pca_terminal and one of an Assembly_module_component_terminal, a Component_interface_terminal, or an Interconnect_module_component_terminal, to provide traceability from the data in the design view to the data in the usage view for the design organization for archival purposes.

The data associated with this application object are the following:

  • assigned_terminal
  • usage_view_terminal

4.2.142.1 assigned_terminal

The assigned_terminal specifies one of an Assembly_module_component_terminal, a Component_interface_terminal, or an Interconnect_module_component_terminal for the Component_terminal_to_pca_terminal_assignment.

4.2.142.2 usage_view_terminal

The usage_view_terminal specifies the role of the Pca_terminal for the Component_terminal_to_pca_terminal_assignment.

Formal constraints:

UR1: The usage_view_terminal shall be unique within a population of Component_terminal_to_pca_terminal_assignments.
