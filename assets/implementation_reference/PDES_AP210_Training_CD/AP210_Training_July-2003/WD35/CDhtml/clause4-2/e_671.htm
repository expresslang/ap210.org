4.2.671 Physical_unit_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_unit_relationship is a type of Managed_design_object. A Physical_unit_relationship is the association of two Physical_unit.

The data associated with this application object are the following:

  • related_unit
  • relating_unit

4.2.671.1 related_unit

The related_unit specifies the dependent Physical_unit that is related to the independent Physical_unit in a Physical_unit_relationship.

4.2.671.2 relating_unit

The relating_unit specifies the independent Physical_unit to which the Physical_unit_relationship is made.
