4.2.878 Substitute_product

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Substitute_product is a type of Managed_design_object. A Substitute_product is an Ee_product that may be substituted for another Ee_product within the context of an assembly or for a specific instance within an assembly. The Substitute_product is a one-way substitution only. Two instances of Substitute_product are required to support the declaration of a totally equivalent substitution relationship.

NOTE: Two different substitutes may be defined within the same assembly for the same base product based on the different usages of that base product within that assembly.

The data associated with this application object are the following:

  • defined_alternate_product
  • substitutable_component

4.2.878.1 defined_alternate_product

The defined_alternate_product specifies the role of the Ee_product for the Substitute_product. There shall be one or more Ee_product for the Substitute_product.

4.2.878.2 substitutable_component

The substitutable_component specifies the role of the Assembly_composition_relationship for the Substitute_product.
