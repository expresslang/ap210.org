4.2.427 Guided_wave_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Guided_wave_terminal is a type of Package_terminal that is explicitly included for the purpose of providing access to the functionality of a Packaged_part using guided waves.

EXAMPLE: Examples of Guided_wave_terminal include fiber-optic terminals and waveguide ports.

Formal constraints:

WR1: A Guided_wave_terminal shall not be a Wire_terminal.

  SUBTYPE OF (package_terminal);
