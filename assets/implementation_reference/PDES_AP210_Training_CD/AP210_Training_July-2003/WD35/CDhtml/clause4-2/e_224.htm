4.2.224 Datum_difference_based_model_parameter

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_difference_based_model_parameter is a type of Model_parameter. In effect, the Datum_difference provides a geometric basis for the definition of the Model_parameter.

The data associated with this application object are the following:

  • of_datum_difference

4.2.224.1 of_datum_difference

An of_datum_difference specifies the role of the Datum_difference for the Datum_difference_based_model_parameter.
