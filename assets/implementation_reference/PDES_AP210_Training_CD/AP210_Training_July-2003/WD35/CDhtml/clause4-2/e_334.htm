4.2.334 Ee_rule_function

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_rule_function is a type of Managed_design_object. An Ee_rule_function is a formal specification for a process that includes the source code and instance values for the process. An Ee_rule_function has a unique combination of identifier and reference_document. An Ee_rule_function shall have at least one parameter.

NOTE: Source code is the computer-interpretable expressions that form a part of the Ee_rule_function. The attribute source_code is the relationship of these expressions to the Ee_rule_function.

The data associated with this application object are the following:

  • identifier
  • parameter
  • reference_document
  • source_code

4.2.334.1 identifier

The identifier specifies the name for the Ee_rule_function.

4.2.334.2 parameter

The parameter specifies the role of the Model_parameter for an Ee_rule_function. There shall be one or more Model_parameter specified for a particular Ee_rule_function.

4.2.334.3 reference_document

The reference_document specifies the role of the Ee_document for an Ee_rule_function.

4.2.334.4 source_code

The source_code specifies the role of the Ee_specification for an Ee_rule_function.

Formal constraints:

UR1: The combination of the identifier and the reference_document shall be unique within a population of Ee_rule_functions.
