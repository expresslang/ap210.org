4.2.261 Design_characteristic_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Design_characteristic_occurrence is a type of Ee_requirement_occurrence that specifies the technical performance values and tolerances for some aspect of a design.

NOTE: The specifications set forth in Design_characteristics may include such items as enterprise or customer specifications on preferred packaging, power design and control, design rules, and preferred design practices.
