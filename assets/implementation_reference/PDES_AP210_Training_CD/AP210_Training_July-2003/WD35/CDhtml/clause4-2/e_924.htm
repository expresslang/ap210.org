4.2.924 Tolerance_zone_form

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_zone_form is a type of Managed_design_object. A Tolerance_zone_form is a definition that characterizes of the form of the Tolerance_zone.

The data associated with this application object are the following:

  • characterized_tolerance_zone
  • name
  • zone_definition

4.2.924.1 characterized_tolerance_zone

The characterized_tolerance_zone specifies the Tolerance_zone for which the Tolerance_zone_form is defined.

4.2.924.2 name

The name specifies the identifying label of the Tolerance_zone_form.

EXAMPLE: The name of the Tolerance_zone_form might be "cylindrical" for a Tolerance_zone that is defined by a Perpendicularity_tolerance.

4.2.924.3 zone_definition

The zone_definition is an inverse relationship that specifies that the existence of the Tolerance_zone_form is dependent on the existence of the Tolerance_zone_definition that specifies the Tolerance_zone_form as its form.

ENTITY tolerance_zone_form
  SUBTYPE OF (managed_design_object);
    characterized_tolerance_zone : tolerance_zone;
    name                         : label;
  INVERSE
    zone_definition : tolerance_zone_definition FOR form;
END_ENTITY; -- tolerance_zone_form
