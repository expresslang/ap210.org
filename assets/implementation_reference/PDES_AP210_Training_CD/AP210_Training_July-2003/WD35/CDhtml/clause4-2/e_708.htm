4.2.706 Pre_defined_datum_plane_3d_symbol

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Pre_defined_datum_plane_3d_symbol is a type of Non_feature_shape_definition and a type of Bound_volume_shape. A Pre_defined_datum_plane_3d_symbol is a standard representation for a Datum_plane in a 3 dimensional coordinate system. A Pre_defined_datum_plane_3d_symbol shall be parallel with the normal vector of the plane, and shall intersect the plane at the origin of the symbol.

The data associated with this application object are the following:

  • associated_element

4.2.706.1 associated_element

The associated_element specifies the role of the Datum_plane for the Pre_defined_datum_plane_3d_symbol.
