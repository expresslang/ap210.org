4.2.759 Radius_edge_feature_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Radius_edge_feature_shape is a type of Edge_shape that describes a circular cross sectioned edge. See Bevel_edge_feature_shape for the additional required geometric information that shall be provided.

EXAMPLE: Figure refs illustrates the shape related information exchanged with this Application object.

The data associated with this application object are the following:

  • maximum_radius_length
  • minimum_radius_length

4.2.759.1 maximum_radius_length

The maximum_radius_length specifies the Length_data_element of the largest allowable radius of the circle that is tangent to the edge surface and to the lateral surface on which the edge surface is a feature for the Radius_edge_feature_shape. The included angle between the points of tangency shall always be 90 degrees.

4.2.759.2 minimum_radius_length

The minimum_radius_length specifies the Length_data_element of the smallest allowable radius of the circle that is tangent to the edge surface and to the lateral surface on which the edge surface is a feature for the Radius_edge_feature_shape. The included angle between the points of tangency shall always be 90 degrees.
