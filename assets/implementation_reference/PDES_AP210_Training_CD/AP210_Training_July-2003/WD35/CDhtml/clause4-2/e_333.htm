4.2.333 Ee_rule_definition

An Ee_rule_definition is a type of Ee_product_definition. An Ee_rule_definition is a formal specification that defines a constraint by relating two elements based upon characteristics of a design, one as premise and one as conclusion. The Ee_rule_definition is for one version of a rule only.

The data associated with this application object are the following:

  • conclusion
  • version
  • premise

4.2.333.1 conclusion

The conclusion specifies the role of the ee_rule_conclusion_definition for the Ee_rule_definition.

4.2.333.2 defined_versioned_rule

The version specifies the role of the Ee_rule_version for the Ee_rule_definition.

4.2.333.3 premise

The premise specifies the role of the ee_rule_premise_definition for the Ee_rule_definition.
