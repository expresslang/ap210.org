4.2.865 Stratum_make_from_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_make_from_relationship is a type of Managed_design_object. A Stratum_make_from_relationship is an association between a Stratum whose design is intended to be reused and a Stratum that results from the reuse. The resultant_stratum may differ from the reusable_stratum for the particular Interconnect_module design.

The data associated with this application object are the following:

  • resultant_stratum
  • reusable_stratum

4.2.865.1 resultant_stratum

The resultant_stratum specifies the Stratum for the result of a design activity that started with reusable_stratum.

4.2.865.2 reusable_stratum

The reusable_stratum specifies the Stratum that is intended to be used as a reference from which to create a resultant_stratum.
