4.2.90 Centre_axis

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Centre_axis is a type of Centre_of_symmetry that is axial in form. The Shape_element from which the Centre_axis is derived are associated with the Centre_axis by the Shape_element_deriving_relationship that are specified as the deriving_relationships.

NOTES:

  1. The attribute deriving_relationships is declared in the Derived_shape_element entity of which this entity is a subtype.
  2. The precise interpretation of a Centre_axis is context dependent. That is, the distinction is not made between an axis of a feature and a derived median line of a feature as defined in 1.3 of ASME Y14.5M.

EXAMPLE: A Position_tolerance and a Straightness_tolerance may be applied to the same Centre_axis of a cylindrical Shape_element.

Informal constraints:

IP1: The form of the Shape_elements from which the Centre_axis is derived shall be consistent with the form of shape elements from which a centre axis can be derived.
