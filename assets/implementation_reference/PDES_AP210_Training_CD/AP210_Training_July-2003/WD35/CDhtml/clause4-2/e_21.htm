4.2.21 Analytical_model_vector_port

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Analytical_model_vector_port is a type of Analytical_model_port that has a size greater than one. An Analytical_model_vector_port may be an Analytical_model_scalar_port. The Analytical_model_vector_port is used in the converse case to the Analytical_model_scalar_port. This part of ISO 10303 does not predefine the elements of a vector port.

NOTE: The cardinality of the structured port is known and a list of names is specified for it. Examples are bus ports, multi-degrees-of-freedom connections, microwave domain modes, and address and data bus definitions.

The data associated with this application object are the following:

  • size

4.2.21.1 size

The size specifies the integer greater than one for the cardinality of the Analytical_model_vector_port.

Formal constraints:

WR1: The size shall be greater than one.
