4.2.360 Explicitly_located_layer_connection_point

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Explicitly_located_layer_connection_point is a type of Layer_connection_point that is not a Dependently_located_layer_connection_point. An Explicitly_located_layer_connection_point may be referenced by a Junction_vertex_allocation to help define a topological structure.

EXAMPLE: Figure ellcp illustrates a detail of a layout drawing that shows the Explicitly_located_layer_connection_points that are included in the design due to internal layout considerations.

The data associated with this application object are the following:

  • location

4.2.360.1 location

The location specifies the Cartesian_point relative to the Cartesian_point origin of the Interconnect_module at which the Explicitly_located_layer_connection_point resides.

Formal constraints:

UR1: The combination of location, and resident_design_layer_stratum inherited from the supertype Layer_connection_point shall be unique within a population of Explicitly_located_layer_connection_point.

WR1: The location shall be in the extent of the resident_design_layer_stratum.
