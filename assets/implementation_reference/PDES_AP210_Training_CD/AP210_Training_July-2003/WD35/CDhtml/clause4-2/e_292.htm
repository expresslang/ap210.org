4.2.299 Dimensional_characteristic_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimensional_characteristic_representation is a type of Managed_design_object. A Dimensional_characteristic_representation is an association between a Dimensional_characteristic and the Shape_dimension_representation that represents the Dimensional_characteristic.

The data associated with this application object are the following:

  • auxiliary_dimension_representation
  • dimension
  • representation

4.2.299.1 auxiliary_dimension_representation

The auxiliary_dimension_representation specifies the boolean value that indicates whether the associated Shape_dimension_representation is to be considered a requirement for the associated Dimensional_characteristic or is specified for informational purposes only. A value of TRUE indicates that the associated Shape_dimension_representation is for informational purposes only. A value of FALSE indicates that the associated Shape_dimension_representation is to be considered a requirement.

NOTE: Auxiliary dimension is defined in 3.1.1.3 of ISO 129.

4.2.299.2 dimension

The dimension specifies either a Dimensional_location or a Dimensional_size that is represented by the specified Shape_dimension_representation for the Dimensional_characteristic_representation.

4.2.299.3 representation

The representation specifies the Shape_dimension_representation that represents the specified Dimensional_characteristic for the Dimensional_characteristic_representation.
