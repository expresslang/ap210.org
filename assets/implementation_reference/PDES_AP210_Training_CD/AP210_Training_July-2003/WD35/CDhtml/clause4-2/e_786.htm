4.2.786 Rule_expiration

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_expiration is a type of Rule_action that states that a version of an Ee_rule has expired.

The data associated with this application object are the following:

  • expiration_rationale

4.2.786.1 expiration_rationale

The expiration_rationale specifies the human-interpretable reason for the Rule_expiration.
