4.2.784 Rule_change_request

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_change_request is a type of Rule_action that states a request has been made for modification to a version of an Ee_rule. The change reason shall be provided as a part of the Rule_change_request.

The data associated with this application object are the following:

  • change_reason

4.2.784.1 change_reason

The change_reason specifies the human-interpretable reason for requesting a Rule_change_request.
