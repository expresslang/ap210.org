4.2.154 Compound_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Compound_characteristic is a type of Characteristic. A Compound_characteristic may be a Characteristic_data_table. A Compound_characteristic is a property of an item. A Compound_characteristic helps to compose a structured characteristic, in conjunction with Compound_characteristic_definition.

NOTE: A Compound_characteristic may describe either a physical item, like a realized part, or a description that is unrealizable, such as a simulation model. A Compound_characteristic is most often used to describe a table of values that may range over a discrete parameter, e.g., a table of response values vs. frequency or time.

The data associated with this application object are the following:

  • property_value

4.2.154.1 property_value

The property_value specifies the role of a Compound_characteristic_definition for the Compound_characteristic.
