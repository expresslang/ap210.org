4.2.356 Evaluated_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Evaluated_characteristic is a type of Managed_design_object. The Evaluated_characteristic provides a mechanism to validate or evaluate a Planned_characteristic.

The data associated with this application object are the following:

  • planned_product_data
  • product_characterization_data
  • product_data_status

4.2.356.1 planned_product_data

The planned_product_data specifies the role of the Planned_characteristic for an Evaluated_characteristic.

4.2.356.2 product_characterization_data

The product_characterization_data specifies one or more Coordinated_characteristic for an Evaluated_characteristic. The product_characterization_data need not be provided for a specific Evaluated_characteristic.

4.2.356.3 product_data_status

The product_data_status provides a status of valid, invalid, not_evaluatable, or to_be_evaluated for an Evaluated_characteristic.

  SUPERTYPE OF (rule_based_evaluated_characteristic)
  SUBTYPE OF (managed_design_object);

    product_data_status           : evaluation_type;
    product_characterization_data : OPTIONAL SET [1:?] OF coordinated_characteristic;
