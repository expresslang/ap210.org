4.2.676 Placement_group_volume_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Placement_group_volume_assignment is a type of Managed_design_object. A Placement_group_volume_assignment is the relationship between a Placement_group and the Mounting_restriction_volume on the interconnect substrate instance to which it is restricted.

The data associated with this application object are the following:

  • volume
  • placed_group

4.2.676.1 volume

The volume specifies the role of the Mounting_restriction_volume for the Placement_group_volume_assignment.

4.2.676.2 placed_group

The placed_group specifies the role of the Placement_group for the Placement_group_volume_assignment.
