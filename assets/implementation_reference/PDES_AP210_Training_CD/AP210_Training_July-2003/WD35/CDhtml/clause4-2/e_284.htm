4.2.284 Device_3d_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Device_3d_position is a type of Managed_design_object. A Device_3d_position is the location in three dimensions of a Part_device within a Packaged_part.

The data associated with this application object are the following:

  • device_shape
  • part_shape
  • placed_device
  • transformation

4.2.284.1 device_shape

The device_shape specifies the Physical_unit_3d_shape of the Part_device for the Device_3d_position.

4.2.284.2 part_shape

The part_shape specifies the Physical_unit_3d_shape of the Packaged_part for the Device_3d_position.

4.2.284.3 placed_device

The placed_device specifies the role of the Part_device for the Device_3d_position.

4.2.284.4 transformation

The transformation specifies the role of the Axis_placement_3d for the Device_3d_position.
