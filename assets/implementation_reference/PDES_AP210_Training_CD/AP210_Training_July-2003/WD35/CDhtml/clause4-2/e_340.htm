4.2.342 Electrical_isolation_laminate_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Electrical_isolation_laminate_component is a type of Filled_area_material_removal_laminate_component that indicates that material is to be removed in the realization of the feature that is in the role of design_intent inherited from Filled_area_material_removal_laminate_component. The requirement shall be interpreted mathematically as a requirement for application of the regularized Boolean subtraction operation.

The data associated with this application object are the following:

  • usage_definition

4.2.342.1 usage_definition

The usage_definition specifies the role of the Electrical_isolation_removal_template for the Electrical_isolation_laminate_component.
