4.2.646 Person

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Person is a type of Managed_design_object. A Person is an individual within the context of a single Organization.

The data associated with this application object are the following:

  • address
  • identifier

4.2.646.1 address

The address specifies the string for the interpretable name for the location of the Person. The address specifies the routing for postal mail or the physical location of the Person. The address need not be specified for a particular Person.

4.2.646.2 identifier

The identifier specifies the name that represents either the personal name or title of the Person.
