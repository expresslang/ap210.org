4.2.952 Via_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Via_template is a type of Inter_stratum_feature_template.

NOTE: A Via_template provides a template for a plated passage that supports the realization of connectivity. The access_mechanisms attribute relates the Via_template to two or more terminals, thereby capturing complete connectivity information.

The data associated with this application object are the following:

  • access_mechanisms
  • of_passage_technology

4.2.952.1 access_mechanisms

The access_mechanisms is an inverse relationship that specifies that the existence of the Via_template is dependent on the existence of the Via_template_terminal that specifies the Via_template as its associated_definition. There shall be two or more Via_template_terminals for a Via_template.

4.2.952.2 of_passage_technology

The of_passage_technology specifies the role of the Default_via_definition for the Via_template.
