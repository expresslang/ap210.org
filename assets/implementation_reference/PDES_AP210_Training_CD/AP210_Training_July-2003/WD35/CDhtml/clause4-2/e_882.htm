4.2.882 Symmetrical_tolerance_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Symmetrical_tolerance_characteristic is a type of Tolerance_characteristic that has symmetrical tolerances around the nominal value.

The data associated with this application object are the following:

  • component_characteristic

4.2.882.1 component_characteristic

The component_characteristic specifies the value of the Numeric_parameter for the Symmetrical_tolerance_characteristic. There shall be exactly two Numeric_parameter for the Symmetrical_tolerance_characteristic.
