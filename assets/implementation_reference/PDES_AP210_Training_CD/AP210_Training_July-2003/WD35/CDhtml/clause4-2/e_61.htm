4.2.61 Bare_die_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Bare_die_component is a type of Assembly_component that is defined by a Bare_die.

The data associated with this application object are the following:

  • usage_definition

4.2.61.1 usage_definition

The usage_definition specifies the role of the Bare_die for the Bare_die_component.
