4.2.923 Tolerance_zone_explicit_opposing_boundary_set

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_zone_explicit_opposing_boundary_set is a type of Tolerance_zone_opposing_boundary_set in which the two opposing boundary members are explicitly identified.

The data associated with this application object are the following:

  • boundary_members

4.2.923.1 boundary_members

The boundary_members specify the Tolerance_zone_boundary_member that comprise the Tolerance_zone_explicit_opposing_boundary_set. There shall be exactly two Tolerance_zone_boundary_members specified as boundary_members for a Tolerance_zone_explicit_opposing_boundary_set.
