4.2.694 Polygonal_area

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Polygonal_area is a type of Primitive_2d.

The data associated with this application object are the following:

  • bounds

4.2.694.1 bounds

The bounds specifies the role of a Cartesian_point for the Polygonal_area. There shall be three or more Cartesian_point for a Polygonal_area. No Cartesian_point shall be specified more than once as a bounds for a particular Polygonal_area.

NOTE: 
*)
ENTITY polygonal_area
  SUBTYPE OF (primitive_2d);
   bounds:  LIST [3:?] OF UNIQUE cartesian_point;
END_ENTITY; -- polygonal_area
