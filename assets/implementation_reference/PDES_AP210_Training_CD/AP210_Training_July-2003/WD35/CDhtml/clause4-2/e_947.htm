4.2.947 Via

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Via is a type of Plated_passage. A Via is one of a Blind_via, a Buried_via, or an Interfacial_connection. Via is defined in IEC 60050-541.

EXAMPLE: Figure via illustrates a padstack that contains a Via.

The data associated with this application object are the following:

  • usage_definition

4.2.947.1 usage_definition

The usage_definition specifies the role of the Via_template for the Via.

Formal constraints:

WR1: A Via shall not interface to an external device. This fact shall be represented by a value of the attribute feature_of_size on the supertype Inter_stratum_feature being set to "FALSE".

ENTITY via
  ABSTRACT SUPERTYPE OF (ONEOF (blind_via,buried_via,interfacial_connection))

    SELF\assembly_component.usage_definition : via_template;
