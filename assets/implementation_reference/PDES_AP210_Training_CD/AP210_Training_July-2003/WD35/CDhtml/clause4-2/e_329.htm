4.2.329 Ee_product_version_supersedence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Ee_product_version_supersedence is a type of Managed_design_object.

The data associated with this application object are the following:

  • superseded_product_version
  • superseding_product_version

4.2.329.1 superseded_product_version

The superseded_product_version specifies an Ee_product_version for the Ee_product_version_supersedence.

4.2.329.2 superseding_product_version

The superseding_product_version specifies an Ee_product_version for the Ee_product_version_supersedence.

Formal constraints:

UR1: The combination of superseded_product_version and superseding_product_version shall be unique within a population of Ee_product_version_supersedence.
