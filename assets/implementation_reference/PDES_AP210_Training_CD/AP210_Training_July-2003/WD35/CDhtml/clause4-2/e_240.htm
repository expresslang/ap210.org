4.2.240 Datum_target

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_target is a type of Physical_feature_or_part_template that is used to define one or more Datum_target_set.

NOTES:

  1. Typically datum targets are used when it is not appropriate to specify an entire surface as a Datum_feature.
  2. The definition of datum target is given in 3.4 of ISO 5459.

The data associated with this application object are the following:

  • datum_target_usages

4.2.240.1 datum_target_usages

The datum_target_usages is an inverse relationship that specifies that the existence of the Datum_target is dependent on the existence of the Datum_target_usage_in_datum_target_set_relationship that specifies the Datum_target as its used_datum_target. There shall be one or more Datum_target_usage_in_datum_target_set_relationship for the Datum_target.

  SUBTYPE OF (physical_feature_or_part_template);
  INVERSE

                          datum_target_usage_in_datum_target_set_relationship FOR 
                          used_datum_target;
