4.2.459 Interconnect_module_secondary_surface

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_secondary_surface is a type of Interconnect_module_surface_feature. An Interconnect_module_secondary_surface identifies the secondary surface of the Interconnect_module for fabrication and assembly purposes. If a design view is populated, and correlation to a Stratum is provided, the Stratum associated with this member of Interconnect_module_primary_surface shall be the last Stratum in the design.

NOTE: The Stratum_concept_to_physical_usage_view_assignment, Adjacent_stratum_surface_definition, and Stratum_surface provide supporting data needed to resolve the position of the Stratum in the design. Because the secondary surface is derived structurally from the primary Stratum, there is no approximation made in the description.
