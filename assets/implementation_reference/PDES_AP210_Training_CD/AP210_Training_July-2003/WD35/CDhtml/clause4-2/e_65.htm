4.2.65 Bare_die_surface

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Bare_die_surface is a type of Part_mounting_feature. A Bare_die_surface is one of a Bare_die_top_surface, a Bare_die_bottom_surface, a Bare_die_edge_segment_surface, or a Bare_die_edge_surface.
