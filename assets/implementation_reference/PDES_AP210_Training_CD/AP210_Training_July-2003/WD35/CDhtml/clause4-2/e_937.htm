4.2.937 Unrouted_conductive_interconnect_element

An Unrouted_conductive_interconnect_element is a type of Unrouted_conductive_interconnect_element. The data provided by Unrouted_conductive_interconnect_element is provided to identify an unrouted portion of a Physical_network.

The data associated with this application object are the following:

  • connected_point

4.2.937.1 connected_point

The connected_point specifies the role of the Layer_connection_point_link for the Unrouted_conductive_interconnect_element. There shall be one or more Layer_connection_point_link for an Unrouted_conductive_interconnect_element. No Layer_connection_point_link shall be specified more than once as a connected_point for an Unrouted_conductive_interconnect_element.
