4.2.700 Positional_boundary_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Positional_boundary_definition is a type of Managed_design_object. A Positional_boundary_definition is a definition that characterizes a Positional_boundary. A Positional_boundary_definition is either a Profile_related_positional_boundary_definition or a Dimension_related_positional_boundary_definition.

The data associated with this application object are the following:

  • defined_positional_boundary

4.2.700.1 defined_positional_boundary

The defined_positional_boundary specifies the Positional_boundary that is characterized by the Positional_boundary_definition.
