4.2.336 Ee_rule_simple_clause

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
The Ee_rule_simple_clause is a type of Managed_design_object. An Ee_rule_simple_clause assigns data specific to the rule definition to the generic algorithm specified by rule_function.

NOTE: This data is not product instance data, but is categorical data helping to compose this definition.

EXAMPLE: An example of this type of data would be "all tracks less than 5 mils."

The data associated with this application object are the following:

  • rule_function
  • parameters
  • identifier

4.2.336.1 rule_function

The rule_function specifies the Ee_rule_definition of the Ee_rule_simple_clause. The rule_function is the rule to which the Ee_simple_clause is assigning parameters.

4.2.336.2 parameters

The parameters specifies the Parameter_assignment for an Ee_rule_simple_clause. There shall be one or more parameters specified for an Ee_rule_simple_clause.

4.2.336.3 identifier

The identifier specifies the name for the Ee_rule_simple_clause.

Formal constraints:

UR1: The identifier shall be unique within a population of Ee_rule_simple_clause.
