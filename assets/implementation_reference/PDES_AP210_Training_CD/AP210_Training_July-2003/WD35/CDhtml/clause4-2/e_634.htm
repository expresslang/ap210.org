4.2.630 Part_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_terminal is a type of Part_feature. A Part_terminal is one of an Assembly_module_terminal, a Bare_die_terminal, an Interconnect_module_terminal, or a Packaged_part_terminal.

The data associated with this application object are the following:

  • connection_requirement
  • functional_usage_view_terminal

4.2.630.1 connection_requirement

The connection_requirement is an inverse relationship that specifies the Part_connected_terminals_definition that specifies the Part_terminal as its connected_terminals. There shall be no more than one Part_connected_terminals_definition for a Part_terminal.

4.2.630.2 functional_usage_view_terminal

The functional_usage_view_terminal is an inverse relationship to the Functional_usage_view_to_part_terminal_assignment that specifies the Part_terminal as its physical_usage_view_terminal. There shall be no more than one Functional_usage_view_to_part_terminal_assignment for a Part_terminal.

ENTITY part_terminal
  ABSTRACT SUPERTYPE OF (ONEOF (minimally_defined_bare_die_terminal,
      assembly_module_terminal,interconnect_module_terminal,
      packaged_part_terminal))
  SUBTYPE OF (part_feature);
INVERSE
    connection_requirement         : SET [0:1] OF 
	part_connected_terminals_definition FOR connected_terminals;
    functional_usage_view_terminal : 
	SET [0:1] OF functional_usage_view_to_part_terminal_assignment FOR 
	physical_usage_view_terminal;
END_ENTITY; -- part_terminal
