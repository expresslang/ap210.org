4.2.417 Geometric_intersection

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Geometric_intersection is a type of Derived_shape_element that is the common intersection of two or more Shape_element. The Shape_element from which the Geometric_intersection is derived are associated with the Geometric_intersection by the Shape_element_deriving_relationship that are specified as the deriving_relationships.

NOTE: The attribute deriving_relationships is declared in the Derived_shape_element entity of which this entity is a subtype.

Formal constraints:

WR1: The Geometric_intersection shall be derived from two or more Shape_element.

Informal constraints:

IP1: The Shape_element from which the Geometric_intersection is derived shall intersect.

NOTE: 
*)
ENTITY geometric_intersection
  SUBTYPE OF (derived_shape_element);
END_ENTITY; -- geometric_intersection
