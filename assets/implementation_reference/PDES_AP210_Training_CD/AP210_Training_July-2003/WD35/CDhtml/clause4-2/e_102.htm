4.2.101 Circular_area

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Circular_area is a type of Primitive_2d.

The data associated with this application object are the following:

  • centre
  • radius

4.2.101.1 centre

The centre specifies a Cartesian_point for the Circular_area.

4.2.101.2 radius

The radius specifies a Length_data_element for the Circular_area.

NOTE: 
*)
ENTITY circular_area
  SUBTYPE OF (primitive_2d);
    centre: cartesian_point;
    radius: length_data_element;
END_ENTITY; -- circular_area
