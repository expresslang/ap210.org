4.2.741 Product_association

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Product_association is a type of Managed_design_object. A Product_association is a usage constraint relationship among some parts that may be used in a design. If the relating part is used in a design then the related parts must also be incorporated in the design and in a prescribed fashion.

The data associated with this application object are the following:

  • constraint_source
  • related_product
  • relating_product

4.2.741.1 constraint_source

The constraint_source specifies the Ee_specification that describes the manner in which Ee_product_version that participate in the Product_association are to be used in the design.

4.2.741.2 related_product

The related_product specifies the Ee_product_version that must be incorporated in the design when the relating_product is used in the design for the Product_association. There shall be one or more Ee_product_version for the Product_association.

4.2.741.3 relating_product

The relating_product specifies the Ee_product_version that, when incorporated in the design, requires the incorporation of one or more related_product to be incorporated in the design to realize the desired functionality of the relating_product for the Product_association.

Formal constraints:

UR1: The constraint_source shall be unique within a population of Product_associations.

WR1: An Ee_product_version shall not be an ancestor or descendent of itself in a population of this object.

ENTITY product_association
  SUBTYPE OF (managed_design_object);
    relating_product  : ee_product_version;
    related_product   : SET [1:?] OF ee_product_version;
    constraint_source : ee_specification;
  UNIQUE
    ur1 : constraint_source, relating_product;
END_ENTITY; -- product_association
