4.2.436 Inter_stratum_join_implementation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Inter_stratum_join_implementation is a type of Managed_design_object. An Inter_stratum_join_implementation is one of a Physical_network_supporting_inter_stratum_feature or a Physical_network_supporting_stratum_feature_conductive_join. An Inter_stratum_join_implementation is a physical feature helping to implement a Physical_network. An Inter_stratum_join_implementation is not intended to be implemented alone. It is one of several Application Objects that together form the physical network implementation model.

NOTES:

  1. The types of Inter_stratum_join_implementation correspond to two physical implementation mechanisms.
  2. The first mechanism consists of creating a passage or edge in the material and then adding conductive material to the passage or edge wall and features on the to_stratum and from_stratum of the Inter_stratum_extent. This mechanism results in a connection between the required features implemented with the passage or edge wall.
  3. The second mechanism does not result in a conducting passage, but relies on a bond (achieved either by direct deposition or some other means) between the features related to the specified layer_connection_points. The second mechanism usually relies on a passage in only the dielectric material separating the bonded materials. A cross-sectional view of the pcb may show a vertical displacement of strata in the area of this object, due to the fact that the dielectric Stratum is much thicker than the conductive Stratum. The shape of the vertical displacement is not a part of the definition of the Inter_stratum_join_implementation when the second mechanism is used.

The data associated with this application object are the following:

  • topological_requirement

4.2.436.1 topological_requirement

The topological_requirement specifies the role of the Inter_stratum_join_relationship for the Inter_stratum_join_implementation.

NOTE: The following is a description of a realization of an Inter_stratum_join_implementation appropriate for some technologies. Features that implement inter layer connection are realized by forming passages in the sequence of stratum that define the extent of the passage. The passage has a nominally uniform cross-section and has a direction normal to the surface specified as the from_stratum. If that surface is not planar, then the surface coordinates of the layer connection points vary layer to layer to account for the finite thickness of the stratum comprising the Stratum.
