4.2.546 Minimally_defined_component_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Minimally_defined_component_terminal is a type of Component_terminal that is included to permit references to terminals without including geometric representations of the terminals.

The data associated with this application object are the following:

  • definition

4.2.546.1 definition

The definition specifies the role of the Part_terminal_external_reference for the Minimally_defined_component_terminal.
