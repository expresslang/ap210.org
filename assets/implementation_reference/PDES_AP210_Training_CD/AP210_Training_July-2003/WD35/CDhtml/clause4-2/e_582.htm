4.2.593 Packaged_part_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Packaged_part_join_terminal is a type of Packaged_part_terminal such that an occurrence of the Packaged_part_terminal shall not be in the role of an external interface when the occurrence of the associated Packaged_connector is a component in an assembly.

The data associated with this application object are the following:

  • internal_connector_function

4.2.593.1 internal_connector_function

The internal_connector_function is an inverse relationship that specifies the Packaged_connector_terminal_relationship that specifies the Packaged_part_join_terminal as its join_terminal. There shall be no more than one Packaged_connector_terminal_relationship for a Packaged_part_join_terminal.
