4.2.849 Stratum_concept_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_concept_relationship is a type of Managed_design_object. A Stratum_concept_relationship is a Dielectric_crossover_area or a Stratum_feature_conductive_join.

The data associated with this application object are the following:

  • resulting_shape
  • technology_requirement

4.2.849.1 resulting_shape

The resulting_shape specifies the role of the Feature_shape_definition for the Stratum_concept_relationship. The resulting_shape need not be specified for a particular Stratum_concept_relationship.

4.2.849.2 technology_requirement

The technology_requirement specifies the Ee_requirement_occurrence that results in the resulting_shape. There shall be one or more Ee_requirement_occurrence for a Stratum_concept_relationship.
