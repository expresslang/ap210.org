4.2.225 Datum_difference_functional_unit_usage_view_terminal_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_difference_functional_unit_usage_view_terminal_assignment is a type of Managed_design_object. A Datum_difference_functional_unit_usage_view_terminal_assignment provides the ability to map from a geometric context into the functional context of a Functional_unit_definition.

The data associated with this application object are the following:

  • assigned_functional_terminal_definition
  • assigned_datum_difference

4.2.225.1 assigned_functional_terminal_definition

An assigned_functional_terminal_definition specifies the role of the Functional_unit_usage_view_terminal_definition for the Datum_difference_functional_unit_usage_view_terminal_assignment.

4.2.225.2 assigned_datum_difference

An assigned_datum_difference specifies the role of the Datum_difference for the Datum_difference_functional_unit_usage_view_terminal_assignment.
