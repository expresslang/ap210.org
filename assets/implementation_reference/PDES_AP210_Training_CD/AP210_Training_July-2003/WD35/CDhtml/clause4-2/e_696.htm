4.2.699 Positional_boundary_and_profile_boundary_member_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Positional_boundary_and_profile_boundary_member_relationship is a type of Managed_design_object. A Positional_boundary_and_profile_boundary_member_relationship is a relationship that associates a Profile_related_positional_boundary with the Tolerance_zone_boundary_member from which the Profile_related_positional_boundary is offset.

The data associated with this application object are the following:

  • base_profile_tolerance_zone_boundary_member
  • offset
  • offset_positional_boundary

4.2.699.1 base_profile_tolerance_zone_boundary_member

The base_profile_tolerance_zone_boundary_member specifies the Tolerance_zone_boundary_member from which the Profile_related_positional_boundary specified as the offset_positional_boundary is offset.

4.2.699.2 offset

The offset is an inverse relationship that specifies that the existence of the Positional_boundary_and_profile_boundary_member_relationship is dependent on the existence of the Positional_boundary_offset that specifies the Positional_boundary_and_profile_boundary_member_relationship as its offset_boundary_relationship.

4.2.699.3 offset_positional_boundary

The offset_positional_boundary specifies the Profile_related_positional_boundary that is offset from the Tolerance_zone_boundary_member that is specified as the base_profile_tolerance_zone_boundary_member.
