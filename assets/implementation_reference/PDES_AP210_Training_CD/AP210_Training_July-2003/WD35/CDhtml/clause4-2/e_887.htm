4.2.887 Template_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Template_definition is a type of Ee_product_definition.

The data associated with this application object are the following:

  • associated_template_occurrence

4.2.887.1 associated_template_occurrence

The associated_template_occurrence is an inverse relationship that specifies that the existence of the Template_definition is dependent on the existence of a Part_template to reference the Template_definition.

NOTE: 
*)
ENTITY template_definition
  SUBTYPE OF (ee_product_definition);
INVERSE
      associated_template_occurrence : 
         part_template FOR associated_definition;
END_ENTITY; -- template_definition
