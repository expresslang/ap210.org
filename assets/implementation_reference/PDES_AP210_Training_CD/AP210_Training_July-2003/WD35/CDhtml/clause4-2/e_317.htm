4.2.317 Ee_document_identifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_document_identifier is a type of Managed_design_object. An Ee_document_identifier is the combination of an identifier name and an assigned value.

EXAMPLE: An example of an Ee_document_identifier is an International Serial Book Number, ISBN, as the identifier_name, and the number associated with a specific publication as the assigned_value. Other examples may be found in the Library of Congress system and the Dewey Decimal System.

The data associated with this application object are the following:

  • assigned_value
  • identifier_name

4.2.317.1 assigned_value

The assigned_value specifies a string identifying the content of the Ee_document_identifier.

4.2.317.2 identifier_name

The identifier_name specifies the string that establishes the context or type for the Ee_document_identifier.

Formal constraints:

UR1: The combination of the identifier_name and assigned_value shall be unique within a population of Ee_document_identifiers.
