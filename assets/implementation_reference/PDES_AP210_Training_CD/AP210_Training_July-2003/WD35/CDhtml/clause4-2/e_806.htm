4.2.806 Shape_and_ee_product_definition_based_constraint_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shape_and_ee_product_definition_based_constraint_occurrence is a type of Ee_product_definition_based_constraint_occurrence that has a basis in the characteristics of one or more Shape_definition within the context of the constraining_definition inherited from Ee_product_definition_based_constraint_occurrence. Each member of Shape_definition referenced shall reference the Ee_product_definition referenced by the constraining_definition attribute inherited from Ee_product_definition_based_constraint_occurrence.

NOTE: This provides a mechanism to specify that one or more of the analysis or keepout shapes on Physical_unit is a constraint.

The data associated with this application object are the following:

  • constraining_shape

4.2.806.1 constraining_shape

The constraining_shape specifies the role of the Shape_definition for the Shape_and_ee_product_definition_based_constraint_occurrence. The interpretation shall be that the constraining_shape specifies the shape_representation and the constraining_definition inherited from the supertype specifies the product_definition. There may be one or more Shape_definition for the Shape_and_ee_product_definition_based_constraint_occurrence.
