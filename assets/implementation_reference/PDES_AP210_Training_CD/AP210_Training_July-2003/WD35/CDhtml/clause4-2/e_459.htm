4.2.473 Interfaced_group_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interfaced_group_component is a type of Assembly_group_component. An Interfaced_group_component includes one or more Interface_component that interfaces a subset of that group of components to the assembly.

EXAMPLE: A component and a socket are both assigned to an Interfaced_group_component. The socket interfaces the other component to the assembly. Both the component and the socket are separately referenced and placed in the assembly.

The data associated with this application object are the following:

  • group_interface_component

4.2.473.1 group_interface_component

The group_interface_component specifies the Interface_component that interfaces the Interfaced_group_component to the assembly. There shall be one or more Interface_component for an Interfaced_group_component.
