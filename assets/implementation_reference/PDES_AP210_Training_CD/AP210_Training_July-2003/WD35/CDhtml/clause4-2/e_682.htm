4.2.682 Planned_lot_effectivity

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Planned_lot_effectivity is a type of Planned_effectivity. A Planned_lot_effectivity is the specification by the design organization of the intended use of an Ee_product or of an Ee_product within an Ee_product_configuration, where the Ee_product or the Ee_product within an Ee_product_configuration is produced as one of a group or lot.

The data associated with this application object are the following:

  • lot_number
  • lot_size

4.2.682.1 lot_number

The lot_number specifies the string for the alphanumeric identification of the group of Ee_product that compose a lot for the Planned_lot_effectivity.

4.2.682.2 lot_size

The lot_size specifies the quantity of Ee_product as a Data_element within the lot with the Planned_lot_effectivity.

Formal constraints:

WR1: The lot_size shall be an integer greater than zero.
