4.2.133 Component_part_2d_geometric_representation_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_part_2d_geometric_representation_relationship is a type of Geometric_representation_relationship. A Component_part_2d_geometric_representation_relationship is the association between a Part_template_planar_shape or a Physical_unit_planar_shape and the Assembly_component_2d_shape that is the occurrence of that planar shape. If the type of the shape representation of the assembly is different from the type of the shape representation of the part, the Component_part_2d_geometric_representation_relationship allows a processor to create a shape rep of the assembly type and establish an equivalency relationship between that shape representation and the part shape representation. If the shape representation of the assembly is of the same type as the part shape representation, the Component_part_2d_geometric_representation_relationship shall reference the same shape representation as both the part_shape and the component_shape. That instance shall be a complex instance of the part shape and of the component shape.

The data associated with this application object are the following:

  • component_shape
  • part_shape

4.2.133.1 component_shape

The component_shape specifies the role of the Assembly_component_2d_shape for the Component_part_2d_geometric_representation_relationship.

4.2.133.2 part_shape

The part_shape specifies either the Part_template_planar_shape or the Physical_unit_planar_shape for the Component_part_2d_geometric_representation_relationship.
