4.2.648 Physical_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_component is a type of Assembly_component. A Physical_component is one of a Cable_component, a Packaged_component, an Interconnect_module_component, an Assembly_module_component, or a Bare_die_component, or additionally may be an Interface_component.
