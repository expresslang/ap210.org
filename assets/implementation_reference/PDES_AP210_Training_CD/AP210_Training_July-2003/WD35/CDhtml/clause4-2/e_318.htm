4.2.318 Ee_material

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_material is a type of Managed_design_object. An Ee_material is the tangible nature of a manufactured or natural product.

The data associated with this application object are the following:

  • material_name
  • registration_document

4.2.318.1 material_name

The material_name specifies the descriptive label that is registered for the Ee_material in the registration_document, and that represents the name of the Ee_material.

4.2.318.2 registration_document

The registration_document specifies a dictionary of registered materials in which the material_name is found.

4.2.318.3 electrical_conductivity_classification

The electrical_conductivity_classification specifies either conductive, non_conductive, resistive, semi_conductive. The electrical_conductivity_classification need not be specified for a particular Ee_material.

4.2.318.4 optical_conductivity_classification

The optical_conductivity_classification specifies either conductive, non_conductive, resistive, or semi_conductive. The optical_conductivity_classification need not be specified for a particular Ee_material.

4.2.318.5 thermal_conductivity_classification

The thermal_conductivity_classification specifies either conductive, non_conductive, resistive, or semi_conductive. The thermal_conductivity_classification need not be specified for a particular Ee_material.

Formal constraints:

UR1: The combination of material_name and registration_document shall be unique within a population of Ee_materials.

WR1: At least one of the following shall be provided: electrical_conductivity_classification, optical_conductivity_classification, thermal_conductivity_classification.
