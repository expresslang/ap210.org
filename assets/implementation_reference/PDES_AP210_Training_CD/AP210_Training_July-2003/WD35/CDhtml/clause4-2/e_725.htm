4.2.725 Printed_connector_component_interface_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_connector_component_interface_terminal is a type of Component_feature. The Printed_connector_component_interface_terminal is an individual access mechanism (e.g., one terminal on a Printed_connector_component). "Interface" means it is not tied to the substrate as part of the fabrication or assembly operation.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.725.1 associated_component

The associated_component specifies the role of the Printed_connector_component for the Printed_connector_component_interface_terminal.

4.2.725.2 definition

The definition specifies the role of the Printed_part_template_interface_terminal for the Printed_connector_component_interface_terminal.

NOTE: Use of this capability occurs when a member of Component_feature_to_physical_usage_view_assignment references a member of Printed_connector_component_interface_terminal.
