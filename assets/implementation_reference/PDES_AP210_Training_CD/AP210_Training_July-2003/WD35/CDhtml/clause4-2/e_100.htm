4.2.100 Characteristic_data_table

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Characteristic_data_table is a type of Compound_characteristic. A Characteristic_data_table describes data as a sequence of columns, where the elements in each successive column in the table are correlated according to the position of the column in the sequence of columns. The recursive construction shall terminate in a simple Characteristic Application object.
