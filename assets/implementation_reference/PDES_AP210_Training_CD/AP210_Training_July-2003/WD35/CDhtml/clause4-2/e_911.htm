4.2.911 Tolerance_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_characteristic is a type of Coordinated_characteristic. A Tolerance_characteristic may be one of Maximum_tolerance_characteristic, Minimum_tolerance_characteristic, Plus_minus_tolerance_characteristic, Statistical_tolerance_characteristic, or Symmetrical_tolerance_characteristic.

The data associated with this application object are the following:

  • component_characteristic

4.2.911.1 component_characteristic

The component_characteristic specifies the role of the Numeric_parameter for the Tolerance_characteristic. There shall be one or more Numeric_parameter for the Tolerance_characteristic.

NOTE: Members of this application object are not exchanged to support geometric tolerancing.
