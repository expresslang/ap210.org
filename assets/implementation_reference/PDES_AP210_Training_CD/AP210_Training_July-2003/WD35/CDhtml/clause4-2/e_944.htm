4.2.944 User_defined_dimension_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A User_defined_dimension_qualifier is a type of Dimension_qualifier that requires a more detailed description than those stated for a Pre_defined_dimension_qualifier.

The data associated with this application object are the following:

  • description
  • name

4.2.944.1 description

The description specifies the text that describes the manner in which the Dimensional_characteristic that are specified as the qualified_dimensions are qualified by the User_defined_dimension_qualifier.

NOTE: The attribute qualified_dimensions is declared in the Dimension_qualifier entity of which this entity is a subtype.

4.2.944.2 name

The name specifies the label for the User_defined_dimension_qualifier by which it is referred.
