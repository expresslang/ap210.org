4.2.287 Diametrical_boundary_size_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Diametrical_boundary_size_characteristic is a type of Boundary_size_characteristic that defines the diameter of a Tolerance_zone_circular_or_cylindrical_or_spherical_boundary.

The data associated with this application object are the following:

  • sized_boundary

4.2.287.1 sized_boundary

The sized_boundary specifies the Tolerance_zone_circular_or_cylindrical_or_spherical_boundary, the diameter of which is defined by the Diametrical_boundary_size_characteristic.

Formal constraints:

WR1: The Tolerance_zone_boundary specified by the sized_boundary shall be the same instance of Tolerance_zone_boundary as specified by the defined_zone_boundary of the Boundary_zone_definition_with_specified_size that is specified as the boundary_definition.

NOTE: The attribute boundary_definition is declared in the Boundary_size_characteristic entity of which this entity is a subtype.
