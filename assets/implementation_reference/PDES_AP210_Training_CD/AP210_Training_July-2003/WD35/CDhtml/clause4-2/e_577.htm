4.2.588 Packaged_connector_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Packaged_connector_component is a type of Packaged_component that is an occurrence of a Packaged_connector.

EXAMPLE: Figure pcc illustrates a member of Packaged_connector_component that is a socket.

The data associated with this application object are the following:

  • interface_access_mechanism
  • usage_definition

4.2.588.1 interface_access_mechanism

The interface_access_mechanism is an inverse relationship that specifies that the existence of the Packaged_connector_component is dependent on the existence of the Packaged_connector_component_interface_terminal that specifies the Packaged_connector_component as its associated_component. There shall be one or more Packaged_connector_component_interface_terminal for a Packaged_connector_component.

4.2.588.2 usage_definition

The usage_definition specifies the role of the Packaged_connector for the Packaged_connector_component.
