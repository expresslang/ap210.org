4.2.285 Device_terminal_map

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Device_terminal_map is a type of Managed_design_object. A Device_terminal_map is the association between a Part_device_terminal and a Packaged_part_terminal. A Device_terminal_map provides the topological equivalent of a bond wire from the package terminal to the terminal of the installed Part_device.

The data associated with this application object are the following:

  • mapped_device_terminal
  • target_terminal

4.2.285.1 mapped_device_terminal

The mapped_device_terminal specifies the Part_device_terminal from which the association is made to the target_terminal by the Device_terminal_map.

4.2.285.2 target_terminal

The target_terminal specifies the targeted Packaged_part_terminal for the Device_terminal_map.

Formal constraints:

UR1: The combination of the target_terminal and mapped_device_terminal shall be unique within a population of Device_terminal_map.

WR1: The mapped_device_terminal and the target_terminal shall be in the context of the same Packaged_part.

NOTE: Reference Part_device_terminal_model_port_assignment for Analytical_model_port_assignment.
