4.2.94 Certification

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Certification is a type of Managed_design_object. A Certification is a granting of approval for use of a Supplied_product_version.

The data associated with this application object are the following:

  • certification_approval
  • certification_date
  • certification_number
  • certified_product_version

4.2.94.1 certification_approval

The certification_approval specifies the Ee_approval of the Certification.

4.2.94.2 certification_date

The certification_date specifies either the Date or the Date_and_time for the Certification.

4.2.94.3 certification_number

The certification_number specifies the Ee_text that identifies the Certification.

4.2.94.4 certified_product_version

The certified_product_version specifies the Supplied_product_version for the subject of the Certification.

Formal constraints:

UR1: The certification_number shall be unique within a population of Certifications.

NOTE: 
*)
ENTITY certification
  SUBTYPE OF (managed_design_object);
    certification_approval    : ee_approval;
    certification_number      : ee_text;
    certified_product_version : supplied_product_version;
    certification_date        : date_or_date_and_time;
  UNIQUE
    ur1 : certification_number;
END_ENTITY; -- certification
