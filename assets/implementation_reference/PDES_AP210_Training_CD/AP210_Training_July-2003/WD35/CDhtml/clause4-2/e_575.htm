4.2.586 Packaged_component_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Packaged_component_join_terminal is a type of Physical_component_terminal. A Packaged_component_join_terminal may be a Movable_packaged_component_join_terminal. A Packaged_component_join_terminal is one termination for a Packaged_component. The terminal provides access to the electrical functionality of the packaged part being terminated.

EXAMPLE: Figure pacjt illustrates a detail of a assembly drawing that shows a microcircuit plugged into a socket, and the role of the Packaged_component_join_terminal.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.586.1 associated_component

The associated_component specifies the Packaged_component being accessed by the Packaged_component_join_terminal.

4.2.586.2 definition

The definition specifies the role of the Packaged_part_join_terminal for the Packaged_component_join_terminal.
