4.2.653 Physical_connectivity_definition_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_connectivity_definition_relationship is a type of Managed_design_object. A Physical_connectivity_definition_relationship is the association between Physical_connectivity_definition that exists because of a Requirement_occurrence.

NOTE: The purpose of the Physical_connectivity_definition_relationship is documented in the Requirement_occurrence, and is typically life cycle dependent, based on design iterations, or variations.

The data associated with this application object are the following:

  • current_definition
  • previous_definition
  • requirement

4.2.653.1 current_definition

The current_definition specifies one role of the Physical_connectivity_definition for the Physical_connectivity_definition_relationship.

4.2.653.2 previous_definition

The previous_definition specifies one role of the Physical_connectivity_definition for the Physical_connectivity_definition_relationship.

4.2.653.3 requirement

The requirement specifies the Ee_requirement_occurrence that causes the Physical_connectivity_definition_relationship to be in existence between the current Physical_connectivity_definition and the previous Physical_connectivity_definition.
