4.2.914 Tolerance_zone_and_viewing_plane_orienting_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_zone_and_viewing_plane_orienting_relationship is a type of Managed_design_object. A Tolerance_zone_and_viewing_plane_orienting_relationship is a relationship that associates a Tolerance_zone with the Viewing_plane by which it is oriented.

The data associated with this application object are the following:

  • oriented_tolerance_zone
  • orienting_viewing_plane
  • zone_orientation

4.2.914.1 oriented_tolerance_zone

The oriented_tolerance_zone specifies the Tolerance_zone that is oriented with respect to the Viewing_plane specified as the orienting_viewing_plane.

4.2.914.2 orienting_viewing_plane

The orienting_viewing_plane specifies the Viewing_plane by which the Tolerance_zone that is specified as the oriented_tolerance_zone is oriented.

4.2.914.3 zone_orientation

The zone_orientation is an inverse relationship that specifies that the existence of the Tolerance_zone_and_viewing_plane_orienting_relationship is dependent on the existence of the Viewing_plane_based_orientation that specifies the Tolerance_zone_and_viewing_plane_orienting_relationship as its orienting_relationship.
