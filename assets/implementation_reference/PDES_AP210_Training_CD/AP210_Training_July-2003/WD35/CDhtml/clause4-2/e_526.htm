4.2.526 Marking

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Marking is a type of Managed_design_object. A Marking is the specification of what, where, and how information that is placed on a Physical_unit for identification purposes is applied. Marking in this part of ISO 10303 is considered to not be part of the definition of a Stratum. Marking is text that is applied to a material object. The message_representation_text shall be an item in the message_representation.

EXAMPLE: Laminate_text_component is text that is realized through the manufacturing process. Reference designators that are created on the board during etching are members of Laminate_text_component.

The data associated with this application object are the following:

  • marked_device_shape
  • marked_message
  • marking_material
  • marking_specification
  • message_representation
  • message_representation_text
  • placement

4.2.526.1 marked_device_shape

The marked_device_shape specifies either a Physical_unit_3d_shape, or a Physical_unit_planar_shape that is the shape definition for the physical device that receives the Marking.

4.2.526.2 marked_message

The marked_message specifies either a Configuration_controlled_message or Structured_text for Marking.

4.2.526.3 marking_material

The marking_material specifies the Ee_material used in the Marking.

4.2.526.4 marking_specification

The marking_specification specifies the Ee_specification of the process and format for properly applying the Marking.

4.2.526.5 message_representation

The message_representation specifies role of the Feature_shape_definition for the Marking.

4.2.526.6 message_representation_text

The message_representation_text specifies role of the Styled_text for the Marking.

4.2.526.7 placement

The placement specifies the Axis_placement of the Marking. The placement specifies the placement of the message_representation in the marked_device_shape.

ENTITY marking
  SUBTYPE OF (managed_design_object);
    marked_message         : marked_message_select;
    marked_device_shape    : view_shape_select;
    marking_material       : ee_material;
    placement              : axis_placement;
    message_representation : feature_shape_definition;
    message_representation_text : styled_text;
    marking_specification  : ee_specification;
WHERE
 WR1: SIZEOF(QUERY(st <* 
       message_representation\shape_definition.element | 
       st = message_representation_text)) = 1;
 WR2: SIZEOF(QUERY(ap <* 
       marked_device_shape\shape_definition.element | 
       ap = placement)) = 1;
 WR3: SIZEOF(TYPEOF(marked_device_shape) * 
      ['AP210_ARM.PHYSICAL_UNIT_3D_SHAPE',
       'AP210_ARM.PHYSICAL_UNIT_PLANAR_SHAPE'])= 1;
END_ENTITY; -- marking
