4.2.12 Alternate_select_product

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Alternate_select_product is a type of Managed_design_object. An Alternate_select_product is one of an Alternate_product, an Assembly_alternate_product, or a Test_select_product. An Alternate_select_product specifies interchangeability relations at the product level. Form, fit and function evaluation by an enterprise is used to inform the decision to establish the relationship.

The data associated with this application object are the following:

  • authorizing_agreement
  • defined_alternate_product
  • defining_base_product

4.2.12.1 authorizing_agreement

The authorizing_agreement specifies the Contract under which the Ee_product was designed and procured for the Alternate_select_product. The authorizing_agreement need not be specified for a particular Alternate_select_product.

4.2.12.2 defined_alternate_product

The defined_alternate_product specifies one role of the Ee_product for the Alternate_select_product. There shall be one or more Ee_product for the Alternate_select_product.

4.2.12.3 defining_base_product

The defining_base_product specifies one role of the Ee_product for the Alternate_select_product. The defining_base_product is the reference with respect to form, fit, and function.
