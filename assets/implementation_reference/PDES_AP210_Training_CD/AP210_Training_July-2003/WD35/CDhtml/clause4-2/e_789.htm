4.2.789 Rule_priority

A Rule_priority is a type of Managed_design_object. A Rule_priority is a statement that an Ee_product that is in the rule category has a priority value. This value is used to order rules. The priority is applied to the rule through the Ee_rule_definition. It is not the Ee_rule_definition that has the priority.

The data associated with this application object are the following:

  • prioritized_rule
  • priority

4.2.789.1 prioritized_rule

The prioritized_rule specifies the role of the Ee_rule_definition for the Rule_priority.

4.2.789.2 priority

The priority specifies the integer for the priority number of the Rule_priority. The lowest numeric value shall have the highest priority.

Formal constraints:

WR1: A number less than zero shall not be allowed.

ENTITY rule_priority
  SUBTYPE OF (managed_design_object);
    priority         : INTEGER;

  WR1: priority >= 0;
