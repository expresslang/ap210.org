4.2.612 Part_feature_placement_link

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_feature_placement_link is a type of a Managed_design_object.

The data associated with this application object are the following:

  • precedent_element
  • referenced_by
  • subsequent_element

4.2.612.1 precedent_element

The precedent_element specifies a Part_feature for the Part_feature_placement_link.

4.2.612.2 referenced_by

The referenced_by is an inverse relationship that specifies that the existence of the Part_feature_placement_link is dependent on the existence of the Part_linear_array_feature that specifies the Part_feature_placement_link as its element.

4.2.612.3 subsequent_element

The subsequent_element specifies a Part_feature for the Part_feature_placement_link.
