4.2.850 Stratum_concept_to_physical_usage_view_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_concept_to_physical_usage_view_assignment is a type of Managed_design_object. A Stratum_concept_to_physical_usage_view_assignment is the relationship between a Part_feature and any one of an Inter_stratum_feature, a Laminate_component, a Stratum, Stratum_feature, or a Stratum_surface for the purpose of traceability between the Part_feature and the underlying design feature.

The data associated with this application object are the following:

  • assigned_design_object
  • defined_part_feature

4.2.850.1 assigned_design_object

The assigned_design_object specifies any one of the Inter_stratum_feature, Laminate_component, Stratum, Stratum_feature, or Stratum_surface as the assigned object for the Stratum_concept_to_physical_usage_view_assignment.

4.2.850.2 defined_part_feature

The defined_part_feature specifies the role of the Part_feature for the Stratum_concept_to_physical_usage_view_assignment.
