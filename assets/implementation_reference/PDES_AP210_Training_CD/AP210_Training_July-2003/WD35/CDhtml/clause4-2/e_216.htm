4.2.216 Datum_axis_and_tolerance_zone_orienting_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_axis_and_tolerance_zone_orienting_relationship is a type of Managed_design_object. A Datum_axis_and_tolerance_zone_orienting_relationship is a relationship that associates a Tolerance_zone with the Datum_axis by which it is oriented.

The data associated with this application object are the following:

  • oriented_tolerance_zone
  • orienting_datum_axis
  • zone_orientation

4.2.216.1 oriented_tolerance_zone

The oriented_tolerance_zone specifies the Tolerance_zone that is oriented with respect to the Datum_axis specified as the orienting_datum_axis.

4.2.216.2 orienting_datum_axis

The orienting_datum_axis specifies the Datum_axis by which the Tolerance_zone that is specified as the oriented_tolerance_zone is oriented.

4.2.216.3 zone_orientation

The zone_orientation is an inverse relationship that specifies that the existence of the Datum_axis_and_tolerance_zone_orienting_relationship is dependent on the existence of the Datum_axis_related_orientation that specifies the Datum_axis_and_tolerance_zone_orienting_relationship as its orienting_relationship.
