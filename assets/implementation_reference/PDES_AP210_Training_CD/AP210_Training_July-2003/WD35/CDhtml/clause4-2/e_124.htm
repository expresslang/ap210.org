4.2.124 Component_group_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_group_assignment is a type of Managed_design_object. A Component_group_assignment is the relationship between an Assembly_component and the Placement_group to which it is assigned. A Component_group_assignment is an incomplete concept by itself and is intended for use with a Placement_group_area_assignment or a Component_placement_restriction_assignment.

The data associated with this application object are the following:

  • assigned_component
  • associated_group

4.2.124.1 assigned_component

The assigned_component specifies the Assembly_component in the Component_group_assignment.

4.2.124.2 associated_group

The associated_group specifies the Placement_group in the Component_group_assignment.
