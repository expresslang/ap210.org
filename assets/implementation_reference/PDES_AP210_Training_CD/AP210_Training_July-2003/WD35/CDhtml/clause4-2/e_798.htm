4.2.798 Runout_zone_orientation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Runout_zone_orientation is a type of Managed_design_object. A Runout_zone_orientation is a definition that characterizes the orientation between a Tolerance_zone and a Directed_axis that is coincident with the Datum_axis about which the Tolerance_zone is centred.

The data associated with this application object are the following:

  • oriented_relationship
  • semi_angle_value
  • zone_definition

4.2.798.1 oriented_relationship

The oriented_relationship specifies the Runout_tolerance_zone_orienting_relationship that associates the Tolerance_zone with the Directed_axis with which the Tolerance_zone is oriented, and that is characterized by the Runout_zone_orientation.

4.2.798.2 semi_angle_value

The semi_angle_value specifies the Angle_data_element that represents the size of the angle between the Tolerance_zone and the Directed_axis.

4.2.798.3 zone_definition

The zone_definition is an inverse relationship that specifies that the existence of the Runout_zone_orientation is dependent on the existence of the Runout_tolerance_zone_definition_with_specified_angle that specifies the Runout_zone_orientation as its runout_orientation.

ENTITY runout_zone_orientation
  SUBTYPE OF (managed_design_object);
    oriented_relationship : runout_tolerance_zone_orienting_relationship;
    semi_angle_value      : angle_data_element;
  INVERSE
    zone_definition : runout_tolerance_zone_definition_with_specified_angle FOR 
                      runout_orientation;
END_ENTITY; -- runout_zone_orientation
