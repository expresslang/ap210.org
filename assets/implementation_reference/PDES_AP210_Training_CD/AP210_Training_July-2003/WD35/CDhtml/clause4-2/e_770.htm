4.2.770 Requirement_composition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Requirement_composition is a type of Managed_design_object. A Requirement_composition helps to compose an aggregation of Ee_requirement_occurrence into a group in order to construct a requirements structure.

EXAMPLES:

  1. A requirements table is a typical example of a requirements structure.
  2. A Requirement_composition would be used to convey an aggregate requirement that is the result of combining two requirements.

The data associated with this application object are the following:

  • composition
  • element

4.2.770.1 composition

The composition specifies the Requirement_group_occurrence for the aggregation for the Requirement_composition.

4.2.770.2 element

The element specifies the role of the Ee_requirement_occurrence for the Requirement_composition. There shall be one or more Ee_requirement_occurrence for a Requirement_composition.
