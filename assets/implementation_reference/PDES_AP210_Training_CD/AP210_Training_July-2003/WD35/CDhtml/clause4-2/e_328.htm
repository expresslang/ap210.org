4.2.328 Ee_product_version

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_product_version is a type of Managed_design_object. An Ee_product_version is the identification of the representation of a product after its design has reached a point of control or has undergone a formal release or change.

The data associated with this application object are the following:

  • authorizing_agreement
  • creator
  • product_definition
  • product_version_approval
  • revision_code
  • security_code
  • versioned_item

4.2.328.1 authorizing_agreement

The authorizing_agreement specifies the role of the Contract for the Ee_product_version. The authorizing_agreement need not be provided for a specific Ee_product_version.

4.2.328.2 creator

The creator specifies the Organization or the Person_and_organization responsible for the Ee_product_version.

4.2.328.3 product_definition

The product_definition is an inverse relationship that specifies that the existence of the Ee_product_version is dependent on the existence of the Ee_product_definition that specifies the Ee_product_version as its version. There shall be one or more Ee_product_definition for an Ee_product_version.

4.2.328.4 product_version_approval

The product_version_approval specifies the Ee_approval associated with the Ee_product_version.

4.2.328.5 revision_code

The revision_code specifies the string identifying an Ee_product_version.

4.2.328.6 security_code

The security_code specifies the Security_classification of an Ee_product_version and is used to control access to design and other information about the Ee_product_version. The security_code is optional and need not be supplied.

4.2.328.7 versioned_item

The versioned_item specifies the Ee_product of which the Ee_product_version identifies the version.

Formal constraints:

UR1: The combination of the versioned_item and the revision_code shall be unique within a population of Ee_product_versions.

ENTITY ee_product_version
  SUPERTYPE OF (ee_rule_version)
  SUBTYPE OF (managed_design_object);
    creator                  : person_organization;
    authorizing_agreement    : OPTIONAL contract;
    revision_code            : ee_name;
    versioned_item           : ee_product;
    security_code            : OPTIONAL security_classification;
    product_version_approval : ee_approval;
  INVERSE
    product_definition : SET [1:?] OF ee_product_definition FOR version;
  UNIQUE
    ur1 : versioned_item, revision_code;
END_ENTITY; -- ee_product_version
