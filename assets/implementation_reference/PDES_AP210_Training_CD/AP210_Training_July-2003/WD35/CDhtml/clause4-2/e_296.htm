4.2.291 Dimension_along_curve

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimension_along_curve is a type of Dimensional_location that defines the location of a Shape_element in relationship to another Shape_element and in which a Shape_element is specified as the path along which the Dimension_along_curve characterizes this relationship.

The data associated with this application object are the following:

  • path

4.2.291.1 path

The path specifies the Shape_element along which the relationship between the two Shape_elements is characterized by the Dimension_along_curve.

Formal constraints:

UR1: The combination of located_elements inherited from Dimensional_location and path shall be unique within a population of Dimension_along_curve.

WR1: The dimension_type of the Dimension_location_with_direction_vector shall be either "arc_length", or "linear".

NOTE: The attribute dimension_type is declared in the Dimensional_characteristic entity of which this entity is a subtype.
