4.2.940 Unsupported_passage_dependent_non_functional_land

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Unsupported_passage_dependent_non_functional_land is a type of Non_functional_land whose shape is dependent upon the shape of an Unsupported_passage.

The data associated with this application object are the following:

  • reference_passage
  • usage_definition

4.2.940.1 reference_passage

The reference_passage specifies the role of the Unsupported_passage for the Unsupported_passage_dependent_non_functional_land.

4.2.940.2 usage_definition

The usage_definition specifies the role of the Default_unsupported_passage_based_land_physical_template for the Unsupported_passage_dependent_non_functional_land.
