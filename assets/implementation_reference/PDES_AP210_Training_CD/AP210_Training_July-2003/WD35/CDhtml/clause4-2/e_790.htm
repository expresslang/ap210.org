4.2.790 Rule_rejection

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_rejection is a type of Rule_action that states a version of an Ee_rule has been rejected. The rejection rationale shall be part of the Rule_rejection.

The data associated with this application object are the following:

  • rejection_reason

4.2.790.1 rejection_reason

The rejection_reason specifies the human-interpretable explanation for the Rule_rejection.

ENTITY rule_rejection
  SUBTYPE OF (rule_action);
    rejection_reason : STRING;
