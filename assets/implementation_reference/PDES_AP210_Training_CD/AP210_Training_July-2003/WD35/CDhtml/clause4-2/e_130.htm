4.2.130 Component_non_planar_location

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_non_planar_location is a type of Managed_design_object. A Component_non_planar_location is the translation and orientation in three dimensions for the positioning of the Assembly_component. This Application Object requires Component_assembly_3d_position for completeness.

The data associated with this application object are the following:

  • transformation
  • placement_fixed

4.2.130.1 transformation

The transformation specifies the role of the Axis_placement_3d for the Component_non_planar_location.

4.2.130.2 placement_fixed

The placement_fixed specifies whether or not the Component_non_planar_location is fixed. A value of TRUE indicates that the Component_non_planar_location is fixed. A value of FALSE indicates that the Component_non_planar_location is not fixed.
