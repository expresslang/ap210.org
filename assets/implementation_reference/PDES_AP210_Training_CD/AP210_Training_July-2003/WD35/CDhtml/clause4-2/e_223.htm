4.2.223 Datum_difference_based_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_difference_based_characteristic is a type of Characteristic. A Datum_difference_based_characteristic is provided to allow characterization based on a geometric model. This is typically used where there is a reference geometric model available.

The data associated with this application object are the following:

  • of_datum_difference

4.2.223.1 of_datum_difference

An of_datum_difference specifies the role of the Datum_difference for the Datum_difference_based_characteristic.
