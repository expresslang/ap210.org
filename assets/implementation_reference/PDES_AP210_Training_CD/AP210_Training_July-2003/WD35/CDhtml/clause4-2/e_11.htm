4.2.11 Alternate_product

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Alternate_product is a type of Alternate_select_product that is completely interchangeable with a specified Ee_product with respect to form, fit, and function for an enterprise.

The data associated with this application object are the following:

  • defined_alternate_product

4.2.11.1 defined_alternate_product

The defined_alternate_product specifies the role of the Ee_product for the Alternate_product.
