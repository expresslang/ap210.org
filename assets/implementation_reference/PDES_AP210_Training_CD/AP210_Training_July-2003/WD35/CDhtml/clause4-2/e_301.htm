4.2.296 Dimension_related_positional_boundary_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimension_related_positional_boundary_definition is a type of Positional_boundary_definition that characterizes a Dimension_related_positional_boundary.

The data associated with this application object are the following:

  • characterized_boundary_member_definitions
  • defined_positional_boundary

4.2.296.1 characterized_boundary_member_definitions

The characterized_boundary_member_definitions specifies the Positional_boundary_member_definition that characterize each Positional_boundary_member that comprise the Dimension_related_positional_boundary. There shall be one or more Positional_boundary_member_definition specified as the characterized_boundary_member_definitions for the Dimension_related_positional_boundary_definition.

4.2.296.2 defined_positional_boundary

The defined_positional_boundary specifies the Dimension_related_positional_boundary that is characterized by the Dimension_related_positional_boundary_definition.

Formal constraints:

WR1: Each Boundary_based_position_tolerance that defines a Positional_boundary_member_definition of the Positional_boundary_definition shall reference the same instance of Datum_system_defining_conditions.

WR2: Each Boundary_based_position_tolerance that defines a Positional_boundary_member_definition of the Positional_boundary_definition shall have the same value for the applied_material_condition_property_name.
