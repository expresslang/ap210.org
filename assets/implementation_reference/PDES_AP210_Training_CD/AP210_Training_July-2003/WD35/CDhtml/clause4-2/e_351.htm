4.2.351 Embedded_component_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Embedded_component_terminal is a type of Laminate_join_terminal included in the design to mate directly with a terminal of a fabricated feature by use of a Fabrication_joint internally in a layout. An Embedded_component_terminal is a model specifically included to bridge between the discrete purchased items used in a design and the custom fabricated interconnect patterns in a substrate design. An Embedded_component_terminal is an instance of a usage definition for access to a discrete component in the context of a substrate design. An Embedded_component_terminal is included to support using Physical_network to define the implementation connections to the associated component.

NOTE: The modeling approach used here does not imply that Embedded_component_terminal is created as part of the wet or dry chemistr process used to create the patterns on the substrate. It is merely identifying a functional role of a terminal of a discrete part when it is embedded in a substrate. By implication, it is also identifying the role of the associated component, i.e. that the associated component is embedded in the substrate. The method or process of achieving this is not defined in this standard. The Embedded_component_terminal will not be included in a Physical_connectivity_definition and is treated herein similarly to the Printed_component terminals. A Fabrication_joint is required to connect the terminal to the Land_join_terminal adjacent to satisfy functional connectivity requirements.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.351.1 associated_component

The associated_component specifies the role of the Bare_die_component or the role of the Packaged_component for the Embedded_component_terminal.

4.2.351.2 definition

The definition specifies the role of the Bare_die_terminal or the role of the Packaged_part_terminal for the Embedded_component_terminal.

Formal constraints:

WR1: An Embedded_component_terminal shall participate in a relationship with a Fabrication_joint as a fabrication_features.
