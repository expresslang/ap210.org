4.2.384 Filled_area_material_removal_laminate_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Filled_area_material_removal_laminate_component is a type of Material_removal_laminate_component that indicates that material is to be removed in the realization of the feature that is in the role of design_intent inherited from Material_removal_laminate_component. A Filled_area_material_removal_laminate_component is either an Electrical_isolation_laminate_component or a Thermal_isolation_laminate_component. The requirement shall be interpreted mathematically as a requirement for application of the regularized Boolean subtraction operation. The design_intent inherited from Material_removal_laminate_component shall not be a Conductor.

NOTE: In some cases, the isolation may be accomplished by complete removal, as in a dielectric removal from a dielectric layer or layers.

The data associated with this application object are the following:

  • reference_padstack

4.2.384.1 description

The reference_padstack specifies the role of the Padstack_definition for the Filled_area_material_removal_laminate_component.
