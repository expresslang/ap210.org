4.2.631 Part_terminal_external_reference

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_terminal_external_reference is a type of Managed_design_object. A Part_terminal_external_reference is an unambiguous reference to a Part_terminal or to a terminal associated with a type of Part_template in an external system. A Part_terminal_external_reference specifies the type of the terminal as well as the identification for the terminal.

EXAMPLE: Items in a design definition may need to reference terminals in a library without duplicating the geometric context, and may need to specify terminal types by reference instead of explicit instantiation.

The data associated with this application object are the following:

  • associated_definition
  • feature_name
  • feature_type

4.2.631.1 associated_definition

The associated_definition specifies the role of either a Part_template or a Physical_unit for the Part_terminal_external_reference.

4.2.631.2 feature_name

The feature_name specifies the role of a string as identification for the Part_terminal_external_reference.

4.2.631.3 feature_type

The feature_type specifies the string that identifies the type of the terminal referenced for the Part_terminal_external_reference. The interpretation of this string by a receiving system shall be in accordance with the interpretation of Part_terminal.

Formal constraints:

UR1: Each member of Part_terminal_external_reference shall have a unique combination of attributes.

WR1: The types of terminals are limited to those types defined in this part of ISO 10303.

WR2: If a Part_terminal type is specified for the feature_type, a Physical_unit type shall be specified. If a Part_terminal type is not specified for the feature_type, a Part_template type shall be specified for the associated_definition.

ENTITY part_terminal_external_reference
  SUBTYPE OF (managed_design_object);
    associated_definition : physical_unit_or_part_template;
    feature_name          : ee_name;
    feature_type          : ee_name;
  UNIQUE
    ur1 : associated_definition, feature_name, feature_type;
END_ENTITY; -- part_terminal_external_reference
