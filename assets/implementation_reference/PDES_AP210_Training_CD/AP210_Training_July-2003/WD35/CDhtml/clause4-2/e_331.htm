4.2.331 Ee_rule_complex_clause

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
The Ee_rule_complex_clause is a type of Managed_design_object. An Ee_rule_complex_clause consist of one or more simple clauses that are ANDed together. The AND operator is provided by the SET aggregation in this application object.

The data associated with this application object are the following:

  • conjunctive_content
  • identifier

4.2.331.1 conjunctive_content

The conjunctive_content specifies specifies the Ee_rule_simple_clause for the Ee_rule_complex_clause. There shall be one or more conjunctive_content specified for an Ee_rule_complex_clause.

4.2.331.2 identifier

The identifier specifies the name for the Ee_rule_complex_clause.

Formal constraints:

UR1: The identifier shall be unique within a population of Ee_rule_complex_clause.

ENTITY ee_rule_complex_clause

     conjunctive_content : SET [1:?] OF ee_rule_simple_clause;
     identifier          : STRING ;
 UNIQUE
     ur1 : identifier;
