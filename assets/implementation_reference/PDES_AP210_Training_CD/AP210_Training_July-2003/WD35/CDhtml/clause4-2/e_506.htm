4.2.506 Layout_macro_definition_terminal_to_usage_terminal_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Layout_macro_definition_terminal_to_usage_terminal_assignment is a type of Managed_design_object. A Layout_macro_definition_terminal_to_usage_terminal_assignment is the association between the Printed_part_template_terminal and one of several fabricated features. The purpose for this assignment is to link the product data in the macro definition to the floor planning usage view to support a recursive definition. The Layout_macro_definition_terminal_to_usage_terminal_assignment shall be referenced by one Layout_macro_floor_plan_template.

NOTES:

  1. In a recursive definition, both a macro definition and the macro interface declaration that is the Layout_macro_floor_plan_template are archived together, with this relationship populated as necessary to provide linkages between the fabricated features in the macro definition and the features in the planning view. Without this linkage populated, there is no way to traverse the recursive definition so as to successfully resolve the terminals at each level of recursion.
  2. This facility is used to support transport of CAD data where geometric subroutines that include layer data and terminal data are defined.

The data associated with this application object are the following:

  • assigned_design_object
  • defined_template_feature

4.2.506.1 assigned_design_object

The assigned_design_object specifies the role of the Component_termination_passage_join_terminal or of the Via_terminal or of the Printed_component_join_terminal or of the Conductive_interconnect_element_terminal or of the Land_join_terminal or of the Non_functional_land_join_terminal for the Layout_macro_definition_terminal_to_usage_terminal_assignment.

4.2.506.2 defined_template_feature

The defined_template_feature specifies the role of the Printed_part_template_terminal for the Layout_macro_definition_terminal_to_usage_terminal_assignment.
