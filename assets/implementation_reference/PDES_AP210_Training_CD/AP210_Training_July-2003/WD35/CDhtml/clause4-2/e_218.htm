4.2.218 Datum_based_direction_vector

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_based_direction_vector is a type of Managed_design_object. A Datum_based_direction_vector is a characterization of a direction away from and normal to a Datum_plane.

The data associated with this application object are the following:

  • direction_vector_representation
  • positive_direction_vector
  • vector_of

4.2.218.1 direction_vector_representation

The direction_vector_representation specifies an Axis_placement that is a representation for the Datum_based_direction_vector.

4.2.218.2 positive_direction_vector

The positive_direction_vector specifies a boolean value that indicates whether the Datum_based_direction_vector is in a positive or negative sense. A value of TRUE indicates that the Datum_based_direction_vector is in the positive sense. A value of FALSE indicates that the Datum_based_direction_vector is in the negative sense.

4.2.218.3 vector_of

The vector_of specifies the Datum_plane from which the Datum_based_direction_vector is based.

Formal constraints:

WR1: The Datum_based_direction_vector shall be specified as the direction by at least one Direction_element.
