4.2.222 Datum_difference_analytical_model_port_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_difference_analytical_model_port_assignment is a type of Managed_design_object. A Datum_difference_analytical_model_port_assignment is provided to map from the geometric context into a specific Analytical_model through its ports. The use of Datum_difference isolates the model ports from a specific geometric representation.

The data associated with this application object are the following:

  • assigned_analytical_model_port
  • of_datum_difference

4.2.222.1 assigned_analytical_model_port

An assigned_analytical_model_port specifies the role of the Analytical_model_port for the Datum_difference_analytical_model_port_assignment.

4.2.222.2 of_datum_difference

An of_datum_difference specifies the role of the Datum_difference for the Datum_difference_analytical_model_port_assignment.
