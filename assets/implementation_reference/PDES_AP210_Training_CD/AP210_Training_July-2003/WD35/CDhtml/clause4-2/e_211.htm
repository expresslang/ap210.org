4.2.211 Data_element

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Data_element is a type of Managed_design_object. A Data_element may be an Angle_data_element or a Length_data_element. A Data_element is a quantitative value that includes the measurement units associated with the value. This part of ISO 10303 predefines base units from which user defined un its may be mapped. ISO 10303-41 defines the mechanism for mapping between user defined units and predefined units.

The data associated with this application object are the following:

  • measure_value
  • unit_of_measure

4.2.211.1 measure_value

The measure_value specifies the REAL type for the Data_element. The interpretation of the measure_value shall be in accordance with the unit_of_measure.

4.2.211.2 unit_of_measure

The unit_of_measure specifies the descriptive label for the unit of measure for the associated measure_value of the Data_element. The units may be SI, derived from SI, or context dependent, and may have prefixes included whose syntax is defined in ISO 10303-41 and whose semantic is defined in ISO 31.
