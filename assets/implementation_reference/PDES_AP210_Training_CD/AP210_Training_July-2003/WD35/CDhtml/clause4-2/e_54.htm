4.2.54 Assembly_separation_requirement_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Assembly_separation_requirement_allocation is a type of Assembly_requirement_allocation. An Assembly_separation_requirement_allocation provides for the physical spacing required among components of the assembly. The Assembly_separation_requirement_allocation is for the purpose of specifying separation between items based on the Ee_requirement_occurrence.

NOTE: Multiple members of this entity may be populated in order to specify all the sets with which a given set will be disjoint, either thermally or electromagnetically.

EXAMPLE: A typical usage of this AO is to reference a member of Layout_spacing_requirement_occurrence as the requirement inherited from Assembly_requirement_allocation.

The data associated with this application object are the following:

  • disjoint_assignment

4.2.54.1 disjoint_assignment

The disjoint_assignment specifies either an Assembly_component or a Component_feature for the Assembly_separation_requirement_allocation. There shall be one or more Assembly_component or Component_feature for the Assembly_separation_requirement_allocation. The disjoint_assignment shall be applied pairwise.

Informal constraints:

IP1: The value of Length_data_element shall be specified as a characteristic of the Ee_requirement_occurrence from Assembly_requirement_allocation.
