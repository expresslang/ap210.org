4.2.332 Ee_rule_conclusion_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_rule_conclusion_definition is a type of Managed_design_object. An Ee_rule_conclusion_definition provides the conclusion part of a rule construct.

The data associated with this application object are the following:

  • rule_definition
  • identifier
  • conjunctive_content

4.2.332.1 rule_definition

The rule_definition specifies the Ee_rule_definition for the Ee_rule_conclusion_definition. The rule_definition is the rule being composed by this Ee_rule_conclusion_definition.

4.2.332.2 identifier

The identifier specifies the name for the Ee_rule_conclusion_definition.

4.2.332.3 conjunctive_content

The conjunctive_content specifies specifies the Ee_rule_simple_clause for the Ee_rule_conclusion_definition. There shall be one or more conjunctive_content specified for an Ee_rule_conclusion_definition.

Formal constraints:

UR1: The combination of the rule_definition and the identifier shall be unique within a population of Ee_rule_conclusion_definition.

ENTITY ee_rule_conclusion_definition
 SUBTYPE OF ( managed_design_object );
     rule_definition     : ee_rule_definition;
     identifier          : STRING ;
     conjunctive_content : SET [1:?] OF ee_rule_simple_clause;
 UNIQUE
     ur1 : rule_definition, identifier;
END_ENTITY; -- ee_rule_conclusion_definition
