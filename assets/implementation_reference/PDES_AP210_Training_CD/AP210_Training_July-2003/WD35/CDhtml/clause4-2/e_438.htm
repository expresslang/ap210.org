4.2.445 Interconnect_module_component_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_component_terminal is a type of Component_terminal. An Interconnect_module_component_terminal may be an Interconnect_module_component_stratum_based_terminal. A member of Interconnect_module_component_terminal may be referenced by a member of Assembly_joint to describe the partial or complete implementation of a connection defined as a requirement by a member of Physical_connectivity_definition.

EXAMPLE: Figure imct illustrates the relationship between an Interconnect_module_component_terminal and its mating feature in an assembly.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.445.1 associated_component

The associated_component specifies the role of the Interconnect_module_component for the Interconnect_module_component_terminal.

4.2.445.2 definition

The definition specifies the role of the Interconnect_module_terminal for the Interconnect_module_component_terminal.
