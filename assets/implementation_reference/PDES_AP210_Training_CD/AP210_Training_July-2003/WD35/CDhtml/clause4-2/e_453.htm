4.2.460 Interconnect_module_stratum_assembly_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_stratum_assembly_relationship is a type of Managed_design_object. An Interconnect_module_stratum_assembly_relationship is the composition association between an Interconnect_module and the Stratum helping to compose it.

The data associated with this application object are the following:

  • assembly
  • component
  • reference_designation
  • stratum_3d_position
  • stratum_planar_position

4.2.460.1 assembly

The assembly specifies the role of the Interconnect_module for the Interconnect_module_stratum_assembly_relationship. There shall be one or more Interconnect_module for the Interconnect_module_stratum_assembly_relationship.

4.2.460.2 component

The component specifies the role of the Stratum for the Interconnect_module_stratum_assembly_relationship.

4.2.460.3 reference_designation

The reference_designation specifies the string for the human-interpretable identifier for the Interconnect_module_stratum_assembly_relationship.

4.2.460.4 stratum_3d_position

The stratum_3d_position is an inverse relationship that specifies the Stratum_interconnect_module_3d_position that specifies the Interconnect_module_stratum_assembly_relationship as its stratum_component. There shall be no more than one Stratum_interconnect_module_3d_position for an Interconnect_module_stratum_assembly_relationship.

4.2.460.5 stratum_planar_position

The stratum_planar_position is an inverse relationship that specifies the Stratum_interconnect_module_planar_position that specifies the Interconnect_module_stratum_assembly_relationship as its stratum_component. There shall be no more than one Stratum_interconnect_module_planar_position for an Interconnect_module_stratum_assembly_relationship.

Formal constraints:

UR1: The combination of the assembly and the reference_designation shall be unique within a population of Interconnect_module_stratum_assembly_relationship.

WR1: The members of assembly shall all be for the same product.

WR2: Each member of assembly shall be for a separate product version.

ENTITY interconnect_module_stratum_assembly_relationship
  SUBTYPE OF (managed_design_object);
    assembly              : interconnect_module;
    component             : stratum;
    reference_designation : ee_name;
  INVERSE
    stratum_planar_position : SET [0:1] OF 
                              stratum_interconnect_module_planar_position FOR 
                              stratum_component;
    stratum_3d_position     : SET [0:1] OF 
                              stratum_interconnect_module_3d_position FOR 
                              stratum_component;
  UNIQUE
    ur1 : assembly, reference_designation;
END_ENTITY; -- interconnect_module_stratum_assembly_relationship
