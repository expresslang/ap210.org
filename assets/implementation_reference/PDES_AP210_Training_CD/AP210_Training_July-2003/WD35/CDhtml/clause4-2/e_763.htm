4.2.763 Rectangular_array_placement_group_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rectangular_array_placement_group_component is a type of Array_placement_group. The location_uncertainty refers to originating CAD system geometric uncertainty, not a material processing tolerance.

The data associated with this application object are the following:

  • element
  • initial_row_positive_y_displacement
  • terminus_row_positive_y_displacement

4.2.763.1 element

The element specifies the role of a Linear_array_placement_group_component_link for the Rectangular_array_placement_group_component. There shall be one or more Linear_array_placement_group_component_link for a Rectangular_array_placement_group.

4.2.763.2 initial_row_positive_y_displacement

The initial_row_positive_y_displacement specifies a Length_data_element for the Rectangular_array_placement_group_component.

4.2.763.3 terminus_row_positive_y_displacement

The terminus_row_positive_y_displacement specifies a Length_data_element for the Rectangular_array_placement_group_component.
