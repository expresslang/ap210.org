4.2.777 Routed_interconnect_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Routed_interconnect_component is a type of Assembly_component. The functionality of a Routed_interconnect_component shall be that of a short.

EXAMPLE: An example of a Routed_interconnect_component is a jumper wire. A jumper wire does not have a predefined placement or shape for its terminal. The shape of a jumper wire is not predefined. The predefined shapes in this example would be those shapes defined in a library.

The data associated with this application object are the following:

  • routed_shape

4.2.777.1 routed_shape

The routed_shape specifies the role of the Feature_shape_definition for the Routed_interconnect_component.

Formal constraints:

WR1: All terminals defined for the Routed_interconnect_component shall be equipotential with respect to the same reference potential.
