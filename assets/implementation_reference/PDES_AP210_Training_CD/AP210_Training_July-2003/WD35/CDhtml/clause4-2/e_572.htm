4.2.571 Owner_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Owner_assignment is a type of Managed_design_object. An Owner_assignment is the specification that one or more Owned_design_object are controlled by a specific entity during design activity.

The data associated with this application object are the following:

  • assigned_owner
  • owned_item

4.2.571.1 assigned_owner

An assigned_owner specifies the role of the Person_and_organization for the Owner_assignment.

4.2.571.2 owned_item

The owned_item specifies an Owned_design_object for which the design is owned by the assigned_owner. There shall be one or more Owned_design_object for a Owner_assignment.

Formal constraints:

WR1: Each Owner_assignment shall not be referenced by any Owner_assignment.
