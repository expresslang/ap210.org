4.2.78 Boundary_size_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Boundary_size_characteristic is a type of Managed_design_object. A Boundary_size_characteristic is either a Diametrical_boundary_size_characteristic or an Opposing_boundary_set_size_characteristic. A Boundary_size_characteristic is a definition that characterizes the size of the Tolerance_zone_boundary.

The data associated with this application object are the following:

  • boundary_definition
  • maximum_tolerance_value
  • tolerance_value

4.2.78.1 boundary_definition

The boundary_definition is an inverse relationship that specifies that the existence of the Boundary_size_characteristic is dependent on the existence of the Boundary_zone_definition_with_specified_size that specifies the Boundary_size_characteristic as its specified_boundary_size.

4.2.78.2 maximum_tolerance_value

The maximum_tolerance_value specifies the Length_data_element that represents the maximum size of the Tolerance_zone_boundary. The maximum_tolerance_value need not be specified for the Boundary_size_characteristic.

NOTE: Fig. 6-42 of ASME Y14.5M illustrates the application of a maximum value to a geometric tolerance.

4.2.78.3 tolerance_value

The tolerance_value specifies the Length_data_element that represents the size of the Tolerance_zone_boundary for the Boundary_size_characteristic.

ENTITY boundary_size_characteristic
  ABSTRACT SUPERTYPE OF (ONEOF (diametrical_boundary_size_characteristic,
      opposing_boundary_set_size_characteristic))
  SUBTYPE OF (managed_design_object);
    tolerance_value         : length_data_element;
    maximum_tolerance_value : OPTIONAL length_data_element;
  INVERSE
    boundary_definition : boundary_zone_definition_with_specified_size FOR 
                          specified_boundary_size;
END_ENTITY; -- boundary_size_characteristic
