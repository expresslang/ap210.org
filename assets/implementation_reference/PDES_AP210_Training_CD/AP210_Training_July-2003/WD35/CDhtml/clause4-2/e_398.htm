4.2.398 Functional_terminal_group

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_terminal_group is a type of a Managed_design_object. A Functional_terminal_group is a group of terminals that have common characteristics and can be acted upon as a group.

The data associated with this application object are the following:

  • group_definition
  • group_type
  • name

4.2.398.1 group_definition

The group_definition specifies the string that provides the meaning for the Functional_terminal_group.

4.2.398.2 group_type

The group_type specifies the string that is the class for the Functional_terminal_group.

4.2.398.3 name

The name specifies the string that is the identifier for the Functional_terminal_group.

Formal constraints:

UR1: The name shall be unique within a population of Functional_terminal_group.
