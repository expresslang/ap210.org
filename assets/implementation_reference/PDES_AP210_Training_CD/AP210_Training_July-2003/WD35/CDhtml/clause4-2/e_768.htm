4.2.768 Registered_font

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Registered_font is a type of Managed_design_object. A Registered_font is a font that is registered by an organization and defined within a document. Support shall be provided for font character representation using outline glyphs and font family exchange.

The data associated with this application object are the following:

  • font_code
  • registration_document

4.2.768.1 font_code

The font_code specifies the string for the identifier of the Registered_font. The font_code shall be an entry in the registration_document.

4.2.768.2 registration_document

The registration_document specifies the Ee_document that contains the information about the Registered_font.

Formal constraints:

UR1: The combination of the font_code and the registration_document shall be unique within all members of a population of Registered_font.

ENTITY registered_font

    ur1 : font_code, registration_document;
