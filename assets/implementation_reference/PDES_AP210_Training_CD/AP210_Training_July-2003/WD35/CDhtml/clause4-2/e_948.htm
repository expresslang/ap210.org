4.2.948 Via_and_contact_size_dependent_land

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Via_and_contact_size_dependent_land is a type of Land whose shape is dependent upon a Via as well as the size of a Component terminal.

The data associated with this application object are the following:

  • interface_access_mechanism
  • reference_via
  • usage_definition

4.2.948.1 interface_access_mechanism

The interface_access_mechanism is an inverse relationship that specifies that the existence of the Via_and_contact_size_dependent_land is dependent on the existence of the Land_interface_terminal that specifies the Via_and_contact_size_dependent_land as its associated_component. This relationship is for external access only.

4.2.948.2 reference_via

The reference_via specifies the role of the Via for the Via_and_contact_size_dependent_land.

4.2.948.3 usage_definition

The usage_definition specifies the role of the Default_attachment_size_and_via_based_land_physical_template for the Via_and_contact_size_dependent_land.
