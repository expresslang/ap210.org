4.2.35 Assembly_component_2d_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Assembly_component_2d_shape is a type of Planar_projected_shape. An Assembly_component_2d_shape is the two-dimensional shape of an Assembly_component.

NOTE: The shape is defined by the Physical_unit_planar_shape, or by the equivalent for Part_template, referenced by the location transformation in Component_assembly_2d_position.

The data associated with this application object are the following:

  • shape_characterized_component

4.2.35.1 shape_characterized_component

The shape_characterized_component specifies the role of the Assembly_component for the Assembly_component_2d_shape.
