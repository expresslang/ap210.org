4.2.679 Planned_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Planned_characteristic is a type of Managed_design_object. A Planned_characteristic associates design intent information based on manufacturing capabilities and component characteristics with an allocated requirement. A Planned_characteristic is a set of properties planned to be valid for all occurrences of the designed product under the conditions defined in the allocated requirement.

The data associated with this application object are the following:

  • planned_coordinated_characteristic
  • reference_requirement

4.2.679.1 planned_coordinated_characteristic

The planned_coordinated_characteristic specifies the role of the Coordinated_characteristic for the Planned_characteristic.

4.2.679.2 reference_requirement

The reference_requirement specifies the Assembly_requirement_allocation, Functional_unit_requirement_allocation, or the Requirement_allocation for the Planned_characteristic.

NOTE: This information is part of the design view for archival purposes of the design organization.
