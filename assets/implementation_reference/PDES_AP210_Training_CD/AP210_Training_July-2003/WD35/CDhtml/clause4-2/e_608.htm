4.2.604 Part_array_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_array_feature is a type of Part_group_feature that is an array of Part_feature whose positions are specified as members of an array, and whose tolerances are applied based upon the array nature of Part_array_feature.

NOTE: Tolerances applied to Part_array_feature are specified by an object external to the Part_array_feature.

NOTE: 
*)
ENTITY part_array_feature
  ABSTRACT SUPERTYPE OF (ONEOF (part_linear_array_feature,
      part_rectangular_array_feature))
  SUBTYPE OF (part_group_feature);
END_ENTITY; -- part_array_feature
