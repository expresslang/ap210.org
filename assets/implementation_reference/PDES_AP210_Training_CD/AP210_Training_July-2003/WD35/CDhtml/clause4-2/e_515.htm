4.2.516 Linear_array_placement_group_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Linear_array_placement_group_component is a type of a Array_placement_group. The placement_uncertainty relates to the originating CAD system uncertainty, not a material processing tolerance.

The data associated with this application object are the following:

  • element
  • initial_node_positive_x_displacement
  • terminus_node_positive_x_displacement

4.2.516.1 element

The element specifies the role of an Assembly_component_placement_link for the Linear_array_placement_group_component. There shall be one or more Assembly_component_placement_link for a Linear_array_placement_group_component.

4.2.516.2 initial_node_positive_x_displacement

The initial_node_positive_x_displacement specifies a Length_data_element for the Linear_array_placement_group_component. The initial_node_positive_x_displacement is the offset in the positive x direction from the local origin of the first element of the array.

4.2.516.3 terminus_node_positive_x_displacement

The terminus_node_positive_x_displacement specifies a Length_data_element for the Linear_array_placement_group_component. The terminun_node_positive_x_displacement is the offset in the positive x direction of the last element of the array.
