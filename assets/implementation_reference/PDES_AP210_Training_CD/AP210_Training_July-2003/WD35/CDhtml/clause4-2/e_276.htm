4.2.276 Design_layer_technology

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Design_layer_technology is a type of Stratum_technology. The Design_layer_technology is a technology specification tailored for those materials intended to be used to implement the direct connectivity as described in a design. The same Design_layer_technology may be used in multiple designs for multiple products, since there is no dependence of the Design_layer_technology on the design referencing it.

The data associated with this application object are the following:

  • design_layer_position
  • design_layer_purpose
  • minimum_finished_feature_size
  • minimum_finished_feature_spacing

4.2.276.1 design_layer_purpose

The design_layer_purpose specifies one of the lands_only, other_signal, or power_or_ground purpose of the Design_layer_stratum for the Design_layer_technology.

4.2.276.2 minimum_finished_feature_size

The minimum_finished_feature_size specifies the Length_data_element of the smallest permissible extent of a feature for the Design_layer_technology.

4.2.276.3 minimum_finished_feature_spacing

The minimum_finished_feature_spacing specifies the Length_data_element of the smallest permissible distance between features for the Design_layer_technology.
