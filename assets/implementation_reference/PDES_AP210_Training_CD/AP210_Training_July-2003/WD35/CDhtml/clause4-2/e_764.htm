4.2.764 Reference_composition_path

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Reference_composition_path is a type of Managed_design_object. A Reference_composition_path is one path through the function reference network graph.

The data associated with this application object are the following:

  • path_element
  • reference_functional_unit

4.2.764.1 path_element

The path_element specifies the role of the Functional_unit for the Reference_composition_path. There shall be one or more Functional_unit for a Reference_composition_path.

4.2.764.2 reference_functional_unit

The reference_functional_unit specifies the Functional_unit that participates as a link to the leaf node in the functional decomposition hierarchical definition in the Reference_composition_path.
