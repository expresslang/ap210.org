4.2.517 Linear_array_placement_group_component_link

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Linear_array_placement_group_component_link is a type of Managed_design_object.

The data associated with this application object are the following:

  • precedent_element
  • referenced_by
  • subsequent_element

4.2.517.1 precedent_element

The precedent_element specifies a Linear_array_placement_group_component for the Linear_array_placement_group_component_link.

4.2.517.2 referenced_by

The referenced_by is an inverse relationship that specifies that the existence of the Linear_array_placement_group_component_link is dependent on the existence of the Rectangular_array_placement_group_component that specifies the Linear_array_placement_group_component_link as its element.

4.2.517.3 subsequent_element

The subsequent_element specifies a Linear_array_placement_group_component for the Linear_array_placement_group_component_link.
