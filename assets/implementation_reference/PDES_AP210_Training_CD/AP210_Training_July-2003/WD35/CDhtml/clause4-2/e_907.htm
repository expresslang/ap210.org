4.2.907 Thermal_requirement_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Thermal_requirement_allocation is a type of Assembly_requirement_allocation for which the Ee_requirement_occurrence has thermal characteristics. A Thermal_requirement_allocation is the relationship between the Thermal_requirement_occurrence and the Thermal_component that satisfies it.

The data associated with this application object are the following:

  • assembly_assignment
  • requirement

4.2.907.1 assembly_assignment

The assembly_assignment specifies the role of the Thermal_component for the Thermal_requirement_allocation.

4.2.907.2 requirement

The requirement specifies the role of the Thermal_requirement_occurrence for the Thermal_requirement_allocation.
