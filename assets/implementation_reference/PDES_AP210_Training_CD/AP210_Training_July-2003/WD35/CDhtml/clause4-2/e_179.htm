4.2.179 Conical_tolerance_zone_boundary_and_surface_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Conical_tolerance_zone_boundary_and_surface_relationship is a type of Managed_design_object. A Conical_tolerance_zone_boundary_and_surface_relationship is a relationship that associates a Conical_tolerance_zone_boundary with one of the two Physical_feature_or_part_template at which the Conical_tolerance_zone_boundary diameter is specified.

The data associated with this application object are the following:

  • size_characteristic
  • sized_tolerance_zone_boundary
  • terminating_surface

4.2.179.1 size_characteristic

The size_characteristic is an inverse relationship that specifies that the existence of the Conical_tolerance_zone_boundary_and_surface_relationship is dependent on the existence of the Conical_tolerance_zone_boundary_diametrical_size_characteristic that specifies the Conical_tolerance_zone_boundary_and_surface_relationship as its sized_boundary_at_surface.

4.2.179.2 sized_tolerance_zone_boundary

The sized_tolerance_zone_boundary specifies the Conical_tolerance_zone_boundary the diameter of which is specified at the Physical_feature_or_part_template specified as the terminating_surface.

4.2.179.3 terminating_surface

The terminating_surface specifies the Physical_feature_or_part_template at which the diameter is specified for the Conical_tolerance_zone_boundary.

  SUBTYPE OF (managed_design_object);
    sized_tolerance_zone_boundary : conical_tolerance_zone_boundary;
    terminating_surface           : physical_feature_or_part_template;
  INVERSE
    size_characteristic : 
	conical_tolerance_zone_boundary_diametrical_size_characteristic FOR 
	sized_boundary_at_surface;
END_ENTITY; -- conical_tolerance_zone_boundary_and_surface_relationship
