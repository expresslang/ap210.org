4.2.182 Conical_tolerance_zone_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Conical_tolerance_zone_definition is a type of Tolerance_zone_definition the boundary of which is defined by a Conical_tolerance_zone_boundary_definition. A Conical_tolerance_zone_definition is defined when it is desired to place a more restrictive specification on one end of a Shape_element than on the other end.

NOTE: Conical tolerance zones are described in 5.8 of ASME Y14.5M.

The data associated with this application object are the following:

  • boundary_definition

4.2.182.1 boundary_definition

The boundary_definition specifies the Conical_tolerance_zone_boundary_definition that defines the boundary for the Conical_tolerance_zone_definition.

Formal constraints:

WR1: The Conical_tolerance_zone_definition shall not be specified as the tolerance_zone_or_boundary_definition by a Geometric_tolerance.
