4.2.680 Planned_date_effectivity

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Planned_date_effectivity is a type of Planned_effectivity. A Planned_date_effectivity is the specification by the design organization of the intended use of an Ee_product or of an Ee_product within an Ee_product_configuration, determined by one or two dates.

The data associated with this application object are the following:

  • end_date
  • start_date

4.2.680.1 end_date

The end_date specifies either the final Date or the final Date_and_time upon which the Ee_product is planned by a design organization to be used in the Ee_product_configuration, as defined by the Planned_date_effectivity. The end_date need not be specified for a particular Planned_date_effectivity.

4.2.680.2 start_date

The start_date specifies either the initial Date or the initial Date_and_time; upon which the Ee_product is planned by a design organization to be used in the Ee_product_configuration, as defined by the Planned_date_effectivity.
