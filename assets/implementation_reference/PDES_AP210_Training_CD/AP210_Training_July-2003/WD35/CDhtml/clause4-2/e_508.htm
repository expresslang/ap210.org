4.2.508 Layout_spacing_requirement_non_conforming_design_object

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Layout_spacing_requirement_non_conforming_design_object is a type of Managed_design_object. A Layout_spacing_requirement_non_conforming_design_object specifies one or more design object that does not meet object to object spacing requirements for the printed circuit board layout.

The data associated with this application object are the following:

  • design_requirement
  • non_conformant_design_object
  • reference_design_object

4.2.508.1 design_requirement

The design_requirement specifies the role of the Layout_spacing_requirement_occurrence for the Layout_spacing_requirement_non_conforming_design_object

4.2.508.2 non_conformant_design_object

The non_conformant_design_object specifies one role of either an Assembly_component, a Component_feature, or one of a Component_termination_passage, a Conductive_filled_area, a Conductor, a Cutout, an Inter_stratum_feature, a Laminate_component, a Land, a Non_functional_land, a Stratum_feature, or a Via for the Layout_spacing_requirement_non_conforming_design_object. There shall be one or more Assembly_component, Component_feature, Component_termination_passage, Conductive_filled_area, Conductor, Cutout, Inter_stratum_feature, Laminate_component, Land, Non_functional_land, Stratum_feature, or Via for each Layout_spacing_requirement_non_conforming_design_object.

4.2.508.3 reference_design_object

The reference_design_object specifies one role of either an Assembly_component, a Component_feature, or one of a Component_termination_passage, a Conductive_filled_area, a Conductor, a Cutout, an Inter_stratum_feature, a Laminate_component, a Land, a Non_functional_land, a Stratum_feature, or a Via for the Layout_spacing_requirement_non_conforming_design_object.

NOTE: This data is the result of an evaluation of the difference between the board layout and the original spacing requirements.
