4.2.305 Direction_element

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Direction_element is a type of Managed_design_object. A Direction_element is a definition of a shape element that has the form of a half-plane and is used in conjunction with the definition of another half-plane shape element to define a Datum_based_vector_orientation. In this instance a half-plane has the form of a plane except it is bounded on one side by a straight line. The half-plane shape element that the Direction_element defines is oriented with respect to three Datum_plane.

The data associated with this application object are the following:

  • angle_value
  • base_datum_plane
  • direction
  • direction_of_half_datum_plane
  • elements_of

4.2.305.1 angle_value

The angle_value specifies the Angle_data_element that represents the magnitude of the angle between the half-plane shape element defined by the Direction_element and half of the Datum_plane specified as the base_datum_plane.

4.2.305.2 base_datum_plane

The base_datum_plane specifies the Datum_plane from which the half-plane shape element defined by the Direction_element is oriented.

4.2.305.3 direction

The direction specifies the Datum_based_direction_vector that defines the orientation of the angle between the half-plane shape element defined by the Direction_element and half of the Datum_plane specified by the base_datum_plane. The angle is measured from half of the base Datum_plane to the defined half-plane shape element in a clockwise direction when viewed in the direction of the Datum_based_direction_vector.

4.2.305.4 direction_of_half_datum_plane

The direction_of_half_datum_plane specifies the Datum_based_direction_vector of a Datum_plane that intersects the base Datum_plane and indicates the half of the base Datum_plane from which the half-plane shape element defined by the Direction_element is oriented.

4.2.305.5 elements_of

The elements_of is an inverse relationship that specifies that the existence of the Direction_element is dependent on the existence of one or more Datum_based_vector_orientation that specifies the Direction_element as one of their direction_components.

Formal constraints:

WR1: The Datum_plane specified as the base_datum_plane, the Datum_plane specified as the vector_of by the Datum_based_direction_vector that is specified as the direction, the Datum_plane specified as the vector_of by the Datum_based_direction_vector that is specified as the direction_of_half_datum_plane shall be different.

WR2: The Datum_plane specified as the base_datum_plane, and the Datum_plane specified as the vector_of by the Datum_based_direction_vector that is specified as the direction, and the Datum_plane specified as the vector_of by the Datum_based_direction_vector that is specified as the direction_of_half_datum_plane shall be Datum_planes of the same Datum_reference_frame.

ENTITY direction_element
  SUBTYPE OF (managed_design_object);
    direction                     : datum_based_direction_vector;
    angle_value                   : angle_data_element;
    base_datum_plane              : datum_plane;
    direction_of_half_datum_plane : datum_based_direction_vector;
  INVERSE
    elements_of : SET [1:?] OF datum_based_vector_orientation FOR direction_components;
END_ENTITY; -- direction_element
