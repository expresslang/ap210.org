4.2.402 Functional_unit_network_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_unit_network_definition is a type of Functional_unit_definition. A Functional_unit_network_definition may be either one or more of an Electrical_network_definition or a Thermal_network_definition or a Functional_specification_definition.

The data associated with this application object are the following:

  • usage_view

4.2.402.1 usage_view

The usage_view specifies the role of the Functional_unit_usage_view for the Functional_unit_network_definition.

Formal constraints:

WR1: The member of Functional_unit_usage_view specified as a usage_view by this member of functional_unit_network_definition shall be an Ee_product_definition of the same version of the product for which this member of Functional_unit_network_definition is an Ee_product_definition.

ENTITY functional_unit_network_definition
  SUPERTYPE OF (thermal_network_definition ANDOR

    usage_view : functional_unit_usage_view;
