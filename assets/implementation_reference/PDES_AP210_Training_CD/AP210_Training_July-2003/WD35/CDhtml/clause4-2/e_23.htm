4.2.23 Angle_data_element

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Angle_data_element is a type of Data_element where the requisite unit type is that of angular displacement. An Angle_data_element may be combined with a Qualified_data_element in a complex instance to include uncertainty in an angle.
