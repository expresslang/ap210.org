4.2.640 Pca_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Pca_terminal is a type of Assembly_module_terminal that is part of the interface between the Pca and its next higher assembly.

The data associated with this application object are the following:

  • associated_definition

4.2.640.1 associated_definition

The associated_definition specifies the role of the Pca_usage_view for the Pca_terminal.
