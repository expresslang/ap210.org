4.2.274 Design_intent_modification_styled_area_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Design_intent_modification_styled_area_representation is a type of Managed_design_object. A Design_intent_modification_styled_area_representation is the representation of one or more Design_intent_modification as one of a Fill_area_occurrence, or a Styled_curve_occurrence. A Design_intent_modification_styled_area_representation is parametric data provided to support an algorithm that calculates explicit geometry. Since the geometry may be discontiguous (fill may included dashed lines), more than one Design_intent_modification may be referenced by Design_intent_modification_styled_area_representation as the resulting product data of the algorithm execution. The geometric data resulting from execution of the algorithm shall be included in Design_intent_modification_planar_shape if it is not desired to exchange the Design_intent_modification_styled_area_representation. Exchanging or sharing both the Design_intent_modification_styled_area_representation and the Design_intent_modification_planar_shape that result from execution of the algorithm may be appropriate. Therefore an optional attribute that specifies the planar shapes that are derived from the Design_intent_modification_styled_area_representation is to be included in an exchange if both the planar shape and the styled area representation is included. The units used in the Design_intent_modification_styled_area_representation shall be those specified in the global geometric context for the Interconnect_module design as specified by Cartesian_coordinate_system. If the geometry is contiguous, there shall be only one Design_intent_modification referenced.

The data associated with this application object are the following:

  • derived_planar_shape
  • represented_design_intent_modification
  • shape_styled_area_representation

4.2.274.1 derived_planar_shape

The derived_planar_shape specifies the Design_intent_modification_planar_shape for the Design_intent_modification_styled_area_representation. There may be more than one Design_intent_modification_planar_shape for the Design_intent_modification_styled_area_representation. A Design_intent_modification_planar_shape need not be provided for the Design_intent_modification_styled_area_representation.

4.2.274.2 represented_design_intent_modification

The represented_design_intent_modification specifies the Design_intent_modification that is characterized by this Design_intent_modification_styled_area_representation. There may be more than one Design_intent_modification for the Design_intent_modification_styled_area_representation.

4.2.274.3 shape_styled_area_representation

The shape_styled_area_representation specifies one of a Fill_area_occurrence, or a Styled_curve_occurrence that represents the modification in the Design_intent.
