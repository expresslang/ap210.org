4.2.5 Adjacent_stratum_surface_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Adjacent_stratum_surface_definition is a type of Managed_design_object. An Adjacent_stratum_surface_definition is a link in a sequence of stratum surfaces that result in the completed design. There may be multiple Adjacent_stratum_surface_definition that reference the same stratum surface in the case where Stratum are laterally adjacent. The use of Adjacent_stratum_surface_definition in combination with the position in 2D space of the Stratum within the interconnect is sufficient to uniquely identify the 3D location of the Stratum.

The data associated with this application object are the following:

  • name
  • precedent_surface
  • subsequent_surface

4.2.5.1 name

The name specifies the string for the human-interpretable identifier for the Adjacent_stratum_surface_definition.

4.2.5.2 precedent_surface

The precedent_surface specifies the previous Stratum_surface for the Adjacent_stratum_surface_definition.

4.2.5.3 subsequent_surface

The subsequent_surface specifies the subsequent Stratum_surface for the Adjacent_stratum_surface_definition.

Formal constraints:

UR1: The name shall be unique with a population of Adjacent_stratum_surface_definition.

UR2: The combination of the precedent_surface and the subsequent_surface shall be unique within a population of Adjacent_stratum_surface_definition.

WR1: The precedent_surface shall be a secondary surface of a Stratum.

WR2: The subsequent_surface shall be a primary surface of a Stratum.

ENTITY adjacent_stratum_surface_definition
  SUBTYPE OF (managed_design_object);
    name               : STRING;
    precedent_surface  : stratum_surface;
    subsequent_surface : stratum_surface;
  UNIQUE
    ur1 : name;
    ur2 : precedent_surface, subsequent_surface;
END_ENTITY; -- adjacent_stratum_surface_definition
