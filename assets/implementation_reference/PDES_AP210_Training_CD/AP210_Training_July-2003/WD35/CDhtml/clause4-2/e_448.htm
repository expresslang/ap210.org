4.2.455 Interconnect_module_macro_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Interconnect_module_macro_component is a type of Assembly_component.

The data associated with this application object are the following:

  • access_mechanisms
  • usage_definition
  • usage_view_definition

4.2.455.1 access_mechanisms

The access_mechanisms is an inverse relationship that specifies that the existence of the Interconnect_module_macro_component is dependent on the existence of the Interconnect_module_component_terminal that specifies the Interconnect_module_macro_component as its associated_component. There shall be two or more Interconnect_module_component_terminal for an Interconnect_module_macro_component.

4.2.455.2 usage_definition

The usage_definition specifies the role of an Interconnect_module for the Interconnect_module_macro_component.

4.2.455.3 usage_view_definition

The usage_view_definition is a derived role of an Interconnect_module_usage_view which specifies the specifies the Assembly_module_usage_view that is identified via the path SELF\assembly_component.usage_definition\physical_unit_design_view.usage_view.

Formal constraints:

WR1: The Interconnect_module_usage_view referenced by the usage_view_definition shall be the same Interconnect_module_usage_view referenced by the associated_definition which is referenced by the definition which is referenced by the access_mechanisms.
