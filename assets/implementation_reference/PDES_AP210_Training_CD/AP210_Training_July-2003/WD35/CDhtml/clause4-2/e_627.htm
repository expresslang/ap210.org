4.2.623 Part_template_location_in_padstack_definition

A Part_template_location_in_padstack_definition is a type of Managed_design_object. A Part_template_location_in_padstack_definition is the relationship that identifies the location of a Part_template in a Padstack_definition. The Application Object references a Template_definition in the formal model since the interpretation of this Application Object is to an Assembly_component_usage, and since there is a one to one relationship between a Part_template and the associated Template_definition.

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
