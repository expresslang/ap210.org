4.2.471 Interface_requirement_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interface_requirement_occurrence is a type of Design_requirement_occurrence in which the assembly interface to the next higher assembly is specified.

The data associated with this application object are the following:

  • interface_to_next_higher_assembly

4.2.471.1 interface_to_next_higher_assembly

The interface_to_next_higher_assembly specifies the role of the Next_higher_assembly_interface_requirement for the Interface_requirement_occurrence. This specifies the component in the next assembly that is the interface.
