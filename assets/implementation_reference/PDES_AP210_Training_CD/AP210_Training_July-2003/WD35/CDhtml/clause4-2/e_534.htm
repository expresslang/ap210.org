4.2.534 Material_removal_feature_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Material_removal_feature_template is a type of Part_template that describes the shape of material to be removed from a feature. A Material_removal_feature_template may be an Electrical_isolation_removal_template or a Thermal_isolation_removal_template.

EXAMPLES:

  1. Figure mrft illustrates a template and associated parameters designed for electrical isolation. For a specific template, values will be assigned for the parameters and the explicit geometry will be representative of the assigned parameter values.
  2. Figure mrft2 illustrates a template and associated parameters designed for thermal isolation. For a specific template, values will be assigned for the parameters and the explicit geometry will be representative of the assigned parameter values. In this example, an instance would be a complex instance including Composite_unit_shape_element and Material_removal_feature_template.

The data associated with this application object are the following:

  • of_stratum_technology

4.2.534.1 of_stratum_technology

The of_stratum_technology specifies the role of the Stratum_technology for the Material_removal_feature_template.
