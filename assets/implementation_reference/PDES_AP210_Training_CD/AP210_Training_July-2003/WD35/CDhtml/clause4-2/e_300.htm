4.2.295 Dimension_related_positional_boundary

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimension_related_positional_boundary is a type of Positional_boundary that is a theoretical boundary that is composed of a set of one or more Positional_boundary_member. Each Positional_boundary_member is defined by a separate Boundary_based_position_tolerance in conjunction with a Dimensional_size.

The data associated with this application object are the following:

  • boundary_members
  • constrained_physical_feature

4.2.295.1 boundary_members

The boundary_members specifies the Positional_boundary_member that comprise the Dimension_related_positional_boundary. There shall be one or more Positional_boundary_member specified as the boundary_members for the Dimension_related_positional_boundary.

4.2.295.2 constrained_physical_feature

The constrained_physical_feature specifies the Physical_feature_or_part_template that is to be constrained by the Dimension_related_positional_boundary.

Formal constraints:

WR1: The Positional_boundary_definition specified as the boundary_definition shall be a Dimension_related_positional_boundary_definition.

NOTE: The attribute boundary_definition is declared in the Positional_boundary entity of which this entity is a subtype.
