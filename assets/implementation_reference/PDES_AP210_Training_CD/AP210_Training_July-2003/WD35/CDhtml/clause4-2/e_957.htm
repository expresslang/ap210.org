4.2.958 Viewing_plane_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Viewing_plane_definition is a type of Managed_design_object. A Viewing_plane_definition is a definition that characterizes a Viewing_plane.

The data associated with this application object are the following:

  • defined_viewing_plane
  • viewing_plane_representation
  • zone_orientation

4.2.958.1 defined_viewing_plane

The defined_viewing_plane specifies the Viewing_plane that is characterized by the Viewing_plane_definition, and with which the specified Tolerance_zone is oriented.

4.2.958.2 viewing_plane_representation

The viewing_plane_representation specifies the Axis_placement that characterizes the Viewing_plane for the Viewing_plane_definition.

4.2.958.3 zone_orientation

The zone_orientation is an inverse relationship that specifies that the existence of the Viewing_plane_definition is dependent on the existence of the Viewing_plane_based_orientation that specifies the Viewing_plane_definition as its viewing_plane_specification.

ENTITY viewing_plane_definition
  SUBTYPE OF (managed_design_object);
    defined_viewing_plane        : viewing_plane;
    viewing_plane_representation : axis_placement;
  INVERSE
    zone_orientation : viewing_plane_based_orientation FOR 
                       viewing_plane_specification;
END_ENTITY; -- viewing_plane_definition
