4.2.872 Stratum_technology_link

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_technology_link is a type of Managed_design_object.

The data associated with this application object are the following:

  • precedent_stratum_technology
  • subsequent_stratum_technology
  • precedent_stratum_technology_link

4.2.872.1 precedent_stratum_technology

The precedent_stratum_technology specifies a Stratum_technology for the Stratum_technology_link.

4.2.872.2 subsequent_stratum_technology

The subsequent_stratum_technology specifies a Stratum_technology for the Stratum_technology_link.

4.2.872.3 precedent_stratum_technology_link

The precedent_stratum_technology_link specifies the role of a Stratum_technology_link for the Stratum_technology_link. The precedent_stratum_technology_link need not be specified for a particular Stratum_technology_link.

Formal constraints:

WR1: The precedent_stratum_technology shall not be the subsequent_stratum_technology within a population of Stratum_technology_link.

WR2: The Stratum_technology_link shall not be in a circular definition.

ENTITY stratum_technology_link
--tt tc1 wd9 modified
 SUPERTYPE OF ( printed_part_template_link)
 SUBTYPE OF (managed_design_object);
   precedent_stratum_technology : stratum_technology;
   subsequent_stratum_technology : stratum_technology;
   precedent_stratum_technology_link : OPTIONAL stratum_technology_link;
WHERE
 WR1: precedent_stratum_technology :<>: subsequent_stratum_technology;
--tt tc1 wd9 modified
 WR2: NOT(EXISTS(precedent_stratum_technology_link)) OR
           (precedent_stratum_technology_link :<>: SELF);
-- Stratum_technology_link shall be acyclic.
END_ENTITY; -- stratum_technology_link
