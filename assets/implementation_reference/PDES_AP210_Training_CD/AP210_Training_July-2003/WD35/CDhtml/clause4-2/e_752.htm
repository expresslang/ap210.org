4.2.752 Projected_tolerance_zone_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Projected_tolerance_zone_definition is a type of Tolerance_zone_definition in which the Tolerance_zone is projected from a Physical_feature_or_part_template of the Physical_unit.

The data associated with this application object are the following:

  • projected_zone_height

4.2.752.1 projected_zone_height

The projected_zone_height specifies the Projected_zone_height_characteristic that defines the height that the defined Tolerance_zone is projected above a base Physical_feature_or_part_template.

Formal constraints:

WR1: The Tolerance_zone specified as the projected_zone of the Projected_zone_and_base_relationship that is specified as the defined_height_of of the projected_zone_height that is specified as the Projected_zone_height shall be the same instance of Projected_tolerance_zone as specified as the defined_tolerance_zone.

NOTE: The attribute defined_tolerance_zone is declared in the Tolerance_zone_definition entity of which this entity is a subtype.

WR2: The Projected_tolerance_zone_definition shall be specified as the tolerance_zone_or_boundary_definition by exactly one Geometric_tolerance, and that Geometric_tolerance shall not be a Circularity_tolerance, a Cylindricity_tolerance, a Flatness_tolerance, a Straightness_tolerance, a Circular_runout_tolerance, a Total_runout_tolerance, a Profile_of_any_line_tolerance, a Profile_of_any_surface_tolerance, or a Boundary_based_position_tolerance.

WR3: The Projected_tolerance_zone_definition shall be specified as the tolerance_zone_or_boundary_definition by exactly one Geometric_tolerance, and that Geometric_tolerance shall specify either a Centre_axis, or Centre_plane as its toleranced_shape_element.
