4.2.722 Printed_component_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_component_join_terminal is a type of Component_terminal. This terminal is a terminus for internal circuitry in the interconnect substrate.

EXAMPLE: Figure pcjt illustrates a detail of a layout drawing that shows the internal terminals of the printed connector (edge connector).

The data associated with this application object are the following:

  • associated_component
  • connection
  • definition
  • stratum_concept_implementation_component

4.2.722.1 associated_component

The associated_component specifies the Printed_component being accessed by the Printed_component_join_terminal.

4.2.722.2 connection

The connection specifies the role of the Connection_zone for the Printed_component_join_terminal.

4.2.722.3 definition

The definition specifies the Printed_part_template_join_terminal that defines the Printed_component_join_terminal.

4.2.722.4 stratum_concept_implementation_component

The stratum_concept_implementation_component specifies either the Inter_stratum_feature, the Laminate_component, the Stratum, the Stratum_feature, or the Stratum_surface for the Printed_component_join_terminal.

ENTITY printed_component_join_terminal
  SUBTYPE OF (laminate_component_join_terminal);
    SELF\component_feature.definition           : printed_part_template_join_terminal;
    SELF\component_feature.associated_component : printed_component;
    stratum_concept_implementation              : stratum_feature;
WHERE
  WR1: SIZEOF(QUERY ( sf <* SELF\component_feature.
            associated_component\printed_component.
                   stratum_feature_implementation |
              sf = stratum_concept_implementation )
         ) = 1;
END_ENTITY; -- printed_component_join_terminal
