4.2.783 Rule_based_evaluated_characteristic

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_based_evaluated_characteristic is a type of Evaluated_characteristic. The Rule_based_evaluated_characteristic identifies the Ee_rule_version used to validate or evaluate a Planned_characteristic.

The data associated with this application object are the following:

  • evaluating_rule

4.2.783.1 evaluating_rule

The evaluating_rule specifies the role of the Ee_rule for a Rule_based_evaluated_characteristic.
