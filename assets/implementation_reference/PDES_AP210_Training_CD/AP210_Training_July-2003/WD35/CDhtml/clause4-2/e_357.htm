4.2.357 Evaluation_ee_product_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Evaluation_ee_product_definition is a type of Ee_product_definition that contains the results of evaluating another Ee_product_definition.

The data associated with this application object are the following:

  • evaluated_product_definition
  • evaluation_data

4.2.357.1 evaluated_product_definition

The evaluated_product_definition specifies the Ee_product_definition that was evaluated.

4.2.357.2 evaluation_data

The evaluation_data specifies the role of the Evaluated_characteristic that contains the detailed product data resulting from the evaluation.
