4.2.221 Datum_difference

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_difference is a type of Composite_unit_shape_element. A Datum_difference is the basis for mapping from a geometric context model into a lexical context model for definitional purposes. A Datum_difference identifies the length or angular difference between two Datum. The difference may be referenced independently of the originating geometric model representation.

NOTES:

  1. One Datum may be located at the origin if a simple location is intended.
  2. The origin reference for a simple angular difference should be the X axis for two-dimensional case, and the XY axis for three-dimesional case.

The data associated with this application object are the following:

  • precedent_datum
  • subsequent_datum

4.2.221.1 precedent_datum

A precedent_datum specifies one role of a Datum for a Datum_difference. The precedent_datum is the reference datum.

4.2.221.2 subsequent_datum

A subsequent_datum specifies one role of a Datum for a Datum_difference. The subsequent_datum is the measureand.

ENTITY datum_difference
  SUBTYPE OF (shape_element_relationship);
    SELF\shape_element_relationship.name : label;

	datum := SELF\shape_element_relationship.relating_shape_element;
    subsequent_datum : 
	datum := SELF\shape_element_relationship.related_shape_element;
END_ENTITY; -- datum_difference
