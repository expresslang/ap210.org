4.2.875 Styled_curve_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Styled_curve_occurrence is a type of Managed_design_object. A Styled_curve_occurrence is an occurrence of a curve with specific parametric styling assigned.

The data associated with this application object are the following:

  • of_curve_style
  • styled_curve

4.2.875.1 of_curve_style

The of_curve_style specifies the Curve_style to apply to the Curve for the Styled_curve_occurrence.

4.2.875.2 styled_curve

The styled_curve specifies the role of the Curve for the Styled_curve_occurrence.
