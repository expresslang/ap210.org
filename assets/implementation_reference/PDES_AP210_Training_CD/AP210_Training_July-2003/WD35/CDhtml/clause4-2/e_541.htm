4.2.541 Maximum_dimension_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Maximum_dimension_representation is a type of Singular_dimension_representation that explicitly describes the maximum limit of one or more Dimensional_characteristic and in which the minimum limit is not specified as it is determined by other elements of the design.

NOTE: Maximum dimensions are discussed in 4.4 of ISO 406.
