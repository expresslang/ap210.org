4.2.935 Type_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Type_qualifier is a type of Managed_design_object which specifies an uncertainty. The uncertainty is defined in clause 2 of "The Guide to the Expression of Uncertainty in Measurement".
