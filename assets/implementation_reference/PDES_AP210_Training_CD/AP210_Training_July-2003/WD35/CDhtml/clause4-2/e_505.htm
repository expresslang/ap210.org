4.2.505 Layout_macro_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Layout_macro_definition is a type of Pcb. A Layout_macro_definition is a design view of a interconnect structure that is intended to be reused as a structure rather than as a component. A Layout_macro_definition may be an aggregation of Stratum. A Layout_macro_definition may be an aggregation of Layout_macro_definition_component A Layout_macro_definition shall not be used as a component in a physical assembly.

EXAMPLE: A Layout_macro_definition may be used as part of a recursive definition or subroutine layout.

The data associated with this application object are the following:

  • floor_layout_view

4.2.505.1 floor_layout_view

The floor_layout_view specifies the Layout_macro_floor_plan_template for the Layout_macro_definition as a layout template for the floor planning layout. This template shall provide the terminal definitions for the Layout_macro_definition and a shape for the floor planning layout to enable the layout design without instantiating the design details of the Layout_macro_definition itself.

NOTE: The floor_layout_view is a replacement for the usage_view that would be inherited from the Interconnect_module.

Formal constraints:

WR1: A Layout_macro_definition shall not be referenced in the role of usage_definition.

WR2: Layout_macro_definition.usage_view (this is an inherited attribute from Physical_unit_design_view) shall not exist.

ENTITY layout_macro_definition
  SUBTYPE OF (pcb);
    floor_layout_view : layout_macro_floor_plan_template;
WHERE
 WR1: NOT(EXISTS(SELF\physical_unit_design_view.usage_view));
 WR2: NOT(EXISTS(SELF\interconnect_module.assembly_design_requirement)); 
 WR3: SIZEOF(USEDIN(SELF,
      'AP210_ARM.ASSEMBLY_COMPONENT.USAGE_DEFINITION')) = 0;
END_ENTITY; -- layout_macro_definition
