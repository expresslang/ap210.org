4.2.45 Assembly_module_component_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Assembly_module_component_terminal is a type of Component_terminal. The Assembly_module_component_terminal is included with the Assembly_module_component to assure the data integrity of references to the terminals of the Assembly_module_component.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.45.1 associated_component

The associated_component specifies the role of the Assembly_module_component for the Assembly_module_component_terminal.

4.2.45.2 definition

The definition specifies the role of the Assembly_module_terminal for the Assembly_module_component_terminal.
