4.2.286 Device_terminal_map_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Device_terminal_map_relationship is a type of Managed_design_object. A Device_terminal_map_relationship is the association of one Device_terminal_map to another.

The data associated with this application object are the following:

  • related_device_terminal_map
  • relating_device_terminal_map

4.2.286.1 related_device_terminal_map

The related_device_terminal_map specifies the target Device_terminal_map in the Device_terminal_map_relationship.

4.2.286.2 relating_device_terminal_map

The relating_device_terminal_map specifies the source Device_terminal_map in the Device_terminal_map_relationship.
