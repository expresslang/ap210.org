4.2.326 Ee_product_related_product_category

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_product_related_product_category is a type of Ee_product_category. An Ee_product_related_product_category relates one or more products to the specific characteristics associated with the category.

The data associated with this application object are the following:

  • of_product

4.2.326.1 of_product

The of_product specifies the Ee_product assigned to this Ee_product_related_product_category. There shall be one or more Ee_product for an Ee_product_related_product_category.
