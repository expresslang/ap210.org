4.2.619 Part_rectangular_array_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_rectangular_array_feature is a type of Part_array_feature. The location_uncertainty refers to originating CAD system uncertainty, not material processing tolerances.

The data associated with this application object are the following:

  • element
  • initial_row_positive_y_displacement
  • terminus_row_positive_y_displacement

4.2.619.1 element

The element specifies the role of a Part_linear_array_feature_link for the Part_rectangular_array_feature. There shall be one or more Part_linear_array_feature_link for a Part_rectangular_array_feature.

4.2.619.2 initial_row_positive_y_displacement

The initial_row_positive_y_displacement specifies a Length_data_element for the Part_rectangular_array_feature.

4.2.619.3 terminus_row_positive_y_displacement

The terminus_row_positive_y_displacement specifies a Length_data_element for the Part_rectangular_array_feature.
