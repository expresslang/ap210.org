4.2.185 Connection_zone

A Connection_zone is a type of Managed_design_object. A Connection_zone is an area on the surface of a design object that may be used to establish electrical or physical connectivity with another design object. A Connection_zone shape is defined in the shape definition of the feature that references the Connection_zone. Since more than one feature may reference the same shape but for different positions, more than one feature may reference the same Connection_zone.

The data associated with this application object are the following:

  • zone_shape_1
  • zone_shape_2
  • representation_colour

4.2.185.1 zone_shape_1

The zone_shape_1 is an inverse relationship that specifies that the Connection_zone is existent dependent on the Connection_zone_shape that specifies the Connection_zone. There shall be zero or more zone_shape_1 for a Connection_zone.

4.2.185.2 zone_shape_2

The zone_shape_2 is an inverse relationship that specifies that the Connection_zone is existent dependent on the Connection_zone_shape that specifies the Connection_zone. There shall be zero or more zone_shape_2 for a Connection_zone.

4.2.185.3 representation_colour

The representation_colour specifies the Ee_colour that depicts the area constituting the Connection_zone. The representation_colour need not be specified.

Formal constraints:

WR1: All points in a Connection_zone shall be topologically connected.

WR2: A particular instance of Connection_zone shall be referenced by at least one shape instantiation.

ENTITY connection_zone
  SUBTYPE OF (managed_design_object);
    representation_colour : OPTIONAL ee_colour;
INVERSE
    zone_shape_1            : SET [0:1] OF 
         design_view_connection_zone_position_relationship for 
                         associated_usage;
    zone_shape_2            : SET [0:1] OF 
         usage_view_connection_zone_terminal_shape_relationship for 
                         associated_usage;
WHERE
  WR1: SIZEOF(USEDIN(SELF,'AP210_ARM.ASSEMBLY_MODULE_TERMINAL.TERMINAL_CONNECTION_ZONE')) + 
       SIZEOF(USEDIN(SELF,'AP210_ARM.BARE_DIE_TERMINAL.EXTERNAL_CONNECTION_AREA') )+ 
       SIZEOF(USEDIN(SELF,'AP210_ARM.COMPONENT_TERMINATION_PASSAGE_TEMPLATE_TERMINAL.TERMINAL_CONNECTION_ZONE') )+
       SIZEOF(USEDIN(SELF,'AP210_ARM.CONDUCTIVE_INTERCONNECT_ELEMENT_TERMINAL.CONNECTION')) +
       SIZEOF(USEDIN(SELF,'AP210_ARM.INTERCONNECT_COMPONENT_INTERFACE_TERMINAL.CONNECTION')) + 
       SIZEOF(USEDIN(SELF,'AP210_ARM.INTERCONNECT_COMPONENT_JOIN_TERMINAL.CONNECTION')) + 
       SIZEOF(USEDIN(SELF,'AP210_ARM.INTERCONNECT_MODULE_TERMINAL.TERMINAL_CONNECTION_ZONE')) +
       SIZEOF(USEDIN(SELF,'AP210_ARM.LAND_TEMPLATE_TERMINAL.TERMINAL_CONNECTION_ZONE')) +
       SIZEOF(USEDIN(SELF,'AP210_ARM.PACKAGE_TERMINAL.EXTERNAL_CONNECTION_ZONE')) + 
       SIZEOF(USEDIN(SELF,'AP210_ARM.PACKAGE_TERMINAL.INTERNAL_CONNECTION_ZONE')) + 
       SIZEOF(USEDIN(SELF,'AP210_ARM.PRINTED_PART_TEMPLATE_TERMINAL.TERMINAL_CONNECTION_ZONE')) + 
       SIZEOF(USEDIN(SELF,'AP210_ARM.VIA_TEMPLATE_TERMINAL.TERMINAL_CONNECTION_ZONE')) + 
       SIZEOF(USEDIN(SELF,'AP210_ARM.PART_MOUNTING_FEATURE.MOUNTING_AREA'))  >= 0;
  WR2: SIZEOF(zone_shape_1) + SIZEOF(zone_shape_2) = 1;
  END_ENTITY; -- connection_zone
