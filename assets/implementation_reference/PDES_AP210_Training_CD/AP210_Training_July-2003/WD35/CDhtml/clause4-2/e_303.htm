4.2.303 Directed_axis

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Directed_axis is a type of Non_feature_shape_element that is an axis in which a direction is defined. A Directed_axis is defined by a base Centre_axis, with which the Directed_axis is congruent, and two Shape_element through which the base Centre_axis passes or would pass if the Shape_elements were extended. One of the two Shape_elements is specified as the start_shape_element and the other one specified as the end_shape_element.

The data associated with this application object are the following:

  • axis_to_be_oriented
  • end_shape_element
  • start_shape_element

4.2.303.1 axis_to_be_oriented

The axis_to_be_oriented specifies the Centre_axis that serves as a base axis by which the Directed_axis is defined.

4.2.303.2 end_shape_element

The end_shape_element specifies the Shape_element through which the Directed_axis passes or would pass if the Shape_element were extended. The Shape_element specified by the end_shape_element is in the positive direction from the Shape_element specified by the start_shape_element.

4.2.303.3 start_shape_element

The start_shape_element specifies the Shape_element through which the Directed_axis passes or would pass if the Shape_element were extended. The Shape_element specified by the start_shape_element is in the negative direction from the Shape_element specified by the end_shape_element.

Formal constraints:

UR1: The combination of axis_to_be_oriented and start_shape_element shall be unique within a population of Directed_axis.

WR1: The Shape_element specified by the start_shape_element shall not be the same Shape_element as specified by the end_shape_element.

Informal constraints:

IP1: The base Centre_axis shall pass through or would pass through the Shape_elements specified by the start_shape_element and end_shape_element if the Shape_elements were extended.

ENTITY directed_axis
  SUBTYPE OF (non_feature_shape_element);
    axis_to_be_oriented : centre_axis;
    start_shape_element : shape_element;
    end_shape_element   : shape_element;
  UNIQUE
    ur1 : axis_to_be_oriented, start_shape_element;
END_ENTITY; -- directed_axis
