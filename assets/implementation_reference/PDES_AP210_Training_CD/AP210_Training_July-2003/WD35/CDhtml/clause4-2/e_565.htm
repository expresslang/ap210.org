4.2.564 Non_functional_land_join_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Non_functional_land_join_terminal is a type of Component_terminal that is included in a design to support the description of the relationship between the Non_functional_land and a plated passage that implements a connection to a feature on a different member of Stratum than the member of Stratum the Non_functional_land_join_terminal is associated with.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.564.1 associated_component

The associated_component specifies the role of the Non_functional_land for the Non_functional_land_join_terminal.

4.2.564.2 definition

The definition specifies the role of the Land_template_join_terminal for the Non_functional_land_join_terminal. The valid purposes for a Non-functional_land_join_terminal are to connect to a Via_terminal or a Component_termination_passage_join_terminal.

Formal constraints:

WR1: The Land_template that is the usage_definition for the Non_functional_land this Non_functional_land_join_terminal is a terminal of is the associated_definition for the definition of this Non_functional_land_join_terminal.

WR2: A Non_functional_land_join_terminal is only to be connected to a passage join terminal, and to nothing else. It is not allowed to be connected to a trace on the same layer; the join is required to be to a passage within the lateral extent of the non-functional land.
