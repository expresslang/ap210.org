4.2.312 Edge_segment_vertex

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Edge_segment_vertex is a type of Managed_design_object. An Edge_segment_vertex supports explicit assignment of the order and connection of edge start and end points but without needing to use an explicit geometric or topological context.

The data associated with this application object are the following:

  • name

4.2.312.1 name

The name specifies the identifier for the Edge_segment_vertex. The name shall be unique within a population of Edge_segment_vertex.

Formal constraints:

UR1: The name shall be unique within a population of Edge_segment_vertex.
