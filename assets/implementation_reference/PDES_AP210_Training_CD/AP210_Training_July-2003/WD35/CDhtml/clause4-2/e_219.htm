4.2.219 Datum_based_length_measure

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_based_length_measure is a type of Managed_design_object. A Datum_based_length_measure is a dimensional measure of length that is taken from a defined Datum.

EXAMPLE: Figure dblm illustrates the usage of a Datum_based_length_measure.

The data associated with this application object are the following:

  • measure
  • measure_orientation
  • of_datum

4.2.219.1 measure

The measure specifies the role of the Length_data_element for the Datum_based_length_measure.

4.2.219.2 measure_orientation

The measure_orientation specifies the role of the Direction for the Datum_based_length_measure or specifies the role of the string 'normal' or specifies the role of the string 'reversed'. 'Normal' shall be interpreted as a description equivalent to the three dimensional concept of a vector parallel to the normal of the plane that is the datum upon which the Datum_based_length_measure is based. 'Reversed' shall be interpreted as a description equivalent to the three dimensional concept of a vector anti-parallel to the normal of the plane that is the datum upon which the Datum_based_length_measure is based. For the case that the Datum is a Datum_plane, the Direction vector shall be in the XZ plane of the Datum_plane, where the Z axis is the normal vector of the Datum_plane. The measure_orientation is measured from the X axis in the XZ plane. For the two-dimensional case, the Datum_plane if used, is considered to be congruent with the XY coordinate system.

NOTE: The use of descriptive items allows the information to be conveyed when the geometric context is 2d, since orientation only supports a description within a 3d geometric context.

4.2.219.3 of_datum

The of_datum specifies the role of the Datum_plane for the Datum_based_length_measure. A Datum_plane is required in order to support equivalent description between the 2d and 3d shape representation cases.
