4.2.621 Part_template_3d_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_template_3d_shape is a type of Bound_volume_shape and a type of Feature_shape_definition. A Part_template_3d_shape is the shape in three dimensions of the Part_template.

The data associated with this application object are the following:

  • shape_characterized_part_template
  • shape_environment
  • shape_material_condition
  • shape_representation

4.2.621.1 shape_characterized_part_template

The shape_characterized_part_template specifies the role of the Part_template for the Part_template_3d_shape.

4.2.621.2 shape_environment

The shape_environment specifies either manufacturing or the end_user_application as the environment in which the Part_template_3d_shape is valid. The shape_environment need not be specified for a particular Part_template_3d_shape.

4.2.621.3 shape_material_condition

The shape_material_condition specifies either the maximum_material_condition, the minimum_material_condition or the nominal_material_condition as the material condition for the Part_template_3d_shape.

4.2.621.4 shape_representation

The shape_representation specifies the Bound_volume_shape that is the model for the Part_template_3d_shape.
