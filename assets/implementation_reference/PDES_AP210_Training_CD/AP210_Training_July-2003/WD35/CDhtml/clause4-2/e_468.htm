4.2.475 Internal_probe_access_area

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Internal_probe_access_area is a type of Probe_access_area included in the design for access to internal conductors for probing purposes.

NOTE: This is included explicitly because probe points are not usually included in a functional nodelist.

The data associated with this application object are the following:

  • stratum_feature_implementation

4.2.475.1 stratum_feature_implementation

The stratum_feature_implementation specifies the role of the Stratum_feature for the Internal_probe_access_area. This Stratum_feature is included only to support the probe requirements and is not necessarily included in a functional nodelist.

Formal constraints:

WR1: Each member of Internal_probe_access_area shall be in the role of connected_probe for exactly one member of Inter_stratum_access.

WR2: The Stratum_feature specified by stratum_feature_implementation shall specify an external Stratum as a resident_stratum.

NOTE: 
*)
ENTITY internal_probe_access_area
  SUBTYPE OF (probe_access_area);
    stratum_feature_implementation : stratum_feature;
END_ENTITY; -- internal_probe_access_area
