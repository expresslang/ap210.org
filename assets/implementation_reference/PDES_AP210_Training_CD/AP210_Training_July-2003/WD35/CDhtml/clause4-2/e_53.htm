4.2.53 Assembly_requirement_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Assembly_requirement_allocation is a type of Managed_design_object. An Assembly_requirement_allocation may be one or more of an Assembly_separation_requirement_allocation, an Electromagnetic_compatibility_requirement_allocation, or a Thermal_requirement_allocation. An Assembly_requirement_allocation is the relationship between the Ee_requirement_occurrence and either an Assembly_component or a Component_feature that satisfies that Ee_requirement_occurrence.

The data associated with this application object are the following:

  • assembly_assignment
  • requirement

4.2.53.1 assembly_assignment

The assembly_assignment specifies either an Assembly_component or a Component_feature for the Assembly_requirement_allocation.

4.2.53.2 requirement

The requirement specifies the role of the Ee_requirement_occurrence for the Assembly_requirement_allocation.
