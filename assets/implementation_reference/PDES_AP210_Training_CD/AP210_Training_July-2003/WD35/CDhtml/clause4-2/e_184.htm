4.2.184 Connected_filled_area

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Connected_filled_area is a type of Conductive_filled_area. The Connected_filled_area is a Conductive_filled_area with which a set of layer connection points are associated (as specified in the implementation_requirement attribute).

The data associated with this application object are the following:

  • implementation_requirement

4.2.184.1 implementation_requirement

The implementation_requirement specifies the Intra_stratum_join_relationship for which the Connected_filled_area is generated.
