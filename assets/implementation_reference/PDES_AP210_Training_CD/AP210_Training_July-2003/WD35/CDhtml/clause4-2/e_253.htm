4.2.253 Default_unsupported_passage_based_land_physical_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Default_unsupported_passage_based_land_physical_template is a type of Default_passage_based_land_physical_template where the passage type is a passage that has no metallized support.

The data associated with this application object are the following:

  • of_passage_technology

4.2.253.1 of_passage_technology

The of_passage_technology specifies the role of the Default_unsupported_passage_definition for the Default_unsupported_passage_based_land_physical_template.
