4.2.372 Feature_shape_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Feature_shape_definition is a type of Shape_definition. A Feature_shape_definition represents real features in the product model. A member of Feature_shape_definition shall be also be a member of another subtype of Shape_definition.

NOTE: The type constraint stated is supported by the information requirements described in the Shape_definition Application Object.
