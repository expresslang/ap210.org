4.2.893 Tertiary_orientation_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tertiary_orientation_feature is both a Part_feature and a Datum_feature. A tertiary_orientation_feature provides anti-rotation capability for unambiguous orientation of the part in the datum_reference_frame.

The data associated with this application object are the following:

  • associated_body_vertical_extent

4.2.893.1 associated_body_vertical_extent

An associated_body_vertical_extent specifies the role of the Package_body_surface for a Tertiary_orientation_feature. The associated_body_vertical_extent helps to identify the surface or surfaces which contain the feature. An associated_body_vertical_extent need not be specified for a Tertiary_orientation_feature.
