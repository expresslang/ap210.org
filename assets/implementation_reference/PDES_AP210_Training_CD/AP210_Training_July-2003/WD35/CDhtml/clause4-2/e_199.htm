4.2.199 Countersunk_passage_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Countersunk_passage_template is a type of Unsupported_passage_template. A Countersunk_passage_template is a combination of two round coaxial passages, one cylindrical and one tapered. The initial diameter of the tapered passage will be larger than the diameter of the cylindrical passage, and will linearly taper to be the same diameter as the cylindrical passage where the two passages meet.

The data associated with this application object are the following:

  • constant_diameter_passage
  • tapered_passage

4.2.199.1 constant_diameter_passage

The constant_diameter_passage specifies the Unsupported_passage_template for the Countersunk_passage_template. The constant_diameter_passage specifies the data that is associated with the constant diameter opening of the Countersunk passage.

4.2.199.2 tapered_passage

The tapered_passage specifies the Blind_passage_template for the Countersunk_passage_template. The tapered_passage specifies the data that is associated with the tapered area of the Countersunk passage.
