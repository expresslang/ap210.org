4.2.315 Ee_colour

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_colour is a type of Managed_design_object. An Ee_colour is the colour characterized by normalized quantities of the colours red, green, and blue in accordance with ISO/IEC 9592-1.

The data associated with this application object are the following:

  • name

4.2.315.1 name

The name specifies the string identifying the Ee_colour.

Formal constraints:

UR1: The name shall be unique within a population of Ee_colour.

NOTE: 
*)
ENTITY ee_colour
  SUBTYPE OF (managed_design_object);
    name : ee_name;
UNIQUE
  ur1: name;
END_ENTITY; -- ee_colour
