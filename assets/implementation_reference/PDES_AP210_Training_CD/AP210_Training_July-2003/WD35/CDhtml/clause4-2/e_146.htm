4.2.146 Component_termination_passage_dependent_land

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_termination_passage_dependent_land is a type of Land. The shape of the Component_termination_passage_dependent_land is dependent only on the passage that is within the shape of the land (where that passage is included for the purpose of connecting the component terminal to a trace on the substrate).

The data associated with this application object are the following:

  • reference_passage
  • usage_definition

4.2.146.1 reference_passage

The reference_passage specifies the role of the Component_termination_passage for the Component_termination_passage_dependent_land.

4.2.146.2 usage_definition

The usage_definition specifies the role of the Default_component_termination_passage_based_land_physical_template for the Component_termination_passage_dependent_land.
