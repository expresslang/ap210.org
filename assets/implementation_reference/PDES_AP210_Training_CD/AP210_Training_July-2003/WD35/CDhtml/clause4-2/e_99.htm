A Characteristic_data_set is a type of Compound_characteristicn. A Characteristic_data_set is a part of the data needed to compose a data table. The items included in the set are related by being in the set but there is no sequence specified. Specific functional relationships are beyond the scope of this Appication object and should be addressed by the Analytical_model Application object.
