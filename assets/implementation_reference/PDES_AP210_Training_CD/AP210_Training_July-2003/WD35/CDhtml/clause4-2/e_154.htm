4.2.155 Composite_curve

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Composite_curve is a type of Curve that combines more than one Curve into its structure. The Composite_curve shall be continuous.

The data associated with this application object are the following:

  • curve_segment

4.2.155.1 curve_segment

The curve_segment specifies the role of the Composite_curve_segment for the Composite_curve. There shall be one or more Composite_curve_segment for a Composite_curve. No Composite_curve_segment shall be specified more than once as a curve_segment for a particular Composite_curve.

NOTE: 
*)
ENTITY composite_curve
  SUBTYPE OF (curve);
    curve_segment : LIST [1:?] OF UNIQUE composite_curve_segment;
END_ENTITY; -- composite_curve
