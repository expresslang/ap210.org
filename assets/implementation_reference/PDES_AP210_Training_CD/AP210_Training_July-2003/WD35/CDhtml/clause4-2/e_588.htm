4.2.578 Package_body_surface

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Package_body_surface is either a Package_body_top_surface or a Package_body_bottom_surface. A Package_body_surface relationship to the seating plane may be explicitly established, helping to unambiguously orient the Packaged_part.

The data associated with this application object are the following:

  • associated_package_body

4.2.578.1 associated_package_body

The associated_package_body specifies the role of the Package_body for the Package_body_surface.
