4.2.232 Datum_plane

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_plane is a type of Datum that is a theoretically exact plane to which toleranced Shape_element are related.

Formal constraints:

WR1: If the Datum_plane is not specified as a reference_planes of a Datum_reference_frame then the Datum_plane shall be either a Common_datum, or a Single_datum.

WR2: A Datum_plane shall be specified as the vector_of by at most two Datum_based_direction_vector.

WR3: If the Datum_plane is specified as the vector_of by two Datum_based_direction_vector, the two Datum_based_direction_vector shall specify different values for the positive_direction_vector.

NOTE: 
*)
ENTITY datum_plane
  SUBTYPE OF (datum);
END_ENTITY; -- datum_plane
