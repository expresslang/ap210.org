4.2.731 Printed_part_template_connected_terminals_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Printed_part_template_connected_terminals_definition is a type of Managed_design_object. A Printed_part_template_connected_terminals_definition is the association between two or more Printed_part_template_terminal that are connected.

The data associated with this application object are the following:

  • connected_terminals
  • name

4.2.731.1 connected_terminals

The connected_terminals specify the Printed_part_template_terminal for the Printed_part_template_connected_terminals_definition. There shall be two or more Printed_part_template_terminal for the Printed_part_template_connected_terminals_definition.

4.2.731.2 name

The name identifies the Printed_part_template_connected_terminals_definition.

Formal constraints:

UR1: The name shall be unique within a population of Printed_part_template_connected_terminals_definitions.
