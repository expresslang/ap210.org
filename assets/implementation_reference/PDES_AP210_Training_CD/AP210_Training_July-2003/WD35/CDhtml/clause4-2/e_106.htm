4.2.106 Component_2d_edge_location

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_2d_edge_location is a type of Component_2d_location. The Component_2d_edge_location places components on the edge of a substrate in order to maintain contact with both the top and bottom circuitry.

NOTE: Connection_zone_interface_plane_relationship and Connection_zone_based_assembly_joint may be used to provide more details if needed by a receiving organization.

The data associated with this application object are the following:

  • reference_terminal_assembly_joint

4.2.106.1 reference_terminal_assembly_joint

The reference_terminal_assembly_joint specifies the Assembly_joint that joins the Primary_reference_terminal of the component being located to the associated mating component feature. The surface the mating component feature is associated with is part of the orientation description for the Component_2d_edge_location.
