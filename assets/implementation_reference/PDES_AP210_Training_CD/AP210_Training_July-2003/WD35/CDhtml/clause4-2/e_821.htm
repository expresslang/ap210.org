4.2.821 Shield

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shield is a type of Assembly_component included for suppression of unwanted electromagnetic coupling among components. An Assembly_component included in the assembly for other purposes might also serve as a Shield, and in that case the other type would be specified as well for that instance.

EXAMPLES:

  1. A Via may be a Shield. More than one shield may shield the same shielded_item for the same purpose.
  2. Figure shield illustrates a detail of a layout drawing that shows a shielded trace (carrying a critical signal) on the several layers it traverses in the design.

The data associated with this application object are the following:

  • shielded_item

4.2.821.1 shielded_item

The shielded_item specifies either an Assembly_component, a Component_feature, or a Physical_connectivity_definition as the shieldable item for a Shield. There shall be one or more Assembly_component, Component_feature, or Physical_connectivity_definition for a Shield.
