4.2.145 Component_termination_passage_and_contact_size_dependent_non_functional_land

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_termination_passage_and_contact_size_dependent_non_functional_land is a type of Non_functional_land. The shape of the Component_termination_passage_and_contact_size_dependent_non_functional_land is dependent both on the passage that is within the shape of the land (where that passage is included for the purpose of connecting the component terminal to a trace on the substrate) and on the size of the terminal that is going to be resting on the board.

NOTE: A Component_termination_passage_and_contact_size_dependent_non_functional_land principally provides a location for the solder joint.

The data associated with this application object are the following:

  • interface_access_mechanism
  • reference_passage
  • usage_definition

4.2.145.1 interface_access_mechanism

The interface_access_mechanism is an inverse relationship that specifies that the existence of the Component_termination_passage_and_contact_size_dependent_non_functional_land is dependent on the existence of the Non_functionl_land_interface_terminal that specifies the Component_termination_passage_and_contact_size_dependent_non_functional_land as its associated_component. This relationship is for external access only.

4.2.145.2 reference_passage

The reference_passage specifies the role of the Component_termination_passage for the Component_termination_passage_and_contact_size_dependent_non_functional_land.

4.2.145.3 usage_definition

The usage_definition specifies the role of the Default_attachment_size_and_component_termination_passage_based_land_physical_template for the Component_termination_passage_and_contact_size_dependent_non_functional_land.
