4.2.636 Partially_routed_physical_network

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Partially_routed_physical_network is a type of Physical_network. A Partially_routed_physical_network is a description of an incompletely specified mapping from requirements to layout. The set of unrouted elements shall be a subset of the total requirement set.

The data associated with this application object are the following:

  • unrouted_connectivity_requirement_element

4.2.636.1 unrouted_connectivity_requirement_element

The unrouted_connectivity_requirement_element specifies the Component_terminal that are not yet implemented by circuitry layout for the Partially_routed_physical_network. There shall be two or more Component_terminal for a Partially_routed_physical_network.
