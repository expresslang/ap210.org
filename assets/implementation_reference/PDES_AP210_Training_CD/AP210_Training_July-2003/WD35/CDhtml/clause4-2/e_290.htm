4.2.290 Digital_analytical_model_port

A Digital_analytical_model_port is a type of Analytical_model_port that defines a digital context.

The data associated with this application object are the following:

  • nominal_signal_flow_direction

4.2.290.1 nominal_signal_flow_direction

The nominal_signal_flow_direction specifies either BIDIRECTIONAL, INPUT_DIRECTION, NOT_APPLICABLE, OUTPUT_DIRECTION, or UNKNOWN_DIRECTION for the Digital_analytical_model_port.

Formal constraints:

WR1: "NOT_APPLICABLE" is not an allowed value.

NOTE: 
*)
ENTITY digital_analytical_model_port
  SUBTYPE OF (analytical_model_port);
WHERE
 wr1: not_applicable <> 
      SELF\analytical_model_port.nominal_signal_flow_direction;
END_ENTITY; -- digital_analytical_model_port
