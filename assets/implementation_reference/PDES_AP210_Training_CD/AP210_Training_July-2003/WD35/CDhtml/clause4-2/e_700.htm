4.2.703 Positional_boundary_offset

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Positional_boundary_offset is a type of Managed_design_object. A Positional_boundary_offset is a definition that characterizes the offset of a Profile_related_positional_boundary from a Tolerance_zone_boundary_member.

The data associated with this application object are the following:

  • boundary_definition
  • offset_boundary_relationship
  • tolerance_value

4.2.703.1 boundary_definition

The boundary_definition is an inverse relationship that specifies that the existence of the Positional_boundary_offset is dependent on the existence of the Profile_related_positional_boundary_definition that specifies the Positional_boundary_offset as its boundary_offset.

4.2.703.2 offset_boundary_relationship

The offset_boundary_relationship specifies the Positional_boundary_and_profile_boundary_member_relationship that associates the Profile_related_positional_boundary with the Tolerance_zone_boundary_member from which it is offset, and which is characterized by the Positional_boundary_offset.

4.2.703.3 tolerance_value

The tolerance_value specifies the Length_data_element for the Positional_boundary_offset. The associated Profile_related_positional_boundary is offset from the associated Tolerance_zone_boundary_member by half the distance specified by the tolerance_value.
