4.2.601 Parallel_shape_element_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Parallel_shape_element_relationship is a type of Shape_element_relationship that asserts that two Shape_element are parallel to each other. One Shape_element is specified as the related_shape_element and the other Shape_element is specified as the relating_shape_element.

NOTES:

  1. The attribute related_shape_element is declared in the Shape_element_relationship entity of which this entity is a subtype.
  2. The attribute relating_shape_element is declared in the Shape_element_relationship entity of which this entity is a subtype.

Formal constraints:

WR1: The Shape_element specified as the relating_shape_element shall not be specified as parallel to itself either directly or through intermediate Parallel_shape_element_relationship.

NOTE:

ENTITY parallel_shape_element_relationship
  SUBTYPE OF (shape_element_relationship);
