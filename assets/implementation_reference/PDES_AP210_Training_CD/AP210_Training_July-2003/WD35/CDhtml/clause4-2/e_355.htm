4.2.355 Equivalent_functional_unit_definition_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Equivalent_functional_unit_definition_assignment is a type of Managed_design_object. An Equivalent_functional_unit_definition_assignment is the association between at least two Functional_unit_definition that are equivalent.

The data associated with this application object are the following:

  • equivalent_functional_unit_definitions
  • name

4.2.355.1 equivalent_functional_unit_definitions

The equivalent_functional_unit_definitions specify the Functional_unit_definition for the Equivalent_functional_unit_definition_assignment. There shall be at least two but may be more than two Functional_unit_definition for the Equivalent_functional_unit_definition_assignment.

4.2.355.2 name

The name specifies the string that is the identifier for the Equivalent_functional_unit_definition_assignment.
