4.2.394 Frozen_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Frozen_assignment is a type of Managed_design_object. A Frozen_assignment is the specification that one or more Frozen_design_object must be unchanged during further design activity.

The data associated with this application object are the following:

  • assigned_approval
  • frozen_item

4.2.394.1 assigned_approval

An assigned_approval specifies the role of the Ee_approval for the Frozen_assignment.

4.2.394.2 frozen_item

The frozen_item specifies a Frozen_design_object for which the design is to remain unchanged in the Frozen_assignment. There shall be one or more Frozen_design_object for a Frozen_assignment.

Formal constraints:

WR1: Each Frozen_assignment shall not be referenced by any Frozen_assignment.

NOTE:

ENTITY frozen_assignment
  SUBTYPE OF (managed_design_object);
    frozen_item       : SET [1:?] OF frozen_design_object;
    assigned_approval : ee_approval;
END_ENTITY; -- frozen_assignment
