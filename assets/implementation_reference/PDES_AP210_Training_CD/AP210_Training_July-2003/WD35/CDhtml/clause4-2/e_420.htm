4.2.418 Geometric_representation_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Geometric_representation_relationship is a type of Managed_design_object. A Geometric_representation_relationship may be a Shape_composition_relationship, or a Component_part_2d_geometric_relationship or a Component_part_3d_relationship or a Component_part_non_planar_relationship. A Geometric_representation_relationship is the association between a Shape_definition and another Shape_definition for predefined purposes.

NOTE: The purposes are described in the subtypes of this Application Object.

The data associated with this application object are the following:

  • shape_1
  • shape_2
  • purpose

4.2.418.1 shape_1

The shape_1 specifies one role of the Shape_definition for the Geometric_representation_relationship.

4.2.418.2 shape_2

The shape_1 specifies one role of the Shape_definition for the Geometric_representation_relationship.
