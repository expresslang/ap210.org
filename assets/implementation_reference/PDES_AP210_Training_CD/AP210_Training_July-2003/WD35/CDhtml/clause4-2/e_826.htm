4.2.826 Simultaneous_geometric_tolerance_requirement

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Simultaneous_geometric_tolerance_requirement is a type of Managed_design_object. A Simultaneous_geometric_tolerance_requirement is a specification that a set of two or more Geometric_tolerance is to be treated as a simultaneous requirement. That is, the shape elements to which composite profile refinement tolerances and composite feature relating position tolerances are applied are treated as separate shape elements with regard to the geometric tolerances. If this type of independence is not desired, the Simultaneous_geometric_tolerance_requirement may be used to specify a set of two or more Geometric_tolerance that are to be treated as a simultaneous requirement.

NOTE: Simultaneous requirements are discussed in 4.5.12.1 of ASME Y14.5M.

The data associated with this application object are the following:

  • grouped_geometric_tolerances

4.2.826.1 grouped_geometric_tolerances

The grouped_geometric_tolerances specifies the Geometric_tolerance that are to be treated as a simultaneous requirement. There shall be two or more Geometric_tolerance for a Simultaneous_geometric_tolerance_requirement.

Formal constraints:

WR1: The Geometric_tolerance that are specified as the grouped_geometric_tolerances either shall all be Geometric_tolerance_without_referenced_datum_system or shall all specify the same instance of Datum_system_defining_conditions as their datum_system_definition.

WR2: The Geometric_tolerance that are specified as the grouped_geometric_tolerances shall either be Composite_linear_profile_refinement_tolerance, Composite_surface_profile_refinement_tolerance, or Composite_feature_relating_position_tolerance.
