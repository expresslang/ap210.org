4.2.162 Composite_shape_element

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
Composite_shape_element is a type of Shape_element that is composed of two or more constituent Shape_elements. A Composite_shape_element is either a Composite_group_shape_element or a Composite_unit_shape_element.

The data associated with this application object are the following:

  • composing_relationships

4.2.162.1 composing_relationships

The composing_relationships is an inverse relationship that specifies that the existence of the Composite_shape_element is dependent on the existence of the Shape_element_composing_relationship that specifies the Composite_shape_element as its relating_shape_element. There shall be two or more Shape_element_composing_relationship for a Composite_shape_element.
