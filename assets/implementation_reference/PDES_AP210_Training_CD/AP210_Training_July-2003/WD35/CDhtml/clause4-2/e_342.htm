4.2.344 Electrical_isolation_removal_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Electrical_isolation_removal_template is a type of Material_removal_feature_template that describes the shape of material to be removed from a feature. The purposes predefined in this part of ISO 10303 include electrical and thermal isolation.

EXAMPLES:

  1. Figure mrft illustrates a template and associated parameters designed for electrical isolation. For a specific template, values will be assigned for the parameters and the explicit geometry will be representative of the assigned parameter values.
  2. Figure mrft2 illustrates a template and associated parameters designed for thermal isolation. For a specific template, values will be assigned for the parameters and the explicit geometry will be representative of the assigned parameter values. In this example, an instance would be a complex instance including Composite_unit_shape_element and Electrical_isolation_removal_template.

The data associated with this application object are the following:

  • electrical_isolation_spacing_requirement

4.2.344.1 electrical_isolation_spacing_requirement

The electrical_isolation_spacing_requirement specifies the role of the Layout_spacing_requirement_occurrence for the Add_design_object_management_relationship.
