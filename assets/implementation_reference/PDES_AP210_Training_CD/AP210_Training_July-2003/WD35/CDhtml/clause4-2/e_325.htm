4.2.325 Ee_product_model

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Ee_product_model is a type of Managed_design_object. An Ee_product_model is the nomenclature that an enterprise marketing function assigns for purposes of communication with customers.

The data associated with this application object are the following:

  • model_name

4.2.325.1 model_name

The model_name specifies the Ee_text identifier of the Ee_product_model.

Formal constraints:

UR1: The model_name shall be unique within a population of Ee_product_models.

NOTE: 
*)
ENTITY ee_product_model
  SUBTYPE OF (managed_design_object);
    model_name : ee_text;
  UNIQUE
    ur1 : model_name;
END_ENTITY; -- ee_product_model
