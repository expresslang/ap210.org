4.2.934 True_profile

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A True_profile is a type of Managed_design_object. A True_profile is a definition that characterizes the theoretically exact shape of a Physical_feature_or_part_template.

The data associated with this application object are the following:

  • boundary_definition
  • profile_representation
  • profiled_surface

4.2.934.1 boundary_definition

The boundary_definition is an inverse relationship that specifies that the existence of the True_profile is dependent on the existence of the Profile_boundary_definition that specifies the True_profile as its basic_profile. There shall be one or more Profile_boundary_definition for a True_profile.

4.2.934.2 profile_representation

The profile_representation specifies one of a Bound_volume_shape, a Dimension_set, an Ee_document, or an Ee_rule_function that represents the True_profile.

4.2.934.3 profiled_surface

The profiled_surface specifies the Physical_feature_or_part_template that is characterized by the True_profile
