4.2.794 Rule_submission

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_submission is a type of Rule_action stating that a version of an Ee_rule has been submitted.

The data associated with this application object are the following:

  • submission_rationale

4.2.794.1 submission_rationale

The submission_rationale specifies the text that describes the rationale for the Rule_submission.
