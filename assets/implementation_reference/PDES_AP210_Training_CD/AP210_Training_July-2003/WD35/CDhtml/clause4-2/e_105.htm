4.2.105 Common_datum

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Common_datum is a type of Datum that is established by two or more Datum_feature.

NOTE: Common datums are described in 6.2.2 of ISO 5459.

Formal constraints:

WR1: Each Common_datum shall participate in establishing at least one Datum_system.

WR2: Each Common_datum shall be specified as the relating_shape_element by two or more Datum_feature_usage_in_common_datum_relationship.

NOTE: 
*)
ENTITY common_datum
  SUBTYPE OF (datum);
END_ENTITY; -- common_datum
