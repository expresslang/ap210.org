4.2.502 Layout_land_width_tolerance_requirement_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Layout_land_width_tolerance_requirement_occurrence is a type of Inspection_requirement_occurrence that characterizes the land width range that is required to be produced for a particular terminal pitch class of mating components.

The data associated with this application object are the following:

  • mating_feature_pitch_class
  • maximum_negative_deviation

4.2.502.1 mating_feature_pitch_class

The mating_feature_pitch_class specifies the role of the Pitch_class for the Layout_land_width_tolerance_requirement_occurrence.

4.2.502.2 maximum_negative_deviation

The maximum_negative_deviation specifies the largest negative Length_data_element allowed width deviation for the Layout_land_width_tolerance_requirement_occurrence. The deviation is measured at the surface of the Land that mates with the external device.

NOTE: The measurement location deviates from normal fabrication practice of measuring at the base of the Land in order to guarantee successful assembly process.
