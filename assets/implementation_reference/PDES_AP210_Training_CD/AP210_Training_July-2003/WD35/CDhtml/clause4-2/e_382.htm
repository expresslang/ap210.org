4.2.380 Fill_area_style_tile

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Fill_area_style_tile is a type of Managed_design_object. A Fill_area_style_tile is the definition of the tile for a Fill_area_style.

The data associated with this application object are the following:

  • tile
  • tiling_pattern
  • tiling_scale

4.2.380.1 tile

The tile specifies either an Externally_defined_tile, a Fill_area_style_tile_curve_with_style, a Fill_area_style_tile_solid_region, or a Fill_area_style_tile_symbol_with_style for the Fill_area_style_tile.

4.2.380.2 tiling_pattern

The tiling_pattern specifies two Vector for the Fill_area_style_tile. There shall be exactly two Vector for a Fill_area_style_tile.

4.2.380.3 tiling_scale

The tiling_scale specifies the role of the Positive_ratio_measure for a Fill_area_style_tile.
