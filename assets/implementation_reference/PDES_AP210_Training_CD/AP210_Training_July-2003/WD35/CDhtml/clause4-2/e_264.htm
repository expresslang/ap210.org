4.2.264 Design_functional_unit_allocation_to_assembly_joint

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Design_functional_unit_allocation_to_assembly_joint is a type of Managed_design_object. A Design_functional_unit_allocation_to_assembly_joint is an assignment of a Functional_unit in a Design_composition_path to its implementation in an Assembly_joint.

The data associated with this application object are the following:

  • functional_design_definition_path
  • implementation

4.2.264.1 functional_design_definition_path

The functional_design_definition_path specifies the role of the Design_composition_path for the Design_functional_unit_allocation_to_assembly_joint.

4.2.264.2 implementation

The implementation specifies the role of the Assembly_joint for the Design_functional_unit_allocation_to_assembly_joint.

NOTE: This application object is included to support detailed analysis of product design.

Formal constraints:

UR1: The functional_design_definition_path shall be unique within a population of Design_functional_unit_allocation_to_assembly_joint.
