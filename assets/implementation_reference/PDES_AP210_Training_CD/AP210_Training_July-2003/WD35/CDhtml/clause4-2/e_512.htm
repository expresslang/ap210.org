4.2.512 Length_trimmed_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Length_trimmed_terminal is a type of Altered_package_terminal. A Length_trimmed_terminal describes the prepared length of a terminal accomplished prior to its assembly to the printed circuit assembly. The prepared length is equivalent to the as delivered length.

The data associated with this application object are the following:

  • maximum_trimmed_length
  • minimum_trimmed_length

4.2.512.1 maximum_trimmed_length

The maximum_trimmed_length specifies the role of the Length_data_element for the longest finished length of the Length_trimmed_terminal.

4.2.512.2 minimum_trimmed_length

The minimum_trimmed_length specifies the role of the Length_data_element for the shortest finished length of the Length_trimmed_terminal.
