4.2.686 Plated_cutout_edge_segment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Plated_cutout_edge_segment is a type of Plated_inter_stratum_feature that helps compose a Partially_plated_cutout. The Plated_cutout_edge_segment is a feature with a specified shape that makes up part of the edge of a Partially_plated_cutout.

The data associated with this application object are the following:

  • composed_cutout
  • end_vertex
  • start_vertex

4.2.686.1 composed_cutout

The composed_cutout specifies the Partially_plated_cutout for whose edge this Plated_cutout_edge_segment is a segment.

4.2.686.2 end_vertex

The end_vertex specifies one role of an Edge_segment_vertex for the Plated_cutout_edge_segment.

4.2.686.3 start_vertex

The start_vertex specifies one role of an Edge_segment_vertex for the Plated_cutout_edge_segment.

Formal constraints:

WR1: The start_vertex shall not be the end_vertex.
