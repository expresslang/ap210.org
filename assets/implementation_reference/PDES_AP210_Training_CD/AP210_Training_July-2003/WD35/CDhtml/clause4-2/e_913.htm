4.2.913 Tolerance_zone

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_zone is a type of Non_feature_shape_element that is a region within which the toleranced Shape_element must lie.

NOTE: Tolerance zones are defined in clause 7 of ISO 1101.

The data associated with this application object are the following:

  • zone_boundary
  • zone_definition

4.2.913.1 zone_boundary

The zone_boundary specifies the Tolerance_zone_boundary that define the limits of the Tolerance_zone.

4.2.913.2 zone_definition

The zone_definition is an inverse relationship that specifies that the existence of the Tolerance_zone is dependent on the existence of the Tolerance_zone_definition that specifies the Tolerance_zone as its defined_tolerance_zone.

ENTITY tolerance_zone
  SUBTYPE OF (non_feature_shape_element);
    zone_boundary : tolerance_zone_boundary;
