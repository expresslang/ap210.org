4.2.354 Equivalent_functional_terminals_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Equivalent_functional_terminals_assignment is a type of Managed_design_object. An Equivalent_functional_terminals_assignment is the assignment that a set of terminals are equivalent in a Functional_unit_usage_view.

The data associated with this application object are the following:

  • equivalent_terminals
  • name

4.2.354.1 equivalent_terminals

The equivalent_terminals specify the equivalent Functional_unit_usage_view_terminal_definition for the Equivalent_functional_terminals_assignment. There shall be at least two Functional_unit_usage_view_terminal_definition for the Equivalent_functional_terminals_assignment.

4.2.354.2 name

The name specifies the string that is the identifier for the Equivalent_functional_terminals_assignment.

Formal constraints:

WR1: One member of equivalent_functional_terminals_assignment shall establish equivalency only between members of functional_unit_usage_view_terminal_definition that are terminals of the same functional_unit_definition.
