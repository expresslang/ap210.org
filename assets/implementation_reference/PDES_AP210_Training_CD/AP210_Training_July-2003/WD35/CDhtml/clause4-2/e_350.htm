4.2.350 Elliptic_area

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Elliptic_area is a type of Primitive_2d.

The data associated with this application object are the following:

  • position
  • semi_axis_1
  • semi_axis_2

4.2.350.1 position

The position specifies an Axis_placement_2d for the Elliptic_area.

4.2.350.2 semi_axis_1

The semi_axis_1 specifies a Length_data_element for the Elliptic_area.

4.2.350.3 semi_axis_2

The semi_axis_2 specifies a Length_data_element for the Elliptic_area.
