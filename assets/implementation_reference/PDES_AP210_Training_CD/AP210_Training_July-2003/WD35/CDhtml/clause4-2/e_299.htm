4.2.294 Dimension_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Dimension_qualifier is a type of Managed_design_object. A Dimension_qualifier is either a Pre_defined_dimension_qualifier or a User_defined_dimension_qualifier. A Dimension_qualifier is a description that qualifies one or more Dimensional_characteristic.

NOTE: Qualifying notes are described in 5.3 of ISO/DIS 1101.

The data associated with this application object are the following:

  • qualified_dimensions

4.2.294.1 qualified_dimensions

The qualified_dimensions specifies the Dimensional_characteristic to which the Dimension_qualifier applies. There shall be one or more Dimensional_characteristic specified as the qualified_dimensions for a Dimension_qualifier.

ENTITY dimension_qualifier
  ABSTRACT SUPERTYPE OF (ONEOF (pre_defined_dimension_qualifier,

  SUBTYPE OF (managed_design_object);
    qualified_dimensions : SET [1:?] OF dimensional_characteristic;
END_ENTITY; -- dimension_qualifier
