4.2.114 Component_assembly_non_planar_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_assembly_non_planar_position is a type of Managed_design_object. A Component_assembly_non_planar_position is the placement location and orientation in three dimensions for the positioning of the Assembly_component_non_planar_shape in the Physical_unit_3d_shape specified by the Next_higher_assembly_relationship.

The data associated with this application object are the following:

  • assembly_model
  • component
  • component_model
  • transformation

4.2.114.1 assembly_model

The assembly_model specifies the role of the Physical_unit_3d_shape for the Component_assembly_non_planar_position.

4.2.114.2 component

The component specifies the role of the Next_higher_assembly_relationship for the Component_assembly_non_planar_position. The Next_higher_assembly_relationship specifies the actual component being placed.

4.2.114.3 component_model

The component_model specifies the role of the Assembly_component_non_planar_shape for the Component_assembly_non_planar_position.

4.2.114.4 transformation

The transformation specifies the role of the Component_non_planar_location for the Component_assembly_non_planar_position.
