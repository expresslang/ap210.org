4.2.269 Design_intent_modification_3d_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Design_intent_modification_3d_shape is a type of Bound_volume_shape and a type of Feature_shape_definition. A Design_intent_modification_3d_shape is the representation of a Design_intent_modification by a Bound_volume_shape.

The data associated with this application object are the following:

  • represented_design_intent_modification
  • shape_representation

4.2.269.1 represented_design_intent_modification

The represented_design_intent_modification specifies the Design_intent_modification that is to be characterized by the Design_intent_modification_3d_shape.

4.2.269.2 shape_representation

The shape_representation specifies the Bound_volume_shape for the representation of the Design_intent_modification.
