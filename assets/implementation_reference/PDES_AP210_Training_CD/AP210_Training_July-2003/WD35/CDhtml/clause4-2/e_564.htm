4.2.563 Non_functional_land_interface_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Non_functional_land_interface_terminal is a type of Component_feature included to identify the interface available for attachment in the next level of assembly.

NOTE: Use of this capability occurs when a member of Component_feature_to_physical_usage_view_assignment references a member of Non_functional_land_interface_terminal to assign a relationship to an Interconnect_module_terminal. A Non_functional_land_interface_terminal shall only be associated to types of Non_functional_land established for the purpose of interface attachment.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.563.1 associated_component

The associated_component specifies the role of the Non_functional_land for the Non_functional_land_interface_terminal.

4.2.563.2 definition

The definition specifies the role of the Land_template_interface_terminal for the Non_functional_land_interface_terminal.

Formal constraints:

WR1: The Land_template that is the usage_definition for the Non_functional_land in the role of associated_component for this Non_functional_land_interface_terminal is the associated_definition for the definition of this Non_functional_land_interface_terminal.
