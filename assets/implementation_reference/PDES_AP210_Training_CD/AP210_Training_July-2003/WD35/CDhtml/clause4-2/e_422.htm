4.2.420 Geometric_tolerance_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Geometric_tolerance_qualifier is a type of Managed_design_object. A Geometric_tolerance_qualifier is either a Pre_defined_geometric_tolerance_qualifier or a User_defined_geometric_tolerance_qualifier. A Geometric_tolerance_qualifier is a description that qualifies one or more Geometric_tolerance.

NOTE: Qualifying notes are described in 5.3 of ISO/DIS 1101.

The data associated with this application object are the following:

  • qualified_geometric_tolerances

4.2.420.1 qualified_geometric_tolerances

The qualified_geometric_tolerances specifies the Geometric_tolerance to which the Geometric_tolerance_qualifier applies. There shall be one or more Geometric_tolerance for a Geometric_tolerance_qualifier.
