4.2.207 Curve_style_font_and_scaling

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Curve_style_font_and_scaling is a type of Managed_design_object. A Curve_style_font_and_scaling is the combination of the computer interpretable font and the scale factor for that font used to present the Curve.

The data associated with this application object are the following:

  • curve_font
  • curve_font_scaling
  • name

4.2.207.1 curve_font

The curve_font specifies either a Curve_style_font, Curve_style_font, an Externally_defined_curve_font, Externally_defined_curve_font, or a Solid_curve, Solid_curve for the Curve_style_font_and_scaling.

4.2.207.2 curve_font_scaling

The curve_font_scaling specifies the Positive_ratio_measure used for the Curve_style_font_and_scaling.

4.2.207.3 name

The name specifies the string used for the identifier for the combination of Curve_style_font_and_scaling.

Formal constraints:

UR1: The name shall be unique within a population of Curve_style_font_and_scaling.
