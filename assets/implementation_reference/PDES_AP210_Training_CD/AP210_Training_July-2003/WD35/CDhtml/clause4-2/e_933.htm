4.2.933 Trimmed_line

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Trimmed_line is a type of Curve that has a defined, finite extent. The Trimmed_line shall be interpreted as a trimmed curve.

NOTE: The Trimmed_line is necessary to support the specification of precise, metric geometry.

NOTE: 
*)
ENTITY trimmed_line
  SUBTYPE OF (curve);
    start_point : cartesian_point;
    end_point : cartesian_point;
END_ENTITY; -- trimmed_line
