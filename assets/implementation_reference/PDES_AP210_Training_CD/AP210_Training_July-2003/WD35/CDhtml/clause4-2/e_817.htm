4.2.817 Shape_formed_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shape_formed_terminal is a type of a Altered_package_terminal that defines a modification of its shape from its original Package_terminal shape. This shape modification is done in order to facilitate the installation of a Packaged_part in an assembly.

The data associated with this application object are the following:

  • terminal_form_specification

4.2.817.1 terminal_form_specification

The terminal_form_specification specifies the Lead_form_specification of the Shape_formed_terminal.
