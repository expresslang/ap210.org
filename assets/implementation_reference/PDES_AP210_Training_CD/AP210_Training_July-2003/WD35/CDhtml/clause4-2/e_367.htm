4.2.361 External_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An External_definition is a type of Managed_design_object. An External_definition is a definition that is supported externally to this part of ISO 10303.

The data associated with this application object are the following:

  • code
  • code_value
  • reference_document

4.2.361.1 code

The code specifies the string as the identifier of the type of the External_definition.

4.2.361.2 code_value

The code_value specifies the text as the content of the External_definition.

4.2.361.3 reference_document

The reference_document specifies either the Ee_document or a string referred to as a standard document identification string, registered by IEC or ISO for the purpose of unambiguously identifying a document published by the respective organization, as the code definition document used by application programmers for the External_definition.

NOTE: The reference_document includes a data dictionary that defines the allowed types that may be used as identifiers.

Informal constraints:

IP1: The string specified by code shall not appear more than once in the reference_document. This constraint is to be interpreted as a requirement against the implementation of document when that implementation is to support External_definition.
