4.2.715 Precision_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Precision_qualifier is a type of Managed_design_object in which there is an associated precision. The precision is defined in clause 2 of "The Guide to the Expression of Uncertainty in Measurement".
