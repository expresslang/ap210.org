4.2.399 Functional_terminal_group_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_terminal_group_assignment is a type of Managed_design_object. A Functional_terminal_group_assignment allocates a Scalar_terminal_definiton to the Functional_terminal_group. This is the mechanism that provides members of the Functional_terminal_group.

The data associated with this application object are the following:

  • composed_group
  • functional_usage_view_terminal

4.2.399.1 composed_group

The composed_group specifies the role of the Functional_terminal_group for the Functional_terminal_group_assignment. The composed_group specifies the group that the terminal is a member.

4.2.399.2 functional_usage_view_terminal

The functional_usage_view_terminal specifies the role of the Scalar_terminal_defintion for the Functional_terminal_group_assignment. The functional_usage_view_terminal specifies the terminal that is a member of the group.
