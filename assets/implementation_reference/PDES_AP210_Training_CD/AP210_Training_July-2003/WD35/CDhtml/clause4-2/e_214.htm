4.2.214 Datum

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum is a type of Non_feature_shape_element that is a theoretically exact geometric reference such as a point, an axis, or a plane, to which toleranced Shape_element are related. Each Datum is either a Datum_point or a Datum_axis or a Datum_plane. Additionally each Datum may be either a Single_datum or a Common_datum.

EXAMPLE: Figure datum illustrates the usage of a Datum.

NOTE: Datums are described in clause 8 of ISO 1101.

The data associated with this application object are the following:

  • datum_feature_usages

4.2.214.1 datum_feature_usages

The datum_feature_usages is an inverse relationship that specifies that the Datum may be referenced by Datum_feature_usage_in_datum_relationship that specifies the Datum as its relating_shape_element. The datum_feature_usages need not be specified for a particular Datum.

Formal constraints:

WR1: A Datum shall be a Datum_axis, Datum_plane, or Datum_point.

ENTITY datum
  ABSTRACT SUPERTYPE OF (ONEOF (datum_point,datum_axis,datum_plane) 
      ANDOR ONEOF (single_datum,common_datum))
  SUBTYPE OF (non_feature_shape_element);
  INVERSE
    datum_feature_usages : SET [0:?] OF
                           datum_feature_usage_in_datum_relationship FOR 
                           relating_shape_element;
END_ENTITY; -- datum
