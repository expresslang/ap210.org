4.2.545 Minimally_defined_bare_die_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Minimally_defined_bare_die_terminal is a type of Part_terminal. A Minimally_defined_bare_die_terminal may be either a Guided_wave_bare_die_terminal or a Bare_die_terminal. The associated_definition specifies the Bare_die to which the Bare_die_terminal provides electrical access. A Minimally_defined_bare_die_terminal does not include location or shape information
