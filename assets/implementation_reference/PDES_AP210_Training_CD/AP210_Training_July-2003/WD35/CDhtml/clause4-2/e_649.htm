4.2.649 Physical_component_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_component_feature is a type of Component_feature and a type of Physical_feature_or_part_template. The Physical_component_feature supports the specification of a Component_feature as a Datum_target or Datum_feature for the purpose of estabishing a Datum_reference_frame for tolerance specification.
