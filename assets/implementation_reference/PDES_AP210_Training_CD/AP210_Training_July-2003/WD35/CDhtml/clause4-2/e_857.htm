4.2.857 Stratum_feature_planar_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_feature_planar_shape is a type of Managed_design_object. A Stratum_feature_planar_shape is the representation of a Stratum_feature by a Planar_projected_shape.

The data associated with this application object are the following:

  • represented_stratum_feature
  • shape_representation

4.2.857.1 represented_stratum_feature

The represented_stratum_feature specifies the Stratum_feature that is to be characterized by the Stratum_feature_planar_shape.

4.2.857.2 shape_representation

The shape_representation specifies the Planar_projected_shape for the representation of the Stratum_feature.
