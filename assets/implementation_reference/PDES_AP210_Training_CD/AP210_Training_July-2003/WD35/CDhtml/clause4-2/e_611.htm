4.2.607 Part_device_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_device_terminal is a type of Managed_design_object. A Part_device_terminal is the surface area on an occurrence of a Bare_die in a Packaged_part that shall allow access to the function of an occurrence of the Bare_die. A Part_device_terminal is included to support detaild modeling and analysis of the internal connection to a Packaged_part_terminal. No connection between Part_device_terminals is supported in this part of ISO 10303. Connections between Bare_die are supported only within Assembly_module.

The data associated with this application object are the following:

  • terminated_device
  • used_bare_die_terminal

4.2.607.1 terminated_device

The terminated_device specifies the Part_device that has this Part_device_terminal.

4.2.607.2 used_bare_die_terminal

The used_bare_die_terminal specifies the Bare_die_terminal that shall be used by the Part_device_terminal.

Formal constraints:

UR1: The combination of the terminated_device and the used_bare_die_terminal shall be unique within a population of Part_device_terminals.
