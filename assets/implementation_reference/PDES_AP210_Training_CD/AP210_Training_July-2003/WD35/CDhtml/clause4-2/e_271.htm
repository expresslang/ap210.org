4.2.271 Design_intent_modification_planar_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Design_intent_modification_planar_position is a type of Managed_design_object. A Design_intent_modification_in_stratum_planar_position is the two dimensional position and orientation of a two dimensional model of a Design_intent_modification in the two dimensional model of the Stratum.

The data associated with this application object are the following:

  • feature_shape
  • stratum_shape
  • transformation

4.2.271.1 feature_shape

The feature_shape specifies either a Design_intent_modification_planar_shape or a Design_intent_modification_styled_area_representation for the Design_intent_modification_planar_position.

4.2.271.2 stratum_shape

The stratum_shape specifies the role of the Stratum_planar_shape for the Design_intent_modification_planar_position.

4.2.271.3 transformation

The transformation specifies the role of the Axis_placement_2d for the Design_intent_modification_planar_position.
