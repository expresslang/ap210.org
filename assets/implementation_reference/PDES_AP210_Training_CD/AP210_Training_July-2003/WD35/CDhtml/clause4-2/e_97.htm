4.2.97 Change_request

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Change_request is a type of Work_request that identifies the proposed extent of work to be done to modify the design of a product.

The data associated with this application object are the following:

  • consequence
  • recommended_solution
  • version

4.2.97.1 consequence

The consequence specifies the Ee_text that describes the impact that the Change_request will have on the product. There shall be one or more Ee_text for a Change_request.

4.2.97.2 recommended_solution

The recommended_solution specifies one or more Design_object_management_relationship, or Ee_text of a potential solution which meets the requirements defined in the Change_request. A recommended_solution need not be specified for a Change_request.

4.2.97.3 version

The version specifies the string identifying a Change_request. The version shall be human interpretable.

Formal constraints:

UR1: The combination of the version and the work_request_id inherited from Work_request shall be unique within a population of Change_request.

ENTITY change_request
  SUBTYPE OF (work_request);
    recommended_solution : OPTIONAL SET [1:?] OF 
                           design_object_management_relationship_or_description;
    consequence          : SET [1:?] OF ee_text;
    version              : ee_name;
  UNIQUE
    ur1 : version, SELF\work_request.work_request_id;
END_ENTITY; -- change_request
