4.2.379 Fill_area_style_hatching

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Fill_area_style_hatching is a type of Managed_design_object. A Fill_area_style_hatching is the parallel line hatch style of a Fill_area. All units associated with Fill_area_style_hatching are defined in the global units for the Interconnect_module being designed. The geometric reference for Fill_area_style_hatching shall be the origin of the Interconnect_module being designed. When this Application Object is used to help compose a Design_intent_modification_styled_area_representation or to help compose a Stratum_feature_styled_area_representation, only the Curve_style_font shall be referenced by the curve_font attribute of the Curve_style Application Object referenced by hatch_line_appearance.

NOTE: In order to obtain cross-hatch, two instances of Fill_are_style_hatching are required, with complementary angles.

The data associated with this application object are the following:

  • hatch_line_angle
  • hatch_line_appearance
  • pattern_start
  • point_of_reference_hatch_line
  • start_of_next_hatch_line

4.2.379.1 hatch_line_angle

The hatch_line_angle specifies the role of the Angle_data_element for the Fill_area_style_hatching.

4.2.379.2 hatch_line_appearance

The hatch_line_appearance specifies the role of the Curve_style for the Fill_area_style_hatching.

NOTE: The curve style may include a discontiguous geometric shape, in which case the hatch cannot be used to represent product data which is contiguous in nature.

EXAMPLE: The curve style may be a dashed line and in that case each of the dashes would correspond to a differenct Stratum_feature.

4.2.379.3 pattern_start

The pattern_start specifies the Cartesian_point for the beginning of the pattern of Fill_area_style_hatching.

4.2.379.4 point_of_reference_hatch_line

The point_of_reference_hatch_line specifies the Cartesian_point for the point of reference and located on the initial hatch line for the Fill_area_style_hatching.

4.2.379.5 start_of_next_hatch_line

The start_of_next_hatch_line specifies the Vector from the end Cartesian_point of the present hatch line to the beginning Cartesian_point of the next hatch line for the Fill_area_style_hatching.
