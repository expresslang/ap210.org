4.2.561 Non_feature_shape_element_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Non_feature_shape_element_relationship is a type of Shape_element_relationship that associates a Derived_shape_element with a Shape_element from which the Derived_shape_element is derived.

The data associated with this application object are the following:

  • related_shape_element
  • relating_shape_element

4.2.561.1 related_shape_element

The related_shape_element specifies the Non_feature_shape_element for the Non_feature_shape_element_relationship.

4.2.561.2 relating_shape_element

The relating_shape_element specifies the Non_feature_shape_element for the Non_feature_shape_element_relationship.

NOTE: The attribute related_shape_element is declared in the Shape_element_relationship entity of which this entity is a subtype.

Formal constraints:

UR1: The combination of the relating_shape_element inherited from Shape_element_relationship and related_shape_element inherited from Shape_element_relationship shall be unique within a population of Non_feature_shape_element_relationship.
