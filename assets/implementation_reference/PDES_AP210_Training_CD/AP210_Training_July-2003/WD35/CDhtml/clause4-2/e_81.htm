4.2.81 Buried_via

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Buried_via is a type of Via. Buried_via is defined in IEC 60050-541.

EXAMPLE: Figure buv illustrates a Buried_via in a cross-section view of an interconnect substrate.

Formal constraints:

WR1: No end of the Buried_via shall be exposed in the Interconnect_module.

NOTE: 
*)
ENTITY buried_via
  SUBTYPE OF (via);
END_ENTITY; -- buried_via
