4.2.514 Limit_dimension_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Limit_dimension_representation is a type of Shape_dimension_representation that explicitly describes the maximum and minimum limits of one or more Dimensional_characteristic.

The data associated with this application object are the following:

  • lower_dimension_value
  • upper_dimension_value

4.2.514.1 lower_dimension_value

The lower_dimension_value specifies the Data_element that describes the minimum limit of the Dimensional_characteristic for which the Limit_dimension_representation is a representation.

4.2.514.2 upper_dimension_value

The upper_dimension_value specifies the Data_element that describes the maximum limit of the Dimensional_characteristic for which the Limit_dimension_representation is a representation.

Formal constraints:

WR1: The value of the upper_dimension_value shall be greater than the value of the lower_dimension_value.

  SUBTYPE OF (shape_dimension_representation);
