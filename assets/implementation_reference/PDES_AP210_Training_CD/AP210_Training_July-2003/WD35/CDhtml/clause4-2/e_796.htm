4.2.796 Runout_tolerance_zone_definition_with_specified_angle

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Runout_tolerance_zone_definition_with_specified_angle is a type of Tolerance_zone_definition that is defined by either a Circular_runout_tolerance or a Total_runout_tolerance, and in which the Tolerance_zone is oriented with respect to a Directed_axis. The Directed_axis is coincident with the Datum_axis about which the Tolerance_zone is centred.

NOTE: Circular run-out tolerance in a specified direction is described in 14.13.4 of ISO 1101.

The data associated with this application object are the following:

  • runout_orientation

4.2.796.1 runout_orientation

The runout_orientation specifies the Runout_zone_orientation that defines the orientation of the Tolerance_zone relative to a specified Directed_axis for the Runout_tolerance_zone_definition_with_specified_angle.

Formal constraints:

WR1: The Tolerance_zone specified as the oriented_zone by the Runout_tolerance_zone_orienting_relationship that is specified as the oriented_relationship by the Runout_zone_orientation that is specified as the runout_orientation, shall be the same instance of Tolerance_zone as specified as the defined_tolerance_zone.

WR2: The Runout_tolerance_zone_definition_with_specified_angle shall be specified as the tolerance_zone_or_boundary_definition by exactly one Geometric_tolerance, and that Geometric_tolerance shall be either a Circular_runout_tolerance or a Total_runout_tolerance.

WR3: The instance of Centre_axis specified as the axis_to_be_oriented by the Directed_axis to which the Tolerance_zone is oriented shall be a Datum_axis that is in the Datum_system the definition of which is specified as the datum_system_definition by the associated Circular_runout_tolerance or Total_runout_tolerance.
