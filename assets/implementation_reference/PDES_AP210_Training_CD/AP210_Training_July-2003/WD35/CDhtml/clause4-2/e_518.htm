4.2.515 Line_width_class

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Line_width_class is a type of Managed_design_object. A Line_width_class is a category which assigns a name to a length range for classification purposes. The Data_element used as the basis for the range shall be a Length_data_element.

EXAMPLE: The Line_width_class may be exchanged so that tolerances may be applied to all members of the same class, and specified only once. E.g., all lines with a design width greater than 50 mils and less than 75 mils are in the 'big' class. All members of 'big' shall have a tolerance of +5/-5 mils. Other Application Objects must be populated in order to apply the tolerances to the class.

The data associated with this application object are the following:

  • class_member
  • class_name

4.2.515.1 class_member

The class_member specifies the role of the Range_characteristic for the Line_width_class. The Range_characteristic shall provide length data.

4.2.515.2 class_name

The class_name specifies the name for the Line_width_class.

ENTITY line_width_class
  SUBTYPE OF (managed_design_object);
    class_member : range_characteristic;
    class_name   : ee_name;
WHERE
 wr1: SIZEOF(QUERY( np <* class_member\coordinated_characteristic.component_characteristic |
      NOT('AP210_ARM.LENGTH_DATA_ELEMENT' IN TYPEOF(np\characteristic.property_value))
         )) = 0;
END_ENTITY; -- line_width_class
