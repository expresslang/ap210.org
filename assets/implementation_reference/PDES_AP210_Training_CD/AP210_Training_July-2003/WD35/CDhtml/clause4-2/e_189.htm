4.2.189 Connection_zone_map_identification

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Connection_zone_map_identification is a type of Managed_design_object. A Connection_zone_map_identification specifies the geometric, causal, relationship between a reference Connection_zone associated with a Package_terminal and a mapped Connection_zone associated with a template interface terminal.
