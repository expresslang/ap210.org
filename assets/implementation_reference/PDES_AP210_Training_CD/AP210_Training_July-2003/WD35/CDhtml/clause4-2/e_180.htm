4.2.180 Conical_tolerance_zone_boundary_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Conical_tolerance_zone_boundary_definition is a type of Tolerance_zone_boundary_definition that characterizes a Conical_tolerance_zone_boundary.

The data associated with this application object are the following:

  • boundary_sizes
  • defined_zone_boundary

4.2.180.1 boundary_sizes

The boundary_sizes specifies the Conical_tolerance_zone_boundary_diametrical_size_characteristic that defines the size of the Conical_tolerance_zone_boundary at specified Physical_feature_or_part_templates. There shall be exactly two Conical_tolerance_zone_boundary_diametrical_size_characteristic specified as the boundary_sizes for the Conical_tolerance_zone_boundary_definition.

4.2.180.2 defined_zone_boundary

The defined_zone_boundary specifies the Conical_tolerance_zone_boundary that is characterized by the Conical_tolerance_zone_boundary_definition.

Formal constraints:

WR1: Each Geometric_tolerance that defines a Conical_tolerance_zone_boundary_diametrical_size_characteristic of the Conical_tolerance_zone_boundary_definition shall be of the same type.

WR2: Each Geometric_tolerance that defines a Conical_tolerance_zone_boundary_diametrical_size_characteristic of the Conical_tolerance_zone_boundary_definition shall reference the same Datum_system_defining_conditions.

WR3: Each Geometric_tolerance that defines a Conical_tolerance_zone_boundary_diametrical_size_characteristic of the Conical_tolerance_zone_boundary_definition shall specify the same value for the applied_material_condition_property_name.

WR4: Each Geometric_tolerance that defines a Conical_tolerance_zone_boundary_diametrical_size_characteristic of the Conical_tolerance_zone_boundary_definition shall specify the same instance of Centre_axis as the toleranced_shape_element.

WR5: The Conical_tolerance_zone_boundary_definition shall be specified as the boundary_definition by a Conical_tolerance_zone_definition.
