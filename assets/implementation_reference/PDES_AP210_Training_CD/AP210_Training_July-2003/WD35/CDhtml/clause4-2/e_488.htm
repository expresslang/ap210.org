4.2.488 Laminate_text_component

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Laminate_text_component is a type of Laminate_component that is designed to be interpreted as a single character portion of a text message by a human or by an artificial vision system. A Laminate_text_component may be used to help compose a Laminate_text_string_component to form a complete message. The composition will be defined by an instance of Assembly_group_component that relates all necessary members of Laminate_component to compose the message embodied in the instance of Laminate_text_string_component. Each Stratum_feature referenced by the stratum_feature_implementation shall be on the same Stratum.

EXAMPLE: Figure ltc illustrates possible arrangements and orientation of the Laminate_text_string_component in the material of the Stratum. The figure includes multiple Laminate_text_component members presenting characters.

The data associated with this application object are the following:

  • of_character
  • stratum_feature_implementation
  • usage_definition

4.2.488.1 of_character

The of_character specifies the message content, a character, for the Laminate_text_component.

4.2.488.2 stratum_feature_implementation

The stratum_feature_implementation specifies the role of the Stratum_feature for the Laminate_text_component. There may be more than one Stratum_feature for the Laminate_text_component.

NOTE: Multiple Stratum_feature may exist in the case that the shape of the character is disjoint, but they are all on the same Stratum.

4.2.488.3 usage_definition

The usage_definition specifies the Part_text_template that contains the font definition and associated parameters to be used for the Laminate_text_component.
