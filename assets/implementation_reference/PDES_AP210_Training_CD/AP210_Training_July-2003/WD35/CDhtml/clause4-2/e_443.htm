4.2.450 Interconnect_module_edge_segment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_edge_segment is a type of Inter_stratum_feature. The Interconnect_module_edge_segment is an entity with a uniquely specified shape that makes up part of the edge of an interconnect module.

EXAMPLE: Usually each side of a four-sided board is an Interconnect_module_edge_segment, or if a board is a round board, one of the Interconnect_module_edge_segment's may be a notch for orientation purposes. The set of Interconnect_module_edge_segment's together make up the total edge.

The data associated with this application object are the following:

  • composed_edge
  • end_surface
  • start_vertex

4.2.450.1 composed_edge

The composed_edge specifies the role of the Interconnect_module_edge for the Interconnect_module_edge_segment.

4.2.450.2 end_vertex

The end_vertex specifies one role of an Edge_segment_vertex for an Interconnect_module_edge_segment.

4.2.450.3 start_vertex

The start_vertex specifies one role of an Edge_segment_vertex for an Interconnect_module_edge_segment.

Formal constraints:

WR1: The end_vertex shall not be the start_vertex.
