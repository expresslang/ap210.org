4.2.414 Functional_usage_view_to_printed_part_template_terminal_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Functional_usage_view_to_printed_part_template terminal_assignment is a type of Managed_design_object. A Functional_usage_view_to_printed_part_template terminal_assignment is the association between a functional terminal definition and a terminal of a Printed_part_template.

EXAMPLE: It is permissible for one functional_usage_view_terminal to be assigned to more than one physical_usage_view_terminal, for example, VCC.

The data associated with this application object are the following:

  • functional_usage_view_terminal
  • physical_usage_view_terminal

4.2.414.1 functional_usage_view_terminal

The functional_usage_view_terminal specifies the role of the Scalar_terminal_definition for the Functional_usage_view_to_printed_part_template_terminal_assignment.

4.2.414.2 physical_usage_view_terminal

The physical_usage_view_terminal specifies the role of the Printed_part_template_terminal for the Functional_usage_view_to_printed_part_template_terminal_assignment.

Formal constraints:

UR1: The combination of functional_usage_view_terminal and physical_usage_view_terminal shall be unique within a population of Functional_usage_view_to_printed_part_template_terminal_assignment.

WR1: The members of functional_usage_view_to_printed_part_template_terminal_assignment that specify the same member of physical_unit_usage_view through the path physical_usage_view_terminal.associated_definition shall all specify the same member of functional_unit_definition through the path functional_usage_view_terminal.associated_functional_unit_definition.
