4.2.86 Cartesian_coordinate_system

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Cartesian_coordinate_system is a type of Managed_design_object. A Cartesian_coordinate_system is a system of two coordinates that locates a point on a plane and measures its distance from either of two intersecting straight-line axes along a line parallel to the other axis, or a system of three coordinates that locates a point in space and measures its distance from any of three intersecting coordinate planes measured parallel to that one of three straight- line axes that is the intersection of the other two planes.

The data associated with this application object are the following:

  • angle_unit_of_measure
  • angular_uncertainty
  • dimensionality
  • length_unit_of_measure
  • length_uncertainty

4.2.86.1 angle_unit_of_measure

The angle_unit_of_measure specifies either degree or radian as the measurement unit of the Cartesian_coordinate_system.

4.2.86.2 angular_uncertainty

The angular_uncertainty specifies the Angle_data_element of the uncertainty of angularity for the Cartesian_coordinate_system.

NOTE: This value is based on the precision of the processor that created the model.

4.2.86.3 dimensionality

The dimensionality specifies either the two-dimensional or three-dimensional nature of the Cartesian_coordinate_system.

4.2.86.4 length_unit_of_measure

The length_unit_of_measure specifies the string for the unit of measurement for the Cartesian_coordinate_system. This shall be interpreted as a length unit of measure.

4.2.86.5 length_uncertainty

The length_uncertainty specifies the Length_data_element of the uncertainty of length for the Cartesian_coordinate_system.

NOTE: This value is based on the precision of the processor that created the model.

ENTITY cartesian_coordinate_system
  SUBTYPE OF (managed_design_object);
    length_unit            : STRING;
    length_unit_prefix     : OPTIONAL si_prefix;
    length_dynamic_range   : OPTIONAL INTEGER;
    dimensionality         : two_or_three_dimensional;
    angle_unit             : degree_or_radian;
    length_uncertainty     : length_data_element;
    angular_uncertainty    : OPTIONAL angle_data_element;
END_ENTITY; -- cartesian_coordinate_system
