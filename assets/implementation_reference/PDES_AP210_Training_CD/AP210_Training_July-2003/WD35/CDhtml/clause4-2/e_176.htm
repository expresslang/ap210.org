4.2.176 Conductor

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Conductor is a type of Stratum_feature realized in a Design_layer_stratum that participates in the implementation of a Physical_network.

EXAMPLE: Figure conduct illustrates a detail of a layout drawing that shows the Conductor in relationship to a component at the next level of the assembly.

The data associated with this application object are the following:

  • implementation_requirement

4.2.176.1 implementation_requirement

The implementation_requirement specifies the Intra_stratum_join_relationship that is used to constrain the Conductor by specifying the Layer_connection_point that are to be connected by the Conductor.

Formal constraints:

WR1: The shape that is a Stratum_feature shape shall not violate the nearest approach constraint to another shape assigned to the same design layer and realized as the stratum material of the design layer stratum.

WR2: A Conductor shall enclose two or more layer connection point in the region between or on the outer bound and an inner bound; or inside or on the outer bound that has no inner bound.
