4.2.93 Centre_point

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Centre_point is a type of Centre_of_symmetry the form of which is that of a point. The Shape_element from which the Centre_point is derived are associated with the Centre_point by the Shape_element_deriving_relationship that are specified as the deriving_relationships.

NOTE: The attribute deriving_relationships is declared in the Derived_shape_element entity of which this entity is a subtype.

Informal constraints:

IP1: The form of the Shape_elements from which the Centre_point is derived shall be consistent with the form of shape elements from which a centre point can be derived.

NOTE: 
*)
ENTITY centre_point
  SUBTYPE OF (centre_of_symmetry);
END_ENTITY; -- centre_point
