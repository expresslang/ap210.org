4.2.343 Electrical_isolation_removal_structure

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Electrical_isolation_removal_structure is a type of Filled_area_material_removal_structure that has specific materials removed from specific subsets of the filled area for the purpose of electrical isolation.

The data associated with this application object are the following:

  • electrical_isolation_spacing_requirement

4.2.343.1 electrical_isolation_spacing_requirement

The electrical_isolation_spacing_requirement specifies the Ee_requirement_occurrence of electrical isolation for the Electrical_isolation_removal_structure.
