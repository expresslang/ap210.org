4.2.425 Geometrically_bounded_surface

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Geometrically_bounded_surface is a type of Bound_volume_shape. The Geometrically_bounded_surface is a surface that does not include topological information as to what edges are shared between surfaces. Only geometric information is supplied, which is useful only for presentation purposes. This entity should not be used to convey connectivity-related data unless independent connectivity reconstruction and verification is performed.
