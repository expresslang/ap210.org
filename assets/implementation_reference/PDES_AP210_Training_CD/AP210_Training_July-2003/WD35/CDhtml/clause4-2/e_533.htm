4.2.533 Material_removal_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Material_removal_feature is a type of Design_intent_modification that indicates that material is to be removed in the realization of the feature that is in the role of design_intent inherited from Design_intent_modification. A Material_removal_feature may be a Filled_area_material_removal_structure. The requirement shall be interpreted mathematically as a requirement for application of the regularized Boolean subtraction operation.

The data associated with this application object are the following:

  • material_removal_shape

4.2.533.1 material_removal_shape

The material_removal_shape specifies the role of the Material_removal_feature_template for the Material_removal_feature.
