4.2.278 Design_object_management_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Design_object_management_relationship is one of an Add_design_object_management_relationship, a Change_design_object_management_relationship, or a Delete_design_object_management_relationship. A Design_object_management_relationship is the association between the Ee_product_definition changed from to the Ee_product_definition changed to.

The data associated with this application object are the following:

  • current_design
  • previous_design

4.2.278.1 current_design

The current_design specifies the current Ee_product_definition, after the change, for the Design_object_management_relationship.

4.2.278.2 previous_design

The previous_design specifies the Ee_product_definition, before the change, for the Design_object_management_relationship.

Formal constraints:

WR1: The current_design shall not be the previous_design.

WR2: The type of both Ee_product_definitions shall be the same (e.g., both assembly_module).

NOTE: 
*)
ENTITY design_object_management_relationship
  ABSTRACT SUPERTYPE OF (ONEOF (
      change_design_object_management_relationship,
      delete_design_object_management_relationship,
      add_design_object_management_relationship));
    previous_design : ee_product_definition;
    current_design  : ee_product_definition;
END_ENTITY; -- design_object_management_relationship
