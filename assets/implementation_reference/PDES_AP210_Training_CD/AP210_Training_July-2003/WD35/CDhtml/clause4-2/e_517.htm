4.2.518 Linear_extension

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Linear_extension is a type of Derived_shape_element that is derived by linearly extending a Shape_element that is a curve or surface. The Shape_element of which the Linear_extension is an extension is associated with the Linear_extension by the Shape_element_deriving_relationship that are specified as the deriving_relationships.

NOTE: The attribute deriving_relationships is declared in the Derived_shape_element entity of which this entity is a subtype.

Formal constraints:

WR1: The Linear_extension shall be derived from only one Shape_element.

Informal constraints:

IP1: The Shape_element from which the Linear_extension is derived shall be a curve or surface.

NOTE: 
*)
ENTITY linear_extension
  SUBTYPE OF (derived_shape_element);
END_ENTITY; -- linear_extension
