4.2.611 Part_feature_model_port_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_feature_model_port_assignment is a type of Managed_design_object. A Part_feature_model_port_assignment is the relationship between an Analytical_model_port and the Part_feature.

NOTE: Usually a simulation model is associated on a one to one basis with the Physical_unit_usage_view.

The data associated with this application object are the following:

  • source
  • target

4.2.611.1 source

The source specifies the Part_feature whose port is to be associated with an Analytical_model_port in a Part_feature_model_port_assignment.

4.2.611.2 target

The target specifies the Analytical_model_port that is to be associated with a Part_feature in a Part_feature_model_port_assignment.
