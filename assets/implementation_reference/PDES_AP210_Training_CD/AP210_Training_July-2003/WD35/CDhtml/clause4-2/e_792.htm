4.2.792 Rule_set

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_set is a type of Managed_design_object. A Rule_set may be a Rule_set_group. A Rule_set is an association among Ee_rule with Rule_priority.

The data associated with this application object are the following:

  • identifier
  • purpose
  • rule_priority_member

4.2.792.1 identifier

The identifier specifies the human-interpretable identification for the Rule_set.

4.2.792.2 purpose

The purpose specifies the human-interpretable string for the purpose for the Rule_set.

4.2.792.3 rule_priority_member

The rule_priority_member specifies the role of the Rule_priority for the Rule_set. There shall be one or more Rule_priority for the Rule_set.

Formal constraints:

UR1: The identifier shall be unique within a population of Rule_sets.
