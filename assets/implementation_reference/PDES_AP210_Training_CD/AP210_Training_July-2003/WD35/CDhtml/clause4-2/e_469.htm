4.2.476 Internal_stratum_access

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Internal_stratum_access is a type of Managed_design_object. An Internal_stratum_access is a relationship between an Internal_probe_access_area and a Via that provides access to a Stratum_feature on an internal Design_layer_stratum. This Application Object is part of a logical structure to identify the special usage of a Via in support of accessing internal conductors in the design.

The data associated with this application object are the following:

  • connected_probe
  • inter_stratum_feature

4.2.476.1 connected_probe

The connected_probe specifies the Internal_probe_access_area that provides probe access to a Stratum_feature on an internal Design_layer_stratum for the Internal_stratum_access.

4.2.476.2 inter_stratum_feature

The inter_stratum_feature specifies the Via that provides electrical connectivity to the Stratum_feature to be probed for the Internal_stratum_access. The external surface of the Via is the area the test probe mates with in a test setup.

Formal constraints:

UR1: The combination of the connected_probe and the inter_stratum_feature shall be unique within a population of Internal_stratum_accesses.

NOTE: 
*)
ENTITY internal_stratum_access
  SUBTYPE OF (managed_design_object);
    connected_probe       : internal_probe_access_area;
    inter_stratum_feature : via;
  UNIQUE
    ur1 : connected_probe, inter_stratum_feature;
END_ENTITY; -- internal_stratum_access
