4.2.307 Documentation_layer_technology

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Documentation_layer_technology is a type of Stratum_technology that defines the purpose for the Documentation_layer_stratum.

The data associated with this application object are the following:

  • documentation_layer_purpose

4.2.307.1 documentation_layer_purpose

The documentation_layer_purpose specifies the string that describes the purpose for the Documentation_layer_technology.

EXAMPLE: Examples of these purposes are "Silkscreen", "Solder Mask", and "Solder Paste".
