4.2.886 Technology

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Technology is a type of Managed_design_object. A Technology is the manufacturing process for the realization of an Ee_product.

The data associated with this application object are the following:

  • classification
  • material_product_system
  • process_technology

4.2.886.1 classification

The classification specifies the External_definition which defines the Technology.

4.2.886.2 material_product_system

The material_product_system specifies the Ee_material which defines the material combination which is necessary for the output of the manufacturing process.

4.2.886.3 process_technology

The process_technology specifies the Process_specification which defines the exact process of the Technology.
