4.2.148 Component_termination_passage_interface_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_termination_passage_interface_terminal is a type of Component_feature. The Component_termination_passage_interface_terminal is the internal wall of the passage only and is the complete internal wall. It does not extend to the exterior top or bottom edges of the passage. It does not include an exterior surface of the passage. This terminal is the definition of an interface mechanism for an interconnect.

NOTE: In normal usage, the interior wall has a smaller diameter than the exterior wall.

The data associated with this application object are the following:

  • associated_component
  • definition

4.2.148.1 associated_component

The associated_component specifies the role of the Component_termination_passage for the Component_termination_passage_interface_terminal.

4.2.148.2 definition

The definition specifies the role of the Component_termination_passage_template_interface_terminal for the Component_termination_passage_interface_terminal.

EXAMPLE: One use of this capability occurs when a member of Component_feature_to_physical_usage_view_assignment references a member of Component_termination_passage_interface_terminal.
