4.2.922 Tolerance_zone_definition_with_specified_orientation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Tolerance_zone_definition_with_specified_orientation is a type of Tolerance_zone_definition in which the orientation of the Tolerance_zone is specified.

The data associated with this application object are the following:

  • zone_orientation

4.2.922.1 zone_orientation

The zone_orientation specifies either a Viewing_plane_based_orientation, a Datum_axis_related_orientation, or a Datum_based_vector_orientation for the Tolerance_zone_definition_with_specified_orientation.

Formal constraints:

WR1: If a Datum_axis_related_orientation is specified as the zone_orientation, the Tolerance_zone specified as the defined_tolerance_zone shall be the same instance of Tolerance_zone as specified as the oriented_tolerance_zone by the Datum_axis_and_tolerance_zone_orienting_relationship that is specified as the orienting_relationship by the Datum_axis_related_orientation that is specified as the zone_orientation.

WR2: If a Viewing_plane_based_orientation is specified as the zone_orientation, the Tolerance_zone specified as the defined_tolerance_zone shall be the same instance of Tolerance_zone as specified as the oriented_tolerance_zone by the Tolerance_zone_and_viewing_plane_orienting_relationship that is specified as the orienting_relationship by the Viewing_plane_based_orientation that is specified as the zone_orientation.

WR3: The Tolerance_zone_definition_with_specified_orientation shall not be specified as a tolerance_zone_or_boundary_definition by either a Circular_runout_tolerance, a Circularity_tolerance, a Concentricity_and_coaxiality_tolerance, a Cylindricity_tolerance, a Flatness_tolerance, a Profile_of_any_surface_tolerance, a Symmetry_tolerance, or a Total_runout_tolerance.
