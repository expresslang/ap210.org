4.2.949 Via_and_contact_size_dependent_non_functional_land

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Via_and_contact_size_dependent_non_functional_land is a type of Non_functional_land whose shape is dependent on a Via as well as the size of a Component terminal.

NOTE: The shape is primarily a function of the mating terminal.

The data associated with this application object are the following:

  • interface_access_mechanism
  • reference_via
  • usage_definition

4.2.949.1 interface_access_mechanism

The interface_access_mechanism is an inverse relationship that specifies that the existence of the Via_and_contact_size_dependent_non_functional_land is dependent on the existence of the Non_functional_land_interface_terminal that specifies the Via_and_contact_size_dependent_non_functional_land as its associated_component. This relationship is for external access only.

4.2.949.2 reference_via

The reference_via specifies the role of the Via for the Via_and_contact_size_dependent_non_functional_land.

4.2.949.3 usage_definition

The usage_definition specifies the role of the Default_via_based_land_physical_template for the Via_and_contact_size_dependent_non_functional_land.
