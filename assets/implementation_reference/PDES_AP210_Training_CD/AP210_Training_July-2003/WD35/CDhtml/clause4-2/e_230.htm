4.2.230 Datum_feature_usage_in_datum_system_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Datum_feature_usage_in_datum_system_relationship is a type of Managed_design_object. A Datum_feature_usage_in_datum_system_relationship is a relationship that associates a Datum_feature with a Datum_system that the Datum_feature participates in establishing.

The data associated with this application object are the following:

  • applied_material_condition_property
  • established_datum_system
  • used_datum_feature

4.2.230.1 applied_material_condition_property

The applied_material_condition_property is an inverse relationship that specifies that the existence of the Datum_feature_usage_in_datum_system_relationship depends on the existence of the Datum_feature_material_condition_property that specifies the Datum_feature_usage_in_datum_system_relationship as its applied_to.

4.2.230.2 established_datum_system

The established_datum_system specifies the Datum_system in which the specified Datum_feature participates in establishing for the Datum_feature_usage_in_datum_system_relationship.

4.2.230.3 used_datum_feature

The used_datum_feature specifies the Datum_feature that is used to establish the specified Datum_system for the Datum_feature_usage_in_datum_system_relationship.

Formal constraints:

WR1: The Datum_feature specified by the used_datum_feature shall establish a Datum that participates in the establishment of the Datum_system specified as the established_datum_system.
