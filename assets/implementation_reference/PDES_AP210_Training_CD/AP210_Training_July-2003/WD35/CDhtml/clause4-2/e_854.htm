4.2.854 Stratum_feature_in_stratum_3d_position

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_feature_in_stratum_3d_position is a type of Managed_design_object. A Stratum_feature_in_stratum_3d_position is the three-dimensional position and orientation of a three-dimensional model of a Stratum_feature in the three-dimensional model of the Stratum.

The data associated with this application object are the following:

  • feature_shape
  • stratum_shape
  • transformation

4.2.854.1 feature_shape

The feature_shape specifies the role of either the Stratum_feature_3d_shape, or the Stratum_feature_non_planar_2d_shape for the Stratum_feature_in_stratum_3d_position.

4.2.854.2 stratum_shape

The stratum_shape specifies the role of the Stratum_3d_shape for the Stratum_feature_in_stratum_3d_position.

4.2.854.3 transformation

The transformation specifies the role of the Axis_placement_3d for the Stratum_feature_in_stratum_3d_position.
