4.2.709 Pre_defined_parallel_datum_axis_2d_symbol

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Pre_defined_parallel_datum_axis_2d_symbol is a type of Non_feature_shape_definition and a type of Planar_shape. A Pre_defined_parallel_datum_axis_2d_symbol is a standard representation for a Datum_axis in a 2 dimensional coordinate system. The axis represented shall be parallel to, or congruent with, the plane that represents the 2 dimensional coordinate system in an associated 3 dimensional coordinate system.

The data associated with this application object are the following:

  • associated_element

4.2.709.1 associated_element

The associated_element specifies the role of the Datum_axis for the Pre_defined_parallel_datum_axis_2d_symbol.
