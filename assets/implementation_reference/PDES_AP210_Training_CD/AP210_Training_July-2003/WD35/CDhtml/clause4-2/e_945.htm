4.2.945 User_defined_geometric_tolerance_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A User_defined_geometric_tolerance_qualifier is a type of Geometric_tolerance_qualifier that requires a more detailed description than those stated for a Pre_defined_geometric_tolerance_qualifier.

The data associated with this application object are the following:

  • description
  • name

4.2.945.1 description

The description specifies the text that describes the manner in which the Geometric_tolerance that are specified as the qualified_geometric_tolerances are qualified by the User_defined_geometric_tolerance_qualifier.

NOTE: The attribute qualified_geometric_tolerances is declared in the Geometric_tolerance_qualifier entity of which this entity is a subtype.

4.2.945.2 name

The name specifies the label for the User_defined_geometric_tolerance_qualifier by which it is referred.
