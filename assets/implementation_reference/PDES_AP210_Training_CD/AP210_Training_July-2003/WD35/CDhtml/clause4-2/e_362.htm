4.2.363 Externally_defined_dimension_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Externally_defined_dimension_representation is a type of Shape_dimension_representation the interpretation of which is dependent on definitions supported externally to this part of ISO 10303.

The data associated with this application object are the following:

  • definitions
  • dimension_value

4.2.363.1 definitions

The definitions specify the External_definition that represents the Dimensional_characteristic. There shall be one or more External_definition specified as definitions for an Externally_defined_dimension_representation.

4.2.363.2 dimension_value

The dimension_value specifies the Data_element that in conjunction with the External_definition, that are specified as the definitions, represent the associated Dimensional_characteristic. The dimension_value need not be specified for an Externally_defined_dimension_representation.

EXAMPLE: The system of limits and fits specifies a way in which to represent dimensions and is defined in ISO 286.
