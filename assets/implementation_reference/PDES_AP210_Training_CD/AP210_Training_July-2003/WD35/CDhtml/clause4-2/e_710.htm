4.2.708 Pre_defined_geometric_tolerance_qualifier

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Pre_defined_geometric_tolerance_qualifier is a type of Geometric_tolerance_qualifier the intent of which is unambiguous and is commonly used in geometric tolerancing.

The data associated with this application object are the following:

  • description

4.2.708.1 description

The description specifies the label that describes the manner in which the Geometric_tolerance that are specified as the qualified_geometric_tolerances are qualified by the Pre_defined_geometric_tolerance_qualifier. The description shall be either "each_element", "each_radial_element", "free_state", "major_diameter", "minor_diameter", "not_concave", "not_convex", or "pitch_diameter".

NOTE: The attribute qualified_geometric_tolerances is declared in the Geometric_tolerance_qualifier entity of which this entity is a subtype.

EXAMPLE: "each element" is a commonly used qualifier that indicates the requirement specified by the geometric tolerance to which the qualifier is applied. "each_element" is on each line element of the feature to which the geometric tolerance is applied rather than on the feature as a whole.
