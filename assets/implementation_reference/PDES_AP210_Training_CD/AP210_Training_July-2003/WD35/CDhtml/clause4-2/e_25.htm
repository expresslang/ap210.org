4.2.24 Angular_dimension_with_direction_vector

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Angular_dimension_with_direction_vector is a type of Dimensional_location that defines the relationship between two Shape_element in terms of the angle between them and a vector that provides an orientation to that angle. The angle is measured in the plane normal to the specified direction vector and in a clockwise direction as viewed in the direction specified by the vector. The measurement is from the Shape_element specified as the locating_shape_element to the Shape_element specified as the located_shape_element by the characterized Shape_element_locating_relationship.

The data associated with this application object are the following:

  • measurement_orientation

4.2.24.1 measurement_orientation

The measurement_orientation specifies either the Datum_based_vector_orientation or an Axis_placement. If the Datum_based_vector_orientation is specified as the measurement_orientation, the specified Datum_based_vector_orientation provides the orientation by which the angle is measured relative to Datum. If an Axis_placement is specified, the orientation is specified in the context of the representation of the Shape_elements the relationship of which is characterized by the Angular_dimension_with_direction_vector.

Formal constraints:

UR1: The combination of located_elements inherited from Dimensional_location and measurement_orientation shall be unique within a population of Angular_dimension_with_direction_vector.

WR1: The dimension_type of the Angular_dimension_with_direction_vector shall be "angular".

NOTE: The attribute dimension_type is declared in the Dimensional_characteristic entity of which this entity is a subtype.

WR2: The boolean value specified as directed shall be TRUE.

NOTE: The attribute directed is declared in the Dimensional_location entity of which this entity is a subtype.

Informal constraints:

IP1: If the measurement_orientation specifies an Axis_placement, the Axis_placement shall be in the same context as the representations of the two Shape_elements that are located relative to each other by the Angular_dimension_with_direction_vector.
