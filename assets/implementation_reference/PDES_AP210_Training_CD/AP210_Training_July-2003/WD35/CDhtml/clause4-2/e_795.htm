4.2.795 Rule_supersedence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_supersedence is a type of Rule_action stating that one Ee_rule_version has been superseded by another Ee_rule_version. Both shall be versions of the same Ee_rule.

The data associated with this application object are the following:

  • superseded_rule

4.2.795.1 superseded_rule

The superseded_rule specifies the Ee_rule_version that has been superseded for the Rule_supersedence.

Formal constraints:

WR1: The Ee_rule in the role of configuration_managed_rule for the Ee_rule_version in the role of superseded_rule specified by this Rule_supersedence shall be the same Ee_rule that is in the role of configuration_managed_rule inherited from the supertype Rule_action that is specified by this Rule_supersedence.
