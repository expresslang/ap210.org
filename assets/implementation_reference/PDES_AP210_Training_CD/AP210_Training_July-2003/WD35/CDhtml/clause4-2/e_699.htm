4.2.702 Positional_boundary_member_definition

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Positional_boundary_member_definition is a type of Managed_design_object. A Positional_boundary_member_definition is a definition that characterizes a Positional_boundary_member. Each Positional_boundary_member_definition is defined by a Boundary_based_position_tolerance in conjunction with a Dimensional_size. The shape of the Positional_boundary_member is identical to that of the toleranced Physical_feature_or_part_template. However, the size of the Positional_boundary_member is based on the associated Dimensional_size, the material conditions specified by the associated Boundary_based_position_tolerance, and whether the toleranced Physical_feature_or_part_template is an internal (e.g., a pair of opposing slot sides) or an external (e.g., a pair of opposing tab sides) Physical_feature_or_part_template.

The data associated with this application object are the following:

  • associated_dimension
  • boundary_definition
  • defined_positional_boundary_member
  • tolerance_value

4.2.702.1 associated_dimension

The associated_dimension specifies the Dimensional_size that defines the maximum material size and minimum material size of the toleranced Physical_feature_or_part_template. If the associated Boundary_based_position_tolerance is specified on a "maximum material principle" basis, the maximum material size is used as the base size to which the Length_data_element specified as the tolerance_value is subtracted if the toleranced Physical_feature_or_part_template is an internal Physical_feature_or_part_template, or added if the toleranced Physical_feature_or_part_template is an external Physical_feature_or_part_template. If the associated Boundary_based_position_tolerance is specified on a "least material requirement" basis, the least material size is used as the base size to which the Length_data_element specified as the tolerance_value is added if the toleranced Physical_feature_or_part_template is an internal Physical_feature_or_part_template, or subtracted if the toleranced Physical_feature_or_part_template is an external Physical_feature_or_part_template.

4.2.702.2 boundary_definition

The boundary_definition is an inverse relationship that specifies that the existence of the Positional_boundary_member_definition is dependent on the existence of the Dimension_related_positional_boundary_definition that specifies the Positional_boundary_member_definition as one of its characterized_boundary_member_definitions.

4.2.702.3 defined_positional_boundary_member

The defined_positional_boundary_member specifies the Positional_boundary_member that is characterized by the Positional_boundary_member_definition.

4.2.702.4 tolerance_value

The tolerance_value specifies the Length_data_element that represents the amount the size of the Positional_boundary_member is reduced or enlarged from the base size that is determined by the associated Dimensional_size and the material condition specified by the associated Boundary_based_position_tolerance.

Formal constraints:

WR1: The Dimension_related_positional_boundary specified as the composed_positional_boundary by the Positional_boundary_member that is specified as the defined_positional_boundary_member shall be the same instance of Dimension_related_positional_boundary as specified as the defined_positional_boundary by the Dimension_related_positional_boundary_definition that is specified as the boundary_definition.

WR2: The Positional_boundary_member_definition shall be specified by exactly one Geometric_tolerance as the tolerance_zone_or_boundary_definition, and that Geometric_tolerance shall be a Boundary_based_position_tolerance.

  SUBTYPE OF (managed_design_object);
    defined_positional_boundary_member : positional_boundary_member;
    tolerance_value                    : length_data_element;
    associated_dimension               : dimensional_size;
  INVERSE
    boundary_definition : dimension_related_positional_boundary_definition FOR
