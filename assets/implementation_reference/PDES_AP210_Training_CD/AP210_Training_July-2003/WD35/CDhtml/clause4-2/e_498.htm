4.2.498 Layer

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Layer is a type of Managed_design_object. A Layer is an extent of contiguous Stratum in a printed circuit board. The Stratum_feature of all Stratum of a Layer are as alike in shape as process tolerances will allow; the design intent is that the shapes of the Stratum_features are to be identical. Standard usage is for exactly one of a Design_layer_stratum or Documentation_layer_stratum to be included in the Inter_stratum_extent referenced by extent.

The data associated with this application object are the following:

  • extent
  • layer_usage
  • name

4.2.498.1 extent

The extent specifies the role of the Inter_stratum_extent for the Layer.

4.2.498.2 layer_usage

The layer_usage specifies the External_definition that characterizes the purpose for incorporating the Layer into a Interconnect_module design. The layer_usage need not be specified for a particular Layer.

4.2.498.3 name

The name identifies the Layer within the product.

Formal constraints:

UR1: The name shall be unique within a population of Layers for a product.

WR1: Each member of Stratum specified by Layer shall be represented by the same geometric representation.

WR2: Each member of Layer shall be related to no more than one Design_layer_stratum or no more than one Documentation_layer_stratum.

ENTITY layer
  SUBTYPE OF (managed_design_object);
    extent      : inter_stratum_extent;
    layer_usage : OPTIONAL external_definition;
    name        : ee_name;
  UNIQUE
    ur1 : name;
END_ENTITY; -- layer
