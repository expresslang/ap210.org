4.2.637 Participant_date_and_time_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Participant_date_and_time_assignment is a type of Managed_design_object. A Participant_date_and_time_assignment is the relationship between a Person_and_organization and a Date_and_time.

The data associated with this application object are the following:

  • action_date_and_time
  • action_participant

4.2.637.1 action_date_and_time

The action_date_and_time specifies the role of the Date_and_time for the Participant_date_and_time_assignment.

4.2.637.2 action_participant

The action_participant specifies the role of the Person_and_organization for the Participant_date_and_time_assignment.
