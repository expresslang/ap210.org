4.2.753 Projected_zone_and_base_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Projected_zone_and_base_relationship is a type of Managed_design_object. A Projected_zone_and_base_relationship is a relationship that associates a Tolerance_zone with the Physical_feature_or_part_template from which the Tolerance_zone is projected.

The data associated with this application object are the following:

  • base_physical_feature
  • projected_zone
  • size_characteristic

4.2.753.1 base_physical_feature

The base_physical_feature specifies the Physical_feature_or_part_template from which the Tolerance_zone that is specified as the projected_zone is projected.

4.2.753.2 projected_zone

The projected_zone specifies the Tolerance_zone that is projected from the Physical_feature_or_part_template that is specified as the base_physical_feature.

4.2.753.3 size_characteristic

The size_characteristic is an inverse relationship that specifies that the existence of the Projected_zone_and_base_relationship is dependent on the existence of the Projected_zone_height_characteristic that specifies the Projected_zone_and_base_relationship as its defined_height_of.

Informal constraints:

IP1: The Physical_feature_or_part_template specified as the base_physical_feature shall be a feature from which it is sensible to project a Tolerance_zone.

ENTITY projected_zone_and_base_relationship
  SUBTYPE OF (managed_design_object);
    base_physical_feature : physical_feature_or_part_template;
    projected_zone        : tolerance_zone;
  INVERSE
    size_characteristic : projected_zone_height_characteristic FOR 
                          defined_height_of;
END_ENTITY; -- projected_zone_and_base_relationship
