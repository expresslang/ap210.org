4.2.812 Shape_element_composing_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shape_element_composing_relationship is a type of Shape_element_constituent_relationship that associates a Composite_shape_element with a Shape_element from which the Composite_shape_element is composed.

The data associated with this application object are the following:

  • relating_shape_element

4.2.812.1 relating_shape_element

The relating_shape_element specifies the Composite_shape_element that is the Shape_element that is composed from the Shape_element specified as the related_shape_element for the Shape_element_composing_relationship.

NOTE: The attribute related_shape_element is declared in the entity Shape_element_relationship of which this entity is a subtype.
