4.2.782 Rule_based_constraint_occurrence

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_based_constraint_occurrence is a type of Constraint_occurrence that limits possible solutions to the requirements specification by identifying boundary conditions for any potential solution. A Rule_based_constraint_occurrence specifies a rule that must be satisfied.

EXAMPLE: A rule might exist that requires junction temperatures of Silicon based integrated devices to be less than 105 degrees Centrigade.

The data associated with this application object are the following:

  • design_constraint

4.2.782.1 design_constraint

The design_constraint specifies the role of the Ee_rule_version for the Rule_based_constraint_occurrence.
