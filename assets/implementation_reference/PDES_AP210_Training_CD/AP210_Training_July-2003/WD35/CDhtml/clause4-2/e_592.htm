4.2.582 Package_terminal_external_reference

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Package_terminal_external_reference is a type of Managed_design_object. A Package_terminal_external_reference is an unambiguous reference to a Package_terminal in an external system.

EXAMPLE: Items in a Physical_connectivity_definition may need to reference terminals in a library without duplicating the geometric context.

The data associated with this application object are the following:

  • associated_definition
  • name

4.2.582.1 associated_definition

The associated_definition specifies the role for a Package_external_reference for the Package_terminal_external_reference.

4.2.582.2 name

The name specifies the role of a string for the Package_terminal_external_reference. The name shall be interpreted as the name attribute inherited by Package_terminal from Shape_element.

Formal constraints:

UR1: Each member of Package_terminal_external_reference shall have a unique combination of attributes.
