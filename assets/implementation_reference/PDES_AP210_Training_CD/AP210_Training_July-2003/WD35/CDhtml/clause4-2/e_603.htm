4.2.603 Parameter_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Parameter_assignment is a type of Managed_design_object. A Parameter_assignment is the association of either a Characteristic or Coordinated_characteristic with a Model_parameter, used to provide actual values for characteristics declared formally by the the Model_parameter.

The data associated with this application object are the following:

  • assigned_parameter
  • parameter_value

4.2.603.1 assigned_parameter

The assigned_parameter specifies the Model_parameter that declares the formal parameters assigned values by the Parameter_assignment.

4.2.603.2 parameter_value

The parameter_value specifies either the role of a Characteristic or a Coordinated_characteristic for the Parameter_assignment.

Formal constraints:

WR1: The type of the parameter_value shall be the same as that of the assigned_parameter.

NOTE: 
*)
ENTITY parameter_assignment
  SUBTYPE OF (managed_design_object);
    parameter_value    : scalar_or_coordinated_characteristics;
    assigned_parameter : model_parameter;
END_ENTITY; -- parameter_assignment
