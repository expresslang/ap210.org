4.2.787 Rule_justification

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Rule_justification is a type of Managed_design_object. A Rule_justification is the justification for an Ee_rule.

The data associated with this application object are the following:

  • justification_rationale
  • justified_action

4.2.787.1 justification_rationale

The justification_rationale specifies the text that describes the Rule_justification.

4.2.787.2 justified_action

The justified_action specifies the Rule_action that results from the Rule_justification.
