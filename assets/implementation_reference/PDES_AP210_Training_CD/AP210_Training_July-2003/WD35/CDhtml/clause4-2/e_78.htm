4.2.79 Boundary_zone_definition_with_specified_size

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Boundary_zone_definition_with_specified_size is a type of Tolerance_zone_boundary_definition in which the size of the Tolerance_zone_boundary is defined by a Boundary_size_characteristic. A Boundary_zone_definition_with_specified_size may be a Profile_boundary_definition.

The data associated with this application object are the following:

  • specified_boundary_size

4.2.79.1 specified_boundary_size

The specified_boundary_size specifies the Boundary_size_characteristic that defines the size of the Tolerance_zone_boundary for the Boundary_zone_definition_with_specified_size.
