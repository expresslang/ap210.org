4.2.487 Laminate_group_component_make_from_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Laminate_group_laminate_group_component_make_from_relationship is a type of Component_make_from_relationship. A Laminate_group_laminate_group_component_make_from_relationship is the association between one Assembly_group_component and another Assembly_group_component, where one Assembly_group_component from full or partial reuse of the other. A Laminate_group_laminate_group_component_make_from_relationship also supports mapping of a Stratum in the reference design to a Stratum in this design.

NOTE: The main use of this concept is use of multi-layer breakout definitions for complex integrated circuits where there is a mapping from a Stratum in the reference_footprint.reference_layout to a stratum in this layout.

The data associated with this application object are the following:

  • resultant_component
  • reusable_component

4.2.487.1 resultant_component

The resultant_component specifies the Assembly_group_component that results from the reuse in the Laminate_group_laminate_group_component_make_from_relationship.

4.2.487.2 reusable_component

The reusable_component specifies the Assembly_group_component that is reused to create the resultant_component in the Laminate_group_laminate_group_component_make_from_relationship.

ENTITY laminate_group_component_make_from_relationship
  SUBTYPE OF (component_make_from_relationship);
    SELF\component_make_from_relationship.reusable_component  : assembly_group_component;
    SELF\component_make_from_relationship.resultant_component : assembly_group_component;
    stratum_mapping : SET [1:?] OF stratum_make_from_relationship;
WHERE
   WR1: SIZEOF(QUERY( sar <* SELF\component_make_from_relationship.reusable_component.assembly |
      NOT('AP210_ARM.LAMINATE_COMPONENT' IN TYPEOF(sar.related_component))
            )) = 0;
   WR2: SIZEOF(QUERY( sar <* SELF\component_make_from_relationship.resultant_component.components |
      NOT('AP210_ARM.LAMINATE_COMPONENT' IN TYPEOF(sar.related_component))
            )) = 0;
END_ENTITY; -- laminate_group_component_make_from_relationship
