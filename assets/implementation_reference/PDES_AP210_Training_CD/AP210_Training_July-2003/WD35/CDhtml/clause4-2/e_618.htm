4.2.614 Part_interface_access_feature

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_interface_access_feature is a type of Part_feature that is identified to facilitate tolerance specification when an occurrence of the associated part is used in an assembly. Specifically the intent is to provide identification that the function of the Part_feature is to provide access via a void in the material to internal components or terminals, which have been identified as having some function at the Part level, e.g., are Part_terminals. The material_state_change attribute inherited from Part_feature shall be material_removal.

EXAMPLE: The reset button in a notebook computer requires a Part_interface_access_feature whose properties are coupled to the reset button to ensure correct functionality of the end product.

The data associated with this application object are the following:

  • accessed_interface
  • interface_access_connection_area

4.2.614.1 accessed_interface

The accessed_interface specifies the role of the Part_feature for the Part_interface_access_feature. The accessed_interface is the feature for which the Part_interface_access_feature provides access.

NOTE: The accessed_interface is at the same level of design hierarchy as the Part_interface_access_feature for consistency purposes. It is anticipated that component_feature_to_physical_usage_view_assignment will be populated as necessary to support the design process peculiar to the enterprise.

4.2.614.2 interface_access_connection_area

The interface_access_connection_area specifies the role of the Connection_zone for the Part_interface_access_feature. The accessed_interface specifies the role of the Part_feature for the Part_interface_access_feature. The interface_access_connection_area provides a means to geometrically specify an area for critical mechanical interface_access properties. The Connection_zone area is void of material.

ENTITY part_interface_access_feature
  SUBTYPE OF (part_feature);
    accessed_interface : part_feature;
    interface_access_area : OPTIONAL SET [1:?] OF connection_zone;
WHERE
 wr1: SELF\part_feature.material_state_change = material_removal;
 wr2: SELF <> accessed_interface;
 wr3: NOT('AP210_ARM.PART_TERMINAL' IN TYPEOF(SELF));
 wr4: NOT('AP210_ARM.PACKAGE_BODY' IN TYPEOF(SELF));
 wr5: NOT('AP210_ARM.PART_MOUNTING_FEATURE' IN TYPEOF(SELF));
 wr6: NOT('AP210_ARM.INTERCONNECT_MODULE_SURFACE_FEATURE' IN TYPEOF(SELF)); 
 wr7: NOT('AP210_ARM.PACKAGE_TERMINAL' IN TYPEOF(SELF)); 
 wr8: NOT('AP210_ARM.PACKAGE_BODY_SURFACE' IN TYPEOF(SELF)); 
 wr9: NOT('AP210_ARM.POLARITY_INDICATION_FEATURE' IN TYPEOF(SELF)); 
END_ENTITY; -- part_interface_access_feature
