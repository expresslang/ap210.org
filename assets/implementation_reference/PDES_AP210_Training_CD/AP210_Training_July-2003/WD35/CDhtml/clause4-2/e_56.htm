4.2.56 Axis_placement

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Axis_placement is a type of Shape_item. An Axis_placement may be an Axis_placement_2d or an Axis_placement_3d. An Axis_placement is the transformation and orientation of one Cartesian_coordinate_system relative to a referenced coordinate system. An Axis_placement is the rigid body transformation that transforms the geometry of the object being placed from its coordinate system to the Cartesian_coordinate_system in which the object is being placed. The dimensionality of Axis_placement is determined by the dimensionality of the object being placed. The interpretation of this Application Object shall be to an axis2_placement_3d or to an axis2_placement_2d.

The data associated with this application object are the following:

  • reference_direction
  • translation

4.2.56.1 reference_direction

The reference_direction specifies a Direction as a reference for an occurrence of a physical device in an Axis_placement.

4.2.56.2 translation

The translation specifies the Cartesian_point that is the origin of a Cartesian_coordinate_system relative to the origin of a reference Cartesian_coordinate_system while maintaining the angular relationship among the axes of the two coordinate systems in the Axis_placement.
