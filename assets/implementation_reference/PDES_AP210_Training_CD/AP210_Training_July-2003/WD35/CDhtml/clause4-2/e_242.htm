4.2.242 Datum_target_usage_in_datum_target_set_relationship

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
Datum_target_usage_in_datum_target_set_relationship is a type of Managed_design_object. A Datum_target_usage_in_datum_target_set_relationship is a relationship that associates a Datum_target with a Datum_target_set that the Datum_target participates in defining.

The data associated with this application object are the following:

  • datum_target_number
  • defined_datum_target_set
  • used_datum_target

4.2.242.1 datum_target_number

The datum_target_number specifies the positive integer that identifies the usage of the Datum_target in the definition of a Datum_target_set for the Datum_target_usage_in_datum_target_set_relationship.

NOTE: The datum target number is described in 7.1.1 of ISO 5459.

4.2.242.2 defined_datum_target_set

The defined_datum_target_set specifies the Datum_target_set in which the specified Datum_target participates in defining.

4.2.242.3 used_datum_target

The used_datum_target specifies the Datum_target that is used to define the Datum_target_set specified as the defined_datum_target_set for the Datum_target_usage_in_datum_target_set_relationship.

NOTE: Since a Datum_target may be used to define multiple Datum_target_sets and the Datum_target number may be different depending on the Datum_target_set that the Datum_target participates in defining, the Datum_target_number is assigned to the Datum_target_usage_in_datum_target_set_relationship rather than on the Datum_target.

EXAMPLE: A datum target is used to define datum target sets A and G. The usage of the datum target in the definition of datum target set A is assigned a datum target number of 3. The usage of the datum target in defining datum target set G is assigned a datum target number of 1. This situation would be represented by a single instance of Datum_target, two instances of Datum_target_set, and two instances of Datum_target_usage_in_datum_target_set_relationship. The instances of Datum_target_set would be assigned the identifications of "A" and "G". The instances of Datum_target_usage_in_datum_target_set_relationship that associate the instance of Datum_target to the instances of Datum_target_set identified as "A" and "G" would have datum_target_number values of 3 and 1, respectively.

Formal constraints:

UR1: The combination of used_datum_target and defined_datum_target_set shall be unique within a population of Datum_target_usage_in_datum_target_set_relationship. That is, a specific Datum_target shall not be used more than once to define a specific Datum_feature.

UR2: The combination of datum_target_number and defined_datum_target_set shall be unique within a population of Datum_target_usage_in_datum_target_set_relationship.

ENTITY datum_target_usage_in_datum_target_set_relationship
  SUBTYPE OF (managed_design_object);
    defined_datum_target_set : datum_target_set;
    used_datum_target        : datum_target;
    datum_target_number      : positive_integer;
  UNIQUE
    ur1 : used_datum_target, defined_datum_target_set;
    ur2 : datum_target_number, defined_datum_target_set;
