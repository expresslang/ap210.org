4.2.201 Csg_primitive

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Csg_primitive is a type of Managed_design_object. A Csg_primitive is either a Right_angular_wedge, or a Right_circular_cylinder, or a Block. A Csg_primitive is used in forming a "CSG" solid model.

NOTE: 
*)
ENTITY csg_primitive
  SUPERTYPE OF (ONEOF (block,right_circular_cylinder,right_angular_wedge))
  SUBTYPE OF (managed_design_object);
END_ENTITY; -- csg_primitive
