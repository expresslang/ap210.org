4.2.304 Direction

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Direction is a type of Managed_design_object. A Direction shall be interpreted as a general direction vector in two-dimensional or three-dimensional space. The actual magnitudes of the components of the vector have no significance; only the ratios among the components are significant.

NOTE: This Application_object does not provide the details of direction.
