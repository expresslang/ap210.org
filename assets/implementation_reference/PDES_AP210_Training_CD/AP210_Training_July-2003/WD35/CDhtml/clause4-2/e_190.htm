4.2.190 Connection_zone_model_port_assignment

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Connection_zone_model_port_assignment is a type of Managed_design_object. A Connection_zone_model_port_assignment is the relationship between the Analytical_model_port and the Connection_zone, for the purpose of mapping the connection points to the model ports directly.

NOTES:

  1. In some circumstances it may be necessary to assign the Connection_zones to model ports in order to characterize the functionality of the terminals more accurately.
  2. Part_device_terminal_model_port_assignment may be used in conjunction with Part_feature_model_port_assignment where this level of detail is not required.

The data associated with this application object are the following:

  • source
  • target

4.2.190.1 source

The source specifies the role of the Connection_zone for the Connection_zone_model_port_assignment.

4.2.190.2 target

The target specifies the role of the Analytical_model_port for the Connection_zone_model_port_assignment.
