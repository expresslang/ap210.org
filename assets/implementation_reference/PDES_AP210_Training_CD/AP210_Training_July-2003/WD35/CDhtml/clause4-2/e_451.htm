4.2.458 Interconnect_module_product_assembly_view_fabrication_joint

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
An Interconnect_module_product_assembly_view_fabrication_joint is a type of Managed_design_object. It specifies the bond among Interconnect_module_component_stratum_based_terminal which is created during an assembly process. The detailed process description of how to create the bond is not included.

NOTE: This Application Object is intended to support a limited view of an interconnect product where the vertical or sequential arrangement of intermediate assemblies is of interest, and the detailed information available in Stratumis not of interest.

The data associated with this application object are the following:

  • joined_terminal

4.2.458.1 joined_terminal

A joined_terminal specifies one or more Interconnect_module_component_stratum_based_terminal for the Interconnect_module_product_assembly_view_fabrication_joint.
