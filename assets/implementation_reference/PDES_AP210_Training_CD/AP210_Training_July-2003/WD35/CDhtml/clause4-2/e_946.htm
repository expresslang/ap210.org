4.2.946 Vector

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Vector is a type of Managed_design_object. A Vector is a quantity that has magnitude and direction and that is commonly represented by a directed line segment whose length represents the magnitude and whose orientation in space represents the direction.
