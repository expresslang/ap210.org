4.2.868 Stratum_surface

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_surface is a type of Managed_design_object. A Stratum_surface specifies the primary_surface or secondary_surface for a Stratum.

The data associated with this application object are the following:

  • name
  • of_stratum

4.2.868.1 name

The name designates a primary_surface or a secondary_surface for a Stratum_surface.

4.2.868.2 of_stratum

An of_stratum specifies the Stratum for a Stratum_surface.

Formal constraints:

UR1: The combination of name and of_stratum shall be unique within a population of Stratum_surface.

NOTE:

ENTITY stratum_surface
  SUBTYPE OF (managed_design_object);
    of_stratum : stratum;
    name       : stratum_surface_designation;
  INVERSE
    of_shape : SET [0:1] OF stratum_surface_shape FOR of_surface;
  UNIQUE
    ur1 : name, of_stratum;
END_ENTITY; -- stratum_surface
