4.2.442 Interconnect_module_component

An Interconnect_module_component is a type of Assembly_component. An Interconnect_module_component is an instance of an Interconnect_module_usage_view.

NOTE: The Interconnect_module_component can be, for example, a circuit board in an assembly. Alternatively, it could be the substrate for a multi-chip module. This entity is included specifically so that the assembly design definition can be queried directly to determine which component in the assembly is the substrate. The Interconnect_module_component is likely the target of connectivity requirement information as allocated to various components. An Interconnect_module_component does not include all the internal details of the design of the Interconnect_module.

The data associated with this application object are the following:

  • usage_definition

4.2.442.1 usage_definition

The usage_definition specifies the role of the Interconnect_module_usage_view for the Interconnect_module_component.

NOTE: The usage_view specifies the definitions of the terminals and other interface elements of the Interconnect_module.

Formal constraints:

WR1: An Interconnect_module_component shall not be mounted onto itself.
