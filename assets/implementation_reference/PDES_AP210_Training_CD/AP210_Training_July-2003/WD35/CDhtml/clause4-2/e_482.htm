4.2.482 Junction_vertex_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Junction_vertex_allocation is a type of Managed_design_object. A Junction_vertex_allocation is the relationship that specifies one or more Layer_connection_point to which a Topological_junction is mapped during the layout routing process. A Junction_vertex_allocation is the result of an assignment process by a designer or a layout routing process that defines the location of Inter_stratum_join_relationship or Explicitly_located_layer_connection_point. A Junction_vertex_allocation maps a node of a Physical_network which may be only partially ordered into a network description which is highly ordered using a tree. A Junction_vertex_allocation associates two Ee_product_definitions, one an Interconnect_module and the other a Physical_unit_network_definition. The Physical_unit_network_definition may be populated for the express purpose of representing the tree structure of the Physical_network in the Interconnect_module, and therefore the version of the design would be the same for both.

The data associated with this application object are the following:

  • implementation_point
  • required_vertex

4.2.482.1 implementation_point

The implementation_point specifies either the Inter_stratum_join_relationship or Explicitly_located_layer_connection_point that, individually, maps the requirement that a junction participates in a connectivity onto points on conducting Stratum for the Junction_vertex_allocation.

NOTES:

  1. This construct allows the Junction_vertex_allocation to consider an Inter_stratum_join_relationship to be a vertex.
  2. The Junction_vertex_allocation allows for the specification of the location of the connections between physical traces and connected areas. It is common practice for the point where a trace, via, pad, etc., connects with other features to be specified by the designer interactively during layout. This location may be constrained for fabrication, layout, or signal integrity purposes. The Junction_vertex allocation specifies these relationships.

4.2.482.2 required_vertex

The required_vertex specifies the Topological_junction, as a vertex of a Physical_connectivity_definition, that is to be mapped to an implementation_point by the Junction_vertex_allocation.

Formal constraints:

UR1: The required_vertex shall be unique within a population of Junction_vertex_allocations.

WR1: The implementation_point and required_vertex shall be for the same product.

NOTE: 
*)
ENTITY junction_vertex_allocation
  SUBTYPE OF (managed_design_object);
    implementation_point : trace_junction_or_via;
    required_vertex      : topological_junction;
  UNIQUE
    ur1 : required_vertex;
END_ENTITY; -- junction_vertex_allocation
