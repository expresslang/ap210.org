4.2.150 Component_termination_passage_template

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_termination_passage_template is a type of Inter_stratum_feature_template. The Component_termination_passage_template defines all shape data for the associated Component_termination_passage (for which it is the usage_definition), and also where the connection (dependency) on passage technology is established for the properties.

NOTE: Properties other than shape are to be found in the of_passage_technology.

The data associated with this application object are the following:

  • access_mechanisms
  • of_passage_technology

4.2.150.1 access_mechanisms

The access_mechanisms is an inverse relationship that specifies that the existence of the Component_termination_passage_template is dependent on the existence of the Component_termination_passage_template_terminal that specifies the Component_termination_passage_template as its associated_definition. There shall be two or more Component_termination_passage_template_terminals for a Component_termination_passage_template.

4.2.150.2 of_passage_technology

The of_passage_technology specifies the role of the Default_component_termination_passage_definition for the Component_termination_passage_template.

Formal constraints:

WR1: One access_mechanisms shall be a Component_termination_passage_template_interface_terminal.

WR2: One access_mechanisms shall be a Component_termination_passage_template_join_terminal.
