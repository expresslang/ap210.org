4.2.669 Physical_unit_planar_keepout_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Physical_unit_planar_keepout_shape is a type of Planar_shape and a type of Non_feature_shape_definition. A Physical_unit_planar_keepout_shape is the projected shape of a keepout region around the Physical_unit in two dimensions.

NOTES:

  1. Keepout regions typically include some minimal spacing around the extreme shape represented by Physical_unit_planar_shape where the exact relationship between the two shapes is unrecorded.
  2. There may be multiple Physical_unit_planar_keepout_shape for a particular purpose. The recommended method to aggregate several Physical_unit_planar_keepout_shape that together describe a volumetric constraint is to reference them from a member of Shape_and_ee_product_definition_based_constraint_occurrence.

The data associated with this application object are the following:

  • application_technology_constraint
  • centroid_location
  • constrained_design_object_category
  • shape_characterized_physical_unit
  • shape_environment
  • shape_distance_from_seating_plane
  • shape_location_with_respect_to_seating_plane
  • shape_material_condition
  • shape_purpose
  • shape_representation

4.2.669.1 application_technology_constraint

The application_technology_constraint specifies the role of Ee_requirement_occurrence for the Physical_unit_3d_shape. The application_technology_constraint is the relationship between keepout shapes and process constraints under which those keepout shapes are valid.

EXAMPLE: An example of an application_technology_constraint is the relationship of the shape of a clearance area to an assembly or testing process which requires the clearance.

4.2.669.2 centroid_location

The centroid_location is a Cartesian_point which is the center of a Physical_unit_planar_keepout_shape. A centroid_location need not be specified for a particular Physical_unit_planar_keepout_shape.

4.2.669.3 constrained_design_object_category

The constrained_design_object_category specifies the category of design object for which the Physical_unit_planar_keepout_shape is a constraint.

4.2.669.4 shape_characterized_physical_unit

The shape_characterized_part_unit specifies the role of the Physical_unit for the Physical_unit_planar_keepout_shape.

4.2.669.5 shape_environment

The shape_environment specifies either manufacturing or an end_user_application as the environment for the Physical_unit_planar_keepout_shape.

4.2.669.6 shape_distance_from_seating_plane

The shape_distance_from_seating_plane specifies the distance of the Physical_unit_planar_keepout_shape from the seating_plane. The shape_distance_from_seating_plane is coordinated with the shape_location_with_respect_to_seating_plane. The shape_distance_from_seating_plane need not be provided with a Physical_unit_planar_keepout_shape.

4.2.669.7 shape_location_with_respect_to_seating_plane

The shape_location_with_respect_to_seating_plane specifies the location of the Physical_unit_planar_keepout_shape as 'above', 'below' or 'congruent' with the seating_plane.

4.2.669.8 shape_material_condition

The shape_material_condition specifies either the maximum_material_condition, the minimum_material_condition, or the nominal_material_condition as the material condition for the Physical_unit_planar_keepout_shape.

4.2.669.9 shape_purpose

The shape_purpose specifies either electro_magnetic_compatibility, shock, thermal, vibration, or an external_definition as the purpose for the Physical_unit_planar_keepout_shape.

Formal constraints:

WR1: If the shape_location_with_respect_to_seating_plane value is 'congruent', the shape_distance_from_seating_plane shall not be supplied.

ENTITY physical_unit_planar_keepout_shape
  SUBTYPE OF (planar_shape, non_feature_shape_definition);
    application_technology_constraint  : OPTIONAL ee_requirement_occurrence;
    constrained_design_object_category : keepout_product_design_object_category;
    shape_characterized_physical_unit  : physical_unit;
    shape_distance_from_seating_plane  : OPTIONAL length_data_element;
    shape_location_with_respect_to_seating_plane: shape_location;
    shape_material_condition           : material_condition;
    centroid_location                  : OPTIONAL cartesian_point;
    shape_environment                  : environment;
    shape_purpose                      : keepout_purpose;
WHERE
  WR1: NOT((congruent = 
           shape_location_with_respect_to_seating_plane) AND 
       (EXISTS(shape_distance_from_seating_plane)));
END_ENTITY; -- physical_unit_planar_keepout_shape
