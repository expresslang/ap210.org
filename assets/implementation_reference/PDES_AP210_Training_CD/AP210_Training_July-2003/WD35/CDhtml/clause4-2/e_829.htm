4.2.829 Singular_dimension_representation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Singular_dimension_representation is a type of Shape_dimension_representation that explicitly describes one or more Dimensional_characteristic with a single value. A Singular_dimension_representation is one of a Maximum_dimension_representation, a Minimum_dimension_representation, or a Theoretically_exact_dimension_representation.

The data associated with this application object are the following:

  • dimension_value

4.2.829.1 dimension_value

The dimension_value specifies the Data_element that describes the Dimensional_characteristic for which the Singular_dimension_representation is a representation.
