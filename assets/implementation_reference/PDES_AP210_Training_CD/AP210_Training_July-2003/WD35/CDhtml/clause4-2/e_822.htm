4.2.822 Shield_allocation

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Shield_allocation is a type of Electromagnetic_compatibility_requirement_allocation. The Shield_allocation enables the assignment of a requirement to a shield. This allocates required properties of shielding effectiveness to the shield.

The data associated with this application object are the following:

  • assembly_assignment
  • requirement

4.2.822.1 assembly_assignment

The assembly_assignment specifies the role of the Shield for the Shield_allocation.

4.2.822.2 requirement

The requirement specifies the role of the Electromagnetic_requirement_occurrence for the Shield_allocation.
