4.2.119 Component_feature_external_reference

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_feature_external_reference is a type of Managed_design_object. A Component_feature_external_reference is an unambiguous reference to a Component_feature in a different design.

EXAMPLE: Items in an Interconnect_module design view may need to reference component features in a reference assembly in order to unambiguously identify source of layout requirements.

The data associated with this application object are the following:

  • associated_component
  • name

4.2.119.1 associated_component

The associated_component specifies the role for a Component_external_reference for the Component_feature_external_reference.

4.2.119.2 name

The name specifies the role of a string for the Component_feature_external_reference.

Formal constraints:

UR1: Each member of Component_feature_external_reference shall have a unique combination of attributes.
