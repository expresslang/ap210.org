4.2.870 Stratum_surface_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Stratum_surface_shape is a type of Open_shell_based_surface. A Stratum_surface_shape is the representation of a Stratum_surface using an open manifold surface.

The data associated with this application object are the following:

  • of_surface

4.2.870.1 of_surface

The of_surface specifies the role of a Stratum_surface for the Stratum_surface_shape.
