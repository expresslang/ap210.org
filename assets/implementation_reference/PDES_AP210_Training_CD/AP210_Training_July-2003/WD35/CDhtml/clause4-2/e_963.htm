4.2.963 Wire_terminal

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Wire_terminal is a type of a Package_terminal whose core_material is inherently flexible and ready for installation to a Pca without prior formal preparation by shape modification. The shape of the Wire_terminal is incompletely defined in the library.

NOTE: The Feature_shape_occurrence inherited by the terminal may be the same Feature_shape_occurrence that is specified by the associated Connection_zone. In this case the whole terminal is the Connection_zone.

The data associated with this application object are the following:

  • internal_connection_zone
  • maximum_wire_terminal_length
  • minimum_wire_terminal_length

4.2.963.1 internal_connection_zone

The internal_connection_zone specifies the role of a Connection_zone for the Wire_terminal. There shall be one or more Connection_zone for the Wire_terminal. The internal_connection_zone is mandatory because it locates the end of the wire that is the attachment area on the package body.

4.2.963.2 maximum_wire_terminal_length

The maximum_wire_terminal_length specifies the Length_data_element for the maximum length of the Wire_terminal.

4.2.963.3 minimum_wire_terminal_length

The minimum_wire_terminal_length specifies the Length_data_element for the minimum length of the Wire_terminal.
