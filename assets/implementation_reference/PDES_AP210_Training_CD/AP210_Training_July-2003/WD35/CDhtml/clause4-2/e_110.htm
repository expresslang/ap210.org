4.2.110 Component_2d_surface_location

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Component_2d_surface_location is a type of Component_2d_location. The Component_2d_surface_location places components on the external surface of a substrate.

NOTE: Connection_zone_interface_plane_relationship and Connection_zone_based_assembly_joint may be used to provide more details if needed by a receiving organization.

4.2.110.1 mounting_surface_assembly_joint

The mounting_surface_assembly_joint specifies the Assembly_joint that joins the Component_mounting_feature of the component being located to the associated mating component feature. The surface the mating component feature is associated with is part of the orientation description for the Component_2d_surface_location.
