4.2.690 Plated_passage

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Plated_passage is a type of Plated_inter_stratum_feature. A Plated_passage is either a Component_termination_passage or a Via.

EXAMPLE: Figure ppass graphically collects the various types of plated passages into a more generic category.

Formal constraints:

WR1: Each Plated_passage shall not be a member of Interconnect_module_edge.

WR2: Each Plated_passage shall not be a member of Interconnect_module_edge_segment.
