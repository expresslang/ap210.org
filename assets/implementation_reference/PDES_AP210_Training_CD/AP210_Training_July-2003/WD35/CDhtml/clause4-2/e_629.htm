4.2.625 Part_template_non_planar_2d_shape

NOTE: The text description for this Application Object has not been fully updated to agree with the EXPRESS.
A Part_template_non_planar_2d_shape is a type of Managed_design_object. A Part_template_non_planar_2d_shape is the shape of a Part_template that is represented in the form of a surface.

The data associated with this application object are the following:

  • shape_characterized_part_template
  • shape_environment
  • shape_material_condition
  • shape_representation
  • transformation

4.2.625.1 shape_characterized_part_template

The shape_characterized_part_template specifies the role of the Part_template for the Part_template_non_planar_2d_shape.

4.2.625.2 shape_environment

The shape_environment specifies either manufacturing or an end_user_application as the valid environment for the Part_template_non_planar_2d_shape. The shape_environment need not be specified for a particular Part_template_non_planar_2d_shape.

4.2.625.3 shape_material_condition

The shape_material_condition specifies either the maximum_material_condition, the minimum_material_condition, or the nominal_material_condition as the material condition for the Part_template_non_planar_2d_shape.

4.2.625.4 shape_representation

The shape_representation specifies the Open_shell_based_surface that is the model for the Part_template_non_planar_2d_shape.

4.2.625.5 transformation

The transformation specifies the role of the Axis_placement_3d for the Part_template_non_planar_2d_shape.
