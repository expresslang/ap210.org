= Test Case Gallery

35 validated test cases from the PDES Test Cases Project, covering the full range of AP 210 entities and constructs. Each test case includes one or more STEP Part 21 files (`.stp`, `.210`) that demonstrate specific aspects of the AP 210 information model.

== Accessing Test Cases

Test case files are available in the repository at `assets/test_cases/PDES_Testcases_Project/`. Each test case directory contains STEP Part 21 files and associated documentation.

== Use in Learning

Test cases are referenced throughout the <<{/learn/reading-data#,Reading AP210 Data>> course module, which provides guided walk-throughs of selected test cases from simple to complex.
