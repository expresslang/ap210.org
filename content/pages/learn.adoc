= Learn

AP 210 learning is organized into five progressive course modules:

== Module 1: STEP & EXPRESS Foundations

Introduction to the STEP standard, the EXPRESS modeling language, and the business context for AP 210.

== Module 2: AP 210 Concepts

Core AP 210 concepts including product structure, connectivity, physical design, and design rules.

== Module 3: Reading AP 210 Data Files

Walk-throughs of AP 210 STEP files, from simple examples to complex multi-layer assemblies.

== Module 4: Querying AP 210 Data

Using EXPRESS queries to extract information from AP 210 datasets.

== Module 5: Programming with AP 210

Building applications with the JSDAI Java framework to work with AP 210 data.
