= Edition 1 — ISO 10303-210:2004

*First edition* of the AP 210 standard for electronic assembly, interconnect, and packaging design.

== Publication

- **ISO reference:** https://www.iso.org/standard/34748.html[ISO 10303-210:2004^]
- **Published:** 2004
- **Status:** Withdrawn (superseded by Edition 2)

== Scope

Edition 1 established the foundational information model for representing printed circuit assembly (PCA) design data. It defined:

- Product structure and classification
- Physical design representation (footprints, board outlines)
- Basic connectivity between components
- Interconnect substrate definition
- Library definitions for components and packages

This edition focused on the exchange of design data between EDA (Electronic Design Automation) tools, enabling interoperability between different vendor systems.
